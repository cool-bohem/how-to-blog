## 아이폰 배터리가 빨리 닳는 이유

아이폰 배터리 소모가 심하다면 대부분 **백그라운드 앱**과 **화면 밝기** 때문이에요. 아래 방법을 순서대로 따라해보세요.

## 방법 1. 백그라운드 앱 새로고침 끄기

1. **설정 → 일반 → 백그라운드 앱 새로고침** 으로 이동
2. 맨 위 토글을 **끔** 으로 변경
3. 자주 쓰지 않는 앱들의 새로고침을 개별로 끌 수도 있어요

## 방법 2. 화면 자동 잠금 시간 줄이기

1. **설정 → 디스플레이 및 밝기 → 자동 잠금**
2. **30초** 로 설정

## 방법 3. 위치 서비스 제한

1. **설정 → 개인 정보 보호 → 위치 서비스**
2. 각 앱을 **앱 사용 중에만** 또는 **안함** 으로 변경
3. 특히 SNS 앱은 위치 서비스가 필요 없어요

## 방법 4. 화면 밝기 낮추기

1. 화면 오른쪽 위에서 아래로 쓸어내려 제어 센터 열기
2. 밝기 슬라이더를 50% 이하로 낮추기
3. **설정 → 손쉬운 사용 → 디스플레이 및 텍스트 크기 → 밝기 줄이기** 도 함께 켜기

## 방법 5. 저전력 모드 활용

- 배터리가 **20% 이하**로 떨어지면 자동으로 알림이 와요
- **설정 → 배터리 → 저전력 모드** 에서 미리 켤 수도 있어요

## 방법 6. 배터리 상태 확인

1. **설정 → 배터리 → 배터리 상태 및 충전**
2. **최대 용량이 80% 이하**라면 배터리 교체를 고려하세요

## 방법 7. 앱별 배터리 사용량 확인

1. **설정 → 배터리**
2. 아래로 내리면 앱별 배터리 사용량이 표시돼요
3. 사용량이 높은 앱을 삭제하거나 알림을 끄세요

배터리 절약 설정만 잘 해도 하루 사용 시간이 2~3시간은 늘어나요.
