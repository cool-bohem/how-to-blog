## 갤럭시 보안 폴더란?

갤럭시 보안 폴더(Secure Folder)는 삼성전자가 제공하는 프라이버시 보호 기능입니다. 보안 폴더 안에 저장된 앱, 사진, 동영상, 연락처, 메시지 등은 일반 화면에서는 완전히 숨겨지며, 별도의 비밀번호나 지문·얼굴 인식을 통해서만 접근할 수 있습니다.

직장 동료나 가족에게 스마트폰을 잠깐 빌려줄 때, 혹은 민감한 사진이나 업무 앱을 따로 관리하고 싶을 때 매우 유용한 기능입니다. 삼성 Knox 보안 기술을 기반으로 하기 때문에 암호화 수준도 매우 높습니다.

이 글에서는 갤럭시 보안 폴더를 처음 설정하는 방법부터 실제 활용 팁, 자주 발생하는 문제 해결 방법까지 상세히 알려드립니다.

## 보안 폴더 설정 전 준비 사항

보안 폴더를 사용하려면 다음 조건이 필요합니다.

- 삼성 갤럭시 스마트폰 (안드로이드 7.0 이상)
- 삼성 계정 로그인 (필수)
- 화면 잠금 설정 (PIN, 패턴, 비밀번호 중 하나)

삼성 계정이 없다면 먼저 설정 > 삼성 계정에서 계정을 만들거나 로그인하세요.

## 보안 폴더 처음 설정하기

### 1단계: 설정 앱 열기

스마트폰 홈 화면에서 **설정(톱니바퀴 아이콘)** 을 탭하거나, 상단 알림창을 두 번 내려서 설정 아이콘을 누릅니다.

### 2단계: 생체 인식 및 보안 메뉴 찾기

설정 화면에서 아래로 스크롤하여 **"생체 인식 및 보안"** 항목을 탭합니다. 갤럭시 모델이나 One UI 버전에 따라 **"보안 및 개인정보 보호"** 라고 표시될 수도 있습니다.

### 3단계: 보안 폴더 선택

생체 인식 및 보안 메뉴에서 **"보안 폴더"** 항목을 찾아 탭합니다. 처음 실행하는 경우 삼성 계정 로그인 화면이 나타납니다. 삼성 계정 이메일과 비밀번호를 입력하고 로그인합니다.

### 4단계: 잠금 방식 선택

보안 폴더에 사용할 잠금 방식을 선택합니다.

- **패턴**: 9개의 점을 연결하는 방식, 직관적이고 빠름
- **PIN**: 숫자 4~6자리 입력 방식
- **비밀번호**: 영문+숫자 조합, 보안 수준 가장 높음
- **지문**: 등록된 지문으로 빠르게 해제
- **얼굴 인식**: 전면 카메라로 얼굴 확인

보안 수준을 높이려면 PIN 또는 비밀번호와 지문을 함께 사용하는 것을 추천합니다. 원하는 방식을 선택하고 설정을 완료합니다.

### 5단계: 보안 폴더 아이콘 확인

설정이 완료되면 홈 화면이나 앱 서랍에 **자물쇠 모양의 보안 폴더 아이콘**이 생성됩니다. 이 아이콘을 탭하면 설정한 잠금 방식으로 인증 후 보안 폴더 내부로 진입할 수 있습니다.

## 보안 폴더에 앱 추가하기

### 방법 1: 보안 폴더 내부에서 추가

1. 보안 폴더를 열고 인증을 완료합니다.
2. 내부 화면에서 **"앱 추가"** 아이콘(+ 버튼)을 탭합니다.
3. **"Play 스토어에서 설치"** 를 선택하면 보안 폴더 전용으로 앱을 설치할 수 있고, **"일반 영역에서 추가"** 를 선택하면 기존에 설치된 앱을 보안 폴더에 복사할 수 있습니다.
4. 원하는 앱을 선택하고 **"추가"** 를 탭합니다.

### 방법 2: 일반 앱에서 이동

1. 앱 서랍에서 보안 폴더로 옮기고 싶은 앱을 **길게 누릅니다**.
2. 나타나는 메뉴에서 **"보안 폴더로 이동"** 을 선택합니다.
3. 앱이 일반 영역에서 제거되고 보안 폴더 안으로 이동됩니다.

> 참고: 일부 앱(예: 카카오톡)은 일반 영역과 보안 폴더 양쪽에 모두 설치해 두 개의 계정을 동시에 사용할 수 있습니다. 직장 카카오톡과 개인 카카오톡을 분리해서 사용하는 분들께 특히 유용합니다.

## 보안 폴더에 사진·파일 추가하기

### 갤러리 사진을 보안 폴더로 이동

1. 갤러리 앱을 열고 숨기고 싶은 사진이나 동영상을 선택합니다.
2. 오른쪽 상단 **점 세 개(더보기)** 를 탭한 후 **"보안 폴더로 이동"** 을 선택합니다.
3. 보안 폴더 인증 후 파일이 보안 폴더 내 갤러리로 이동됩니다.
4. 원본 갤러리에서는 해당 사진이 완전히 사라집니다.

### 파일을 보안 폴더로 이동

1. **내 파일** 앱을 열고 이동할 파일을 선택합니다.
2. **더보기 > 보안 폴더로 이동** 을 탭합니다.
3. 인증 후 파일이 이동됩니다.

## 보안 폴더 숨기기 및 표시

보안 폴더 자체를 아예 홈 화면에서 감추고 싶다면:

1. 보안 폴더를 열고 우측 상단 **점 세 개** 를 탭합니다.
2. **"보안 폴더 숨기기"** 를 선택합니다.
3. 홈 화면과 앱 서랍에서 보안 폴더 아이콘이 사라집니다.

다시 표시하려면:
1. **설정 > 생체 인식 및 보안 > 보안 폴더** 로 이동합니다.
2. **"보안 폴더 표시"** 토글을 켭니다.

또는 빠른 설정 패널(상단 알림창)에 보안 폴더 버튼을 추가해두면 더 편리하게 숨기기/표시 전환이 가능합니다.

## 보안 폴더 잠금 비밀번호를 잊어버렸을 때

보안 폴더 비밀번호를 잊어버렸을 때는 삼성 계정을 통해 재설정할 수 있습니다.

1. 보안 폴더 잠금 화면에서 **"비밀번호 찾기"** 또는 **"잠금 해제 방법을 모르시나요?"** 를 탭합니다.
2. **삼성 계정으로 재설정** 을 선택합니다.
3. 삼성 계정 비밀번호를 입력하면 보안 폴더 잠금을 초기화할 수 있습니다.

> 주의: 삼성 계정 비밀번호도 잊어버린 경우 보안 폴더 내부 데이터를 복구하기 매우 어렵습니다. 삼성 계정 비밀번호는 반드시 별도로 기록해두세요.

## 자주 묻는 질문 (Q&A)

**Q. 보안 폴더 안의 카카오톡은 일반 카카오톡과 별개인가요?**
A. 네, 완전히 별개의 앱으로 작동합니다. 다른 계정으로 로그인하거나, 같은 번호라도 독립된 환경에서 사용됩니다. 직장용/개인용 계정 분리에 많이 활용됩니다.

**Q. 보안 폴더의 앱을 업데이트하려면 어떻게 하나요?**
A. 보안 폴더 안으로 들어간 후 보안 폴더 내부의 Play 스토어에서 업데이트하거나, 일반 Play 스토어에서 "보안 폴더 앱 업데이트"를 함께 진행하면 됩니다.

**Q. 스마트폰을 초기화하면 보안 폴더 데이터는 어떻게 되나요?**
A. 초기화 시 보안 폴더의 모든 데이터가 삭제됩니다. 중요한 파일은 삼성 클라우드나 외부 저장소에 미리 백업해두세요.

**Q. 보안 폴더를 완전히 삭제하고 싶습니다.**
A. 보안 폴더 내부 > 점 세 개 > 설정 > 보안 폴더 삭제를 선택하면 폴더와 내부 모든 데이터가 삭제됩니다. 삭제 전에 중요 파일을 일반 영역으로 이동하세요.

**Q. 보안 폴더 사진이 갤러리 앱에도 보이나요?**
A. 보안 폴더로 이동한 사진은 일반 갤러리에서는 보이지 않습니다. 보안 폴더 내부 갤러리에서만 확인 가능합니다.

## 정리

갤럭시 보안 폴더는 한 번 설정해두면 프라이버시 관리가 훨씬 편리해지는 기능입니다. 민감한 사진, 개인 메모, 업무용 앱 등을 안전하게 분리 보관하고, 필요할 때만 꺼내 쓸 수 있어 실용적입니다.

핵심 정리:
- **설정 > 생체 인식 및 보안 > 보안 폴더** 에서 간단히 활성화
- 삼성 계정 필수, 별도 잠금 방식 설정
- 앱·사진·파일 모두 보안 폴더로 이동 가능
- 비밀번호 분실 시 삼성 계정으로 재설정
- 홈 화면에서 아이콘 자체를 숨길 수 있음

스마트폰에 개인 정보가 많다면 지금 바로 보안 폴더를 활용해 보세요!
