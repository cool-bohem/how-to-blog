## 윈도우 11이란? 업그레이드해야 할까요?

윈도우 11은 마이크로소프트가 2021년 출시한 운영체제로, 윈도우 10의 다음 버전이에요. 2026년 현재 마이크로소프트는 윈도우 10 지원을 종료할 예정이기 때문에, 아직 윈도우 10을 사용 중이라면 지금이 업그레이드 적기예요.

**윈도우 11의 주요 특징**
- 새롭게 디자인된 시작 메뉴와 작업 표시줄 (중앙 정렬)
- 둥근 모서리와 투명 효과로 더 깔끔한 디자인
- **스냅 레이아웃**: 창을 자동으로 배치해 멀티태스킹 편의성 향상
- 보안 기능 강화 (TPM 2.0, 안전 부팅 등)
- 안드로이드 앱 일부 지원 (Xbox 게임패스 통합)
- 성능 및 배터리 효율 개선

**윈도우 10 지원 종료 일정**
마이크로소프트는 2025년 10월 14일에 윈도우 10 공식 지원을 종료했어요. 더 이상 보안 업데이트를 받지 못하기 때문에 사이버 공격에 취약해질 수 있어요. 윈도우 11로 업그레이드가 강력히 권장돼요.

## 내 PC가 윈도우 11을 지원하는지 확인하는 방법

윈도우 11은 최소 사양 요구사항이 있어요. 먼저 내 PC가 호환되는지 확인해야 해요.

### 방법 1. PC 상태 검사 앱 사용 (가장 간편)

마이크로소프트가 제공하는 공식 호환성 확인 앱이에요.

1. 현재 PC에서 **윈도우 + I** 키 눌러 **설정** 열기
2. **Windows 업데이트** 클릭
3. **Windows 11로 업그레이드하기** 배너가 보이면 클릭
4. 또는 설정 → **시스템 → 정보 → PC 상태 검사** 클릭
5. **지금 확인** 버튼 클릭
6. 결과에서 호환 여부 확인

### 방법 2. 최소 사양 직접 확인

**윈도우 11 최소 시스템 요구사항 (2026년 기준)**

| 항목 | 최소 요구사항 |
|------|-------------|
| 프로세서 | 1GHz 이상, 2코어 이상 64비트 프로세서 |
| RAM | 4GB 이상 |
| 저장 공간 | 64GB 이상 |
| 시스템 펌웨어 | UEFI, 보안 부팅 지원 |
| TPM | TPM 버전 2.0 |
| 그래픽 카드 | DirectX 12 이상 |
| 디스플레이 | 9인치 이상, 720p 이상 해상도 |

**내 PC 사양 확인 방법**
1. **윈도우 + R** → `msinfo32` 입력 후 Enter
2. 시스템 정보 창에서 프로세서, RAM, 운영체제 버전 등 확인

### TPM 2.0 확인 방법

TPM은 보안 칩으로, 윈도우 11의 핵심 요구사항이에요.

1. **윈도우 + R** 키 → `tpm.msc` 입력 후 Enter
2. TPM 관리 창이 열리면:
   - **"이 컴퓨터에서 신뢰할 수 있는 플랫폼 모듈(TPM)을 찾을 수 없습니다"** → TPM 없음
   - **"TPM 제조업체 정보"에 버전이 2.0으로 표시** → 조건 충족

**TPM이 없거나 1.2인 경우**: BIOS/UEFI 설정에서 TPM을 활성화할 수 있는 경우도 있어요. PC 제조사에 따라 다르므로 제조사 홈페이지에서 확인하세요.

## 윈도우 11 무료 업그레이드 방법

### 방법 1. Windows Update를 통한 업그레이드 (권장)

가장 안전하고 간편한 방법이에요.

1. **설정 → Windows 업데이트** 열기 (윈도우 + I → Windows 업데이트)
2. **업데이트 확인** 클릭
3. **Windows 11로 업그레이드 준비 완료** 항목 확인
4. **다운로드 및 설치** 클릭
5. 다운로드 진행 (파일 크기 약 5~8GB, 시간이 걸릴 수 있어요)
6. 다운로드 완료 후 **지금 다시 시작** 또는 **예약된 시간에 다시 시작** 선택
7. PC가 재시작되며 설치 진행 (약 20~40분 소요)
8. 설치 완료 후 윈도우 11 환경으로 부팅

**주의**: 업그레이드 중에는 PC를 끄지 마세요. 전원이 끊기면 시스템에 오류가 생길 수 있어요.

### 방법 2. 설치 도우미(Installation Assistant)를 통한 업그레이드

Windows Update에 업그레이드 옵션이 안 보일 때 사용해요.

1. 마이크로소프트 공식 사이트(microsoft.com/ko-kr/software-download/windows11)에 접속
2. **Windows 11 설치 도우미** 다운로드
3. 다운로드한 파일 실행
4. 안내에 따라 업그레이드 진행
5. 재시작 후 설치 완료

### 방법 3. ISO 파일을 이용한 클린 설치 (고급 사용자용)

기존 데이터를 모두 지우고 깨끗하게 새로 설치하는 방법이에요. 컴퓨터가 많이 느려졌거나 오래된 경우 추천해요.

**주의**: 클린 설치 시 모든 데이터가 삭제돼요. 반드시 중요 파일을 백업하세요.

1. 마이크로소프트 공식 사이트에서 **미디어 만들기 도구** 다운로드
2. USB 드라이브(8GB 이상) 준비
3. 미디어 만들기 도구 실행 → USB 부팅 드라이브 만들기
4. PC를 USB로 부팅
5. 윈도우 11 설치 마법사 따라 진행

## 업그레이드 전 반드시 해야 할 것

### 데이터 백업

업그레이드는 대부분 기존 데이터를 유지하지만, 예상치 못한 오류가 생길 수 있어요.

**백업 방법**
- 중요 문서, 사진, 동영상을 외장 하드 또는 USB에 복사
- **OneDrive** 동기화 켜기 (설정 → 계정 → Windows 백업)
- 구글 드라이브, 네이버 MYBOX 등 클라우드 백업

### 현재 윈도우 정품 인증 확인

1. **설정 → 시스템 → 정품 인증** 확인
2. **Windows 정품 인증됨** 메시지 확인

정품 인증된 윈도우 10은 업그레이드 후에도 윈도우 11이 자동으로 정품 인증돼요.

### 드라이버 업데이트

오래된 드라이버는 윈도우 11과 충돌할 수 있어요.

1. **장치 관리자** 열기 (윈도우 + X → 장치 관리자)
2. 느낌표(!) 표시된 항목 있으면 우클릭 → **드라이버 업데이트**

## 윈도우 11 업그레이드 후 첫 설정

업그레이드 완료 후 몇 가지 설정을 확인하세요.

**시작 메뉴 레이아웃 변경**
1. 빈 화면 우클릭 → **개인 설정**
2. **시작** 메뉴 → 레이아웃 선택 (더 많은 핀 고정 / 더 많은 권장 사항)
3. 또는 작업 표시줄을 왼쪽 정렬로 바꿀 수 있어요: 설정 → 개인 설정 → 작업 표시줄 → 작업 표시줄 동작 → 작업 표시줄 맞춤: 왼쪽

**필요한 앱 재확인**
일부 오래된 앱은 윈도우 11에서 작동하지 않을 수 있어요. 설치된 앱을 확인하고 최신 버전으로 업데이트하세요.

## 자주 묻는 질문

**Q. 윈도우 10으로 다시 돌아갈 수 있나요?**
업그레이드 후 10일 이내에는 롤백이 가능해요. 설정 → 시스템 → 복구 → 이전 버전의 Windows로 돌아가기를 클릭하면 돼요. 10일이 지나면 클린 설치만 가능해요.

**Q. 업그레이드하면 기존 파일과 프로그램이 유지되나요?**
업그레이드(덮어쓰기 설치) 방식은 개인 파일과 설치된 프로그램이 유지돼요. 단, 일부 앱은 재설치나 재인증이 필요할 수 있어요.

**Q. 업그레이드 비용이 있나요?**
윈도우 10 정품 사용자라면 완전 무료예요. 따로 비용을 내지 않아도 돼요.

**Q. 업그레이드 후 컴퓨터가 느려지지 않나요?**
윈도우 11은 최소 4GB RAM을 요구해요. 4GB 미만이거나 오래된 HDD를 사용 중이라면 체감상 느리게 느껴질 수 있어요. SSD 교체나 RAM 추가를 고려해보세요.

**Q. 기업용 윈도우 10 라이선스도 무료 업그레이드가 되나요?**
개인 PC 기준 Windows 10 Home/Pro는 무료 업그레이드 대상이에요. 기업 볼륨 라이선스는 별도 절차가 필요할 수 있으니 IT 담당자에게 문의하세요.

**Q. 업그레이드 도중 인터넷이 끊어지면 어떻게 되나요?**
다운로드 중 끊기면 중단되지만, 재연결 후 이어받기가 가능한 경우가 많아요. 설치 중 끊기는 것은 문제가 될 수 있으니 안정적인 Wi-Fi 또는 유선 인터넷 연결 상태에서 진행하는 게 좋아요.

## 정리

윈도우 11 업그레이드는 윈도우 10 정품 사용자라면 무료로 할 수 있어요. 업그레이드 전에 내 PC가 호환되는지 확인하고, 중요한 데이터는 꼭 백업해두세요. Windows Update를 통한 업그레이드가 가장 쉽고 안전하므로 이 방법을 먼저 시도해보세요. 윈도우 10 지원이 종료된 지금, 보안을 위해서라도 윈도우 11로 넘어가는 것이 좋아요.
