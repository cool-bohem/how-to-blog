## 카카오페이 송금이란?

카카오페이는 카카오톡 안에 내장된 간편 금융 서비스예요. 은행 앱을 따로 열지 않아도, 계좌번호를 몰라도 카카오톡 친구에게 바로 돈을 보낼 수 있어요.

**카카오페이 송금의 장점**:
- **수수료 무료** (횟수 제한 없음)
- 상대방 계좌번호 불필요 (카카오톡 친구면 됨)
- 24시간 즉시 송금
- 소액도 부담 없이 사용 가능

밥값, 커피값 더치페이나 축의금, 용돈 송금 등에 자주 활용돼요.

## 카카오페이 처음 설정하기

카카오페이를 처음 사용한다면 계좌 연결이 필요해요.

1. 카카오톡 하단 **더보기(···)** 탭 클릭
2. **페이(💰)** 아이콘 클릭
3. **카카오페이 시작하기** 클릭
4. **본인 인증**: 이름, 생년월일, 휴대폰 번호 입력
5. **계좌 연결**: 은행 선택 → 계좌번호 입력 → 1원 인증
6. **간편 비밀번호** 6자리 설정 (지문/얼굴 인식도 설정 가능)

**팁**: 1원 인증이란 연결할 계좌로 1원을 보내고, 입금자 이름에 표시된 인증번호를 입력하는 방식이에요. 본인 계좌임을 확인하는 과정이에요.

## 송금 방법

**방법 1. 채팅방에서 바로 송금**

1. 카카오톡에서 **송금할 친구와의 채팅방** 열기
2. 하단 **+ 버튼** 클릭
3. **송금** 선택
4. 금액 입력
5. **보내기** 클릭
6. 지문/얼굴 인식 또는 비밀번호로 확인

**방법 2. 카카오페이 앱에서 송금**

1. 카카오페이 메인 화면
2. **송금** 클릭
3. 받을 사람 검색 (이름, 전화번호, 카카오 ID)
4. 금액 입력 후 **보내기**

## 카카오페이 머니 충전

카카오페이 머니는 카카오페이 전용 잔액이에요. 연결된 은행 계좌에서 충전해서 사용해요.

1. 카카오페이 메인 화면
2. **충전** 클릭
3. 충전 금액 입력 (최소 1,000원)
4. 연결된 계좌에서 즉시 이체

**팁**: 자동 충전 기능을 설정하면 잔액이 일정 금액 이하로 떨어질 때 자동으로 충전돼요.

## 카카오페이 머니 출금

카카오페이 머니를 다시 은행 계좌로 뺄 수 있어요.

1. **출금** 클릭
2. 출금할 계좌 선택
3. 금액 입력 후 출금 (즉시 처리)

## 송금 받기

상대방이 송금하면 카카오톡 메시지로 알림이 와요.

- 카카오페이 계좌 연결이 된 경우: 자동으로 카카오페이 머니로 입금
- 카카오페이 미사용자: 수령 링크 클릭 후 계좌번호 입력해야 받을 수 있어요

**팁**: 송금받을 계좌를 미리 설정해두면 자동으로 내 통장으로 입금돼요.

## 이용 한도 및 주의사항

| 항목 | 한도 |
|---|---|
| 1회 송금 한도 | 200만원 |
| 1일 송금 한도 | 500만원 |
| 카카오페이머니 최대 보유 | 2,000만원 |

**보이스피싱 주의**: 모르는 사람이 카카오톡으로 송금을 요청하거나, 급하게 돈이 필요하다며 연락이 오면 절대 송금하지 마세요. 가족이나 지인을 사칭하는 경우도 많으니 전화로 직접 확인하세요.

## 자주 묻는 질문

**Q. 카카오페이 계좌가 없는 친구에게도 송금할 수 있나요?**
네. 카카오페이 미사용자에게도 송금할 수 있어요. 받는 분이 수령 링크를 통해 계좌번호를 입력하면 받을 수 있어요.

**Q. 잘못 보냈을 때 취소할 수 있나요?**
송금 직후 취소 요청이 가능해요. 카카오페이 → 거래 내역 → 해당 송금 → 취소 요청. 단, 상대방이 이미 출금했다면 취소가 불가능해요.

**Q. 카카오페이 사용 내역은 어디서 볼 수 있나요?**
카카오페이 앱 → **전체 서비스 → 거래 내역** 에서 모든 입출금 내역을 확인할 수 있어요.
