## C드라이브 용량이 부족하면 생기는 문제

윈도우 PC를 오래 사용하다 보면 어느 순간 C드라이브의 여유 공간이 빨간색으로 바뀌면서 "디스크 공간이 부족합니다"라는 경고가 뜨는 경우가 있습니다. 이 상태가 되면 다음과 같은 문제가 생깁니다.

- 윈도우 업데이트가 설치되지 않음
- 프로그램 설치나 저장이 안 됨
- 컴퓨터 속도가 급격히 느려짐
- 임시 파일이 계속 쌓여 악순환 반복

다행히 윈도우 11에는 저장공간을 정리하는 여러 가지 방법이 내장되어 있습니다. 순서대로 따라 하면 생각보다 많은 공간을 쉽게 확보할 수 있습니다.

## 방법 1: 기본 디스크 정리 도구 사용하기

윈도우 기본 제공 도구인 **디스크 정리**는 임시 파일, 휴지통, 인터넷 캐시 등을 한 번에 삭제해줍니다.

### 1단계: 디스크 정리 실행

1. **파일 탐색기**를 엽니다 (단축키: **Win + E**)
2. 왼쪽 패널에서 **내 PC** 클릭
3. **C: 드라이브**를 마우스 오른쪽 버튼으로 클릭
4. **속성** 선택
5. **일반** 탭에서 **디스크 정리** 버튼 클릭

또는 **시작 메뉴** 검색창에 "디스크 정리"를 입력해서 바로 실행할 수도 있습니다.

### 2단계: 정리할 항목 선택

디스크 정리 창이 열리면 정리 가능한 파일 목록이 나타납니다. 각 항목 앞의 체크박스를 선택하거나 해제하세요.

- **임시 인터넷 파일**: 브라우저 캐시 파일로 삭제해도 무방
- **다운로드한 프로그램 파일**: 삭제 가능
- **임시 파일**: 삭제 가능
- **휴지통**: 휴지통에 있는 파일 영구 삭제
- **미리 보기 그림**: 폴더 미리보기 캐시로 삭제 가능
- **오류 보고 파일**: 삭제 가능

모두 체크한 뒤 **확인** 버튼을 클릭하고 **파일 삭제** 를 눌러 진행합니다.

### 3단계: 시스템 파일 정리 (더 많은 공간 확보)

일반 디스크 정리보다 훨씬 많은 공간을 확보하려면 **시스템 파일 정리** 기능을 활용하세요.

1. 디스크 정리 창에서 **시스템 파일 정리** 버튼 클릭
2. 관리자 권한 허용 후 잠시 대기
3. 새로운 항목들이 추가로 나타남:
   - **Windows 업데이트 정리**: 이전 윈도우 업데이트 파일 (수 GB 절약 가능)
   - **이전 Windows 설치**: 윈도우 업그레이드 전 파일 (10GB 이상 절약 가능)
   - **Windows 업그레이드 로그 파일**
4. 원하는 항목 체크 후 **확인** 클릭

특히 **Windows 업데이트 정리**와 **이전 Windows 설치** 항목은 수 GB에서 수십 GB까지 확보할 수 있어 가장 효과적입니다.

## 방법 2: 저장소 감지(Storage Sense) 자동화 설정

**저장소 감지**는 윈도우 11이 자동으로 주기적으로 불필요한 파일을 삭제해주는 기능입니다.

### 설정 방법

1. **설정** 앱 열기 (단축키: **Win + I**)
2. **시스템** → **저장소** 선택
3. **저장소 감지** 토글을 **켬**으로 전환
4. **저장소 감지 구성** 클릭하여 세부 설정
   - **저장소 감지 실행 빈도**: 매일 / 매주 / 매월 중 선택 (매주 권장)
   - **임시 앱 파일 삭제**: 켬 설정
   - **휴지통 자동 비우기**: 30일 이상 된 항목 자동 삭제
   - **다운로드 폴더**: 사용하지 않는 파일 자동 삭제 (중요 파일이 있다면 주의)

### 지금 바로 정리 실행하기

저장소 감지 설정 화면 하단의 **지금 저장소 감지 실행** 버튼을 클릭하면 즉시 정리가 실행됩니다.

## 방법 3: 대용량 파일 찾아서 삭제하기

저장소 설정에서 어떤 파일이 공간을 많이 차지하는지 확인할 수 있습니다.

### 1단계: 저장소 사용 현황 확인

1. **설정** → **시스템** → **저장소** 이동
2. **C: 드라이브** 클릭
3. 카테고리별 용량 사용 현황이 나타남:
   - 앱 및 기능
   - 임시 파일
   - 문서, 사진, 음악, 동영상
   - 기타

### 2단계: 앱 및 기능 정리

**앱 및 기능** 항목을 클릭하면 설치된 프로그램 목록이 용량순으로 정렬됩니다. 사용하지 않는 프로그램을 찾아 **제거** 버튼으로 삭제하세요.

**특히 확인해야 할 항목:**
- 오래된 게임 클라이언트(스팀, 배틀넷 등)
- 사용하지 않는 오피스 프로그램
- 예전에 설치했다 잊어버린 앱들

### 3단계: 개인 파일 정리

**동영상** 카테고리는 의외로 공간을 많이 차지합니다. 파일 탐색기에서 다운로드 폴더, 동영상 폴더, 바탕화면을 직접 확인하고 필요 없는 파일을 삭제하세요.

## 방법 4: 임시 파일 직접 삭제하기

윈도우의 임시 파일 폴더를 직접 비우면 추가 공간을 확보할 수 있습니다.

### %temp% 폴더 정리

1. 키보드에서 **Win + R** 키를 누름
2. 실행 창에 **%temp%** 입력 후 **Enter**
3. 폴더 안의 모든 파일을 선택 (Ctrl + A)
4. Delete 키로 삭제 (일부 사용 중인 파일은 삭제가 안 될 수 있으니 건너뛰기 선택)

### temp 폴더 정리

1. Win + R 키 누름
2. **temp** 입력 후 Enter
3. 동일하게 전체 선택 후 삭제

이 두 폴더는 프로그램이 임시로 만들어놓은 파일들이 모이는 곳으로, 삭제해도 PC 사용에 전혀 지장이 없습니다.

## 방법 5: 휴지통 비우기

삭제한 파일은 휴지통에 임시 보관되어 있어 실제 저장공간은 그대로 차지합니다.

1. 바탕화면의 **휴지통** 아이콘을 마우스 오른쪽 버튼으로 클릭
2. **휴지통 비우기** 선택
3. 확인 버튼 클릭

또는 휴지통을 열어서 내용물을 확인한 뒤, 필요한 파일은 복원하고 나머지만 삭제할 수도 있습니다.

## 방법 6: OneDrive 또는 외장 드라이브로 파일 이동

중요하지만 자주 사용하지 않는 파일(사진, 동영상, 문서 등)은 외부 저장소로 이동하는 것이 근본적인 해결책입니다.

- **OneDrive**: 윈도우 11에 기본 포함. 5GB 무료 제공. 파일을 클라우드에 올리고 로컬에서 삭제 가능
- **Google 드라이브 / 네이버 마이박스**: 추가 클라우드 스토리지 옵션
- **외장 HDD / USB**: 오래된 파일, 영상 등 대용량 파일 이동에 적합

### OneDrive로 파일 이동 후 로컬 사본 삭제

1. 파일 탐색기에서 OneDrive 폴더에 파일 복사
2. 동기화가 완료되면(파일 아이콘에 파란 체크 표시) 원본 파일 삭제
3. OneDrive 설정에서 **공간 확보** 옵션으로 클라우드에만 보관 가능

## 자주 묻는 질문 (Q&A)

**Q. 디스크 정리를 해도 용량이 별로 확보되지 않아요.**
A. 기본 디스크 정리보다 **시스템 파일 정리** 버튼을 클릭해서 나오는 "Windows 업데이트 정리"와 "이전 Windows 설치" 항목을 삭제해보세요. 이 항목들이 수 GB ~ 수십 GB까지 차지하는 경우가 많습니다.

**Q. C드라이브가 항상 부족해요. 파티션을 늘릴 수 없나요?**
A. D드라이브 등 다른 파티션에 여유 공간이 있다면 윈도우 디스크 관리 도구로 C드라이브를 확장할 수 있습니다. 다만 이 작업은 다소 위험할 수 있으므로 중요한 파일은 반드시 미리 백업하세요.

**Q. 프로그램을 삭제했는데도 폴더가 남아 있어요.**
A. 일부 프로그램은 제거 후에도 설정 파일이나 사용자 데이터 폴더를 남겨놓습니다. C:\Program Files, C:\Program Files (x86), C:\Users\[사용자명]\AppData 폴더를 확인해서 해당 프로그램 폴더를 수동으로 삭제하면 됩니다. AppData는 숨김 폴더이므로 파일 탐색기 상단 **보기 → 숨긴 항목 표시**를 먼저 체크하세요.

**Q. %temp% 폴더 정리 시 "다른 프로세스가 사용 중"이라는 오류가 나요.**
A. 현재 실행 중인 프로그램이 사용 중인 임시 파일은 삭제가 되지 않습니다. "건너뛰기"를 선택해 삭제 가능한 파일만 지우면 됩니다. 컴퓨터를 재시작한 뒤 다시 시도하면 더 많이 삭제됩니다.

**Q. 바탕화면에 파일을 많이 저장하면 안 되나요?**
A. 바탕화면 파일도 C드라이브에 저장됩니다(C:\Users\[사용자명]\Desktop). 특히 고화질 사진이나 동영상을 바탕화면에 두면 C드라이브 용량에 바로 영향을 줍니다. 중요한 파일은 D드라이브나 문서 폴더로 이동하세요.

## 요약

C드라이브 저장공간을 확보하는 방법을 단계별로 정리하면 다음과 같습니다.

1. **디스크 정리** → 시스템 파일 정리 포함해서 실행 (윈도우 업데이트 정리 항목 꼭 체크)
2. **저장소 감지** 켜두기 → 자동 정리 설정
3. **임시 파일 폴더** (%temp%, temp) 직접 정리
4. **사용 안 하는 앱** 제거
5. **휴지통** 비우기
6. **대용량 파일** (사진, 영상)은 클라우드나 외장 드라이브로 이동

이 과정을 순서대로 진행하면 대부분의 경우 수 GB에서 수십 GB까지 공간을 확보할 수 있습니다. PC를 쾌적하게 사용하려면 C드라이브 여유 공간을 최소 전체 용량의 10~15% 이상 유지하는 것이 좋습니다.
