## 아이폰 → 갤럭시 이전 전에 확인할 것

기기를 바꾸기 전에 아래 항목들을 미리 준비해두면 이전 과정이 훨씬 수월해요.

**이전 전 체크리스트**:
- 아이폰에서 iCloud 백업 최신화 (설정 → 이름 → iCloud → iCloud 백업)
- 카카오톡 대화 백업 완료 확인
- 중요한 사진 별도 백업
- 이중 인증 앱(Google Authenticator 등) 새 기기로 이전

아이폰과 안드로이드는 운영체제가 달라서 앱 자체를 통째로 옮기는 건 불가능해요. 항목별로 하나씩 이전해야 해요.

## 연락처 이전

**방법 1. 구글 계정 동기화 (추천)**

이 방법이 가장 안정적이고 나중에도 두 기기 간 동기화가 유지돼요.

1. 아이폰 **설정 → 연락처 → 계정 → 계정 추가**
2. **Google** 선택 후 구글 계정으로 로그인
3. **연락처 동기화** 켜기 → 아이폰 연락처가 구글 계정에 업로드됨
4. 갤럭시에서 같은 구글 계정으로 로그인하면 자동으로 연락처 불러옴

**방법 2. vCard 파일로 내보내기**

1. 아이폰 **연락처 앱 → 메뉴 → 연락처 내보내기**
2. vCard(.vcf) 파일로 저장
3. 파일을 이메일/카카오톡으로 갤럭시에 전송
4. 갤럭시에서 파일 열기 → 연락처로 가져오기

## 사진 이전

**방법 1. 구글 포토 (무료 15GB)**

1. 아이폰에 **구글 포토** 앱 설치
2. 구글 계정으로 로그인 후 **백업 켜기**
3. 모든 사진/영상 백업 완료까지 대기 (사진이 많으면 시간이 걸려요)
4. 갤럭시에서 같은 계정으로 구글 포토 앱 로그인하면 모두 확인 가능

**방법 2. PC를 통한 직접 이전**

1. 아이폰을 USB로 PC에 연결 → 사진 폴더 복사
2. PC에서 갤럭시 연결 후 붙여넣기

**팁**: 사진이 15GB 이상이면 구글 포토 유료 요금제(월 2,900원, 100GB)를 고려해볼 수 있어요.

## 카카오톡 대화 이전

카카오톡은 기기 변경 시 대화 백업과 복구가 가장 신경 쓰이는 부분이에요.

**1단계. 아이폰에서 백업**
1. 카카오톡 **더보기(···) → 설정 → 채팅 → 대화 백업**
2. **카카오 계정으로 백업** 선택
3. 카카오 계정 비밀번호 입력 후 백업 완료까지 대기

**2단계. 갤럭시에서 복구**
1. 갤럭시에 카카오톡 설치 후 **같은 카카오 계정**으로 로그인
2. 로그인 과정에서 **"이전 대화를 불러오시겠어요?"** 팝업 확인
3. **불러오기** 클릭 → 복구 완료

**주의사항**:
- **오픈채팅 내역**은 이전되지 않아요
- 아이폰에서 백업한 데이터는 안드로이드에서 복구 가능하지만, 사진/동영상 첨부파일은 포함되지 않을 수 있어요

## Samsung Smart Switch 활용 (한 번에 이전)

삼성 공식 앱으로 대부분의 데이터를 한 번에 이전할 수 있어요.

1. **아이폰**에 Smart Switch 앱 설치 (App Store)
2. **갤럭시**에서 Smart Switch 앱 실행 (기본 설치 또는 Play Store)
3. 아이폰과 갤럭시를 같은 와이파이 또는 케이블로 연결
4. 이전할 항목 선택 (연락처, 메시지, 사진, 앱 목록 등)
5. 전송 시작

**이전 가능한 항목**: 연락처, SMS, 사진, 동영상, 캘린더, 메모, 일부 앱 데이터

**주의**: 아이폰 전용 앱(iMessage, FaceTime 등)은 안드로이드로 이전되지 않아요.

## 자주 묻는 질문

**Q. iMessage가 갤럭시로 안 와요.**
갤럭시 전환 후 아이폰에서 iMessage를 비활성화해야 해요. **설정 → 메시지 → iMessage 끄기**. 그렇지 않으면 다른 아이폰 사용자가 보낸 문자가 iMessage로 전송되어 갤럭시에서 수신이 안 돼요.

**Q. 앱 자체도 이전되나요?**
iOS 앱은 안드로이드에서 실행되지 않아요. Smart Switch가 "이 앱의 안드로이드 버전이 있음"을 알려주면 Play Store에서 다시 설치해야 해요.

**Q. 카카오톡 오픈채팅은 정말 이전이 안 되나요?**
현재 기준으로 오픈채팅 내역은 백업/복구 대상이 아니에요. 중요한 정보는 미리 따로 저장해두세요.
