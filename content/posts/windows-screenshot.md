## 윈도우 화면 캡처, 여러 방법이 있어요

윈도우에는 화면을 캡처하는 방법이 생각보다 다양해요. 단순히 전체 화면을 찍는 것부터, 원하는 부분만 정확하게 캡처하고 바로 편집하는 것까지 가능해요. 상황에 맞는 방법을 알아두면 훨씬 편리해요.

## 방법 1. Print Screen 키 — 전체 화면 캡처

가장 기본적인 방법이에요.

- **PrtSc** 키를 누르면 현재 화면이 클립보드에 복사됨
- 그림판, Word, 카카오톡 등에 **Ctrl + V** 로 붙여넣기 후 사용

**주의**: 이 방법은 파일로 바로 저장되지 않아요. 반드시 다른 곳에 붙여넣기 후 저장해야 해요.

## 방법 2. Win + Print Screen — 자동 파일 저장

파일로 바로 저장하고 싶을 때 사용해요.

- **Win + PrtSc** 동시에 누르기
- 화면이 잠깐 어두워지면 캡처 성공
- **내 PC → 사진 → Screenshots** 폴더에 자동으로 PNG 파일로 저장

**팁**: 가장 자주 쓰게 될 방법이에요. 바로 파일로 저장되어 나중에 찾기도 쉬워요.

## 방법 3. Alt + Print Screen — 현재 창만 캡처

여러 창이 열려 있을 때, 현재 사용 중인 창만 캡처하고 싶을 때 사용해요.

- **Alt + PrtSc** 동시에 누르기
- 현재 활성화된 창만 클립보드에 복사됨
- **Ctrl + V** 로 붙여넣기

**팁**: 브라우저 하나, 엑셀 하나를 따로 캡처할 때 유용해요.

## 방법 4. Win + Shift + S — 원하는 영역만 캡처 (최추천)

가장 실용적인 방법이에요. 원하는 부분만 정확하게 잘라서 캡처할 수 있어요.

1. **Win + Shift + S** 동시에 누르기
2. 화면이 어두워지면서 상단에 툴바 표시
3. 캡처 방식 선택:
   - **직사각형 캡처**: 드래그로 영역 선택
   - **자유형 캡처**: 자유롭게 그려서 선택
   - **창 캡처**: 특정 창 클릭으로 캡처
   - **전체 화면 캡처**: 전체 화면 바로 캡처
4. 캡처 후 오른쪽 하단 알림 클릭 → 편집 및 저장 가능

**팁**: 캡처한 이미지는 클립보드에도 복사되어 있어서 카카오톡, 이메일 등에 바로 붙여넣기 가능해요.

## 방법 5. 캡처 도구 앱 — 편집 기능 포함

윈도우 기본 앱으로 더 다양한 캡처와 간단한 편집이 가능해요.

1. 작업표시줄 검색창에 **캡처 도구** 또는 **Snipping Tool** 입력
2. 앱 실행 후 **새로 만들기** 클릭
3. 캡처 방식 선택 (모드):
   - 직사각형 캡처
   - 자유형 캡처
   - 창 캡처
   - 전체 화면 캡처
4. 영역 선택 후 캡처
5. 편집 도구로 강조, 크롭, 텍스트 추가 가능
6. 파일로 저장

**새 기능**: 윈도우 11 기준 캡처 도구는 화면 녹화 기능도 추가됐어요.

## 방법별 비교

| 방법 | 자동 저장 | 영역 선택 | 편집 기능 |
|---|---|---|---|
| PrtSc | X (붙여넣기 필요) | X | X |
| Win + PrtSc | O (PNG 파일) | X | X |
| Alt + PrtSc | X | 현재 창만 | X |
| Win + Shift + S | X (클립보드) | O | 간단히 |
| 캡처 도구 | O | O | O |

## 자주 묻는 질문

**Q. 캡처한 화면을 JPG로 저장하고 싶어요.**
Win + PrtSc는 PNG로만 저장돼요. JPG로 저장하려면 그림판이나 캡처 도구에서 **다른 이름으로 저장 → JPEG** 선택 후 저장하세요.

**Q. 캡처 단축키가 작동하지 않아요.**
한글 입력 모드에서 단축키가 안 먹히는 경우가 있어요. 영어 입력 상태에서 시도해보세요. 또한 노트북에서는 **Fn** 키와 함께 눌러야 하는 경우도 있어요.

**Q. 스크롤 캡처(긴 웹페이지 전체 캡처)도 가능한가요?**
윈도우 기본 기능으로는 스크롤 캡처가 지원되지 않아요. 크롬 브라우저 확장 프로그램(GoFullPage 등)이나 외부 앱(ShareX, Greenshot)을 사용하면 가능해요.
