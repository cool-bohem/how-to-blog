## 윈도우 화면 캡처 방법

## 방법 1. Print Screen 키 (전체 화면)

- **PrtSc** 키를 누르면 클립보드에 복사됨
- 그림판이나 워드에 **Ctrl+V** 로 붙여넣기 후 저장

## 방법 2. Win + Print Screen (자동 저장)

- **Win + PrtSc** 동시에 누르기
- 화면이 잠깐 어두워지면 캡처 성공
- **내 PC → 사진 → Screenshots** 폴더에 자동 저장

## 방법 3. Alt + Print Screen (현재 창만)

- **Alt + PrtSc** 로 현재 활성화된 창만 캡처
- 클립보드에 복사됨

## 방법 4. Win + Shift + S (원하는 영역만)

1. **Win + Shift + S** 동시에 누르기
2. 화면이 어두워지면 캡처할 영역 드래그
3. 클립보드에 복사됨
4. 알림 클릭하면 편집 가능

## 방법 5. 캡처 도구 앱

1. 검색창에 **캡처 도구** 입력
2. 앱 실행 후 **새로 만들기** 클릭
3. 영역, 창, 전체 화면 등 선택하여 캡처
4. 바로 편집 및 저장 가능
