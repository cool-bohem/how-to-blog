## 애플 ID 비밀번호 찾기

애플 ID 비밀번호를 잊어버렸을 때 복구하는 방법이에요.

## 방법 1. 아이폰에서 직접 복구 (가장 쉬움)

1. **설정 → 상단 이름** 클릭
2. **비밀번호 및 보안 → 비밀번호 변경**
3. 기기 잠금 해제 후 새 비밀번호 설정

## 방법 2. iforgot.apple.com 에서 복구

1. **iforgot.apple.com** 접속
2. 애플 ID(이메일) 입력
3. 복구 방법 선택:
   - **신뢰할 수 있는 기기로 재설정**
   - **신뢰할 수 있는 전화번호로 재설정**
4. 인증 완료 후 새 비밀번호 설정

## 방법 3. 계정 복구 키 사용

미리 계정 복구 키를 설정한 경우:

1. **비밀번호를 잊으셨나요?** 선택
2. **계정 복구 키 사용** 선택
3. 28자리 복구 키 입력

## 이중 인증 설정 (분실 예방)

1. **설정 → 이름 → 비밀번호 및 보안**
2. **이중 인증 켜기**
3. 신뢰할 수 있는 전화번호 등록

이중 인증을 켜두면 나중에 복구가 훨씬 쉬워요.

## 주의사항

- 애플 ID는 아이폰, 맥, 아이패드 등 모든 애플 기기와 연결되어 있어요
- 복구가 안 될 경우 **애플 고객센터(1-800-275-2273)** 에 문의하세요
