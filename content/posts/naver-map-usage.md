## 네이버 지도가 카카오맵, 구글 지도보다 좋은 이유

네이버 지도는 국내 이용자에게 최적화된 지도 앱이에요. 대중교통 정보, 실시간 교통량 반영, 버스/지하철 도착 정보 등이 국내에서 가장 정확하다는 평가를 받아요.

**특히 뛰어난 기능**:
- 실시간 버스 위치 추적
- 길 찾기 시 예상 소요 시간 정확도
- 도보 길찾기 골목길 안내
- 식당, 카페 등 장소 리뷰와 영업시간 정보

## 기본 길찾기 방법

1. 네이버 지도 앱 실행
2. 하단 **길찾기** 버튼 클릭 또는 장소 검색 후 **길찾기** 클릭
3. **출발지, 도착지** 입력
   - 현재 위치를 출발지로 설정하려면 **내 위치** 선택
4. 이동 수단 선택 (대중교통/자동차/도보/자전거)
5. 여러 경로 중 원하는 것 선택

## 대중교통 길찾기 팁

대중교통 결과 화면에서 다양한 필터를 활용할 수 있어요.

**필터 활용**
- **최소 환승**: 환승 횟수를 줄인 경로
- **최단 시간**: 가장 빠른 경로
- **최소 도보**: 걷는 거리를 줄인 경로

**막차 시간 확인**
- 경로 결과 하단에 막차 시간 표시
- 늦은 귀가 전 미리 확인하세요

**도착 시간 기준 역방향 검색**
- 특정 시간까지 도착해야 할 때 역산해서 출발 시간을 알려줘요
- 길찾기 화면에서 **출발/도착 시간** 을 탭하면 설정 가능

**버스 실시간 위치 확인**
- 정류장 검색 후 버스별 도착 예정 시간 실시간 확인
- 몇 정거장 전인지, 몇 분 후에 오는지 정확하게 파악 가능

## 자동차 길찾기 팁

**실시간 교통 반영**
- 현재 막히는 구간을 반영해서 예상 도착 시간을 계산해줘요
- 빨간색(정체), 노란색(서행), 초록색(원활)로 도로 상태 표시

**회피 도로 설정**
- **고속도로 제외**: 통행료를 아끼고 싶을 때
- **유료도로 제외**: 고속도로 + 유료 터널/다리 제외
- **최단 경로**: 거리 기준으로 최적화

**출발 예약**
- 미래 특정 시간 기준으로 예상 소요 시간을 계산해줘요
- "내일 아침 8시에 출발하면 얼마나 걸릴까?" 같은 상황에서 유용해요

## 유용한 기능들

**집/회사 등록하기**
- 자주 가는 곳을 등록해두면 홈 화면에서 바로 길찾기 가능
- **지도 홈 → 집/회사 등록** 클릭

**즐겨찾기 저장**
- 자주 가는 식당, 카페 등을 별표(★)로 저장
- **즐겨찾기 탭**에서 한눈에 확인 가능

**주차장 검색**
- 목적지 근처 주차장 정보 확인
- 요금, 운영 시간, 빈 자리 수까지 실시간 표시

**거리뷰**
- 처음 가는 곳의 외관을 미리 사진으로 확인
- 장소 카드 → **거리뷰** 탭 클릭

## 네이버 지도 vs 카카오맵 비교

| | 네이버 지도 | 카카오맵 |
|---|---|---|
| 대중교통 정보 | 매우 정확 | 정확 |
| 장소 리뷰 | 블로그 리뷰 연동 | 풍부한 별점 리뷰 |
| 길찾기 정확도 | 국내 최고 수준 | 우수 |
| 해외 지원 | 제한적 | 제한적 |
| 실시간 버스 위치 | 지원 | 지원 |

## 자주 묻는 질문

**Q. 네이버 지도에서 실시간 내 위치를 친구에게 공유할 수 있나요?**
네이버 지도는 특정 장소 공유는 가능하지만 실시간 위치 공유 기능은 카카오맵이 더 편리해요.

**Q. 지하철 노선도를 보려면?**
앱 하단 **더보기** 또는 검색창에 **지하철 노선도** 검색하면 전국 지하철 노선도를 확인할 수 있어요.

**Q. 자전거 길찾기도 지원하나요?**
네. 길찾기 이동 수단에서 **자전거** 를 선택하면 자전거 전용도로와 적합한 경로를 안내해줘요. 공공 자전거 대여소 위치도 확인 가능해요.

**Q. 오프라인에서도 사용할 수 있나요?**
인터넷이 없으면 실시간 교통 정보나 대중교통 정보는 이용할 수 없어요. 해외 여행 시에는 구글 지도의 오프라인 저장 기능이 더 유용해요.
