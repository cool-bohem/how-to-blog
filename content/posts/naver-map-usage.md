## 네이버 지도 길찾기 기본

네이버 지도는 대중교통, 자동차, 도보, 자전거까지 다양한 이동 수단의 길찾기를 지원해요.

## 기본 길찾기 방법

1. 네이버 지도 앱 실행
2. 하단 **길찾기** 버튼 클릭
3. **출발지, 도착지** 입력
4. 이동 수단 선택 (대중교통/자동차/도보)
5. 원하는 경로 선택

## 대중교통 길찾기 팁

- **최소 환승** / **최단 시간** / **최소 도보** 필터 활용
- 시간대별 막차 시간 확인 가능
- **도착 시간 설정**으로 역방향 검색 가능 (몇 시에 출발해야 하는지)

## 자동차 길찾기 팁

- **실시간 교통** 반영으로 막히는 길 피하기
- **회피 도로** 설정: 고속도로, 유료도로 제외 가능
- **주차장 검색** 기능으로 도착지 근처 주차장 확인

## 유용한 기능들

**즐겨찾기 등록**
- 자주 가는 곳은 별표(★)로 저장해두면 빠르게 검색 가능

**내 장소 저장**
- 집, 회사 등록하면 홈 화면에서 바로 길찾기 가능

**거리뷰**
- 목적지 주변을 미리 사진으로 확인 가능
