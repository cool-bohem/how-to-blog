파워포인트(Microsoft PowerPoint)는 발표 자료, 교육 자료, 업무 보고서를 만들 때 가장 많이 사용하는 프레젠테이션 프로그램입니다. 처음에는 어렵게 느껴질 수 있지만, 기본 기능만 익혀도 깔끔한 자료를 만들 수 있습니다. 이 글에서는 파워포인트를 처음 열어 보는 분도 따라 할 수 있도록 처음부터 차근차근 설명합니다.

## 파워포인트 실행 및 새 프레젠테이션 만들기

1. 윈도우 시작 버튼을 클릭하고 검색창에 **PowerPoint**를 입력한 후 엔터를 누릅니다.
2. 파워포인트가 실행되면 시작 화면이 나타납니다.
3. **새 프레젠테이션**을 클릭하면 빈 슬라이드가 열립니다.
4. 또는 미리 만들어진 디자인 템플릿을 사용하려면 검색창에 원하는 키워드(예: '비즈니스', '교육')를 입력하고 마음에 드는 템플릿을 클릭합니다.

**화면 구성 살펴보기:**
- **슬라이드 창:** 가운데 큰 영역으로 실제 슬라이드 내용을 편집하는 공간입니다.
- **슬라이드 패널:** 왼쪽에 슬라이드 목록이 작게 표시됩니다.
- **리본 메뉴:** 상단에 홈, 삽입, 디자인 등 탭이 있습니다.
- **노트 영역:** 슬라이드 아래쪽에 발표자 메모를 입력하는 공간입니다.

## 슬라이드에 텍스트 입력하기

1. 빈 슬라이드에는 **'제목을 추가하려면 클릭하십시오'** 라는 안내 문구가 있습니다. 이 영역을 클릭하면 텍스트를 입력할 수 있습니다.
2. 원하는 제목을 입력합니다. 예: **2026년 1분기 업무 보고**
3. 아래 **'부제목을 추가하려면 클릭하십시오'** 영역을 클릭하고 부제목을 입력합니다.
4. 텍스트 크기, 글꼴, 색상을 바꾸려면 텍스트를 드래그하여 선택한 후, 상단 **홈** 탭에서 원하는 스타일을 클릭합니다.

**텍스트 상자 새로 추가하기:**
1. 상단 **삽입** 탭을 클릭합니다.
2. **텍스트 상자**를 클릭합니다.
3. 슬라이드에서 원하는 위치를 드래그하면 텍스트 상자가 생성됩니다.
4. 바로 텍스트를 입력할 수 있습니다.

## 새 슬라이드 추가 및 레이아웃 변경

1. 왼쪽 슬라이드 패널에서 마지막 슬라이드를 오른쪽 클릭합니다.
2. **새 슬라이드**를 클릭합니다. 또는 상단 **홈** 탭 → **새 슬라이드** 버튼을 클릭해도 됩니다.
3. 슬라이드 레이아웃(구성 방식)을 바꾸려면 **홈** → **레이아웃**을 클릭하여 원하는 형태를 선택합니다. 예: 제목 슬라이드, 제목 및 내용, 두 가지 내용 등

**슬라이드 순서 바꾸기:**
왼쪽 패널에서 슬라이드를 클릭한 채로 위아래로 드래그하면 순서를 바꿀 수 있습니다.

## 이미지 삽입하기

시각적인 자료가 있으면 발표가 훨씬 생동감 있어집니다.

1. 상단 **삽입** 탭을 클릭합니다.
2. **그림**을 클릭합니다.
3. **이 디바이스...**를 클릭하면 내 PC에 저장된 이미지를 불러올 수 있습니다.
4. 원하는 이미지 파일을 선택하고 **삽입**을 클릭합니다.
5. 삽입된 이미지는 모서리나 가장자리의 조절점을 드래그하여 크기를 조절하고, 이미지 중앙을 드래그하여 위치를 옮길 수 있습니다.

**온라인 이미지 삽입하기:**
1. **삽입** → **그림** → **온라인 그림...**을 클릭합니다.
2. 검색창에 원하는 이미지 키워드를 입력합니다.
3. 마음에 드는 이미지를 클릭하고 **삽입**을 클릭합니다.

**이미지 스타일 꾸미기:**
이미지를 클릭하면 상단에 **그림 형식** 탭이 나타납니다. 여기서 이미지에 그림자, 테두리, 반사 효과 등을 적용할 수 있습니다.

## 도형 및 스마트아트 삽입하기

도형을 이용하면 정보를 시각적으로 구조화할 수 있습니다.

1. **삽입** → **도형**을 클릭합니다.
2. 사각형, 원, 화살표 등 원하는 도형을 클릭합니다.
3. 슬라이드에서 드래그하여 도형을 그립니다.
4. 도형을 더블클릭하면 도형 안에 텍스트를 입력할 수 있습니다.
5. 도형을 클릭하면 상단에 **도형 형식** 탭이 나타납니다. 여기서 색상, 테두리 스타일, 효과를 바꿀 수 있습니다.

**스마트아트(SmartArt) 사용하기:**
스마트아트는 조직도, 흐름도, 목록 등을 예쁘게 만드는 기능입니다.
1. **삽입** → **SmartArt**를 클릭합니다.
2. 원하는 형태(목록, 프로세스, 주기, 계층 구조 등)를 선택하고 **확인**을 클릭합니다.
3. 텍스트 입력 창에서 내용을 채워 넣으면 자동으로 예쁜 다이어그램이 만들어집니다.

## 디자인 테마 적용하기

슬라이드 전체의 색상과 폰트를 통일감 있게 바꾸고 싶다면 테마를 사용합니다.

1. 상단 **디자인** 탭을 클릭합니다.
2. 테마 갤러리에서 마음에 드는 디자인 위에 마우스를 올려 보면 미리보기가 표시됩니다.
3. 원하는 테마를 클릭하면 모든 슬라이드에 일괄 적용됩니다.
4. 오른쪽 **변형** 옵션에서 같은 테마의 다른 색상 조합을 선택할 수 있습니다.

**배경 색상 변경하기:**
1. **디자인** → **배경 서식**을 클릭합니다.
2. 오른쪽에 패널이 열립니다. 단색, 그라데이션, 그림으로 배경을 설정할 수 있습니다.
3. 특정 슬라이드만 바꾸려면 해당 슬라이드를 선택한 상태에서 **현재 슬라이드에 적용**을 클릭합니다.

## 애니메이션 효과 추가하기

슬라이드 내 텍스트나 이미지가 순서대로 나타나게 하고 싶다면 애니메이션을 사용합니다.

1. 애니메이션을 적용할 텍스트 상자나 이미지를 클릭합니다.
2. 상단 **애니메이션** 탭을 클릭합니다.
3. **나타나기** 효과(페이드, 날아오기 등) 중 원하는 것을 클릭합니다.
4. **애니메이션 창** 버튼을 클릭하면 오른쪽에 패널이 열려서 각 효과의 순서와 시간을 조절할 수 있습니다.
5. **미리 보기** 버튼을 클릭하면 적용된 애니메이션을 확인할 수 있습니다.

**슬라이드 전환 효과 추가하기:**
슬라이드와 슬라이드 사이의 전환 효과입니다.
1. **전환** 탭을 클릭합니다.
2. 원하는 전환 효과(페이드, 밀어내기 등)를 클릭합니다.
3. **모두 적용**을 클릭하면 전체 슬라이드에 같은 효과가 적용됩니다.

## 파일 저장하기

1. 상단 **파일** 탭을 클릭합니다.
2. **저장** 또는 **다른 이름으로 저장**을 클릭합니다.
3. 저장 위치를 선택하고 파일 이름을 입력합니다.
4. **파일 형식**은 기본값인 **PowerPoint 프레젠테이션(.pptx)**이 좋습니다.
5. 구버전 파워포인트(2003 이하) 사용자와 공유해야 한다면 **.ppt** 형식을 선택하세요.
6. PDF로 저장하려면 파일 형식에서 **PDF**를 선택합니다. PDF는 수정이 어렵지만 어떤 기기에서도 동일하게 보여 발표 자료를 배포할 때 유용합니다.

**자동 저장 기능 활용:**
OneDrive에 파일을 저장하면 상단 왼쪽 **자동 저장** 토글을 켤 수 있습니다. 내용이 변경될 때마다 자동으로 저장되어 작업 중 실수로 닫혀도 잃어버리지 않습니다.

## 슬라이드 쇼 진행하기

완성된 발표 자료를 슬라이드 쇼 모드로 실행하는 방법입니다.

1. **슬라이드 쇼** 탭을 클릭합니다.
2. **처음부터**를 클릭하면 첫 번째 슬라이드부터 전체 화면으로 발표가 시작됩니다.
3. 현재 선택된 슬라이드부터 시작하려면 **현재 슬라이드부터**를 클릭합니다.
4. 슬라이드 쇼 중에는 마우스 클릭 또는 키보드 방향키(→, ←)로 슬라이드를 넘깁니다.
5. 발표를 종료하려면 **ESC** 키를 누릅니다.

**발표자 보기 활용:**
모니터가 두 개(빔 프로젝터 포함) 연결되어 있을 때 유용합니다.
1. **슬라이드 쇼** → **발표자 보기 사용**을 체크합니다.
2. 발표자 화면에는 현재 슬라이드, 다음 슬라이드 미리보기, 발표자 메모가 표시됩니다.
3. 청중 화면에는 슬라이드만 전체 화면으로 보입니다.

## 자주 묻는 질문 (Q&A)

**Q. 파워포인트가 없어도 pptx 파일을 열 수 있나요?**
A. 네, 가능합니다. **Google 슬라이드**(구글 계정 필요), **리브레오피스 Impress**(무료 설치), 또는 **Microsoft 365 온라인**(웹브라우저 무료)으로 열고 편집할 수 있습니다.

**Q. 작업 중에 파워포인트가 꺼져서 파일이 사라졌어요.**
A. 파워포인트는 기본적으로 10분마다 자동 복구 파일을 저장합니다. 파워포인트를 다시 실행하면 왼쪽에 **복구된 파일** 목록이 나타납니다. 또는 파일 → 정보 → 문서 관리 → 저장하지 않은 프레젠테이션 복구에서 찾을 수 있습니다.

**Q. 슬라이드 크기를 A4 용지에 맞게 바꾸고 싶어요.**
A. **디자인** → **슬라이드 크기** → **사용자 지정 슬라이드 크기**를 클릭합니다. 크기 드롭다운에서 **A4 용지(210x297mm)**를 선택하고 확인을 클릭합니다.

**Q. 폰트가 다른 PC에서는 다르게 보여요.**
A. 파일 저장 시 폰트를 포함시키면 해결됩니다. **파일** → **옵션** → **저장** → **파일에 글꼴 포함**을 체크하세요. 단, 파일 크기가 커집니다.

**Q. 여러 슬라이드에 같은 로고나 텍스트를 넣고 싶어요.**
A. **보기** → **슬라이드 마스터**를 사용합니다. 마스터 슬라이드에 로고를 삽입하면 모든 슬라이드에 자동으로 표시됩니다. 편집 완료 후 **마스터 보기 닫기**를 클릭합니다.

## 정리

파워포인트 기초는 ①새 프레젠테이션 만들기 → ②텍스트와 이미지 삽입 → ③디자인 테마 적용 → ④애니메이션 추가 → ⑤저장 및 발표 순서로 익히면 됩니다. 처음에는 기본 기능만 써도 충분히 좋은 자료를 만들 수 있습니다. 스마트아트와 발표자 보기 같은 고급 기능은 기본이 익숙해지면 하나씩 도전해 보세요. 이 가이드를 참고해서 첫 번째 프레젠테이션을 만들어 보시기 바랍니다!
