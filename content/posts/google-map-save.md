## 구글 지도 즐겨찾기 기능이란?

구글 지도는 단순한 길찾기 앱을 넘어, 장소를 저장하고 관리하는 기능이 매우 뛰어나요. 맛집, 카페, 가고 싶은 여행지 등을 미리 저장해두면 나중에 빠르게 찾을 수 있고, 친구에게 공유하거나 해외 여행 시 오프라인으로도 사용할 수 있어요.

구글 계정에 연동되어 있어서 스마트폰을 바꿔도 저장한 장소는 그대로 남아 있어요.

## 장소 저장 방법

1. 구글 지도에서 저장할 장소 검색
2. 장소 카드 하단 **저장** 클릭
3. 리스트 선택:
   - **즐겨찾기**: 자주 가는 곳
   - **가고 싶은 곳**: 나중에 방문할 장소
   - **여행지**: 여행 계획 중인 곳
   - **+ 새 목록 만들기**: 원하는 이름으로 직접 생성
4. **완료**

**팁**: 리스트를 여러 개 만들어 분류하면 편해요. 예: '서울 맛집', '제주 여행', '가고 싶은 카페' 등으로 구분해보세요.

## 저장한 장소 확인

1. 하단 **저장됨** 탭 클릭
2. 리스트별로 저장된 장소 확인 가능
3. 저장된 장소를 클릭하면 바로 길찾기로 연결

**팁**: 저장된 장소는 지도에서도 표시돼요. 해당 지역으로 지도를 이동하면 저장한 장소에 작은 아이콘이 나타나요.

## 장소 친구에게 공유하기

약속 장소를 정확하게 알려주고 싶을 때 유용해요.

1. 공유할 장소 검색
2. 장소 카드에서 **공유** 클릭
3. 카카오톡, 문자, 링크 복사 등으로 공유

**팁**: 링크를 받은 사람은 구글 지도 앱이 없어도 웹브라우저에서 바로 위치를 확인할 수 있어요.

## 오프라인 지도 다운로드

해외 여행 시 데이터 로밍 없이 지도를 쓰고 싶을 때 정말 유용한 기능이에요. 미리 해당 지역 지도를 다운받아두면 인터넷 없이도 길찾기가 가능해요.

1. 지도에서 원하는 지역 검색
2. 프로필 아이콘 → **오프라인 지도**
3. **나만의 지도 선택** → 다운로드할 지역 범위 조정
4. **다운로드**

**주의사항**:
- 오프라인 지도는 용량이 크게는 수백 MB까지 돼요. 와이파이 연결 상태에서 다운받는 걸 권장해요
- 오프라인 지도는 주기적으로 업데이트가 필요해요. 오래된 지도는 정보가 다를 수 있어요
- 대중교통 실시간 정보는 인터넷이 있어야 사용 가능해요

## 실시간 내 위치 공유하기

친구를 만날 때 "나 지금 어디야?" 대신 실시간 위치를 공유할 수 있어요.

1. 프로필 아이콘 → **위치 공유**
2. 공유할 사람 선택 (연락처 또는 링크)
3. 공유 시간 설정 (1시간, 종료할 때까지 등)
4. **공유 시작**

상대방도 구글 계정이 있으면 상대방 위치도 서로 볼 수 있어요.

## 유용한 구글 지도 기능

**내 집/회사 등록**
- 프로필 → **집/직장 설정**
- 등록 후 "집으로" 또는 "회사로" 검색하면 바로 길찾기 시작

**주변 검색**
- 하단 **탐색** 탭에서 현재 위치 주변 음식점, 카페, ATM 등 검색 가능

**거리뷰**
- 처음 가는 장소의 외관을 미리 확인할 수 있어요
- 장소 카드에서 **거리뷰** 사진 클릭

## 자주 묻는 질문

**Q. 네이버 지도와 구글 지도 중 어느 게 더 좋아요?**
국내에서는 네이버 지도가 대중교통 정보, 실시간 교통량이 더 정확한 편이에요. 해외 여행 시에는 구글 지도가 훨씬 강력해요.

**Q. 저장한 장소를 PC에서도 볼 수 있나요?**
네. 같은 구글 계정으로 maps.google.com에 접속하면 PC에서도 저장한 장소를 확인할 수 있어요.

**Q. 오프라인 지도에서도 길찾기가 되나요?**
자동차 경로는 오프라인에서도 가능해요. 단, 대중교통 정보와 실시간 교통량은 인터넷이 필요해요.
