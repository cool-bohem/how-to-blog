## 구글 지도 장소 저장하기

자주 가는 곳이나 가고 싶은 곳을 저장해두면 나중에 빠르게 찾을 수 있어요.

## 장소 저장 방법

1. 구글 지도에서 저장할 장소 검색
2. 장소 카드 하단 **저장** 클릭
3. 리스트 선택 (즐겨찾기, 가고 싶은 곳 등)
4. **완료**

## 저장한 장소 확인

1. 하단 **저장됨** 탭 클릭
2. 리스트별로 저장된 장소 확인 가능

## 장소 친구에게 공유하기

1. 공유할 장소 검색
2. 장소 카드에서 **공유** 클릭
3. 카카오톡, 문자 등으로 공유

## 오프라인 지도 다운로드

인터넷 없이도 지도를 사용하고 싶을 때:

1. 지도에서 원하는 지역 검색
2. 프로필 아이콘 → **오프라인 지도**
3. **나만의 지도 선택** → 지역 선택
4. **다운로드**

데이터 없는 환경에서도 길찾기 가능해요.

## 내 위치 공유하기

실시간으로 내 위치를 친구에게 보여줄 수 있어요.

1. 프로필 아이콘 → **위치 공유**
2. 공유할 사람 선택
3. 공유 시간 설정 후 시작
