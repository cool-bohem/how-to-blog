## 공장 초기화란 무엇인가요?

공장 초기화(Factory Reset)는 스마트폰을 처음 구입했을 때의 상태로 되돌리는 작업이에요. 저장된 모든 데이터(사진, 연락처, 앱, 설정 등)가 삭제되고, 기기가 깨끗하게 초기화돼요.

**공장 초기화가 필요한 상황**
- 스마트폰 동작이 너무 느리고 자주 멈출 때
- 중고로 판매하거나 양도할 때
- 바이러스나 악성 앱 감염이 의심될 때
- 소프트웨어 오류가 반복되어 해결이 안 될 때
- 개인정보 보호를 위해 완전히 지우고 싶을 때

초기화 전에는 **반드시 중요한 데이터를 백업**해야 해요. 초기화 후에는 복구가 불가능하기 때문에 이 단계를 절대 건너뛰지 마세요.

## 초기화 전 백업 방법

### 방법 1. 삼성 클라우드 백업 (가장 간편)

삼성 계정이 있다면 클라우드에 자동으로 백업할 수 있어요.

1. **설정** 앱 열기
2. **Samsung 계정** 탭하기
3. **내 데이터 백업** 선택
4. 백업할 항목 선택 (연락처, 통화 기록, 메시지, 앱 데이터, 설정 등)
5. **지금 백업** 버튼 탭하기
6. 백업 완료 확인 후 진행

**팁**: 삼성 클라우드는 무료로 15GB까지 제공해요. 사진이 많다면 추가 용량이 필요할 수 있어요.

### 방법 2. 구글 계정 백업

1. **설정 → 계정 및 백업 → 계정 → Google** 선택
2. **계정 동기화** 탭하기
3. 연락처, 캘린더, Gmail 등 동기화 항목 모두 켜기
4. 동기화 완료 확인

### 방법 3. 사진·동영상 백업 (Google 포토)

1. **Google 포토** 앱 열기
2. 프로필 사진 → **사진 백업** 탭
3. **백업 켜기** 선택
4. 백업 완료될 때까지 Wi-Fi 연결 상태 유지

### 방법 4. 연락처 파일로 내보내기

1. **연락처** 앱 열기
2. 우측 상단 **더보기(⋮)** 탭
3. **가져오기/내보내기 → 내보내기** 선택
4. **.vcf 파일로 저장** 후 USB 또는 이메일로 전송

### 중요: 초기화 전 확인 목록

- [ ] 사진·동영상 백업 완료
- [ ] 연락처 백업 완료
- [ ] 카카오톡 채팅 백업 완료 (카카오톡 설정 → 채팅 → 대화 백업)
- [ ] 중요한 메모/문서 저장
- [ ] 인증서·공인인증서 등 금융 앱 재인증 준비
- [ ] 기기에 설치된 앱 목록 메모 (구글 플레이 → 앱 및 게임 → 관리에서 확인 가능)

## 갤럭시 공장 초기화 방법 (설정 메뉴 이용)

### 1단계: 설정 앱 열기

홈 화면 또는 앱 서랍에서 **설정(톱니바퀴 아이콘)** 앱을 열어요.

### 2단계: 일반 관리 메뉴 찾기

설정 화면을 아래로 스크롤해서 **일반 관리**를 탭해요.

### 3단계: 초기화 메뉴 진입

**일반 관리 → 초기화** 탭해요.

### 4단계: 공장 초기화 선택

**공장 초기화(초기화)** 항목을 탭해요. 화면에 삭제될 항목 목록이 표시돼요. 꼼꼼히 확인하고 이상이 없으면 아래로 스크롤해요.

### 5단계: 초기화 실행

**초기화** 버튼을 탭해요. PIN, 패턴, 비밀번호 등 화면 잠금 인증을 요구해요. 설정한 잠금 방식으로 인증하세요.

### 6단계: 최종 확인 및 삭제

**모두 삭제** 버튼을 탭하면 초기화가 시작돼요. 이 단계 이후에는 되돌릴 수 없어요.

초기화는 보통 3~10분 정도 걸리며, 완료되면 스마트폰이 자동으로 재시작돼요. 재시작 후 초기 설정 화면이 나타나요.

## 복구 모드에서 초기화하는 방법 (화면이 안 켜질 때)

스마트폰이 부팅되지 않거나 화면이 응답하지 않을 때 사용하는 방법이에요.

1. 스마트폰 전원을 완전히 끄기
2. **볼륨 높임 버튼 + 전원 버튼** 동시에 길게 누르기 (약 10초)
3. 삼성 로고가 뜨면 두 버튼 모두 손가락 떼기
4. Android Recovery 화면(녹색 로봇 화면)이 나타남
5. **볼륨 버튼**으로 이동, **전원 버튼**으로 선택
6. **Wipe data/factory reset** 선택
7. **Factory data reset** 또는 **Yes** 선택
8. 초기화 완료 후 **Reboot system now** 선택

**주의**: 일부 갤럭시 모델은 버튼 조합이 다를 수 있어요. 예를 들어 갤럭시 S24 시리즈는 **볼륨 높임 + 측면 버튼**을 사용해요.

## 초기화 후 새 기기 설정하는 방법

초기화 후 처음 켜면 기기 설정 화면이 나와요.

1. **언어 선택**: 한국어 선택 후 다음
2. **Wi-Fi 연결**: 집 Wi-Fi에 연결 (빠른 복원을 위해 필수)
3. **데이터 복원 선택**: 삼성 클라우드 또는 Google 백업에서 복원 가능
4. **Samsung 계정 로그인**: 기존 계정으로 로그인하면 백업 데이터 불러오기 가능
5. **Google 계정 로그인**: Gmail, 연락처, 앱 등 동기화
6. **잠금 화면 설정**: PIN, 패턴, 생체인식(지문/얼굴) 새로 등록
7. 설정 완료 후 앱 재설치 및 개인 설정 복원

## 자주 묻는 질문

**Q. 초기화하면 SD카드 데이터도 지워지나요?**
기본적으로는 지워지지 않아요. 하지만 초기화 옵션에서 'SD카드 포맷' 항목이 있으면 확인하고 해제하세요. 확실하게 하려면 초기화 전에 SD카드를 꺼내 두는 게 안전해요.

**Q. 초기화 후 삼성페이, 카카오페이 등 간편결제 앱은 어떻게 되나요?**
모두 삭제돼요. 재설치 후 본인 인증을 다시 거쳐 등록해야 해요. 카드 정보도 다시 입력해야 해요.

**Q. 초기화하면 통신사 약정이나 번호가 바뀌나요?**
아니요. 공장 초기화는 소프트웨어만 초기화해요. 통신사 계약, 전화번호, USIM 정보에는 전혀 영향이 없어요.

**Q. 초기화 중 배터리가 없으면 어떻게 되나요?**
초기화 도중 전원이 꺼지면 기기가 망가질 수 있어요. 반드시 **배터리 60% 이상** 또는 충전기 연결 상태에서 진행하세요.

**Q. 카카오톡 채팅 내역도 복원할 수 있나요?**
초기화 전에 카카오톡에서 백업을 해두면 돼요. 카카오톡 앱 → 더보기 → 설정 → 채팅 → 대화 백업으로 카카오 계정에 저장할 수 있어요. 복원 시 카카오톡 재설치 후 로그인하면 백업 복원 안내가 나와요.

## 정리

공장 초기화는 스마트폰 문제를 해결하거나 기기를 양도하기 전에 꼭 필요한 작업이에요. 핵심은 **초기화 전 백업**이에요. 사진, 연락처, 카카오톡 대화 등 중요한 데이터를 모두 백업한 뒤에 초기화를 진행하면 소중한 데이터를 잃지 않을 수 있어요. 천천히 순서대로 따라 하면 누구나 쉽게 할 수 있으니 걱정하지 마세요.
