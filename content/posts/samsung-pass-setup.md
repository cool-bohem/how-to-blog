## 삼성 패스란?

지문이나 얼굴 인식으로 각종 앱 로그인과 비밀번호를 자동으로 입력해주는 갤럭시 기본 기능이에요.

## 삼성 패스 설정하기

1. **설정 → 보안 및 개인 정보 보호 → 삼성 패스**
2. 삼성 계정 로그인
3. 지문 또는 얼굴 등록

## 앱 로그인에 삼성 패스 연결

1. 원하는 앱 실행 후 로그인 화면으로 이동
2. 아이디/비밀번호 입력 후 로그인
3. **삼성 패스로 저장하시겠습니까?** 팝업 → **저장**
4. 다음부터는 지문/얼굴로 자동 로그인

## 웹사이트 비밀번호 저장

1. 삼성 인터넷 브라우저에서 로그인
2. **비밀번호 저장** 팝업 → **저장**
3. 이후 삼성 패스로 자동 입력

## 저장된 비밀번호 관리

1. 삼성 패스 앱 실행
2. **로그인 정보** 탭에서 저장된 계정 목록 확인
3. 불필요한 계정 삭제 가능

## 주의사항

- 삼성 패스 데이터는 삼성 계정에 암호화되어 저장돼요
- 기기 분실 시 삼성 계정에서 원격으로 삭제 가능
