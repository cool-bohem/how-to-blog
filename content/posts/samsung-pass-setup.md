## 삼성 패스란?

삼성 패스(Samsung Pass)는 갤럭시 스마트폰에 기본 탑재된 비밀번호 관리 및 생체인증 서비스예요. 수십 개의 앱과 웹사이트 비밀번호를 기억하지 않아도, 지문이나 얼굴 인식 한 번으로 자동 로그인할 수 있어요.

**삼성 패스의 주요 기능**:
- 앱 로그인 자동 입력 (지문/얼굴 인식)
- 웹사이트 아이디/비밀번호 저장 및 자동 완성
- 신용카드 정보 저장 (온라인 결제 자동 완성)
- 개인 정보 저장 (배송지 등)
- 삼성 계정에 암호화 저장 (기기 분실 시에도 안전)

## 삼성 패스 설정하기

1. **설정 → 보안 및 개인 정보 보호 → 삼성 패스** 이동
2. **삼성 계정 로그인** (이미 로그인되어 있으면 생략)
3. **생체 인식 등록**:
   - **지문 인식**: 손가락을 여러 각도로 스캔 (2~3회)
   - **얼굴 인식**: 정면을 바라보며 스캔

**팁**: 지문과 얼굴을 모두 등록해두면 어두운 환경에서도 지문으로, 손이 젖어 있을 때는 얼굴 인식으로 대체 사용 가능해요.

## 앱 로그인에 삼성 패스 연결하기

처음에는 수동으로 로그인하면 삼성 패스가 저장할지 물어봐요.

1. 원하는 앱 실행 후 로그인 화면으로 이동
2. 아이디/비밀번호 직접 입력 후 로그인
3. 화면 하단에 **삼성 패스로 저장하시겠습니까?** 팝업 확인
4. **저장** 클릭
5. 다음 로그인부터는 지문/얼굴 인식으로 자동 입력

**직접 추가하는 방법**:
1. 삼성 패스 앱 실행
2. **로그인 정보** 탭 → **+ 추가** 클릭
3. 앱 선택 → 아이디/비밀번호 입력 후 저장

## 웹사이트 비밀번호 저장

삼성 인터넷 브라우저를 사용할 때 웹사이트 비밀번호도 저장할 수 있어요.

1. 삼성 인터넷으로 웹사이트 접속 후 로그인
2. **비밀번호 저장** 팝업 → **저장** 클릭
3. 이후 같은 사이트 방문 시 자동으로 입력

**팁**: 크롬 브라우저를 사용한다면 크롬의 비밀번호 관리자를 활용하거나, 삼성 인터넷으로 전환하는 것을 고려해보세요.

## 신용카드 정보 저장

온라인 쇼핑 시 카드 번호 입력을 간편하게 할 수 있어요.

1. 삼성 패스 앱 → **결제 정보** 탭
2. **+ 추가** 클릭
3. 카드 번호, 유효기간, CVC 입력 또는 카드 스캔
4. 저장 완료

**주의**: 카드 정보는 삼성 계정에 암호화 저장되지만, 타인에게 기기 비밀번호를 알려주면 접근이 가능해요.

## 저장된 비밀번호 관리

1. 삼성 패스 앱 실행
2. **로그인 정보** 탭에서 저장된 계정 목록 확인
3. 항목 선택 → 비밀번호 확인, 수정, 삭제 가능
4. **생체 인증** 후에만 비밀번호를 볼 수 있어요

**팁**: 주기적으로 오래된 계정이나 더 이상 사용하지 않는 서비스는 삭제해두는 게 좋아요.

## 기기 분실 시 대처

삼성 패스 데이터는 삼성 계정 서버에 암호화 저장되기 때문에 기기 분실 시에도 데이터가 보호돼요.

1. 다른 기기에서 **galaxy.samsung.com/findmymobile** 접속
2. 분실 기기에서 삼성 계정 로그아웃 처리
3. 새 기기에서 같은 삼성 계정으로 로그인하면 데이터 복원

## 자주 묻는 질문

**Q. 삼성 패스에 저장된 비밀번호를 누군가가 볼 수 있나요?**
저장된 비밀번호를 보려면 생체 인증 또는 삼성 패스 비밀번호가 필요해요. 잠금화면만 해제해서는 볼 수 없어요.

**Q. 아이폰에서 삼성 패스를 사용할 수 있나요?**
아니요. 삼성 패스는 갤럭시 전용 기능이에요. 아이폰에서는 아이클라우드 키체인이 유사한 기능을 해요.

**Q. 앱이 업데이트되면 저장된 로그인 정보가 사라지나요?**
아니요. 앱이 업데이트되어도 삼성 패스 데이터는 유지돼요. 단, 앱 개발사가 로그인 방식을 변경하면 재저장이 필요할 수 있어요.
