## Face ID란 무엇인가요?

Face ID는 아이폰의 얼굴 인식 잠금 해제 기능으로, 아이폰 X 이후 모든 노치(또는 다이나믹 아일랜드) 탑재 모델에 기본 내장되어 있어요. 비밀번호나 지문 인식 없이 얼굴만 바라보면 잠금이 자동으로 풀려 매우 편리합니다.

Face ID는 단순히 잠금 해제뿐 아니라, 앱스토어 결제, 카카오페이·토스 등 금융 앱 인증, 애플페이, 그리고 iCloud 키체인 자동 로그인 등 다양한 곳에서 활용돼요. 제대로 설정해두면 일상 속 번거로움이 확 줄어들 거예요.

## Face ID 최초 설정 방법

### 준비 사항

Face ID 설정 전에 아이폰이 충전이 일정 수준 이상 되어 있는지 확인하고, 밝은 환경에서 진행하는 것이 좋아요. 설정 중에는 화면에서 눈을 떼지 말고 따라오세요.

### 1단계: 설정 앱 열기

아이폰 홈 화면에서 회색 톱니바퀴 모양의 **설정** 앱을 탭합니다. 앱을 찾기 어려우시면 홈 화면을 아래로 쓸어내려 검색창에 "설정"을 입력하면 바로 나와요.

### 2단계: Face ID 및 암호 메뉴 진입

설정 앱이 열리면 화면을 조금 아래로 스크롤하면 **"Face ID 및 암호"** 항목이 보여요. 이 항목을 탭합니다. 이미 암호가 설정된 경우 기존 암호를 입력하라는 화면이 나타납니다. 숫자 6자리 또는 영숫자 암호를 입력하세요.

### 3단계: Face ID 설정 시작

"Face ID 설정" 버튼을 탭합니다. "시작하기" 안내 화면이 나오면 **시작하기**를 탭해주세요.

### 4단계: 첫 번째 얼굴 스캔

화면 중앙에 원형 프레임이 나타나고, 그 안에 얼굴이 들어오도록 화면을 얼굴 앞에 세워 주세요. 안내에 따라 **고개를 천천히 원을 그리듯 회전**시킵니다. 마치 '머리로 원을 그린다'는 느낌으로 천천히 360도 돌려주시면 돼요. 이 과정을 통해 아이폰이 다양한 각도에서 얼굴 데이터를 수집합니다.

> **팁**: 안경을 평소에 착용한다면 착용한 채로 스캔하는 것이 좋아요. 나중에 '대체 모습 추가'로 안경 없는 얼굴도 등록할 수 있습니다.

### 5단계: 두 번째 얼굴 스캔

"첫 번째 스캔 완료" 메시지 후 **계속하기**를 탭하면 동일한 방식으로 두 번째 스캔이 시작됩니다. 조금 다른 각도나 조명에서 다시 고개를 천천히 돌려주세요. 두 번 스캔하는 이유는 다양한 환경(조명 변화, 약간 다른 각도)에서도 정확하게 인식하기 위해서예요.

### 6단계: 완료 및 암호 설정

두 번째 스캔까지 완료되면 "Face ID 설정 완료" 메시지가 나타납니다. 아직 암호가 없다면 **암호 설정** 단계로 넘어가요. Face ID 인식이 실패할 때 대신 사용하는 비상용 암호이니 꼭 기억하기 쉬우면서도 남이 유추하기 어려운 6자리 번호로 설정하세요.

## Face ID 사용 범위 설정

설정이 완료된 뒤 "Face ID 및 암호" 화면으로 돌아오면, Face ID를 어디에 사용할지 개별로 켜고 끌 수 있어요.

| 항목 | 설명 |
|------|------|
| **아이폰 잠금 해제** | 화면을 켤 때 자동 잠금 해제 |
| **iTunes 및 App Store** | 앱 다운로드·결제 시 인증 |
| **지갑 및 Apple Pay** | 애플페이 결제 인증 |
| **암호 자동 완성** | 저장된 계정·비밀번호 자동 입력 |
| **기타 앱** | Face ID를 지원하는 서드파티 앱 |

보안이 걱정된다면 금융 관련 항목만 켜두고 나머지는 꺼두는 방법도 있어요.

## 대체 모습(얼굴) 추가하기

가족이나 연인과 공유하거나, 안경 착용 여부에 따른 인식률 차이를 줄이고 싶을 때 **대체 모습**을 추가할 수 있어요.

1. **설정 → Face ID 및 암호** 진입
2. **"대체 모습 설정"** 탭
3. 위와 동일하게 두 차례 스캔 진행

대체 모습은 1개만 등록 가능합니다. 두 번째 사람의 얼굴을 등록하면 그 사람도 내 아이폰을 잠금 해제할 수 있으니 신중하게 사용하세요.

## Face ID 인식이 안 될 때 해결법

### 문제 1. 마스크 착용 시 인식 실패

아이폰 iOS 15.4 이후부터는 마스크 착용 상태에서도 Face ID를 사용할 수 있어요.

1. **설정 → Face ID 및 암호**
2. **"마스크 착용 시 Face ID"** 토글을 켜기
3. 마스크를 착용한 상태로 얼굴을 다시 스캔

단, 마스크 착용 Face ID는 눈 주위 특징만 분석하므로 일반 모드보다 보안이 다소 낮아질 수 있어요.

### 문제 2. 어두운 환경에서 인식 불량

Face ID는 적외선 카메라를 사용하므로 완전히 어두운 곳에서도 대체로 잘 작동해요. 하지만 직사광선이 카메라를 정면으로 비추거나 화면이 너무 가까울 때 오류가 날 수 있습니다.

- 화면에서 **25~50cm 거리**를 유지하세요
- 직사광선을 피하거나 몸으로 그늘을 만들어 시도해보세요

### 문제 3. Face ID를 5회 이상 실패했을 때

연속으로 5번 인식 실패하면 암호 입력 화면으로 전환돼요. 암호를 입력하면 Face ID가 다시 활성화됩니다.

### 문제 4. Face ID 재설정 필요

외모가 많이 변했거나(수술, 큰 체중 변화 등) 인식률이 현저히 낮아졌다면 Face ID를 초기화하고 다시 등록하는 것이 좋아요.

1. **설정 → Face ID 및 암호**
2. **"Face ID 재설정"** 탭
3. 위에서 설명한 최초 설정 과정을 처음부터 다시 진행

## Face ID 관련 보안 팁

- **잠들었을 때 무단 잠금 해제 방지**: Face ID는 눈이 떠져 있고 화면을 응시하는 상태에서만 작동해요. **설정 → Face ID 및 암호 → 주시 확인 필요**가 기본으로 켜져 있는지 확인하세요.
- **강제로 Face ID 비활성화**: 긴급 상황에서 아이폰을 잠그려면 전원 버튼과 볼륨 버튼을 동시에 5초간 누르면 SOS 화면이 나타나고 Face ID가 일시 비활성화돼요. 이후에는 암호를 입력해야만 잠금이 풀려요.
- **분실·도난 시**: iCloud의 **나의 찾기**를 통해 즉시 분실 모드를 활성화하면 Face ID와 암호 모두 잠겨요.

## 자주 묻는 질문 (Q&A)

**Q. Face ID가 지원되는 아이폰 모델은 어디서부터인가요?**
A. 아이폰 X(2017년)부터 지원됩니다. 아이폰 SE 시리즈(1·2·3세대)는 Face ID 대신 Touch ID(지문 인식)가 탑재돼 있어요.

**Q. 쌍둥이 형제가 내 아이폰을 열 수 있나요?**
A. 일란성 쌍둥이처럼 외모가 매우 흡사한 경우 인식될 수 있다는 보고가 있어요. 이 경우에는 암호를 추가 보안 수단으로 반드시 설정해두세요.

**Q. Face ID 데이터는 어디에 저장되나요?**
A. 얼굴 데이터는 아이폰 내부의 **보안 격리 영역(Secure Enclave)**에 암호화되어 저장되며, 애플 서버나 iCloud에는 업로드되지 않아요.

**Q. 앱에서 Face ID 인증 요청이 뜨지 않아요.**
A. **설정 → Face ID 및 암호** 하단 '기타 앱' 목록에서 해당 앱의 Face ID 허용 토글이 켜져 있는지 확인하세요.

**Q. 자다가 누군가 내 얼굴에 폰을 들이대면 열릴까요?**
A. 기본 설정인 **주시 확인 필요** 옵션이 켜져 있으면, 눈을 감고 있는 상태에서는 Face ID가 작동하지 않아요.

## 마무리 정리

Face ID는 한 번만 제대로 설정해두면 하루에도 수십 번 쓰는 가장 편리한 기능 중 하나예요. 설정 순서를 다시 정리하면 다음과 같아요.

1. 설정 → Face ID 및 암호 진입
2. 암호 입력 후 "Face ID 설정" 시작
3. 고개를 천천히 원을 그리며 두 차례 스캔
4. 원하는 사용 범위(잠금 해제, App Store, 애플페이 등) 켜기
5. 필요하다면 대체 모습 추가, 마스크 착용 모드 활성화

처음 설정이 잘 안 된다면 밝은 조명 아래서 화면과 25~50cm 거리를 유지하며 천천히 시도해보세요. Face ID를 잘 활용하면 아이폰 보안과 편의를 동시에 잡을 수 있답니다.
