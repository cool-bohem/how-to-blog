## 인스타그램 릴스 저장 방법

인스타그램은 기본적으로 다른 사람 릴스를 기기에 저장하는 기능이 제한돼요. 하지만 몇 가지 방법이 있어요.

## 방법 1. 내 릴스 저장 (자신의 영상)

1. 내가 올린 릴스 열기
2. 오른쪽 하단 **점 세 개(···)** 클릭
3. **동영상 저장** 클릭
4. 갤러리에 저장됨

## 방법 2. 컬렉션에 저장 (북마크)

삭제되지 않는 한 나중에 다시 볼 수 있어요.

1. 릴스 오른쪽 하단 **책갈피 아이콘(🔖)** 클릭
2. 저장할 컬렉션 선택 또는 새 컬렉션 만들기
3. **프로필 → 저장됨** 에서 나중에 확인 가능

## 방법 3. 링크 복사 후 다운로드 사이트 활용

1. 릴스 오른쪽 하단 **점 세 개(···)** 클릭
2. **링크 복사** 선택
3. 인스타그램 영상 다운로드 사이트에 링크 붙여넣기

## 방법 4. 화면 녹화

직접 저장이 안 될 때 화면 녹화로 저장할 수 있어요.

- **갤럭시**: 빠른 설정 패널에서 **화면 녹화** 버튼
- **아이폰**: 제어 센터에서 **화면 녹화** 버튼

## 주의사항

- 다른 사람의 콘텐츠를 무단으로 재배포하는 것은 저작권 위반이에요
- 개인 감상 목적으로만 저장하세요
