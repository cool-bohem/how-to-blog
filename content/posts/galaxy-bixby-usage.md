## 빅스비(Bixby)란?

빅스비(Bixby)는 삼성전자가 개발한 갤럭시 스마트폰 전용 AI 음성 비서입니다. 애플의 시리(Siri), 구글의 어시스턴트와 비슷한 역할을 하지만, 갤럭시 기기와 삼성 앱에 훨씬 더 깊게 통합되어 있다는 것이 특징입니다. 빅스비를 잘 활용하면 스마트폰을 손으로 직접 조작하지 않아도 전화 걸기, 메시지 보내기, 알람 설정, 앱 실행 등 거의 모든 작업을 음성 명령만으로 처리할 수 있습니다.

2026년 현재 갤럭시 S25, S24, A55 등 대부분의 갤럭시 기기에 빅스비가 기본 탑재되어 있으며, 삼성의 AI 기능인 **갤럭시 AI**와 결합하여 더욱 스마트해졌습니다. 이 글에서는 빅스비를 처음 쓰는 분부터 더 잘 활용하고 싶은 분까지 모두를 위한 단계별 사용법을 알려드립니다.

## 빅스비 초기 설정하기

### 1단계: 빅스비 앱 실행

갤럭시 스마트폰의 **측면 버튼(전원 버튼)을 길게 누르면** 빅스비가 실행됩니다. 처음 실행하면 약관 동의 화면이 나타납니다.

1. 전원 버튼(또는 측면 버튼)을 길게 누릅니다.
2. 빅스비 환영 화면이 나타나면 **"동의 및 계속"** 버튼을 누릅니다.
3. 삼성 계정으로 로그인하라는 안내가 나오면 계정 정보를 입력합니다. 삼성 계정이 없다면 먼저 만들어야 합니다.
4. 로그인이 완료되면 빅스비 기본 설정이 자동으로 적용됩니다.

> **참고:** 기기 모델에 따라 빅스비 버튼이 별도로 있는 경우도 있습니다 (갤럭시 S10 이전 모델). 측면에 있는 빅스비 전용 버튼을 한 번 누르면 됩니다.

### 2단계: 목소리 인식 설정 (Voice Wake-up)

"하이 빅스비"라고 말하는 것만으로 빅스비를 호출하려면 목소리 인식을 활성화해야 합니다.

1. 빅스비 화면에서 우측 상단 **⋮(더보기)** 아이콘을 누릅니다.
2. **설정 → 목소리 인식** 항목으로 이동합니다.
3. **"하이 빅스비"로 깨우기** 토글을 켭니다.
4. 화면 안내에 따라 **"하이 빅스비"를 4번** 말해 내 목소리를 등록합니다.
5. 등록이 완료되면 잠금 화면에서도 "하이 빅스비"만 말하면 음성으로 빅스비를 바로 사용할 수 있습니다.

## 빅스비 기본 음성 명령 활용하기

빅스비를 호출한 뒤 다음과 같은 다양한 명령을 말할 수 있습니다.

### 전화 및 메시지

- **"홍길동에게 전화해줘"** — 연락처에 저장된 사람에게 바로 전화
- **"엄마한테 카카오톡으로 '조금 늦을 것 같아' 보내줘"** — 특정 앱으로 메시지 전송
- **"부재중 전화 확인해줘"** — 부재중 통화 목록 바로 표시

### 앱 실행 및 기기 제어

- **"유튜브 열어줘"** — 앱 바로 실행
- **"밝기 낮춰줘"** — 화면 밝기 조절
- **"블루투스 켜줘"** / **"와이파이 꺼줘"** — 연결 설정 제어
- **"내일 오전 7시에 알람 맞춰줘"** — 알람 설정
- **"삼성 인터넷으로 네이버 열어줘"** — 특정 앱으로 웹사이트 열기

### 정보 검색

- **"오늘 서울 날씨 어때?"** — 현재 날씨 확인
- **"1달러 원화로 얼마야?"** — 환율 검색
- **"가장 가까운 편의점 어디야?"** — 주변 장소 검색

## 빅스비 루틴(Routines) 설정하기

빅스비 루틴은 특정 조건이 충족될 때 자동으로 여러 동작을 실행하도록 설정하는 기능입니다. 예를 들어 "출근할 때마다 와이파이 끄고, 알람 취소하고, 출근 재생목록 자동 시작"과 같은 복잡한 자동화를 한 번에 설정할 수 있습니다.

### 루틴 설정 방법

1. **설정 앱**을 열고 **"유용한 기능"** 또는 **"빅스비 루틴"**을 탭합니다.
2. **"+" 버튼**을 눌러 새 루틴을 추가합니다.
3. **조건(IF)** 설정: 루틴이 시작될 조건을 선택합니다.
   - 시간 (예: 매일 오전 7시)
   - 위치 (예: 집에서 나갈 때)
   - 앱 실행 (예: 유튜브 실행 시)
   - 배터리 잔량 (예: 20% 이하일 때)
4. **동작(THEN)** 설정: 조건이 충족되면 실행할 동작을 추가합니다.
   - 블루투스/와이파이 켜기·끄기
   - 화면 밝기·볼륨 변경
   - 특정 앱 실행
   - 알림 모드 변경 (방해 금지 등)
5. 루틴 이름을 입력하고 **저장**합니다.

### 추천 루틴 예시

| 루틴 이름 | 조건 | 동작 |
|---|---|---|
| 수면 모드 | 매일 밤 11시 | 밝기 최소, 방해금지 ON, 블루투스 OFF |
| 출근 모드 | 집 위치 이탈 시 | 와이파이 OFF, 데이터 ON, 음악 앱 실행 |
| 충전 모드 | 충전기 연결 시 | 밝기 올리기, 알림음 켜기 |

## 빅스비 비전(Bixby Vision) 사용하기

빅스비 비전은 카메라를 활용한 AI 인식 기능입니다. 텍스트 번역, 사물 인식, QR코드 스캔 등을 카메라로 바로 처리할 수 있습니다.

### 빅스비 비전 실행 방법

1. **카메라 앱**을 실행합니다.
2. 상단에 있는 **빅스비 비전 아이콘(눈 모양)** 을 탭합니다.
3. 하단 탭에서 원하는 기능을 선택합니다.

### 주요 기능

- **번역**: 외국어 간판, 메뉴판, 문서에 카메라를 갖다 대면 실시간으로 한국어로 번역됩니다. 해외여행 시 매우 유용합니다.
- **텍스트 인식**: 종이 문서나 책의 텍스트를 인식해 스마트폰으로 복사할 수 있습니다. 복잡한 주소나 긴 URL을 직접 타이핑할 필요가 없어집니다.
- **쇼핑**: 상품 바코드나 제품 자체를 비추면 관련 쇼핑 정보와 가격 비교 결과를 보여줍니다.
- **음식 칼로리**: 음식에 카메라를 겨누면 해당 음식의 칼로리와 영양 정보를 대략적으로 알려줍니다.
- **QR 코드**: 별도 앱 없이 QR 코드를 바로 스캔할 수 있습니다.

## 빅스비 텍스트 콜(Bixby Text Call) 사용하기

빅스비 텍스트 콜은 목소리를 사용하기 어려운 상황(회의 중, 도서관 등)에서도 전화를 받을 수 있게 해주는 기능입니다. 상대방이 말하는 내용을 실시간으로 텍스트로 변환해주고, 내가 타이핑한 내용을 목소리로 변환해 상대방에게 전달합니다.

1. 전화가 오면 **"텍스트 콜"** 슬라이더를 선택합니다.
2. 상대방의 말이 화면에 텍스트로 실시간 표시됩니다.
3. 하단 입력창에 답변을 타이핑하면 AI 목소리로 상대방에게 전달됩니다.
4. 미리 저장된 답변(예: "잠깐만요", "나중에 전화할게요")을 버튼 하나로 빠르게 보낼 수도 있습니다.

## 자주 묻는 질문 (Q&A)

**Q. 빅스비가 내 말을 자꾸 못 알아들어요. 어떻게 하면 되나요?**
A. 빅스비 설정 → 목소리 인식 → "목소리 다시 등록"을 눌러 목소리를 다시 학습시켜 보세요. 또한 주변 소음이 적은 곳에서 다시 등록하면 인식률이 크게 높아집니다.

**Q. 빅스비 버튼을 실수로 자꾸 누르게 돼요. 비활성화할 수 있나요?**
A. 설정 → 유용한 기능 → 측면 버튼 → "길게 누를 때"에서 **"전원 끄기 메뉴"**로 변경하면 빅스비 대신 전원 메뉴가 열립니다.

**Q. 빅스비가 한국어 외에 다른 언어도 지원하나요?**
A. 네, 빅스비 설정 → 언어에서 영어, 중국어, 일본어 등 다양한 언어로 변경할 수 있습니다.

**Q. 빅스비 루틴과 구글 어시스턴트 루틴의 차이는 무엇인가요?**
A. 빅스비 루틴은 갤럭시 기기의 시스템 설정(밝기, 사운드 프로파일, 삼성 앱 등)을 더 깊이 제어할 수 있습니다. 구글 어시스턴트는 구글 서비스 연동에 특화되어 있습니다.

**Q. 빅스비를 완전히 삭제할 수 있나요?**
A. 빅스비는 갤럭시 시스템 앱이라 완전 삭제는 어렵습니다. 다만 설정에서 비활성화하거나, 측면 버튼 기능을 다른 기능으로 바꿔서 실수로 열리지 않게 할 수 있습니다.

## 정리 요약

| 기능 | 사용법 | 활용 팁 |
|---|---|---|
| 음성 호출 | "하이 빅스비" 또는 측면 버튼 길게 누름 | 잠금 화면에서도 사용 가능 |
| 기기 제어 | 음성으로 설정 변경 | 블루투스, 와이파이, 밝기 등 |
| 루틴 | 조건+동작 자동화 설정 | 출퇴근·수면 패턴에 맞게 |
| 빅스비 비전 | 카메라로 번역·인식 | 해외 메뉴판, QR 스캔 |
| 텍스트 콜 | 음성 없이 전화 응대 | 조용한 장소에서 유용 |

빅스비는 처음엔 낯설게 느껴질 수 있지만, 한 번 루틴을 잘 세팅해두면 매일 반복되는 스마트폰 조작 시간을 크게 줄일 수 있습니다. 특히 운전 중이나 요리 중처럼 손이 자유롭지 않을 때 음성 명령은 정말 유용합니다. 오늘부터 "하이 빅스비!"를 시작해 보세요.
