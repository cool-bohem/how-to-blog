## 윈도우 가상 데스크톱이란?

윈도우 10과 11에 내장된 **가상 데스크톱(Virtual Desktop)** 기능은 여러 개의 독립된 작업 공간을 만들어 용도별로 관리할 수 있게 해주는 기능입니다.

예를 들어, **데스크톱 1**에는 업무 관련 창(엑셀, 이메일, 업무 문서)을, **데스크톱 2**에는 공부 관련 창(인터넷 강의, PDF, 노트 앱)을, **데스크톱 3**에는 개인 용도(유튜브, 게임, SNS)를 배치해 두고 필요할 때마다 빠르게 전환할 수 있습니다.

창을 최소화하고 다시 찾는 번거로움 없이, 마치 여러 대의 모니터를 쓰는 것처럼 깔끔하게 작업할 수 있습니다.

---

## 가상 데스크톱 기본 조작법

### 작업 보기 열기

가상 데스크톱을 관리하는 화면인 **"작업 보기"**를 여는 방법은 두 가지입니다.

- **단축키**: `Windows 키 + Tab` 을 누릅니다.
- **작업 표시줄 아이콘**: 작업 표시줄(화면 하단)에서 창이 겹쳐진 모양의 **"작업 보기"** 아이콘을 클릭합니다. (윈도우 11에서는 검색창 오른쪽에 위치)

작업 보기가 열리면 현재 열려 있는 모든 창이 미리 보기로 표시되고, 화면 상단에는 현재 생성된 가상 데스크톱 목록이 나타납니다.

---

## 새 가상 데스크톱 만들기

### 1단계: 작업 보기 열기

`Windows 키 + Tab` 단축키로 작업 보기를 엽니다.

### 2단계: 새 데스크톱 추가

화면 상단 데스크톱 목록 오른쪽에 있는 **"새 데스크톱"** 또는 **"+ 데스크톱 추가"** 버튼을 클릭합니다. 윈도우 11에서는 **"+"** 아이콘으로 표시됩니다.

### 3단계: 새 데스크톱 확인

클릭하면 데스크톱 목록에 **"데스크톱 2"**가 새로 생성됩니다. 이 과정을 반복하면 데스크톱 3, 4 등 여러 개를 만들 수 있습니다. 최대 생성 개수에는 실질적인 제한이 없습니다.

---

## 가상 데스크톱 간 전환하기

생성된 가상 데스크톱 사이를 빠르게 오가는 방법입니다.

### 방법 1: 단축키로 전환 (가장 빠름)

- `Ctrl + Windows 키 + 오른쪽 화살표`: 다음 데스크톱으로 이동
- `Ctrl + Windows 키 + 왼쪽 화살표`: 이전 데스크톱으로 이동

이 단축키를 익혀두면 마우스 없이도 초고속으로 전환할 수 있어 생산성이 크게 향상됩니다.

### 방법 2: 작업 보기에서 선택

`Windows 키 + Tab`으로 작업 보기를 열고 원하는 데스크톱을 클릭합니다.

### 방법 3: 작업 표시줄 아이콘 마우스오버

작업 보기 아이콘 위에 마우스를 올리면 데스크톱 미리 보기 썸네일이 나타납니다. 원하는 데스크톱을 클릭하면 즉시 전환됩니다.

---

## 창을 다른 가상 데스크톱으로 이동하기

현재 데스크톱에 있는 창을 다른 데스크톱으로 옮기고 싶을 때:

### 1단계: 작업 보기 열기

`Windows 키 + Tab`으로 작업 보기를 엽니다. 현재 데스크톱의 창들이 미리 보기로 표시됩니다.

### 2단계: 이동할 창 찾기

이동하고 싶은 창의 미리 보기 위에 마우스를 올리고 **오른쪽 클릭**합니다.

### 3단계: 이동 대상 선택

오른쪽 클릭 메뉴에서 **"이동"** 또는 **"다음으로 이동"**을 선택하면 데스크톱 목록이 나타납니다. 이동할 데스크톱을 선택합니다.

### 방법 2: 드래그 앤 드롭

작업 보기에서 이동할 창의 미리 보기를 마우스로 드래그하여 상단 데스크톱 목록의 원하는 데스크톱 위에 드롭합니다.

## 가상 데스크톱 이름 바꾸기

기본 이름인 "데스크톱 1", "데스크톱 2" 대신 알기 쉬운 이름을 붙일 수 있습니다.

### 윈도우 11에서 이름 바꾸기

1. `Windows 키 + Tab`으로 작업 보기를 엽니다.
2. 상단 데스크톱 목록에서 이름을 바꿀 데스크톱 이름을 **클릭**합니다.
3. 이름이 편집 가능한 상태로 바뀌면 원하는 이름을 입력합니다. (예: "업무", "공부", "개인")
4. `Enter`를 눌러 저장합니다.

### 윈도우 10에서 이름 바꾸기

윈도우 10에서는 기본적으로 이름 변경 기능이 제한적입니다. 작업 보기에서 데스크톱 썸네일 위에 마우스를 올리면 **연필 아이콘**이 나타납니다. 해당 아이콘을 클릭하면 이름을 변경할 수 있습니다.

---

## 가상 데스크톱 삭제하기

더 이상 필요 없는 데스크톱을 삭제하는 방법:

1. `Windows 키 + Tab`으로 작업 보기를 엽니다.
2. 삭제할 데스크톱 썸네일 위에 마우스를 올리면 오른쪽 상단에 **"X"** 버튼이 나타납니다.
3. **"X"**를 클릭하면 해당 데스크톱이 삭제됩니다.

**주의**: 데스크톱을 삭제해도 그 안에 열려 있던 창들은 닫히지 않고, 바로 이전 데스크톱으로 이동합니다. 작업 내용이 사라지지 않으니 안심하세요.

## 가상 데스크톱 활용 꿀팁

### 배경화면 다르게 설정하기 (윈도우 11)

윈도우 11에서는 각 가상 데스크톱마다 다른 배경화면을 설정할 수 있어, 현재 어느 데스크톱에 있는지 시각적으로 바로 구분할 수 있습니다.

1. 원하는 데스크톱으로 전환합니다.
2. 바탕화면을 **오른쪽 클릭** → **"개인 설정"**을 선택합니다.
3. **"배경"** 항목에서 원하는 사진을 선택합니다.
4. 현재 데스크톱에만 적용됩니다. 다른 데스크톱으로 전환 후 같은 방법으로 다른 배경을 설정하면 됩니다.

### 앱을 모든 데스크톱에서 표시하기

특정 앱(예: 카카오톡, 알람 앱)을 모든 데스크톱에서 항상 보이게 할 수 있습니다.

1. 작업 보기에서 해당 앱의 미리 보기를 오른쪽 클릭합니다.
2. **"모든 데스크톱에 표시"**를 선택합니다.
3. 이제 어느 데스크톱으로 전환해도 해당 앱이 항상 표시됩니다.

### 작업 표시줄 앱 표시 설정

윈도우 11에서는 가상 데스크톱별로 작업 표시줄에 어떤 앱을 표시할지 설정할 수 있습니다.

설정 → 시스템 → 멀티태스킹 → "가상 데스크톱" 섹션에서 **"작업 표시줄에 창 표시"** 옵션을 조정하세요.

- **사용 중인 데스크톱만**: 현재 데스크톱의 앱만 작업 표시줄에 표시됩니다.
- **모든 데스크톱**: 모든 데스크톱의 앱이 작업 표시줄에 표시됩니다.

## 자주 묻는 질문 (Q&A)

**Q. 가상 데스크톱이 작업 표시줄에 아이콘이 없어요.**

A. 윈도우 10에서 작업 보기 아이콘이 숨겨져 있을 수 있습니다. 작업 표시줄을 오른쪽 클릭 → **"작업 보기 단추 표시"**를 클릭하면 아이콘이 나타납니다. 또는 `Windows 키 + Tab` 단축키를 사용하면 됩니다.

**Q. 가상 데스크톱을 만들어도 PC가 느려지나요?**

A. 가상 데스크톱 자체는 시스템 자원을 거의 소모하지 않습니다. 다만, 각 데스크톱에 많은 앱과 창을 열어두면 당연히 RAM 사용량이 늘어 PC가 느려질 수 있습니다. 가상 데스크톱은 창 관리 방식의 변화이지, 실제로 더 많은 메모리를 사용하도록 만드는 것은 아닙니다.

**Q. PC를 재시작하면 가상 데스크톱 설정이 초기화되나요?**

A. 네, 기본적으로 PC를 재시작하면 가상 데스크톱은 초기화됩니다. 하지만 윈도우 11 최신 버전에서는 재시작 후에도 데스크톱 구성을 유지하는 기능이 개선되고 있습니다. 재시작 후 데스크톱과 창을 다시 배치하는 것이 현재로서는 일반적입니다.

**Q. 맥OS의 Mission Control과 비슷한 기능인가요?**

A. 네, 윈도우의 가상 데스크톱은 맥OS의 Spaces(스페이스) 기능과 매우 유사합니다. 개념과 사용 방법이 거의 동일하므로, 맥에서 윈도우로 전환한 사용자라면 쉽게 적응할 수 있습니다.

**Q. 특정 앱을 항상 같은 데스크톱에서 열리게 할 수 있나요?**

A. 윈도우 자체 기능만으로는 앱별 기본 데스크톱 지정이 어렵습니다. 이 기능을 원한다면 타사 유틸리티(예: DisplayFusion)를 활용하거나, 각 데스크톱에 미리 앱을 열어두고 사용하는 방식을 권장합니다.

## 정리

윈도우 가상 데스크톱은 설치나 별도 프로그램 없이 바로 사용할 수 있는 강력한 생산성 도구입니다. 업무와 개인 작업을 깔끔하게 분리하면 집중력이 높아지고 작업 전환도 훨씬 수월해집니다.

핵심 단축키를 정리하면 다음과 같습니다.

- `Windows 키 + Tab`: 작업 보기 열기
- `Windows 키 + Ctrl + D`: 새 가상 데스크톱 즉시 생성
- `Windows 키 + Ctrl + →/←`: 데스크톱 간 이동
- `Windows 키 + Ctrl + F4`: 현재 데스크톱 삭제
