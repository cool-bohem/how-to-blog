## 에어드롭(AirDrop)이란?

에어드롭은 애플 기기 간에 사진, 동영상, 연락처, 링크, 문서 등을 **블루투스와 Wi-Fi를 이용해 케이블 없이 주고받는 기능**입니다. 카카오톡이나 이메일로 보내는 것보다 훨씬 빠르고, 화질이나 파일 크기가 압축되지 않아 원본 그대로 전달됩니다.

아이폰 ↔ 아이폰, 아이폰 ↔ 아이패드, 아이폰 ↔ 맥북/iMac 사이에서 모두 사용할 수 있습니다.

---

## 에어드롭 사용 전 확인할 사항

에어드롭을 사용하려면 아래 조건이 모두 충족되어야 합니다.

1. **블루투스와 Wi-Fi가 모두 켜져 있어야 합니다.** (Wi-Fi 네트워크에 연결되지 않아도 되지만, Wi-Fi 자체는 활성화 상태여야 합니다)
2. **핫스팟(개인용 핫스팟)이 꺼져 있어야 합니다.** 핫스팟이 켜져 있으면 에어드롭이 작동하지 않습니다.
3. **두 기기가 서로 약 9미터(30피트) 이내**에 있어야 합니다.
4. **상대방의 에어드롭 수신 설정이 켜져 있어야 합니다.**

---

## 에어드롭 수신 설정 켜는 방법

보내기 전에 받는 쪽 기기의 에어드롭 수신을 활성화해야 합니다.

### 방법 1: 제어 센터에서 설정

1. 아이폰 화면 오른쪽 상단 모서리에서 아래로 **스와이프**해 제어 센터를 엽니다. (아이폰 SE 등 홈 버튼 기기는 하단에서 위로 스와이프)
2. 왼쪽 상단의 **네트워크 블록**(비행기 모드, Wi-Fi, 블루투스, 셀룰러 아이콘이 모인 네모 영역)을 **꾹 눌러** 확장합니다.
3. **에어드롭** 아이콘을 누릅니다.
4. 수신 대상을 선택합니다:
   - **수신 끔**: 아무도 보낼 수 없음
   - **연락처만**: 내 연락처에 저장된 사람만 보낼 수 있음
   - **모든 사람 (10분)**: 근처 모든 아이폰 사용자가 보낼 수 있음 (10분 후 자동으로 연락처만으로 변경)

> **팁:** 평소에는 "연락처만"으로 설정해두고, 낯선 사람과 파일을 주고받을 때만 "모든 사람"으로 바꾸세요. 모르는 사람이 보내는 불쾌한 사진을 방지할 수 있습니다.

### 방법 2: 설정 앱에서 변경

1. **설정** 앱을 실행합니다.
2. **일반** → **에어드롭**을 차례로 누릅니다.
3. 원하는 수신 설정을 선택합니다.

---

## 사진·동영상 에어드롭으로 보내는 방법

### 사진 앱에서 전송

1. **사진** 앱을 실행합니다.
2. 보내려는 사진이나 동영상을 선택합니다. 여러 장을 선택하려면 **"선택"** 버튼을 누른 뒤 각 사진을 탭합니다.
3. 화면 왼쪽 하단의 **공유 아이콘**(네모 위에 화살표 모양)을 누릅니다.
4. 공유 시트 상단에 근처 에어드롭 기기 목록이 표시됩니다.
5. 보내려는 **상대방의 이름 또는 기기 이름**을 누릅니다.
6. 상대방 기기에 수락/거절 알림이 뜹니다. 상대방이 **"수락"**을 누르면 전송이 시작됩니다.
7. 전송이 완료되면 공유 버튼 아래에 "전송됨"이라고 표시됩니다.

### 여러 앱에서 전송

사진 앱 외에도 파일 앱, 메모 앱, Safari 등 거의 모든 앱에서 공유 버튼(↑) → 에어드롭 순으로 동일하게 전송할 수 있습니다.

---

## 파일 앱에서 문서 전송하는 방법

PDF, Word 파일, 압축 파일 등을 전송할 때는 파일 앱을 이용합니다.

1. **파일** 앱을 실행합니다.
2. 보내려는 파일을 찾아 **길게 누릅니다** (꾹 누르기).
3. 팝업 메뉴에서 **"공유"**를 선택합니다.
4. 공유 시트에서 **에어드롭**을 선택합니다.
5. 상대방 기기를 선택하면 전송이 시작됩니다.

---

## 맥(Mac)으로 에어드롭 보내는 방법

아이폰에서 맥북이나 iMac으로 파일을 보낼 수도 있습니다.

### 맥 쪽 설정

1. 맥에서 **Finder**를 실행합니다.
2. 왼쪽 사이드바에서 **에어드롭**을 클릭합니다.
3. 화면 하단 "다음 사용자로부터 검색 허용"에서 **"모든 사람"** 또는 **"연락처만"**을 선택합니다.

### 아이폰에서 맥으로 보내기

1. 위에서 설명한 방법과 동일하게 공유 버튼(↑)을 누릅니다.
2. 에어드롭 항목에서 **맥의 이름**(예: "홍길동의 MacBook")을 선택합니다.
3. 맥에서 수락 알림이 나타나면 클릭합니다.
4. 전송된 파일은 맥의 **다운로드 폴더**에 저장됩니다.

---

## 에어드롭 이름(기기 이름) 변경 방법

에어드롭 목록에 표시되는 이름은 기기 이름을 따릅니다. 변경하려면 아래 순서를 따르세요.

1. **설정** 앱을 실행합니다.
2. 맨 위의 **Apple 계정 이름** 아래 **"내 이름 및 iPhone"** 또는 직접 **일반 → 정보 → 이름**으로 이동합니다.
3. **이름** 항목을 눌러 원하는 이름으로 변경합니다.
4. 변경 후 에어드롭 목록에서 새 이름으로 표시됩니다.

## 에어드롭이 안 될 때 해결 방법

### 상대방 기기가 목록에 안 보여요

1. **블루투스와 Wi-Fi가 모두 켜져 있는지** 확인하세요.
2. **개인용 핫스팟**이 켜져 있다면 꺼주세요.
3. 두 기기가 **9미터 이내**에 있는지 확인하세요.
4. 상대방의 에어드롭 수신 설정이 **"모든 사람"** 또는 **"연락처만"**으로 설정되어 있는지 확인하세요.
5. 두 기기 모두 블루투스를 껐다 켜보세요.

### 전송 중에 실패해요

1. **기기를 재시작**해보세요.
2. **iOS 업데이트**가 필요한지 설정 앱에서 확인하세요.
3. 용량이 큰 파일(수 GB)은 에어드롭보다 케이블 연결 또는 iCloud를 통한 전송이 더 안정적입니다.

### "수락 대기 중"에서 멈춰요

- 상대방이 알림을 못 봤을 수 있습니다. 상대방 기기 화면을 확인해 수락을 눌러달라고 요청하세요.
- 알림이 없다면 상대방 기기의 에어드롭 수신 설정을 다시 확인하세요.

## 자주 묻는 질문 (Q&A)

**Q. 에어드롭으로 보낸 사진은 화질이 유지되나요?**
A. 네, 에어드롭은 원본 파일을 그대로 전송합니다. 카카오톡이나 문자로 보낼 때처럼 화질이 낮아지거나 압축되지 않습니다.

**Q. 안드로이드폰으로도 에어드롭을 사용할 수 있나요?**
A. 아니요, 에어드롭은 애플 기기(아이폰, 아이패드, 맥)끼리만 사용할 수 있습니다. 안드로이드와 파일을 주고받으려면 카카오톡이나 구글 드라이브를 이용하세요.

**Q. 에어드롭 사용 시 데이터 요금이 발생하나요?**
A. 아니요, 에어드롭은 블루투스와 Wi-Fi 다이렉트(Direct) 방식을 사용해 인터넷을 거치지 않으므로 모바일 데이터 요금이 발생하지 않습니다.

**Q. 한 번에 몇 개까지 보낼 수 있나요?**
A. 사진은 수백 장도 한 번에 선택해 전송할 수 있습니다. 단, 파일 수와 용량이 많을수록 시간이 더 오래 걸립니다.

**Q. 에어드롭으로 받은 파일은 어디에 저장되나요?**
A. 사진·동영상은 **사진 앱**에 저장되고, 문서·파일은 **파일 앱**의 "다운로드" 폴더에 저장됩니다.

**Q. 모르는 사람이 에어드롭으로 불쾌한 사진을 보내요.**
A. 즉시 **"거절"**을 누르고, 에어드롭 설정을 **"연락처만"** 또는 **"수신 끔"**으로 변경하세요. iOS 17 이상에서는 "모든 사람" 설정이 10분 후 자동으로 "연락처만"으로 바뀝니다.

## 마무리 정리

에어드롭은 애플 기기 간 파일 전송에서 가장 빠르고 편리한 방법입니다. 블루투스와 Wi-Fi를 켜두고, 핫스팟만 꺼두면 언제든지 사용할 수 있습니다.

- **사진 여러 장 전송**: 사진 앱 → 선택 → 공유 → 에어드롭
- **문서 전송**: 파일 앱 → 길게 누르기 → 공유 → 에어드롭
- **안 될 때**: 블루투스·Wi-Fi 재활성화, 핫스팟 끄기, 거리 확인

평소에는 수신 설정을 **"연락처만"**으로 해두고 필요할 때만 변경하는 습관을 들이면 개인정보 보호에도 도움이 됩니다.
