## 신한 SOL이란?

신한 SOL은 신한은행의 공식 모바일 뱅킹 앱입니다. 과거에는 신한 쏠(Shinhan SOL)이라는 이름으로도 불렸으며, 현재는 **신한 SOL**로 통합 운영되고 있습니다. 이 앱 하나로 계좌 잔액 조회, 이체, 공과금 납부, 카드 관리, 대출 신청, 그리고 **신한페이판**을 통한 QR·NFC 결제까지 모두 처리할 수 있습니다.

신한은행 계좌가 있는데 창구나 ATM만 이용하고 있다면, 이 앱을 설치하는 것만으로도 은행 업무의 대부분을 집에서 처리할 수 있게 됩니다. 이 글에서는 앱 설치와 가입부터 일상적인 이체·결제까지 처음 사용하는 분도 쉽게 따라할 수 있도록 안내합니다.

---

## 1단계: 신한 SOL 앱 설치하기

1. 스마트폰에서 **Play 스토어(안드로이드)** 또는 **App Store(아이폰)** 를 엽니다.
2. 검색창에 **"신한 SOL"** 을 입력합니다.
3. 검색 결과에서 **신한은행** 공식 앱을 선택합니다. 앱 아이콘은 주황색 바탕에 흰색 S 모양입니다.
4. **설치(또는 받기)** 버튼을 눌러 설치합니다.
5. 설치 완료 후 앱을 실행합니다.

> **주의:** 유사한 이름의 피싱 앱이 존재할 수 있습니다. 개발자가 반드시 **"신한은행"** 인지 확인하고 설치하세요.

---

## 2단계: 본인인증 및 서비스 가입

신한 SOL을 처음 사용하려면 본인인증을 통해 서비스에 가입해야 합니다.

1. 앱을 실행하면 **"서비스 가입"** 또는 **"로그인"** 화면이 표시됩니다.
2. **서비스 가입** 버튼을 탭합니다.
3. 이름과 **주민등록번호 앞 7자리(생년월일+성별)** 를 입력합니다.
4. **휴대폰 본인인증** 을 선택하고, 통신사를 선택한 뒤 인증번호를 입력합니다.
5. 신한은행 계좌번호와 출금 비밀번호(계좌 비밀번호)를 입력합니다.
6. **로그인 비밀번호**를 설정합니다. 숫자 6자리로 설정하며, 타인이 추측하기 어려운 번호로 설정하세요.
7. 이용약관에 동의 후 **가입 완료**.

> **팁:** 계좌 비밀번호가 기억나지 않는다면 가까운 신한은행 지점을 방문하거나 고객센터(1599-8000)에 문의하세요.

---

## 3단계: 간편 로그인 설정 (지문·Face ID)

매번 비밀번호를 입력하는 번거로움을 줄이려면 생체 인증을 등록하세요.

1. 가입 완료 후 로그인 화면에서 **"간편 로그인 설정"** 팝업이 나타납니다.
2. **지문 인증** (안드로이드) 또는 **Face ID** (아이폰)를 선택합니다.
3. 스마트폰에 등록된 생체 정보를 인증하면 설정 완료됩니다.
4. 다음부터는 앱 실행 시 지문이나 얼굴 인식만으로 즉시 로그인됩니다.

나중에 설정하려면: 앱 하단 **메뉴** → **설정** → **보안/인증** → **간편 로그인** 에서 변경 가능합니다.

---

## 4단계: 계좌 잔액 조회하기

1. 앱에 로그인하면 **홈 화면**에 보유 계좌가 목록으로 표시됩니다.
2. 잔액을 확인하고 싶은 계좌를 탭하면 잔액과 최근 거래내역이 표시됩니다.
3. **거래내역** 탭을 누르면 입금·출금 내역을 날짜별로 확인할 수 있습니다.
4. 특정 기간의 내역만 보고 싶다면 **기간 설정** 버튼으로 조회 기간을 지정하세요.

---

## 5단계: 이체(송금)하기

### 일반 이체
1. 홈 화면 상단 또는 하단 메뉴에서 **이체** 를 탭합니다.
2. **출금 계좌**를 선택합니다 (내 계좌 목록에서 선택).
3. **입금 은행**을 선택합니다 — 드롭다운 메뉴에서 은행명을 탭하세요.
4. **입금 계좌번호**를 입력합니다. 숫자를 직접 입력하거나 최근 이체 계좌에서 선택할 수 있습니다.
5. **이체 금액**을 입력합니다. 예) 50,000
6. 메모(받는 분 통장에 표시될 문구)를 원하면 입력합니다.
7. **다음** 버튼 → 이체 내용 확인 → **이체하기** 버튼 → 간편 비밀번호 또는 지문 인증으로 완료.

### 자주 쓰는 계좌 등록
- 이체 완료 후 **"자주 쓰는 계좌에 저장"** 을 선택하면 다음에 더 빠르게 이체할 수 있습니다.
- 저장된 계좌는 이체 화면의 **즐겨찾기** 탭에서 바로 선택 가능합니다.

## 6단계: 공과금·납부 처리하기

신한 SOL에서는 각종 공과금을 납부할 수 있습니다.

1. 하단 메뉴 → **이체/납부** → **공과금납부** 를 탭합니다.
2. 납부 종류를 선택합니다:
   - 국세(소득세, 부가세 등)
   - 지방세(재산세, 자동차세 등)
   - 전기요금, 도시가스, 수도료
   - 국민연금, 건강보험료
   - 아파트 관리비
3. 종류 선택 후 **고지서 번호** 또는 **납부자 번호**를 입력합니다. 고지서에 적힌 번호를 그대로 입력하면 됩니다.
4. 납부 금액 확인 후 **납부하기** 버튼 → 인증 완료.

> **팁:** OCR(카메라 인식) 기능을 지원하는 경우 고지서를 카메라로 찍으면 번호가 자동 입력됩니다. 납부 화면에서 **카메라 아이콘**을 탭해보세요.

---

## 7단계: 신한페이판으로 QR 결제하기

신한 SOL에는 **신한페이판** 이라는 간편결제 기능이 내장되어 있습니다. 카드를 들고 다니지 않아도 QR코드 또는 NFC로 결제할 수 있습니다.

1. 앱 홈 화면 상단의 **페이판** 버튼을 탭합니다.
2. **결제 QR코드**가 화면에 표시됩니다.
3. 가맹점의 결제 단말기에 QR코드를 스캔하면 결제 완료됩니다.
4. NFC 결제를 지원하는 단말기에서는 스마트폰을 단말기에 가까이 대면 자동으로 결제됩니다.

페이판에 등록할 카드 또는 계좌를 설정하려면: 페이판 화면 → **결제수단 관리** → 원하는 신한카드 또는 계좌 선택.

---

## 8단계: 보안 설정 점검하기

모바일 뱅킹은 보안이 매우 중요합니다. 아래 보안 설정을 반드시 확인하세요.

1. **앱 잠금 설정:** 앱을 사용하지 않을 때 자동으로 잠기도록 설정합니다. 메뉴 → 설정 → 보안 → 앱 잠금 → 켜기
2. **이체 한도 확인:** 하루 이체 한도를 본인 필요에 맞게 설정합니다. 불필요하게 한도를 높게 설정하지 마세요. 메뉴 → 이체 → 이체 한도 관리
3. **알림 서비스 설정:** 입출금 시 즉시 알림을 받도록 설정합니다. 메뉴 → 알림 설정 → 입출금 알림 → 켜기
4. **출처 불명 링크 주의:** 문자나 카카오톡으로 온 링크를 클릭해 앱을 설치하지 마세요. 반드시 공식 스토어(Play 스토어, App Store)에서만 설치하세요.

## 자주 묻는 질문 (Q&A)

**Q. 신한은행 계좌가 없어도 신한 SOL을 쓸 수 있나요?**
A. 기본 뱅킹 기능(이체, 잔액 조회)은 신한은행 계좌가 있어야 사용 가능합니다. 하지만 환율 조회나 이벤트 확인 등 일부 기능은 비회원도 이용할 수 있습니다.

**Q. 비밀번호를 5번 이상 틀렸어요.**
A. 보안상 서비스 이용이 잠깁니다. 앱에서 **"비밀번호 초기화"** 를 선택한 뒤 본인인증을 통해 새 비밀번호를 설정할 수 있습니다. 또는 신한은행 고객센터(1599-8000)로 전화하여 도움을 받으세요.

**Q. 이체 후 상대방 계좌에 돈이 안 들어갔어요.**
A. 이체는 즉시 처리되는 경우가 대부분이지만, 은행 점검 시간(보통 새벽 23:30~00:30) 중에는 처리가 지연될 수 있습니다. 이체 내역에 "완료" 상태인데 수신자에게 미반영이라면 수신 은행 고객센터에 확인하세요.

**Q. 공인인증서(구 공인인증서)는 필요 없나요?**
A. 신한 SOL은 공인인증서 없이도 이용 가능합니다. 간편 비밀번호, 지문 인증, 패턴 인증으로 대체할 수 있습니다. 다만 일부 고액 이체나 공공기관 연계 업무에는 공동인증서가 필요할 수 있습니다.

## 마무리 정리

신한 SOL 앱을 제대로 설정해 두면 은행 창구에 갈 일이 크게 줄어듭니다. 핵심 과정을 요약하면 다음과 같습니다:

1. Play 스토어 / App Store에서 **신한 SOL** 공식 앱 설치
2. 본인인증 및 계좌 비밀번호로 **서비스 가입**
3. 지문 또는 Face ID **간편 로그인 설정**
4. 홈 화면에서 **잔액 조회** 및 **거래내역** 확인
5. **이체** 메뉴로 간편하게 송금
6. 공과금 납부는 **공과금 납부** 메뉴에서 고지서 번호 입력
7. **신한페이판**으로 QR·NFC 간편결제 이용

보안 설정도 꼭 확인하여 안전하게 사용하세요. 신한 SOL을 활용하면 금융 업무의 대부분을 스마트폰 하나로 해결할 수 있습니다.
