## 애플워치 처음 설정하기

박스에서 꺼낸 애플워치를 아이폰과 연결하는 방법이에요.

## 아이폰과 연결하기

1. 애플워치 전원 켜기 (옆면 버튼 꾹 누르기)
2. 아이폰에서 **Watch 앱** 실행
3. **페어링 시작** 클릭
4. 아이폰 카메라로 워치 화면의 패턴 스캔
5. 손목 선택 (왼쪽/오른쪽) 후 설정 진행

## 기본 설정

- **페이스 ID 또는 암호** 설정
- **애플페이** 카드 추가
- **건강 데이터** 입력 (키, 몸무게 등)
- **알림 설정** (받을 앱 선택)

## 유용한 앱 설치

Watch 앱에서 앱 스토어 탭:

- **카카오톡** - 메시지 확인
- **네이버 지도** - 손목에서 길찾기
- **Workout** - 운동 기록 (기본 내장)

## 워치 페이스 변경

1. 워치 화면을 꾹 누르기
2. 좌우로 스와이프해서 페이스 선택
3. **편집** 으로 색상, 컴플리케이션 변경

## 배터리 관리 팁

- 충전은 자는 동안 하기
- **극저전력 모드** 켜면 3일까지 사용 가능
