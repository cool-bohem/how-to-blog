## 애플워치 개봉 후 처음 할 일

박스에서 꺼낸 애플워치는 아이폰과 페어링해야 사용할 수 있어요. 준비물은 애플워치, 아이폰(iPhone 6s 이상), 그리고 와이파이 연결이면 돼요. 처음 설정은 15~20분 정도 걸려요.

**주의**: 애플워치는 안드로이드 기기와 연결되지 않아요. 아이폰 전용이에요.

## 아이폰과 연결하기 (페어링)

1. 애플워치 전원 켜기 — 옆면 버튼(크라운 아래 버튼)을 꾹 누르기
2. 아이폰에서 **Watch 앱** 실행 (기본 설치되어 있어요)
3. **페어링 시작** 클릭
4. 아이폰 카메라로 워치 화면의 패턴 스캔
5. 이전 애플워치 백업 복원 여부 선택 (처음이라면 새로 설정)
6. 손목 선택 후 설정 계속 진행

**팁**: 페어링 중 워치와 아이폰을 가까이 두세요. 멀어지면 연결이 끊길 수 있어요.

## 기본 설정

페어링 후 차례대로 설정을 진행해요.

**암호 설정**
- 워치를 손목에서 벗을 때마다 잠금이 걸려요
- 간단한 4자리 암호 또는 6자리 숫자 암호 설정 가능
- 암호를 설정해야 애플페이 사용 가능

**애플페이 카드 추가**
- 기존 아이폰에 등록된 카드를 워치에도 추가할 수 있어요
- 카드를 손목에 가져다 대면 바로 결제 가능

**건강 데이터 입력**
- 키, 몸무게, 생년월일을 입력하면 칼로리 소모량 계산이 정확해져요
- 나중에 **건강 앱**에서도 수정 가능

**알림 설정**
- 아이폰의 알림 중 워치로 받을 앱을 선택할 수 있어요
- 모든 알림을 다 받으면 손목이 너무 자주 진동해요. 중요한 앱만 선택하는 걸 추천해요

## 워치 페이스 변경

워치 페이스는 시계판 디자인이에요. 원하는 스타일로 바꿀 수 있어요.

1. 워치 화면을 **꾹 누르기**
2. 좌우로 스와이프해서 페이스 선택
3. **편집** 으로 색상, 컴플리케이션 변경
4. 컴플리케이션: 시계 화면에 날씨, 배터리, 걸음 수 등 정보 추가 가능

**추천 페이스**: Modular(정보량 많음), Infograph(심플함), Siri(일정 자동 표시)

## 유용한 앱 설치

Watch 앱에서 App Store 탭을 열면 워치용 앱을 설치할 수 있어요.

**추천 앱**
- **카카오톡**: 손목에서 메시지 확인 및 빠른 답장
- **네이버 지도**: 손목에서 방향 안내 (진동으로 좌우 알림)
- **Workout**: 운동 기록, 기본 내장 앱
- **Spotify / 멜론**: 음악 재생 제어

## 배터리 관리 팁

- 충전은 자는 동안 하는 게 가장 편해요 (8시간 충전으로 하루 이상 사용 가능)
- **극저전력 모드** 켜면 3일까지 사용 가능 (단, 기능이 제한됨)
- **설정 → 배터리**에서 현재 잔량과 절약 모드 확인 가능
- 워치를 충전기에 올려두면 자동으로 충전 시작

## 알아두면 좋은 조작법

| 동작 | 기능 |
|---|---|
| 크라운 돌리기 | 스크롤 또는 확대/축소 |
| 크라운 클릭 | 홈 화면 |
| 크라운 두 번 클릭 | 최근 앱 전환 |
| 측면 버튼 클릭 | Dock(자주 쓰는 앱) |
| 측면 버튼 두 번 클릭 | 애플페이 실행 |

## 자주 묻는 질문

**Q. 애플워치가 아이폰과 자꾸 연결이 끊겨요.**
워치와 아이폰은 블루투스로 연결돼요. 블루투스 범위(약 10m)를 벗어나면 끊겨요. 근처 와이파이가 있으면 와이파이로 연결을 유지해요.

**Q. 워치로 전화를 받을 수 있나요?**
네. 아이폰이 근처에 있으면 워치에서 바로 전화를 받거나 걸 수 있어요. 스피커로 통화해요.

**Q. 애플워치를 샤워할 때 차도 되나요?**
애플워치 Series 2 이후 모델은 방수(50m)가 돼요. 샤워는 괜찮지만, 온천이나 사우나는 피하는 게 좋아요.
