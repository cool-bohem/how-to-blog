## 크롬이 느려지는 이유

크롬은 세계에서 가장 많이 쓰이는 브라우저지만, 메모리(RAM)를 많이 사용하는 것으로 유명해요. 시간이 지날수록 캐시가 쌓이고, 확장 프로그램이 늘고, 탭을 많이 열어두면 컴퓨터 전체가 느려지는 것처럼 느껴질 수 있어요.

크롬이 느려지는 주요 원인:
- **캐시와 쿠키 누적**: 오랫동안 지우지 않으면 수 GB씩 쌓여요
- **확장 프로그램 과다**: 하나하나는 작아 보여도 여러 개가 합쳐지면 메모리를 많이 써요
- **탭 과다**: 탭 하나당 별도의 메모리를 사용해요
- **구형 버전**: 업데이트하지 않으면 성능 최적화가 적용되지 않아요

## 방법 1. 캐시 및 쿠키 삭제

가장 효과적인 방법이에요. 쌓인 캐시를 지우면 페이지 로딩 속도가 눈에 띄게 빨라져요.

1. 크롬 오른쪽 상단 **점 세 개(⋮)** 클릭
2. **설정 → 개인 정보 보호 및 보안**
3. **인터넷 사용 기록 삭제** 클릭
4. 기간: **전체 기간** 선택
5. **캐시된 이미지 및 파일**, **쿠키** 체크 후 삭제

**단축키**: `Ctrl + Shift + Delete` 를 누르면 바로 삭제 창이 열려요.

**주의사항**: 쿠키를 삭제하면 자동 로그인이 해제돼요. 자주 쓰는 사이트 비밀번호를 미리 확인해두세요.

## 방법 2. 불필요한 확장 프로그램 삭제

확장 프로그램은 유용하지만 많을수록 크롬이 무거워져요. 안 쓰는 건 과감히 제거하세요.

1. 주소창에 `chrome://extensions` 입력
2. 사용하지 않는 확장 프로그램 **제거** 클릭
3. 자주 쓰지 않는 건 **비활성화** 만 해도 돼요

**팁**: 광고 차단 확장 프로그램(예: uBlock Origin)은 오히려 페이지 로딩을 빠르게 해줄 수 있어요. 광고를 불러오지 않아도 되기 때문이에요.

## 방법 3. 탭 개수 줄이기

열린 탭이 많을수록 크롬이 사용하는 메모리가 늘어나요. 탭 20개를 열어두면 크롬 혼자 램 2~4GB를 쓰기도 해요.

- 나중에 볼 페이지는 **북마크**에 저장하고 탭 닫기
- **OneTab** 확장 프로그램으로 탭을 리스트로 변환해 저장 (메모리 절약 효과 큼)

**팁**: 크롬 주소창에 `chrome://discards` 를 입력하면 각 탭의 메모리 사용량을 볼 수 있어요.

## 방법 4. 하드웨어 가속 켜기

GPU(그래픽카드)를 활용해 크롬의 렌더링 부담을 줄여주는 기능이에요. 꺼져 있는 경우 켜면 속도가 개선될 수 있어요.

1. **설정 → 시스템**
2. **가능한 경우 하드웨어 가속 사용** 토글 켜기
3. 크롬 재시작

**주의사항**: 일부 구형 PC에서는 하드웨어 가속이 오히려 화면 깜빡임이나 오류를 유발할 수 있어요. 문제가 생기면 다시 꺼주세요.

## 방법 5. 크롬 최신 버전 유지

업데이트에는 보안 패치뿐만 아니라 성능 개선도 포함돼요.

1. 주소창에 `chrome://settings/help` 입력
2. 자동으로 업데이트 확인 및 설치
3. 업데이트 후 재시작

## 추가 팁: 메모리 절약 모드 활성화

크롬에는 비활성 탭의 메모리를 자동으로 해제하는 기능이 있어요.

1. 주소창에 `chrome://settings/performance` 입력
2. **메모리 절약** 모드 켜기
3. 사용하지 않는 탭은 자동으로 메모리에서 해제됨

## 자주 묻는 질문

**Q. 크롬 말고 다른 브라우저가 더 빠른가요?**
Edge(Microsoft), Firefox, Brave 등이 메모리 효율이 더 좋다는 평가를 받아요. 특히 Brave는 광고 차단 기능이 내장돼 있어 빠른 편이에요.

**Q. 크롬을 재설치하면 더 빨라지나요?**
캐시와 설정이 초기화되기 때문에 효과가 있을 수 있어요. 단, 북마크와 로그인 정보를 미리 백업해두세요.

**Q. 크롬이 갑자기 응답 없음이 되면 어떻게 하나요?**
`Shift + Esc` 를 누르면 크롬 내 작업 관리자가 열려요. 여기서 특정 탭이나 확장 프로그램을 강제 종료할 수 있어요.

위 방법을 순서대로 적용하면 크롬 속도가 확실히 개선될 거예요. 특히 캐시 삭제와 확장 프로그램 정리는 효과가 즉각적으로 느껴지는 방법이에요.
