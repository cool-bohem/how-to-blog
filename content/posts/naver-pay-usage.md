## 네이버페이란?

네이버페이는 네이버에서 운영하는 간편 결제 서비스예요. 네이버쇼핑, 스마트스토어에서의 구매는 물론 편의점, 카페 등 오프라인 가맹점에서도 사용할 수 있어요. 네이버 포인트 적립과 연계되어 자주 쓸수록 혜택이 커지는 구조예요.

**네이버페이 주요 기능**:
- 온라인 쇼핑 간편 결제
- 오프라인 QR/바코드 결제
- 친구에게 송금
- 포인트 적립 및 사용
- 청구서 납부 (공과금, 통신비 등)

## 네이버페이 처음 등록하기

1. 네이버 앱 실행 → 하단 **페이** 탭 클릭
2. **네이버페이 시작하기** 클릭
3. 결제 수단 등록:
   - 신용/체크카드 추가
   - 계좌 연결 (네이버페이 포인트 충전용)
4. **간편 비밀번호 6자리** 설정
5. 생체 인증(지문/얼굴) 설정 (선택)

**팁**: 카드를 여러 장 등록해두면 결제 시 원하는 카드를 선택할 수 있어요.

## 온라인 결제 방법

네이버쇼핑, 스마트스토어에서 쇼핑할 때:

1. 상품 선택 후 **구매하기** 클릭
2. 결제 수단에서 **네이버페이** 선택
3. **지문 인식** 또는 **간편 비밀번호** 입력
4. 결제 완료

네이버페이로 결제하면 **포인트가 적립**돼요. 적립률은 카드 종류와 이벤트에 따라 달라요.

## 오프라인 결제 방법

편의점(CU, GS25, 세븐일레븐), 카페, 음식점 등 네이버페이 가맹점에서 사용 가능해요.

1. 네이버 앱 실행 → **페이** 탭
2. **결제** 클릭
3. **QR코드/바코드** 표시
4. 점원에게 보여주거나 QR 스캐너에 스캔

**팁**: 결제 전 오프라인 가맹점 여부는 매장 입구 네이버페이 스티커로 확인할 수 있어요.

## 네이버페이 포인트 충전 및 사용

**포인트 충전**
1. 페이 탭 → **충전** 클릭
2. 연결된 계좌에서 금액 입력 후 충전

**포인트 사용**
- 네이버쇼핑 결제 시 포인트를 현금처럼 사용
- 1포인트 = 1원

**팁**: 네이버 멤버십 구독자는 쇼핑 포인트 적립률이 높아져요.

## 송금 방법

1. 페이 탭 → **송금** 클릭
2. 받는 사람 검색:
   - 네이버 아이디
   - 전화번호 (네이버 앱 설치된 경우)
3. 금액 입력 후 **보내기**
4. 인증 후 완료

**송금 한도**: 1일 200만원 (본인 인증 수준에 따라 한도가 달라요)

## 공과금/청구서 납부

전기요금, 통신비, 아파트 관리비 등을 네이버페이로 납부할 수 있어요.

1. 페이 탭 → **청구서** 클릭
2. 납부할 청구서 확인 (자동으로 내 명의 청구서 표시)
3. **납부하기** 클릭

## 네이버페이 안전하게 사용하기

**이용 한도 설정**
- 페이 설정에서 1회/1일 결제 한도를 직접 설정할 수 있어요
- 분실에 대비해 한도를 낮게 설정해두는 게 좋아요

**분실 시 대처**
- 네이버 앱 접속 → 페이 → 설정 → **페이 잠금** 으로 즉시 결제 차단

## 자주 묻는 질문

**Q. 네이버페이와 카카오페이 중 어떤 걸 써야 하나요?**
네이버쇼핑을 자주 이용한다면 네이버페이, 카카오톡 친구에게 송금을 많이 한다면 카카오페이가 더 편해요. 둘 다 등록해두고 상황에 맞게 쓰는 분도 많아요.

**Q. 포인트는 언제 소멸되나요?**
네이버페이 포인트는 종류에 따라 소멸 기간이 다릅니다. 적립 내역에서 소멸 예정 포인트를 확인할 수 있어요.

**Q. 해외에서도 네이버페이를 쓸 수 있나요?**
현재 네이버페이는 국내 가맹점 위주예요. 해외 결제는 연결된 카드로 직접 결제하는 방법을 이용하세요.
