## 네이버페이란?

네이버 앱에서 바로 결제, 송금, 포인트 적립이 가능한 간편 결제 서비스예요.

## 네이버페이 처음 등록하기

1. 네이버 앱 실행 → 하단 **페이** 탭
2. **네이버페이 시작하기**
3. 카드 또는 계좌 등록
4. 간편 비밀번호 6자리 설정

## 온라인 결제 방법

네이버쇼핑, 스마트스토어 등에서:

1. 결제 시 **네이버페이** 선택
2. 지문 또는 비밀번호 인증
3. 결제 완료 (포인트 자동 적립)

## 오프라인 결제 방법

편의점, 카페 등 가맹점에서:

1. 네이버페이 앱 실행
2. **바코드 결제** 클릭
3. 바코드를 카운터에 제시

## 포인트 충전 및 사용

1. **페이 → 포인트 충전**
2. 계좌에서 충전
3. 네이버쇼핑 결제 시 포인트 사용 가능

## 송금 방법

1. **페이 → 송금**
2. 받는 사람 검색 (네이버 아이디 또는 전화번호)
3. 금액 입력 후 송금
