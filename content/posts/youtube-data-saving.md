## 유튜브 데이터 절약 설정

유튜브는 스마트폰 데이터를 가장 많이 쓰는 앱 중 하나예요. 아래 설정으로 데이터를 아껴보세요.

## 방법 1. 데이터 절약 모드 켜기

1. 유튜브 앱 실행 후 오른쪽 상단 **프로필 아이콘** 클릭
2. **설정** 으로 이동
3. **동영상 화질 기본 설정** 클릭
4. **모바일 데이터** 항목에서 **데이터 절약** 선택

## 방법 2. 자동재생 끄기

자동재생은 영상이 끝나면 다음 영상을 자동으로 틀어줘서 데이터를 많이 써요.

1. 유튜브 **설정 → 자동재생**
2. **다음 동영상 자동재생** 토글 끄기

## 방법 3. 와이파이에서만 HD 재생

1. **설정 → 동영상 화질 기본 설정**
2. **모바일 데이터**: 360p 또는 480p 선택
3. **와이파이**: 고화질(HD) 선택

## 방법 4. 오프라인 저장 활용

자주 보는 영상은 **와이파이 연결 시 미리 저장**해두면 이동 중에 데이터 없이 볼 수 있어요.

1. 영상 아래 **다운로드 버튼(↓)** 클릭
2. 화질 선택 후 저장
3. **보관함 → 다운로드** 에서 오프라인으로 시청

## 방법 5. 백그라운드 재생 제한

유튜브 프리미엄이 없어도 화면을 끄면 재생이 멈춰요. 이게 오히려 데이터 절약에 도움이 돼요.

화면이 꺼진 상태에서 재생되지 않도록 앱 알림과 백그라운드 권한을 제한하는 방법:

- **설정 → 앱 → 유튜브 → 배터리 → 백그라운드 활동 제한**
