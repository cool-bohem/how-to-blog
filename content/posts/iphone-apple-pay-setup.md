## 애플페이란?

애플페이(Apple Pay)는 아이폰, 애플워치, 맥 등 애플 기기에서 사용할 수 있는 간편결제 서비스입니다. 실물 카드 없이도 스마트폰만 있으면 오프라인 매장과 온라인 쇼핑몰에서 결제할 수 있어 많은 분들이 사용하고 있습니다. 한국에서는 2023년 3월부터 정식 서비스가 시작되었으며, 2026년 현재는 대부분의 주요 카드사에서 애플페이를 지원하고 있습니다.

이 글에서는 아이폰에서 애플페이를 처음 설정하는 방법부터, 오프라인 매장 결제, 교통카드 등록까지 모든 것을 상세하게 안내해 드리겠습니다.

## 애플페이 사용 가능 기기 확인

애플페이는 모든 아이폰에서 사용 가능한 것은 아닙니다. 아이폰 6 이상 모델에서 사용 가능하지만, 한국에서 서비스되는 애플페이의 경우 페이스 아이디(Face ID) 또는 터치 아이디(Touch ID)가 탑재된 기기가 필요합니다.

지원 기기 목록:
- 아이폰 SE(2세대 이상)
- 아이폰 8 이상
- 아이폰 X 이상 모든 모델

또한 최신 iOS 버전으로 업데이트되어 있어야 합니다. 설정 → 일반 → 소프트웨어 업데이트에서 현재 버전을 확인하고, 업데이트가 있다면 먼저 진행해 두세요.

## 카드 등록 전 준비사항

애플페이에 카드를 등록하기 전에 다음 사항을 미리 준비해 두세요.

1. **지원 카드사 확인**: 현대카드, 신한카드, 국민카드, 우리카드, 하나카드, 롯데카드 등 주요 카드사에서 애플페이를 지원합니다. 본인이 사용하는 카드가 지원되는지 카드사 공식 홈페이지나 고객센터를 통해 확인하세요.

2. **카드 정보 준비**: 카드 번호 16자리, 유효기간(월/년), CVV 번호(카드 뒷면 서명란 옆 3자리 숫자)를 미리 확인해 두세요.

3. **Apple ID 로그인 확인**: 설정 앱 최상단에서 본인의 Apple ID가 로그인되어 있는지 확인하세요.

4. **이중 인증 설정**: 보안을 위해 Apple ID 이중 인증이 활성화되어 있어야 합니다. 설정 → [이름] → 로그인 및 보안에서 확인할 수 있습니다.

## 아이폰 애플페이 카드 등록 방법

### 1단계: 지갑(Wallet) 앱 열기

아이폰 홈 화면에서 **지갑(Wallet)** 앱을 찾아 실행합니다. 지갑 앱은 기본적으로 설치되어 있으며, 삭제했다면 App Store에서 "Apple 지갑"을 검색하여 다시 설치할 수 있습니다.

지갑 앱 아이콘은 여러 장의 카드가 겹쳐진 모양으로, 보통 홈 화면 하단이나 앱 보관함에서 찾을 수 있습니다.

### 2단계: 카드 추가 버튼 클릭

지갑 앱이 열리면 오른쪽 상단의 **'+'(더하기) 버튼**을 탭합니다. 카드가 하나도 없는 경우에는 화면 중앙이나 하단에 "카드 추가" 버튼이 크게 표시됩니다.

### 3단계: 카드 종류 선택

카드 추가 화면에서 다음 중 원하는 항목을 선택합니다:
- **체크카드 또는 신용카드**: 일반 결제용 카드 등록
- **교통카드**: 지하철, 버스 등 대중교통 이용용 카드 등록

신용카드나 체크카드를 먼저 등록하려면 "체크카드 또는 신용카드"를 선택하세요.

### 4단계: 카드 스캔 또는 수동 입력

카드 정보를 입력하는 두 가지 방법이 있습니다.

**카메라로 스캔하는 방법:**
실물 카드를 카메라 화면 안에 맞추면 카드 번호와 유효기간이 자동으로 인식됩니다. 빛 반사가 심한 환경에서는 인식이 잘 안 될 수 있으므로 밝은 자연광 아래에서 시도해 보세요.

**수동으로 입력하는 방법:**
화면 하단의 **"카드 정보를 수동으로 입력하십시오"**를 탭한 후, 카드 번호 16자리를 입력하고 이름과 유효기간을 차례로 입력합니다.

### 5단계: 보안 코드(CVV) 입력

카드 정보 입력 후 다음 화면에서 CVV 보안 코드를 입력합니다. CVV는 카드 뒷면 서명란 오른쪽에 적혀 있는 3자리 숫자입니다. 아멕스 카드의 경우 카드 앞면에 4자리로 표시됩니다.

### 6단계: 카드사 약관 동의

해당 카드사의 애플페이 이용약관이 표시됩니다. 내용을 확인한 후 **"동의"**를 탭하여 진행합니다.

### 7단계: 카드 인증 완료

카드사에 따라 추가 본인 인증이 필요할 수 있습니다. 인증 방법은 카드사마다 다릅니다.

- **문자(SMS) 인증**: 카드에 등록된 휴대폰 번호로 인증번호 6자리가 전송됩니다. 입력하면 인증이 완료됩니다.
- **카드사 앱 인증**: 해당 카드사 앱(예: 현대카드 앱)을 통해 인증을 진행합니다.
- **고객센터 전화**: 카드사 고객센터에 직접 전화하여 상담사를 통해 인증합니다.

인증이 완료되면 카드가 지갑 앱에 등록되며, "활성화 완료" 메시지가 표시됩니다.

## 오프라인 매장에서 애플페이 결제하기

카드 등록이 완료되었다면 이제 실제 결제를 해볼 차례입니다.

### 결제 단말기 확인

애플페이는 NFC 기반 비접촉 결제로, 매장에서 **비접촉 결제(컨택리스) 단말기**를 지원해야 합니다. 단말기에 와이파이 신호 모양의 비접촉 아이콘이나 "NFC" 표시가 있으면 사용 가능합니다.

2026년 현재 이마트, 롯데마트, 홈플러스, 스타벅스, 맥도날드, 롯데리아, GS25, CU, 세븐일레븐, 올리브영 등 대부분의 대형 매장과 편의점에서 애플페이를 사용할 수 있습니다.

### 결제 방법 - Face ID 탑재 기기 (아이폰 X 이상)

1. 아이폰 **오른쪽 측면 버튼을 두 번 빠르게** 클릭합니다
2. 지갑 앱이 열리면서 기본 카드가 표시됩니다
3. 아이폰을 들고 **페이스 아이디로 얼굴을 인식**합니다
4. 화면에 "준비됨"이 표시되면 아이폰 상단 부분을 결제 단말기에 **가까이** 댑니다 (약 2~3cm 거리)
5. 결제 완료 소리와 함께 화면에 체크 표시가 나타납니다

다른 카드로 결제하려면 기본 카드가 표시된 상태에서 카드를 위로 스와이프하여 원하는 카드를 선택하세요.

### 결제 방법 - Touch ID 탑재 기기 (아이폰 SE 등)

1. 아이폰 **오른쪽 측면 버튼을 두 번 빠르게** 클릭합니다
2. **홈 버튼에 손가락을 올려** 지문을 인식합니다
3. 화면에 "준비됨"이 표시되면 아이폰 상단을 단말기에 가까이 댑니다
4. 결제가 완료됩니다

## 온라인 결제에서 애플페이 사용하기

애플페이는 온라인 쇼핑에서도 사용할 수 있습니다. 사파리 브라우저 또는 애플페이를 지원하는 쇼핑 앱에서 결제 시 **"Apple Pay"** 버튼을 탭하면 별도로 카드 번호를 입력할 필요 없이 페이스 아이디 또는 터치 아이디로 빠르게 결제가 완료됩니다.

지원 쇼핑몰과 앱은 계속 늘어나고 있으며, 결제 화면에서 애플 로고와 함께 "Apple Pay"가 표시되는 곳에서 사용 가능합니다.

## 교통카드 등록 방법

애플페이 교통카드 기능을 이용하면 실물 교통카드 없이도 지하철과 버스를 이용할 수 있습니다.

1. 지갑 앱에서 오른쪽 상단 **'+'** 버튼을 탭합니다
2. **"교통카드"**를 선택합니다
3. 사용할 교통카드 종류를 선택합니다 (티머니 등)
4. 충전 금액을 선택하고 등록된 카드로 결제합니다
5. 충전이 완료되면 지하철 개찰구나 버스 단말기에 아이폰 상단을 태그하여 이용합니다

교통카드는 특별한 배터리 절약 기능인 **"예비 전원(Power Reserve)"**을 지원합니다. 아이폰 배터리가 완전히 방전된 상태에서도 최대 5시간까지 교통카드 사용이 가능하므로 배터리 걱정 없이 대중교통을 이용할 수 있습니다.

## 기본 카드 변경 및 카드 관리

### 기본 결제 카드 변경

지갑 앱에서 기본 카드로 사용할 카드를 길게 누른 후 맨 앞으로 드래그합니다. 또는 설정 → 지갑 및 Apple Pay에서 "기본 카드"를 변경할 수 있습니다.

### 카드 삭제

지갑 앱에서 삭제할 카드를 탭한 후 화면 하단까지 스크롤하여 **"카드 제거"**를 탭합니다. 카드를 제거해도 실제 카드는 삭제되지 않으며, 애플페이에서만 제거됩니다.

## 자주 묻는 질문(Q&A)

**Q. 카드 등록이 안 됩니다. 어떻게 해야 하나요?**
A. 먼저 해당 카드사가 애플페이를 지원하는지 확인하세요. 지원하는 카드라면 카드 번호, 유효기간, CVV 정보를 정확히 입력했는지 다시 확인하고, 인터넷 연결 상태를 점검해 보세요. 그래도 안 된다면 해당 카드사 고객센터(카드 뒷면 번호)에 문의하세요.

**Q. 결제 시 "결제 거부" 또는 "인식 불가" 메시지가 나옵니다.**
A. 해당 매장이 NFC 비접촉 결제를 지원하지 않거나, 단말기 설정이 올바르지 않을 수 있습니다. 점원에게 "컨택리스 결제 가능한가요?" 또는 "NFC 결제 되나요?"라고 물어보세요. 또한 아이폰 케이스가 두꺼운 경우 NFC 인식이 방해받을 수 있으니 케이스를 벗고 시도해 보세요.

**Q. 애플페이는 안전한가요?**
A. 애플페이는 매우 높은 수준의 보안을 제공합니다. 실제 카드 번호 대신 가상의 기기 계정 번호(DAN, Device Account Number)를 사용하며, 매 결제마다 일회성 보안 코드를 생성합니다. 따라서 카드 번호가 유출될 위험이 없고, 일반 카드 결제보다 훨씬 안전합니다. 또한 페이스 아이디 또는 터치 아이디로만 결제가 승인되므로 타인이 사용하기 어렵습니다.

**Q. 아이폰을 분실했을 때 애플페이는 어떻게 되나요?**
A. Apple ID로 iCloud.com에 접속하거나 "나의 찾기" 앱을 통해 기기를 분실 모드로 설정하면 애플페이 결제가 즉시 중단됩니다. 기기를 초기화해도 애플페이 카드 정보는 자동으로 삭제됩니다.

**Q. 카드를 여러 장 등록할 수 있나요?**
A. 네, 기기당 최대 12장까지 등록할 수 있습니다. 지갑 앱에서 원하는 카드를 선택하여 결제할 수 있으며, 기본 카드를 설정해 두면 결제 시 해당 카드가 자동으로 선택됩니다.

**Q. 애플페이 결제 내역은 어디서 확인하나요?**
A. 지갑 앱에서 해당 카드를 탭하면 최근 결제 내역을 확인할 수 있습니다. 더 자세한 내역은 해당 카드사 앱이나 카드사 홈페이지에서 확인하세요.

## 마무리 정리

애플페이는 한 번 설정해 두면 매우 편리하게 사용할 수 있는 간편결제 수단입니다. 지갑 앱에서 카드를 등록하고, 페이스 아이디나 터치 아이디로 인증하면 빠르고 안전하게 결제할 수 있습니다.

오늘 안내해 드린 내용을 정리하면 다음과 같습니다:
- 지갑(Wallet) 앱에서 '+' 버튼으로 카드 추가
- 카드 카메라 스캔 또는 수동 입력 후 카드사 인증 완료
- 오프라인 결제: 측면 버튼 두 번 클릭 → 페이스 아이디 인증 → 단말기에 태그
- 교통카드도 지갑 앱에서 등록하여 편리하게 대중교통 이용 가능
- 분실 시 "나의 찾기"로 즉시 이용 정지 가능

애플페이와 함께 더 스마트하고 안전한 결제 생활을 즐기세요!
