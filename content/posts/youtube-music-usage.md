유튜브 뮤직(YouTube Music)은 구글이 운영하는 음악 스트리밍 서비스로, 유튜브의 방대한 음악 콘텐츠를 이용할 수 있는 앱입니다. 기존 유튜브처럼 뮤직비디오도 볼 수 있고, 음악만 따로 스트리밍하는 것도 가능합니다. 특히 안드로이드 스마트폰에는 기본 음악 앱으로 설치되어 있는 경우가 많아 한국에서도 사용자가 꾸준히 늘고 있습니다. 이 글에서는 유튜브 뮤직의 기본 사용법부터 실용적인 팁까지 모두 설명합니다.

## 유튜브 뮤직 앱 설치하기

### 안드로이드 (갤럭시 등)

1. **Play 스토어** 앱을 실행합니다.
2. 검색창에 **"유튜브 뮤직"** 또는 **"YouTube Music"** 을 입력합니다.
3. 검색 결과에서 **YouTube Music** 앱(개발사: Google LLC)을 선택합니다.
4. **설치** 버튼을 누릅니다.
5. 설치가 완료되면 **열기** 버튼을 눌러 실행합니다.

갤럭시 스마트폰은 기본 앱으로 유튜브 뮤직이 이미 설치되어 있는 경우가 많습니다. 앱 서랍에서 "YouTube Music"을 찾아보세요.

### 아이폰 (iPhone)

1. **App Store** 를 실행합니다.
2. 검색창에 **"YouTube Music"** 을 입력합니다.
3. **YouTube Music** 앱을 찾아 **받기** 버튼을 누릅니다.
4. Face ID 또는 비밀번호로 인증 후 설치합니다.

## 구글 계정으로 로그인하기

1. 유튜브 뮤직 앱을 실행합니다.
2. 시작 화면에서 **"로그인"** 버튼을 누릅니다.
3. 구글 계정(Gmail) 이메일 주소를 입력합니다.
4. 비밀번호를 입력하고 **다음** 을 누릅니다.
5. 로그인이 완료되면 메인 화면이 나타납니다.

> 구글 계정이 없다면 먼저 구글 홈페이지에서 무료로 계정을 만드세요. 계정 생성 시 Gmail 주소도 함께 발급됩니다.

로그인하지 않고도 일부 기능을 사용할 수 있지만, 재생목록 저장이나 개인화 추천 등의 기능은 로그인이 필요합니다.

## 음악 검색하고 재생하기

1. 앱 하단의 **검색(돋보기 아이콘)** 을 누릅니다.
2. 검색창에 노래 제목, 가수 이름, 또는 가사 일부를 입력합니다.
3. 검색 결과에서 **"노래"** 탭을 선택하면 음원만 필터링됩니다.
4. 원하는 노래를 누릅니다.
5. 노래가 바로 재생됩니다.

### 재생 화면 조작법

재생 중에는 화면 하단에 미니 플레이어가 표시됩니다. 미니 플레이어를 위로 스와이프하면 전체 재생 화면이 열립니다.

- **이전/다음 곡**: 재생 화면에서 화살표 버튼을 누릅니다.
- **셔플**: 섞기 아이콘(화살표 두 개가 교차하는 모양)을 눌러 랜덤 재생합니다.
- **반복 재생**: 반복 아이콘을 눌러 한 곡 반복 또는 전체 반복을 설정합니다.
- **좋아요**: 하트 아이콘을 눌러 즐겨찾기에 추가합니다.

> **팁**: 잘 모르는 노래의 가사 일부를 검색해도 찾을 수 있습니다. "사랑이 뭔지 알 것 같아" 처럼 기억나는 가사를 그대로 검색하면 됩니다.

## 플레이리스트 만들기

나만의 음악 목록을 만들어 보세요.

1. 원하는 노래를 검색해 찾습니다.
2. 노래 제목 오른쪽의 **점 세 개(⋯)** 를 누릅니다.
3. **"재생목록에 저장"** 을 선택합니다.
4. **"새 재생목록 만들기"** 를 선택합니다.
5. 재생목록 이름을 입력하고 (예: "드라이브 음악", "공부할 때") **만들기** 를 누릅니다.
6. 이후 노래를 추가할 때는 같은 방법으로 **기존 재생목록**을 선택하면 됩니다.

재생목록은 **보관함** 탭에서 확인하고 관리할 수 있습니다. 재생목록 이름과 설명을 수정하거나 순서를 변경하는 것도 가능합니다.

## 오프라인 저장하기 (유료 프리미엄 기능)

유튜브 뮤직 프리미엄 구독 시 인터넷 없이도 음악을 들을 수 있도록 다운로드할 수 있습니다.

1. 오프라인으로 저장할 노래나 앨범, 재생목록을 찾습니다.
2. 다운로드 아이콘(아래 화살표 모양)을 누릅니다.
3. 다운로드가 완료되면 아이콘 색이 바뀝니다.
4. 이후 인터넷이 없는 환경에서도 해당 음악을 재생할 수 있습니다.

Wi-Fi 환경에서 미리 다운로드해 두면 지하철이나 비행기 안에서도 데이터 없이 음악을 즐길 수 있습니다.

## 배경 재생 설정하기

유튜브 뮤직 무료 버전은 화면을 끄거나 다른 앱을 실행하면 음악이 멈춥니다. 배경 재생을 유지하려면 **유튜브 뮤직 프리미엄** 또는 **유튜브 프리미엄** 구독이 필요합니다.

프리미엄 구독 후에는 화면을 끈 상태에서도, 다른 앱을 사용하면서도 음악이 계속 재생됩니다. 잠금화면에서도 재생/일시정지, 다음 곡 넘기기 등 기본 조작이 가능합니다.

## 유튜브 뮤직 프리미엄 구독하기

프리미엄 기능을 이용하려면 구독이 필요합니다.

1. 앱 우측 상단의 **프로필 사진** 을 누릅니다.
2. **"유튜브 뮤직 프리미엄 시작하기"** 를 선택합니다.
3. 요금제를 선택합니다 (개인/가족/학생 요금제 중 선택).
4. 결제 수단을 등록합니다 (신용카드, 통신사 결제, 구글 페이 등).
5. **"구독 시작"** 을 누릅니다.

2026년 기준 유튜브 뮤직 프리미엄 가격은 개인 월 8,690원이며, 유튜브 프리미엄(영상+음악 통합) 묶음은 월 14,900원입니다. 처음 가입 시 한 달 무료 체험을 제공하는 경우가 있으니 확인해 보세요.

## 음악 취향 기반 추천 받기

유튜브 뮤직은 사용 기록을 분석해 음악을 추천해 줍니다.

1. 앱 하단 **홈** 탭에서 추천 음악을 확인합니다.
2. 처음 사용 시 좋아하는 아티스트를 선택하면 더 정확한 추천을 받을 수 있습니다.
3. 노래를 들을 때 **좋아요(엄지 위)** 또는 **싫어요(엄지 아래)** 를 표시하면 추천 알고리즘이 점점 개선됩니다.

오래 사용할수록 내 취향에 맞는 음악을 더 잘 추천해주니, 처음에는 다소 맞지 않더라도 계속 피드백을 주면서 사용해 보세요.

## 음질 설정 변경하기

1. 프로필 사진을 누른 후 **설정** 으로 들어갑니다.
2. **"오디오 품질"** 항목을 선택합니다.
3. Wi-Fi와 모바일 데이터 각각의 음질을 설정합니다.
   - **낮음 (48kbps)**: 데이터를 가장 적게 씁니다.
   - **보통 (128kbps)**: 일반적인 사용에 적합합니다.
   - **높음 (256kbps)**: 좋은 음질로 듣고 싶을 때 선택합니다.
   - **항상 높음**: 데이터를 많이 사용하지만 가장 좋은 음질을 제공합니다.

## 자주 묻는 질문 (Q&A)

**Q. 유튜브 뮤직과 멜론, 지니뮤직의 차이점은 무엇인가요?**
A. 멜론과 지니뮤직은 국내 스트리밍 서비스로 한국 음악(K-POP, 트로트 등) 최신곡을 빠르게 제공하며, 음원 차트와 국내 독점 콘텐츠가 강점입니다. 유튜브 뮤직은 국내외 모든 장르의 음악을 커버하며 뮤직비디오와 라이브 영상도 볼 수 있습니다. 해외 음악을 즐겨 듣거나 유튜브를 이미 사용 중이라면 유튜브 뮤직이 유리합니다.

**Q. 무료로 사용하면 광고가 많이 나오나요?**
A. 네, 무료 사용 시 음악 재생 중 광고가 표시되며, 배경 재생이 불가능합니다. 광고 없이 끊김 없이 듣고 싶다면 프리미엄 구독을 권장합니다.

**Q. 데이터를 아끼면서 음악을 들을 수 있나요?**
A. 설정 > 오디오 품질에서 음질을 '낮음' 또는 '보통'으로 설정하면 데이터 사용량을 줄일 수 있습니다. 프리미엄 사용자는 음악을 미리 Wi-Fi로 다운로드해 두면 이동 중에 데이터 없이 들을 수 있습니다.

**Q. 가족 요금제는 몇 명까지 쓸 수 있나요?**
A. 유튜브 뮤직 패밀리 요금제는 최대 6명(계정 소유자 포함)까지 사용할 수 있으며, 2026년 기준 월 17,990원입니다.

**Q. 구독을 해지하면 저장한 음악은 어떻게 되나요?**
A. 프리미엄 구독을 해지하면 오프라인 저장된 음악은 더 이상 재생되지 않습니다. 단, 재생목록이나 좋아요 표시한 음악 목록은 계정에 그대로 남아 있습니다.

**Q. 자동으로 다음 곡이 추천되는데 끌 수 있나요?**
A. 재생 화면에서 '자동재생' 아이콘을 탭하면 현재 재생목록이 끝난 후 자동 추천 재생을 끌 수 있습니다.

## 마무리 정리

유튜브 뮤직은 구글 계정 하나로 쉽게 시작할 수 있는 음악 스트리밍 서비스입니다. 무료로도 기본 기능을 사용할 수 있고, 프리미엄 구독 시 광고 없이 오프라인 저장과 배경 재생까지 가능합니다. 특히 한국과 해외 음악을 모두 즐기는 분들, 또는 유튜브 프리미엄을 이미 구독 중인 분들께 적극 추천합니다. 처음 사용한다면 좋아하는 가수를 검색해 노래를 들어보고, 마음에 드는 곡은 플레이리스트에 저장해 나만의 음악 컬렉션을 만들어 보세요.
