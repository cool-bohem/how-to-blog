## 삭제된 파일, 복구할 수 있을까요?

중요한 파일을 실수로 삭제했을 때의 당황스러움은 누구나 한 번쯤 경험해본 적이 있을 것입니다. 다행히 윈도우 11에는 삭제된 파일을 복구할 수 있는 여러 가지 방법이 내장되어 있습니다. 파일을 삭제한 직후라면 복구 성공률이 매우 높으니, 당황하지 말고 아래 방법들을 차례로 시도해보세요.

**중요**: 파일 삭제 후 새로운 파일을 저장하거나 프로그램을 설치하면 삭제된 파일이 저장되어 있던 공간이 덮어씌워져 복구 가능성이 낮아집니다. 파일을 삭제한 사실을 알아챘다면 최대한 빨리 복구를 시도하는 것이 좋습니다.

## 방법 1: 휴지통에서 복구하기

가장 간단하고 빠른 방법입니다. 일반적으로 Delete 키로 삭제한 파일은 즉시 사라지는 것이 아니라 휴지통에 임시 보관됩니다.

1. 바탕화면에서 **휴지통** 아이콘을 더블클릭합니다.
2. 복구할 파일을 찾습니다. 파일 이름이 기억나지 않는 경우 **삭제한 날짜** 기준으로 정렬해보세요. (상단 열 이름 클릭)
3. 복구할 파일을 **마우스 오른쪽 버튼**으로 클릭합니다.
4. 메뉴에서 **"복원"**을 선택합니다.
5. 파일이 원래 있던 위치로 자동 복원됩니다.

여러 파일을 한 번에 복원하려면 Ctrl 키를 누른 상태로 여러 파일을 선택한 후 우클릭 → 복원을 누릅니다.

**주의**: 다음의 경우 휴지통에 없을 수 있습니다.
- **Shift + Delete**로 삭제한 경우 (휴지통을 거치지 않는 영구 삭제)
- USB, 외장하드 등 외부 저장장치에서 삭제한 경우
- 휴지통을 이미 비운 경우
- 파일 크기가 매우 큰 경우 (휴지통 용량 초과 시 자동 삭제)

## 방법 2: 파일 히스토리(백업)에서 복구하기

파일 히스토리가 사전에 설정되어 있다면 이전 시점의 파일을 복구할 수 있습니다.

1. 복구하려는 파일이 있던 **폴더**를 엽니다. 파일이 삭제되었어도 폴더는 남아 있을 수 있습니다.
2. 폴더 내 빈 공간에서 **마우스 오른쪽 버튼**을 클릭합니다.
3. **"이전 버전 복원"**을 선택합니다.
4. 이전 버전 목록이 나타납니다. 파일이 있었던 날짜의 버전을 선택합니다.
5. 버전을 선택한 후:
   - **"열기"** 버튼: 해당 시점의 폴더 내용을 먼저 확인
   - **"복원"** 버튼: 전체 폴더를 해당 시점으로 복원 (주의: 이후에 추가된 파일이 사라질 수 있음)
6. 특정 파일만 복구하려면 버전을 열어 해당 파일을 직접 복사해 현재 위치로 붙여넣습니다.

**파일 히스토리 설정 방법** (나중에 대비해 미리 설정해두세요):
1. **시작 메뉴** → **설정** → **시스템** → **저장소**
2. **고급 저장소 설정** → **백업 옵션**
3. 또는 제어판에서 **파일 히스토리** 검색
4. 외장하드나 USB를 연결한 상태에서 **"켜기"** 버튼을 클릭합니다.

## 방법 3: 이전 버전으로 복구하기

시스템 복원 지점이나 백업이 있는 경우 개별 파일 또는 폴더의 이전 버전을 복구할 수 있습니다.

1. 복구하려는 파일이 있던 **폴더**로 이동합니다.
2. 해당 **폴더**를 **마우스 오른쪽 버튼**으로 클릭합니다.
3. **"속성"**을 선택합니다.
4. **"이전 버전"** 탭을 클릭합니다.
5. 복구 가능한 이전 버전 목록이 표시됩니다. (시스템 복원 지점 또는 백업이 있는 경우에만 나타납니다)
6. 원하는 날짜의 버전을 선택하고 **"열기"**로 먼저 내용을 확인합니다.
7. 복구할 파일이 있으면 직접 복사해 현재 폴더에 붙여넣거나 **"복원"** 버튼을 사용합니다.

## 방법 4: Windows 파일 복구 도구 사용하기

마이크로소프트에서 공식 제공하는 복구 도구입니다. 휴지통을 비운 경우나 Shift+Delete로 영구 삭제된 파일도 복구를 시도할 수 있습니다.

1. **시작 메뉴**에서 **"Microsoft Store"**를 엽니다.
2. 검색창에 **"Windows File Recovery"**를 입력해 설치합니다.
3. 설치 후 **시작 메뉴**에서 **"Windows File Recovery"**를 검색합니다.
4. **마우스 오른쪽 버튼** 클릭 → **"관리자 권한으로 실행"**을 선택합니다.
5. 검은색 명령 프롬프트 창이 열립니다.
6. 복구 명령어를 입력합니다. 예시:
   ```
   winfr C: D:\복구결과 /regular /n \Users\홍길동\Documents\중요파일.xlsx
   ```
   - `C:` : 복구할 드라이브
   - `D:\복구결과` : 복구된 파일을 저장할 경로 (원본과 다른 드라이브 권장)
   - `/regular` : 일반 모드
   - `/n` : 특정 파일 또는 폴더 지정
7. **Y**를 눌러 확인합니다.
8. 복구 작업이 완료되면 지정한 폴더에서 복구된 파일을 확인합니다.

**모드 선택 가이드**:
- `/regular`: 최근 삭제된 파일 복구에 적합 (NTFS 파티션)
- `/extensive`: 오래 전에 삭제됐거나 포맷된 드라이브에서 복구 시 적합 (시간이 더 오래 걸림)

**팁**: 파일 형식을 모를 경우 `/n *.docx` 처럼 확장자로 검색하거나, `/n \Users\홍길동\Desktop\` 처럼 폴더 전체를 지정할 수 있습니다.

## 방법 5: 시스템 복원을 통한 복구

시스템 전체를 이전 상태로 되돌리는 방법입니다. 파일뿐 아니라 시스템 설정과 프로그램 상태도 함께 복원됩니다.

1. **시작 버튼**을 클릭하고 **"시스템 복원"** 또는 **"복원 지점 만들기"**를 검색합니다.
2. **시스템 속성** 창이 열리면 **"시스템 보호"** 탭에서 **"시스템 복원"** 버튼을 클릭합니다.
3. 복원 마법사가 시작됩니다. **"다음"**을 클릭합니다.
4. 파일이 있었던 날짜 이전의 **복원 지점**을 선택합니다. "영향받는 프로그램 검색" 버튼으로 어떤 변경이 되돌려지는지 미리 확인할 수 있습니다.
5. **"다음"** → **"완료"**를 클릭하면 확인 창이 나타납니다.
6. **"예"**를 누르면 시스템이 재시작되고 복원이 진행됩니다.

**주의사항**:
- 시스템 복원은 개인 파일(문서, 사진 등)을 직접 삭제하지 않습니다.
- 다만 복원 지점 이후에 설치한 프로그램이나 드라이버는 삭제될 수 있습니다.
- 시스템 보호 기능이 꺼져 있으면 복원 지점이 없을 수 있습니다.

## 파일 삭제 예방을 위한 습관

복구보다는 예방이 훨씬 중요합니다. 아래 습관을 평소에 들여두면 파일 분실 걱정을 크게 줄일 수 있습니다.

1. **클라우드 자동 동기화**: OneDrive, 구글 드라이브 등에 중요 폴더를 자동 동기화 설정하면 클라우드에 항상 최신 버전이 보관됩니다.
2. **파일 히스토리 설정**: 외장하드를 연결하고 파일 히스토리를 활성화해 정기 자동 백업을 설정합니다.
3. **삭제 전 확인 습관**: 특히 Shift+Delete 사용 시 한 번 더 확인합니다.
4. **복원 지점 생성**: 중요한 작업(프로그램 설치, 시스템 변경 등) 전에 수동으로 복원 지점을 만들어두세요.
5. **중요 파일 이중 저장**: USB, 외장하드, 클라우드 등 두 군데 이상 보관하는 습관을 들이세요.

## 자주 묻는 질문 (Q&A)

**Q. 복구 후 파일이 열리지 않아요.**
A. 파일이 부분적으로만 복구된 경우 파일이 손상(corrupt)될 수 있습니다. 워드 파일이라면 워드의 "파일 복구" 기능을 시도해보거나, Recuva 같은 전문 복구 프로그램을 추가로 사용해볼 수 있습니다.

**Q. 회사 공유 드라이브의 파일을 삭제했어요.**
A. 회사 IT 담당자나 네트워크 관리자에게 즉시 문의하세요. 서버 측 백업이나 휴지통이 따로 있는 경우가 많아 복구가 가능할 수 있습니다.

**Q. 삭제한 지 한 달이 지났는데도 복구가 가능한가요?**
A. 해당 저장 공간에 새 파일이 덮어쓰여지지 않았다면 가능성이 있습니다. Windows File Recovery의 `/extensive` 모드나 Recuva(무료 복구 프로그램) 같은 전문 도구를 시도해보세요.

**Q. 외장하드에서 삭제한 파일도 복구할 수 있나요?**
A. 네, 가능합니다. 외장하드를 PC에 연결한 후 Windows File Recovery를 사용하거나, Recuva(무료)를 이용하면 됩니다. 외장하드 연결 후 새 파일을 쓰지 않는 것이 중요합니다.

**Q. SSD의 파일도 복구할 수 있나요?**
A. SSD는 TRIM 기능 때문에 HDD보다 복구가 훨씬 어렵습니다. 삭제 즉시 시도하는 것이 중요하며, 성공률은 HDD보다 낮습니다. 그래도 일부는 복구 가능하므로 시도해볼 가치가 있습니다.

**Q. Windows 파일 복구 명령어가 너무 어려워요. 더 쉬운 방법이 있나요?**
A. Recuva(무료), EaseUS Data Recovery(유료) 같은 그래픽 인터페이스 복구 프로그램을 사용하면 명령어 없이도 쉽게 파일을 복구할 수 있습니다. 복구 프로그램 자체는 삭제된 파일이 있던 드라이브가 아닌 다른 곳에 설치하는 것이 좋습니다.

## 정리

윈도우 11에서 파일을 삭제했을 때 당황하지 말고 다음 순서로 시도해보세요:

1. **휴지통 확인** (가장 먼저, 가장 간단)
2. **파일 히스토리 / 이전 버전 복구** (백업이 설정되어 있는 경우)
3. **Windows File Recovery** (영구 삭제된 경우, 마이크로소프트 공식 도구)
4. **시스템 복원** (파일이 삭제된 날짜를 알고 있는 경우)
5. **Recuva 등 전문 복구 프로그램** (위 방법으로 해결 안 될 경우)

파일 삭제 후 가능한 한 빨리 시도하는 것이 성공률을 높이는 가장 중요한 포인트입니다. 평소 OneDrive나 외장하드를 이용한 정기 백업 습관을 들이면 이런 상황 자체를 미리 예방할 수 있습니다.
