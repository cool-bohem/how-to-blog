## 마이크로소프트 계정이란?

마이크로소프트 계정은 윈도우 11, 엑셀·워드 등 Microsoft 365, 원드라이브(OneDrive) 클라우드 저장소, 마이크로소프트 스토어, Xbox 게임 서비스 등 마이크로소프트의 모든 서비스를 하나의 계정으로 이용할 수 있는 통합 계정입니다. 이메일 주소(Outlook, Hotmail, Live 등)를 아이디로 사용합니다.

2026년 현재 윈도우 11은 기본적으로 마이크로소프트 계정으로 로그인하도록 유도하지만, **로컬 계정**(인터넷 연결 없이 해당 PC에서만 사용하는 계정)으로도 사용할 수 있습니다. 이 글에서는 두 가지 방법을 모두 설명합니다.

---

## 마이크로소프트 계정 vs 로컬 계정 차이

| 항목 | 마이크로소프트 계정 | 로컬 계정 |
|------|------|------|
| 인터넷 연결 | 필요 (최초 로그인 시) | 불필요 |
| OneDrive 동기화 | 자동 지원 | 수동 설정 필요 |
| 앱 설정 동기화 | 여러 PC 간 자동 동기화 | 해당 PC에만 적용 |
| 비밀번호 분실 시 | 온라인으로 재설정 가능 | PC 초기화 필요할 수 있음 |
| 개인정보 | 마이크로소프트 서버에 일부 저장 | PC에만 저장 |

개인 정보 보호가 중요하거나 인터넷 없이 사용하는 PC라면 로컬 계정이 적합합니다. 반면 여러 대의 PC를 사용하거나 OneDrive·Office를 적극 활용한다면 마이크로소프트 계정이 유리합니다.

## 마이크로소프트 계정 새로 만들기

아직 마이크로소프트 계정이 없다면 먼저 계정을 만들어야 합니다.

### 1단계: 계정 생성 페이지 접속

1. 인터넷 브라우저(엣지, 크롬 등)를 열고 **account.microsoft.com** 에 접속합니다.
2. 화면 중앙의 **계정 만들기** 버튼을 클릭합니다.

### 2단계: 이메일 주소 입력

1. 기존에 사용하는 이메일 주소(Gmail, 네이버 메일 등)를 입력해 마이크로소프트 계정으로 등록할 수 있습니다.
2. 또는 **새 이메일 주소 받기**를 클릭해 @outlook.com 또는 @hotmail.com 주소를 새로 만들 수 있습니다.
3. 원하는 방법 선택 후 **다음**을 클릭합니다.

### 3단계: 비밀번호 설정

1. 사용할 비밀번호를 입력합니다. 최소 8자 이상, 대문자·소문자·숫자·특수문자를 조합하는 것을 권장합니다.
2. **다음**을 클릭합니다.

### 4단계: 이름 및 생년월일 입력

1. 이름(성, 이름)을 입력합니다.
2. 생년월일을 선택하고 **다음**을 클릭합니다.

### 5단계: 이메일 인증

1. 입력한 이메일 주소로 인증 코드가 전송됩니다.
2. 이메일을 열어 코드를 확인하고 입력한 뒤 **다음**을 클릭합니다.
3. 보안 문자(CAPTCHA) 인증을 완료하면 계정 생성이 완료됩니다.

## 윈도우 11에 마이크로소프트 계정 연결하기

현재 로컬 계정으로 윈도우 11을 사용 중이라면, 마이크로소프트 계정을 연결하면 OneDrive 동기화와 앱 설정 백업 등의 혜택을 받을 수 있습니다.

### 6단계: 설정 앱 열기

1. 화면 왼쪽 하단의 **시작 버튼(윈도우 로고)**을 클릭합니다.
2. **설정(톱니바퀴 아이콘)** 을 클릭합니다. 또는 키보드에서 **Windows 키 + I** 를 동시에 누릅니다.

### 7단계: 계정 설정으로 이동

1. 설정 창 왼쪽 메뉴에서 **계정**을 클릭합니다.
2. 오른쪽 화면에서 **사용자 정보**를 클릭합니다.
3. "Microsoft 계정으로 로그인" 또는 **Microsoft 계정으로 전환**이라는 링크를 클릭합니다.

### 8단계: 마이크로소프트 계정 로그인

1. 마이크로소프트 계정의 이메일 주소를 입력하고 **다음**을 클릭합니다.
2. 비밀번호를 입력하고 **로그인**을 클릭합니다.
3. 현재 로컬 계정의 비밀번호를 한 번 더 입력해 신원을 확인합니다.
4. **다음** → **로그인**을 클릭하면 마이크로소프트 계정 연결이 완료됩니다.

연결 완료 후 PC를 재시작하면 마이크로소프트 계정으로 로그인하게 됩니다. OneDrive가 자동으로 활성화되며, 마이크로소프트 스토어에서 앱 구매 내역도 공유됩니다.

## 로컬 계정으로 전환하기

반대로 마이크로소프트 계정에서 로컬 계정으로 전환하고 싶다면 다음 단계를 따르세요.

### 9단계: 설정 → 계정 → 사용자 정보

1. **Windows 키 + I** 를 눌러 설정을 엽니다.
2. **계정** → **사용자 정보**로 이동합니다.

### 10단계: 로컬 계정으로 전환

1. "로컬 계정으로 로그인"이라는 링크를 클릭합니다. (마이크로소프트 계정으로 현재 로그인 중인 경우에만 표시됩니다.)
2. 팝업 창에서 **다음**을 클릭합니다.
3. 마이크로소프트 계정 비밀번호를 입력해 본인 확인을 합니다.
4. 새로 사용할 **로컬 계정 사용자 이름**과 **비밀번호**를 설정합니다. 비밀번호 없이 사용하려면 비밀번호 입력란을 비워두세요.
5. **다음** → **로그아웃 및 마치기**를 클릭합니다.
6. PC가 자동으로 로그아웃되고, 로컬 계정으로 다시 로그인합니다.

> **참고**: 로컬 계정 전환 후에도 이전에 설치한 앱이나 파일은 그대로 유지됩니다. 다만 OneDrive 자동 동기화는 중단됩니다.

## PIN 및 생체 인증 설정

마이크로소프트 계정이든 로컬 계정이든, 매번 긴 비밀번호를 입력하는 대신 **PIN**이나 **지문 인증**으로 편리하게 로그인할 수 있습니다.

### PIN 설정

1. **설정** → **계정** → **로그인 옵션**으로 이동합니다.
2. **PIN(Windows Hello)** 항목을 클릭한 뒤 **설정**을 누릅니다.
3. 마이크로소프트 계정 비밀번호로 인증 후, 4~6자리 숫자 PIN을 설정합니다.
4. 이후 PC 로그인 시 PIN만 입력하면 됩니다. PIN은 인터넷 연결 없이도 동작합니다.

### 지문 인식 설정 (지원 PC에 한함)

1. **설정** → **계정** → **로그인 옵션**으로 이동합니다.
2. **지문 인식(Windows Hello)** 항목을 클릭하고 **설정**을 누릅니다.
3. 화면의 안내에 따라 지문 센서에 손가락을 여러 번 올려 등록합니다.
4. 등록 완료 후 지문만으로 PC를 잠금 해제할 수 있습니다.

## 자주 묻는 질문 (Q&A)

**Q. 마이크로소프트 계정 비밀번호를 잊어버렸어요. 어떻게 하나요?**
A. 인터넷 브라우저에서 **account.microsoft.com** 에 접속해 **로그인** → **비밀번호를 잊으셨나요?** 를 클릭합니다. 등록된 이메일 또는 전화번호로 인증 코드를 받아 비밀번호를 재설정할 수 있습니다. 이것이 마이크로소프트 계정의 큰 장점 중 하나입니다.

**Q. 마이크로소프트 계정 없이 윈도우 11을 초기 설정(초기화 후)할 수 있나요?**
A. 가능합니다. 초기 설정 화면에서 인터넷 연결 단계를 건너뛰거나, 이메일 입력 화면에서 **로그인 옵션** → **도메인에 가입** 또는 **오프라인 계정** 을 선택하면 로컬 계정으로 설정할 수 있습니다. 다만 윈도우 11 버전에 따라 화면 구성이 다를 수 있습니다.

**Q. 가족이 함께 쓰는 PC에서 각자 계정을 만들 수 있나요?**
A. 네. **설정** → **계정** → **가족 및 다른 사용자** 에서 다른 사용자 계정을 추가할 수 있습니다. 자녀 보호 기능이 필요한 경우 **가족 구성원 추가**로 자녀 계정을 만들고 사용 시간과 콘텐츠를 제한할 수 있습니다.

**Q. PIN을 설정하면 마이크로소프트 계정 비밀번호는 필요 없나요?**
A. PC 로그인 시에는 PIN만 사용해도 됩니다. 단, 특정 보안 관련 설정을 변경할 때나 앱 설치 등 관리자 권한이 필요한 경우 마이크로소프트 계정 비밀번호 입력이 필요할 수 있습니다.

**Q. 윈도우 11에서 마이크로소프트 계정으로 로그인하면 개인 파일이 자동으로 마이크로소프트 서버에 올라가나요?**
A. OneDrive 동기화가 활성화되어 있으면 지정한 폴더(기본값: 문서, 사진, 바탕화면)의 파일이 클라우드에 업로드됩니다. OneDrive 동기화를 원하지 않으면 시스템 트레이의 OneDrive 아이콘을 클릭해 동기화를 중지하거나 비활성화할 수 있습니다.

## 정리

윈도우 11에서의 계정 설정을 요약하면 다음과 같습니다.

- **마이크로소프트 계정**: OneDrive 동기화, 여러 PC 설정 공유, 비밀번호 온라인 재설정 가능. 적극적으로 PC를 활용하는 사용자에게 추천
- **로컬 계정**: 인터넷 없이 사용 가능, 개인정보 서버 전송 최소화. 단순 작업용 PC나 개인정보 보호가 중요한 경우 추천
- **PIN 설정은 필수**: 어느 계정이든 매번 긴 비밀번호 대신 PIN이나 지문 인증을 설정하면 훨씬 편리합니다.

계정 전환은 언제든지 설정에서 할 수 있으므로, 먼저 마이크로소프트 계정으로 사용해 보고 불편하면 로컬 계정으로 전환해도 됩니다.
