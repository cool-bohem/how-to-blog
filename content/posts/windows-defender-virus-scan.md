## 윈도우 디펜더란?

**Windows Defender(Microsoft Defender 바이러스 백신)**는 윈도우 10·11에 기본 내장된 무료 보안 프로그램입니다. 별도의 백신 소프트웨어를 설치하지 않아도 바이러스, 악성코드, 랜섬웨어, 스파이웨어 등 다양한 위협으로부터 PC를 보호합니다. 과거에는 유료 백신보다 성능이 떨어진다는 평가가 있었지만, 2026년 현재 마이크로소프트의 꾸준한 업데이트로 업계 최고 수준의 탐지율을 자랑합니다.

이 글에서는 윈도우 디펜더를 사용해 수동으로 바이러스 검사를 실행하는 방법, 실시간 보호 설정 확인, 예외 폴더 등록, 바이러스 격리 및 삭제 방법을 순서대로 알려드립니다.

---

## 윈도우 디펜더 열기

### 방법 1: 시작 메뉴에서 열기

1. 화면 왼쪽 하단 **시작 버튼(윈도우 로고)**을 클릭합니다.
2. 검색창에 **"Windows 보안"** 을 입력합니다.
3. 검색 결과에서 **"Windows 보안"** 앱을 클릭합니다.

### 방법 2: 작업 표시줄에서 열기

1. 화면 오른쪽 하단 **알림 영역(시스템 트레이)**을 클릭합니다.
2. 방패 모양 아이콘 **"Windows 보안"**을 더블클릭합니다.

### 방법 3: 설정 앱에서 열기

1. **윈도우 키 + I**를 눌러 설정을 엽니다.
2. **개인 정보 보호 및 보안 → Windows 보안**을 클릭합니다.
3. **"Windows 보안 열기"** 버튼을 클릭합니다.

---

## 바이러스 및 위협 보호 상태 확인하기

Windows 보안 앱이 열리면 대시보드에서 현재 PC의 보안 상태를 한눈에 볼 수 있습니다.

1. 왼쪽 메뉴에서 **"바이러스 및 위협 방지"**를 클릭합니다.
2. 화면 상단에 **"현재 위협 없음"** 이라고 표시되면 정상입니다.
3. 마지막 검사 날짜와 시간, 검사한 파일 수도 확인할 수 있습니다.
4. **보안 인텔리전스 버전** 항목에서 바이러스 정의 파일이 최신인지 확인합니다. 오래됐다면 **"업데이트 확인"**을 눌러 최신 버전으로 갱신합니다.

> **중요:** 바이러스 정의(시그니처) 파일이 최신이어야 새로운 악성코드도 탐지할 수 있습니다. 가능하면 자동 업데이트를 켜두세요.

## 수동 바이러스 검사 실행하기

### 빠른 검사 (Quick Scan)

가장 흔히 감염되는 시스템 폴더와 실행 중인 프로세스를 빠르게 검사합니다. 소요 시간은 보통 **5~10분**입니다.

1. **"바이러스 및 위협 방지"** 화면에서 **"빠른 검사"** 버튼을 클릭합니다.
2. 검사가 시작되면 진행률 표시줄과 함께 현재 검사 중인 파일 경로가 표시됩니다.
3. 검사 완료 후 결과가 나타납니다. 위협이 발견되지 않으면 "현재 위협 없음"이라고 표시됩니다.

### 전체 검사 (Full Scan)

PC에 저장된 모든 파일과 프로그램을 검사합니다. 시간이 오래 걸리지만(보통 **30분~수 시간**) 가장 철저하게 검사할 수 있습니다. 처음으로 검사하거나 의심스러운 상황일 때 권장합니다.

1. **"빠른 검사"** 버튼 아래 **"검사 옵션"** 링크를 클릭합니다.
2. **"전체 검사"**를 선택하고 **"지금 검사"** 버튼을 클릭합니다.
3. 검사 중에도 PC를 계속 사용할 수 있지만, 검사 속도가 느려질 수 있으므로 자리를 비울 때 실행하는 것이 좋습니다.

### 사용자 지정 검사 (Custom Scan)

특정 폴더나 드라이브만 검사하고 싶을 때 사용합니다. 예를 들어 USB 드라이브나 다운로드 폴더만 검사할 수 있습니다.

1. **"검사 옵션"** → **"사용자 지정 검사"** 선택 후 **"지금 검사"** 클릭합니다.
2. 팝업 창에서 검사할 폴더나 드라이브를 선택합니다.
3. **"폴더 선택"**을 클릭하면 해당 위치만 빠르게 검사합니다.

### 오프라인 검사 (Microsoft Defender Offline Scan)

일반 검사로 제거되지 않는 악성코드는 **오프라인 검사**를 사용합니다. PC를 재부팅해 윈도우가 시작되기 전 단계에서 검사하므로, 악성코드가 활성화되기 전에 탐지·제거할 수 있습니다.

1. **"검사 옵션"** → **"Microsoft Defender 오프라인 검사"** 선택
2. **"지금 검사"** 클릭
3. 경고 메시지가 나타나면 **"검사"** 버튼 클릭 — PC가 **즉시 재부팅**됩니다.
4. 재부팅 후 자동으로 검사가 진행되고, 완료되면 다시 윈도우로 부팅됩니다.

## 실시간 보호 설정 확인하기

실시간 보호는 PC에서 파일을 열거나 다운로드할 때 즉시 검사하는 기능입니다. 반드시 켜져 있어야 합니다.

1. **"바이러스 및 위협 방지"** → **"바이러스 및 위협 방지 설정"** 아래 **"설정 관리"** 클릭
2. 다음 항목들이 **켜짐** 상태인지 확인합니다.
   - **실시간 보호**: 악성코드를 실시간으로 탐지
   - **클라우드 제공 보호**: 마이크로소프트 클라우드를 통해 최신 위협 탐지
   - **자동 샘플 제출**: 의심 파일을 마이크로소프트에 자동 보고
   - **변조 방지**: 외부 프로그램이 Windows 보안 설정을 무단 변경하지 못하도록 차단

> **주의:** 실시간 보호가 꺼져 있다면 즉시 켜주세요. 악성코드가 이미 실시간 보호를 강제로 끈 경우일 수도 있으니 오프라인 검사를 실행하는 것이 좋습니다.

---

## 위협 발견 시 처리 방법

검사 후 악성코드가 발견되면 다음과 같이 처리합니다.

1. 검사 결과 화면에서 **"위협 보기"** 또는 **"조치 시작"** 버튼을 클릭합니다.
2. 발견된 파일 목록이 표시됩니다. 각 위협 항목을 클릭하면 파일 경로와 위협 종류를 확인할 수 있습니다.
3. 처리 방법을 선택합니다.
   - **제거**: 악성 파일을 완전히 삭제합니다. 가장 일반적인 방법입니다.
   - **격리**: 파일을 격리 보관소로 옮겨 실행되지 않게 합니다. 나중에 확인 후 복원하거나 영구 삭제할 수 있습니다.
   - **허용**: 탐지된 파일이 실제로 필요한 파일임을 확인한 경우에만 선택하세요.
4. **"조치 시작"** 버튼을 클릭해 선택한 처리를 적용합니다.

## 예외(제외) 폴더 등록하기

특정 프로그램이나 폴더를 디펜더가 반복적으로 위협으로 오탐(False Positive)하는 경우, 해당 경로를 예외로 등록할 수 있습니다. 단, 신뢰할 수 있는 파일과 폴더만 예외로 등록해야 합니다.

1. **"바이러스 및 위협 방지 설정"** → **"제외 항목"** 섹션에서 **"제외 항목 추가 또는 제거"** 클릭
2. **"제외 항목 추가"** → **"폴더"** 선택
3. 예외로 등록할 폴더를 선택하고 **"폴더 선택"**을 클릭합니다.
4. 이후 해당 폴더 내 파일은 실시간 보호 검사에서 제외됩니다.

## 자주 묻는 질문 (Q&A)

**Q. 윈도우 디펜더 말고 다른 백신을 설치해도 되나요?**
A. 가능합니다. 다른 유료·무료 백신(알약, V3, 아바스트 등)을 설치하면 Windows Defender의 실시간 보호는 자동으로 비활성화됩니다. 두 가지를 동시에 켜두면 충돌이 생길 수 있으니 하나만 사용하는 것이 좋습니다.

**Q. 바이러스 검사 중 PC 속도가 너무 느려져요.**
A. 전체 검사는 CPU와 디스크 자원을 많이 사용합니다. Windows 보안 → 바이러스 및 위협 방지 설정에서 "낮은 우선순위로 검사 실행"을 켜거나, PC를 사용하지 않는 밤에 예약 검사를 실행하는 것을 추천합니다.

**Q. 검사 예약은 어떻게 설정하나요?**
A. 작업 스케줄러(Task Scheduler)에서 설정할 수 있습니다. 시작 메뉴 검색창에 "작업 스케줄러"를 입력해 실행한 후, "작업 스케줄러 라이브러리 → Microsoft → Windows → Windows Defender"에서 "Windows Defender Scheduled Scan"을 더블클릭해 원하는 주기로 설정하세요.

**Q. 격리된 파일은 어떻게 복원하나요?**
A. Windows 보안 → 바이러스 및 위협 방지 → "보호 기록" → 해당 항목 클릭 → "복원" 버튼으로 복구할 수 있습니다.

**Q. 디펜더가 계속 꺼진다고 알림이 와요.**
A. 일부 악성코드는 보안 프로그램을 무력화합니다. 오프라인 검사를 실행하거나, 안전 모드에서 검사해 보세요. 그래도 해결되지 않으면 윈도우 초기화를 고려해야 할 수 있습니다.

## 정리 요약

| 검사 종류 | 소요 시간 | 권장 상황 |
|---|---|---|
| 빠른 검사 | 5~10분 | 정기 점검, 의심스러울 때 |
| 전체 검사 | 30분~수 시간 | 처음 검사, 감염 의심 시 |
| 사용자 지정 검사 | 선택 영역에 따라 다름 | USB, 다운로드 폴더 등 특정 위치 |
| 오프라인 검사 | 15~20분 (재부팅 포함) | 일반 검사로 해결 안 될 때 |

윈도우 디펜더는 무료이면서도 강력한 보안 도구입니다. 별도의 백신을 구매하지 않아도 실시간 보호와 정기 검사만 잘 유지하면 대부분의 위협으로부터 PC를 안전하게 지킬 수 있습니다. 최소 한 달에 한 번은 전체 검사를 실행하는 습관을 들여보세요.
