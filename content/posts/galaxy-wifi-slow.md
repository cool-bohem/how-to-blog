## 갤럭시 와이파이가 느린 이유

와이파이 속도가 느리거나 연결이 불안정한 경우 대부분 **설정 문제**로 간단히 해결할 수 있어요.

## 방법 1. 와이파이 껐다 켜기

1. 화면 위에서 아래로 쓸어내려 빠른 설정 패널 열기
2. **와이파이 아이콘** 을 눌러 끄기
3. 5초 후 다시 켜기

## 방법 2. 네트워크 설정 초기화

1. **설정 → 일반 관리 → 초기화**
2. **네트워크 설정 초기화** 선택
3. 확인 후 재시작 — 저장된 와이파이 비밀번호가 삭제되니 미리 메모해두세요

## 방법 3. 와이파이 주파수 변경 (5GHz → 2.4GHz)

- **5GHz**: 속도 빠르지만 거리에 약함
- **2.4GHz**: 속도는 느리지만 멀리서도 잘 터짐

공유기와 거리가 멀다면 **2.4GHz** 네트워크에 연결해보세요.

## 방법 4. 절전 모드 해제

절전 모드에서는 와이파이 속도가 제한될 수 있어요.

1. **설정 → 배터리 및 디바이스 케어 → 배터리**
2. **전원 모드** 를 **최적화** 로 변경

## 방법 5. DNS 변경

1. **설정 → 연결 → 와이파이**
2. 연결된 네트워크 옆 **톱니바퀴 아이콘** 클릭
3. **IP 설정 → 고정** 선택
4. DNS 1: `8.8.8.8` / DNS 2: `8.8.4.4` 입력 후 저장

구글 DNS로 변경하면 속도가 개선되는 경우가 많아요.
