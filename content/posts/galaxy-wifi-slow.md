## 갤럭시 와이파이가 느린 이유

같은 와이파이를 쓰는데 노트북이나 다른 기기는 빠른데 갤럭시만 느리다면, 대부분 **기기 설정 문제**예요. 공유기 문제라면 모든 기기가 느려지지만, 스마트폰만 느리다면 아래 방법들로 대부분 해결할 수 있어요.

갤럭시 와이파이가 느려지는 주요 원인:
- **캐시된 네트워크 설정이 충돌**하는 경우
- **절전 모드**에서 와이파이 속도가 자동으로 제한되는 경우
- **5GHz/2.4GHz 대역** 선택이 현재 환경에 맞지 않는 경우
- **DNS 서버**가 느린 경우

## 방법 1. 와이파이 껐다 켜기

가장 간단하면서 의외로 효과가 있는 방법이에요. 연결 캐시가 초기화되면서 속도가 회복되는 경우가 많아요.

1. 화면 위에서 아래로 쓸어내려 빠른 설정 패널 열기
2. **와이파이 아이콘** 을 눌러 끄기
3. 5초 후 다시 켜기

**팁**: 껐다 켜도 안 된다면 스마트폰 자체를 재시작해보세요. 재시작만으로 해결되는 경우도 많아요.

## 방법 2. 네트워크 설정 초기화

저장된 와이파이 설정에 오류가 생긴 경우 초기화로 해결할 수 있어요.

1. **설정 → 일반 관리 → 초기화**
2. **네트워크 설정 초기화** 선택
3. 확인 후 재시작

**주의사항**: 저장된 모든 와이파이 비밀번호가 삭제돼요. 집 와이파이 비밀번호를 미리 메모해두세요. 블루투스 페어링 기기도 재등록이 필요해요.

## 방법 3. 와이파이 주파수 변경 (5GHz ↔ 2.4GHz)

와이파이에는 두 가지 주파수 대역이 있어요.

| | 5GHz | 2.4GHz |
|---|---|---|
| 속도 | 빠름 | 보통 |
| 거리 | 약함 (10~15m) | 강함 (30m 이상) |
| 장애물 | 취약 | 강함 |

공유기와 가까이 있다면 5GHz, 방이 여러 개이거나 공유기에서 멀리 있다면 2.4GHz가 유리해요.

공유기에서 거리가 멀다면 **2.4GHz** 네트워크 이름을 찾아서 연결해보세요.

## 방법 4. 절전 모드 해제

갤럭시의 절전 모드는 배터리를 아끼기 위해 와이파이 속도를 제한할 수 있어요.

1. **설정 → 배터리 및 디바이스 케어 → 배터리**
2. **전원 모드** 를 **최적화** 로 변경

**팁**: 절전 모드가 아닌데도 느리다면 **와이파이 파워 절약 모드**가 따로 켜져 있을 수 있어요. **설정 → 연결 → 와이파이 → 우측 상단 점 세 개 → 고급**에서 **와이파이 파워 절약 모드**를 꺼보세요.

## 방법 5. DNS 변경 (구글 DNS)

기본 DNS 서버가 느린 경우, 구글 DNS로 바꾸면 웹페이지 로딩 속도가 빨라질 수 있어요.

1. **설정 → 연결 → 와이파이**
2. 연결된 네트워크 옆 **톱니바퀴 아이콘** 클릭
3. **IP 설정 → 고정** 선택
4. DNS 1: `8.8.8.8` / DNS 2: `8.8.4.4` 입력 후 저장

**팁**: 구글 DNS 외에도 Cloudflare DNS(`1.1.1.1`)도 속도가 빠르기로 유명해요.

## 위 방법으로 해결 안 될 때

스마트폰 설정 문제가 아닌 경우도 있어요.

- **공유기 재시작**: 공유기 전원을 뽑았다가 30초 후 다시 연결해보세요
- **공유기 위치 변경**: 공유기와 스마트폰 사이에 전자레인지, 무선 전화기 등이 있으면 간섭이 생길 수 있어요
- **통신사 문의**: 인터넷 자체 속도 문제라면 통신사 고객센터(KT 100, SKT 106, LGU+ 101)에 문의하세요

## 자주 묻는 질문

**Q. 와이파이 속도를 직접 측정하려면?**
네이버나 구글에서 '와이파이 속도 측정'을 검색하거나, Speedtest 앱을 사용해보세요.

**Q. 5GHz와 2.4GHz를 어떻게 구분하나요?**
공유기 설정에서 두 대역의 이름을 다르게 설정해야 해요. 보통 '이름_5G'처럼 구분해두는 경우가 많아요.

**Q. 갤럭시가 유독 와이파이 신호를 잘 못 잡는 것 같아요.**
기기 케이스가 두껍거나 금속 재질이면 신호를 약하게 할 수 있어요. 케이스를 벗고 테스트해보세요.
