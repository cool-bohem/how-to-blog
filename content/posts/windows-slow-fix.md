## 컴퓨터가 느려지는 이유

처음 샀을 때는 빠르던 컴퓨터가 시간이 지나면서 느려지는 이유는 크게 세 가지예요.

- **시작 프로그램 누적**: 프로그램을 설치할수록 켤 때 자동으로 실행되는 것들이 쌓여요
- **디스크 용량 부족**: C드라이브 여유 공간이 20% 이하로 줄면 속도가 급격히 느려져요
- **메모리(RAM) 부족**: 많은 프로그램을 동시에 실행하면 메모리가 부족해져요
- **악성 프로그램**: 모르게 설치된 광고/악성 프로그램이 백그라운드에서 실행되는 경우도 있어요

재설치 없이 아래 방법들로 대부분 해결할 수 있어요.

## 방법 1. 시작 프로그램 줄이기 (효과 최고)

컴퓨터를 켤 때 자동으로 실행되는 프로그램들이 많을수록 부팅이 느리고 전반적으로 무거워져요.

1. **Ctrl + Shift + Esc** 로 작업 관리자 열기
2. **시작 프로그램** 탭 클릭
3. 목록에서 불필요한 프로그램 선택
4. 오른쪽 클릭 → **사용 안 함** 클릭

**끄면 좋은 것들**: 카카오톡, 웨일, 각종 게임 런처, 프린터 드라이버 알림 앱

**끄면 안 되는 것들**: 백신 프로그램, 윈도우 보안 관련 항목

**팁**: 시작 프로그램 탭에서 **CPU 영향**, **디스크 영향** 항목이 "높음"으로 표시된 것부터 확인해보세요.

## 방법 2. 디스크 정리

불필요한 임시 파일, 윈도우 업데이트 잔여 파일 등을 삭제해서 공간을 확보해요.

1. **내 PC** 에서 C드라이브 우클릭 → **속성**
2. **디스크 정리** 클릭
3. 정리할 항목 모두 체크 후 **확인**
4. 추가로 **시스템 파일 정리** 클릭 → 윈도우 업데이트 파일 삭제 가능

**팁**: 시스템 파일 정리까지 하면 수 GB까지 확보되는 경우도 있어요. 특히 윈도우 업데이트 후에 하면 효과가 커요.

## 방법 3. 전원 옵션 변경

윈도우 기본 설정이 절전 모드인 경우 성능이 제한될 수 있어요.

1. 작업표시줄 검색창에 **전원 옵션** 입력
2. **전원 및 절전 설정** 클릭
3. **추가 전원 설정** 클릭
4. **고성능** 선택

**주의**: 노트북의 경우 고성능 모드는 배터리 소모가 빨라요. 충전기를 꽂은 상태에서만 사용하세요.

## 방법 4. 시각 효과 끄기

애니메이션, 그림자 등 시각 효과는 오래된 PC에서 성능을 잡아먹어요.

1. 작업표시줄 검색창에 **시스템의 고급 설정 보기** 입력
2. **시스템 속성** 창에서 **성능** 섹션의 **설정** 클릭
3. **최적 성능으로 조정** 선택
4. **적용** 후 확인

**팁**: 전부 끄면 화면이 좀 밋밋해지지만 속도가 눈에 띄게 빨라져요. 일부만 끄고 싶다면 **사용자 지정** 에서 선택할 수 있어요.

## 방법 5. 악성 프로그램 검사

모르는 사이에 설치된 프로그램이 속도를 잡아먹는 경우도 많아요.

1. **윈도우 보안** 앱 열기 (작업표시줄 검색)
2. **바이러스 및 위협 방지** 클릭
3. **빠른 검사** 또는 **전체 검사** 실행
4. 발견된 위협 제거

**추가 팁**: 제어판 → **프로그램 및 기능**에서 설치된 프로그램 목록을 확인하고, 설치한 기억이 없는 프로그램은 삭제하세요.

## 방법 6. 재시작 습관 들이기

컴퓨터를 며칠씩 계속 켜두면 메모리에 캐시가 쌓여서 점점 느려져요. 하루에 한 번 또는 이틀에 한 번 재시작하는 것만으로도 체감 속도가 달라져요.

**절전 모드 대신 종료**: 퇴근 시 절전 모드가 아닌 완전 종료를 권장해요.

## 자주 묻는 질문

**Q. 이 방법들을 다 해봤는데도 느려요.**
RAM 용량이 부족한 경우일 수 있어요. 작업 관리자 → 성능 탭에서 메모리 사용률을 확인해보세요. 항상 70% 이상이라면 RAM 증설을 고려해야 해요.

**Q. SSD로 교체하면 얼마나 빨라지나요?**
하드디스크(HDD)를 SSD로 교체하면 부팅 속도가 2~5배, 프로그램 실행 속도가 2~3배 빨라지는 경우가 많아요. 가장 효과적인 업그레이드 방법 중 하나예요.

**Q. 윈도우를 재설치하면 무조건 빨라지나요?**
재설치하면 초기 상태로 돌아가기 때문에 빨라지는 것은 맞아요. 하지만 백업 없이 하면 데이터를 잃을 수 있고, 프로그램도 다시 설치해야 해요. 위 방법들을 먼저 시도해보고 그래도 안 되면 재설치를 고려하세요.
