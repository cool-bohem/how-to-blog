## 컴퓨터가 느려지는 이유

시간이 지날수록 불필요한 프로그램, 파일, 시작 프로그램이 쌓여 속도가 느려져요. 아래 순서대로 해보세요.

## 방법 1. 시작 프로그램 줄이기

1. **Ctrl + Shift + Esc** 로 작업 관리자 열기
2. **시작 프로그램** 탭 클릭
3. 불필요한 프로그램 선택 후 **사용 안 함** 클릭
4. 카카오톡, 웨일, 각종 업데이터 등은 꺼도 돼요

## 방법 2. 디스크 정리

1. **내 PC** 에서 C드라이브 우클릭 → **속성**
2. **디스크 정리** 클릭
3. 모두 체크 후 **확인**
4. **시스템 파일 정리** 도 함께 실행하면 더 많이 확보돼요

## 방법 3. 전원 옵션 변경

1. 작업표시줄 검색에 **전원 옵션** 입력
2. **고성능** 선택

## 방법 4. 시각 효과 끄기

1. 검색창에 **시스템 고급 설정** 입력
2. **성능 → 설정** 클릭
3. **최적 성능으로 조정** 선택

## 방법 5. 재시작 습관 들이기

컴퓨터를 며칠씩 켜두면 메모리가 쌓여요. 하루에 한 번 재시작하는 것만으로도 속도가 달라져요.
