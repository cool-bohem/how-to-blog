## 스팸 전화, 얼마나 위험할까요?

하루에도 몇 통씩 오는 광고 전화, 대출 권유 전화, 그리고 최근 급증한 보이스피싱 전화까지. 스팸 전화는 단순히 귀찮은 것을 넘어 금전적 피해로 이어질 수도 있어요.

**스팸 전화의 유형**:
- 대출, 보험, 주식 등 금융 광고 전화
- 인터넷 서비스 해지 및 재가입 유도
- 택배/금융기관 사칭 보이스피싱
- 국제 전화를 이용한 사기 전화 (+1, +44 등)

아이폰과 갤럭시 모두 기본 기능 + 추천 앱을 활용하면 대부분의 스팸을 차단할 수 있어요.

## 갤럭시에서 스팸 차단하기

**특정 번호 차단**

이미 온 스팸 전화 번호를 바로 차단할 수 있어요.

1. **전화 앱** 열기 → **최근 통화** 탭
2. 차단할 번호 클릭
3. 오른쪽 상단 **점 세 개(⋮)** 클릭
4. **번호 차단** 선택

또는 전화가 오는 중에도 차단 가능해요:
- 통화 화면에서 **차단** 버튼 클릭

**자동 스팸 차단 켜기**

갤럭시에는 알려진 스팸 번호를 자동으로 필터링하는 기능이 있어요.

1. 전화 앱 열기 → 오른쪽 상단 **점 세 개(⋮)** 클릭
2. **설정** → **번호 차단**
3. **스팸 차단** 토글 켜기
4. **스팸 방어** 수준 선택 (낮음/중간/높음)

**차단 번호 목록 관리**

1. 전화 앱 → 설정 → **번호 차단**
2. **차단된 번호 목록** 에서 확인 및 해제 가능

## 아이폰에서 스팸 차단하기

**특정 번호 차단**

1. **전화 앱** → **최근 통화** 탭
2. 차단할 번호 옆 **정보(i) 아이콘** 클릭
3. 아래로 내려서 **이 발신자 차단** 클릭
4. **연락처 차단** 확인

**알 수 없는 발신자 무음 처리 (강력 추천)**

저장되지 않은 번호에서 오는 전화를 모두 무음으로 처리하는 기능이에요.

1. **설정 → 전화**
2. **알 수 없는 발신자 무음 처리** 켜기
3. 저장되지 않은 번호에서 오는 전화는 무음으로 처리 + 부재중 통화 목록에 기록됨

**주의**: 중요한 번호인데 연락처에 없는 경우 놓칠 수 있어요. 필요하다면 부재중 통화 목록을 주기적으로 확인하세요.

## 후후 앱 활용 (갤럭시/아이폰 공통, 강력 추천)

후후는 통화 연구소에서 만든 스팸 전화 차단 앱이에요. 수백만 건의 스팸 데이터베이스를 기반으로 전화가 오기 전에 스팸 여부를 알려줘요.

**앱 설치 및 설정**
1. 앱스토어/플레이스토어에서 **후후** 검색 후 설치
2. 앱 실행 후 권한 허용 (통화 정보 접근)
3. 자동으로 스팸 감지 시작

**후후의 기능**:
- 전화 올 때 **스팸/대출/보이스피싱** 등 유형 표시
- 사용자 신고 기반 실시간 업데이트
- 번호 직접 검색하여 스팸 여부 확인 가능

## 문자 스팸 차단

전화뿐만 아니라 스팸 문자도 차단할 수 있어요.

**갤럭시 문자 차단**
1. 메시지 앱에서 스팸 문자 열기
2. 오른쪽 상단 **점 세 개** 클릭
3. **스팸으로 신고** 또는 **번호 차단** 선택

**아이폰 문자 차단**
1. 메시지 앱에서 스팸 문자 열기
2. 상단 발신자 클릭 → **정보** 클릭
3. **이 발신자 차단** 클릭

## 자주 묻는 질문

**Q. 차단한 번호에서 전화하면 어떻게 되나요?**
갤럭시는 전화가 연결되지 않고, 아이폰은 전화가 보이스메일로 바로 연결돼요. 상대방에게 차단되었다는 것이 알려지지 않아요.

**Q. 해외 번호(+82, +1 등)로 오는 스팸을 차단하려면?**
후후 앱이나 갤럭시 스팸 차단 설정에서 국제 전화 자동 차단 옵션을 활용해보세요.

**Q. 차단된 번호의 문자는 수신되나요?**
아니요. 전화와 문자 모두 차단돼요. 부재중 통화 목록에도 표시되지 않아요.
