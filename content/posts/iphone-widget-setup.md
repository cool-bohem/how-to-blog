## 아이폰 위젯이란?

위젯(Widget)은 앱을 직접 열지 않아도 홈 화면이나 잠금 화면에서 날씨, 배터리 잔량, 일정, 뉴스 등 앱의 핵심 정보를 바로 확인할 수 있도록 해주는 미니 화면입니다. 2026년 현재 iOS에서는 홈 화면 위젯, 잠금 화면 위젯, 대기 화면 위젯까지 다양한 위치에 위젯을 배치할 수 있어 스마트폰을 훨씬 편리하게 사용할 수 있습니다.

이 글에서는 아이폰 홈 화면과 잠금 화면에 위젯을 추가하고 커스터마이징하는 방법을 처음 사용하는 분도 이해할 수 있도록 단계별로 안내합니다.

---

## 홈 화면에 위젯 추가하기

### 1단계: 편집 모드(흔들기 모드) 진입

홈 화면에서 아무 빈 곳이나 **길게 누릅니다(약 1~2초)**. 화면이 흔들리며 앱 아이콘이 떨리는 편집 모드로 진입합니다. 이 상태에서 앱과 위젯을 자유롭게 이동하거나 추가할 수 있습니다.

또는 홈 화면 왼쪽 상단의 **편집** 버튼이 보이면 탭합니다.

### 2단계: 위젯 추가 버튼 탭

편집 모드에서 화면 왼쪽 상단의 **+** 버튼을 탭합니다. 위젯 선택 화면이 아래에서 올라옵니다.

### 3단계: 원하는 위젯 검색 또는 선택

위젯 화면 상단 검색창에 원하는 앱 이름을 검색합니다(예: 날씨, 캘린더, 배터리, 시계 등). 또는 아래로 스크롤하면 설치된 앱 목록이 나타납니다.

추천 위젯 앱:
- **날씨**: 현재 날씨와 주간 예보 표시
- **캘린더**: 오늘의 일정 한눈에 보기
- **배터리**: 아이폰·에어팟·애플워치 배터리 상태
- **시계**: 세계 시계 또는 알람 표시
- **미리 알림**: 할 일 목록 빠른 확인
- **사진**: 추억의 사진 자동 표시
- **지도**: 집·직장 등 즐겨찾기 도착 시간

### 4단계: 위젯 크기 선택

앱을 선택하면 해당 앱이 제공하는 위젯 크기가 표시됩니다:
- **소형(2×2)**: 가장 작은 크기, 핵심 정보만 표시
- **중형(4×2)**: 조금 더 넓어 더 많은 정보 표시
- **대형(4×4)**: 가장 크고 많은 정보 제공

원하는 크기로 좌우 스와이프해 선택한 뒤 **위젯 추가**를 탭합니다.

### 5단계: 위젯 위치 조정

추가된 위젯이 홈 화면에 나타납니다. 위젯을 길게 누른 채 드래그해 원하는 위치로 이동합니다. 다른 페이지로도 이동 가능합니다.

### 6단계: 편집 모드 종료

오른쪽 상단 **완료**를 탭하거나 홈 화면 빈 곳을 탭하면 편집 모드가 종료됩니다.

## 위젯 내용(스타일) 변경하기

일부 위젯은 표시하는 내용을 직접 설정할 수 있습니다.

### 설정 방법

1. 편집 모드에서 위젯을 **길게 누릅니다**.
2. 팝업 메뉴에서 **위젯 편집**을 탭합니다.
3. 위젯마다 다른 옵션이 나타납니다. 예를 들어 날씨 위젯은 표시할 도시를 변경할 수 있고, 시계 위젯은 표시할 시간대를 바꿀 수 있습니다.
4. 설정 후 **완료**를 탭합니다.

---

## 스택 위젯 만들기 (여러 위젯을 한 자리에)

스택 위젯은 같은 크기의 위젯 여러 개를 겹쳐 놓는 기능입니다. 손가락으로 위아래로 스와이프해 위젯을 전환할 수 있습니다.

### 스택 만드는 방법

1. 편집 모드에서 위젯 하나를 다른 위젯 위로 **드래그해 겹칩니다**.
2. 두 위젯이 합쳐져 스택이 생성됩니다.
3. 같은 방법으로 최대 10개까지 위젯을 스택에 추가할 수 있습니다.

### 스마트 스택 활용하기

스마트 스택은 애플이 기본 제공하는 위젯 묶음입니다. 시간대와 사용 패턴에 따라 가장 유용한 위젯을 자동으로 앞에 표시해줍니다. 예를 들어 출근 시간에는 교통 정보를, 저녁에는 뉴스를 보여주는 식입니다.

**스마트 스택 추가 방법:**
위젯 추가 화면에서 **스마트 스택**을 검색하거나 목록 상단에서 찾아 추가합니다.

**스마트 스택 자동 회전 설정:**
1. 편집 모드에서 스마트 스택을 길게 누릅니다.
2. **위젯 편집**을 탭합니다.
3. **스마트 순환** 스위치를 켜면 iOS가 자동으로 적절한 위젯을 앞에 표시합니다.

## 잠금 화면에 위젯 추가하기

잠금 화면 위젯은 화면을 켜기만 해도 날씨, 배터리, 운동 기록 등을 바로 확인할 수 있는 기능입니다.

### 1단계: 잠금 화면 편집 모드 진입

잠금 화면에서 아무 곳이나 **길게 누릅니다**. 잠금 해제가 필요하면 Face ID 또는 비밀번호를 입력합니다. 잠금 화면 편집 화면이 나타납니다.

### 2단계: 사용자화 탭

하단의 **사용자화** 버튼을 탭합니다.

### 3단계: 위젯 영역 탭

잠금 화면에는 두 곳에 위젯을 추가할 수 있습니다:
- **시계 위 영역**: 날짜 표시 줄 (작은 위젯 1개)
- **시계 아래 영역**: 최대 4개의 작은 위젯 또는 2개의 중형 위젯 배치 가능

시계 아래 위젯 영역을 탭합니다.

### 4단계: 위젯 선택

위젯 선택 패널이 나타납니다. 원하는 앱의 위젯을 탭하면 잠금 화면에 추가됩니다. 배터리, 날씨, 피트니스, 알람 등이 인기 있는 잠금 화면 위젯입니다.

### 5단계: 저장

오른쪽 상단 **추가** → **완료**를 탭해 저장합니다.

## 대기 화면(StandBy) 위젯 설정하기

아이폰을 가로로 눕혀 충전기에 연결하면 대기 화면(StandBy) 모드가 활성화됩니다. 이 화면에도 위젯을 배치해 시계, 날씨, 알림 등을 큰 화면으로 볼 수 있습니다.

### 대기 화면 위젯 변경 방법

1. 충전 중인 아이폰을 가로로 놓아 대기 화면을 활성화합니다.
2. 화면에서 위젯 영역을 **길게 누릅니다**.
3. 위젯을 좌우로 스와이프해 원하는 위젯 세트로 변경합니다.
4. **+** 버튼을 탭하면 새 위젯을 추가할 수 있습니다.

> **팁:** 대기 화면은 침실 탁상 시계처럼 활용할 수 있어 야간 모드(빨간 화면)와 함께 사용하면 눈에 편합니다.

## 자주 묻는 질문 (Q&A)

**Q. 위젯을 추가했는데 "앱을 다운로드하세요"라고 표시돼요.**
A. 해당 위젯이 연결된 앱이 삭제된 경우 표시됩니다. 앱을 다시 설치하거나 위젯을 삭제해주세요.

**Q. 위젯 크기를 나중에 바꿀 수 있나요?**
A. 위젯 크기를 직접 수정하는 기능은 없습니다. 기존 위젯을 삭제하고 원하는 크기로 다시 추가해야 합니다.

**Q. 서드파티 앱(네이버, 카카오 등)도 위젯을 제공하나요?**
A. 앱 개발사가 위젯을 지원해야 사용 가능합니다. 네이버, 카카오톡, 유튜브 뮤직 등 많은 앱이 위젯을 제공합니다. 앱 업데이트 후 위젯 추가 화면에서 해당 앱을 검색해보세요.

**Q. 위젯이 업데이트가 안 돼서 오래된 정보가 표시돼요.**
A. iOS는 배터리 절약을 위해 위젯을 일정 간격으로 업데이트합니다. 위젯을 탭해 앱을 열면 즉시 최신 정보로 갱신됩니다. 또는 설정 → 일반 → 백그라운드 앱 새로고침에서 해당 앱을 허용하면 더 자주 업데이트됩니다.

**Q. 위젯을 삭제하는 방법은?**
A. 편집 모드에서 위젯 왼쪽 상단의 **-** 버튼을 탭하거나, 위젯을 길게 눌러 **위젯 제거**를 선택하면 됩니다.

**Q. 잠금 화면 위젯이 보이지 않아요.**
A. 설정 → Face ID 및 암호 → 잠금 시 접근 허용 항목에서 **잠금 화면 위젯**이 켜져 있는지 확인하세요.

**Q. 동일한 앱 위젯을 여러 개 추가할 수 있나요?**
A. 네, 가능합니다. 예를 들어 날씨 위젯을 두 개 추가하고 각각 다른 도시로 설정해 두 곳의 날씨를 동시에 확인할 수 있습니다.

## 추천 위젯 조합

처음 위젯을 설정한다면 다음 조합을 참고해보세요.

**실용형 홈 화면**
- 상단: 스마트 스택(중형) - 날씨·캘린더·미리 알림 포함
- 중간: 배터리(소형) + 시계(소형)

**감성형 홈 화면**
- 대형 사진 위젯 1개로 배경처럼 활용
- 아날로그 시계 위젯 + 날씨 위젯

**잠금 화면**
- 시계 위: 날짜(기본)
- 시계 아래: 날씨 / 배터리 / 피트니스 링 / 알람

## 마무리 정리

아이폰 위젯은 홈 화면, 잠금 화면, 대기 화면에 모두 추가할 수 있어 필요한 정보를 앱을 열지 않고도 즉시 확인할 수 있습니다. 스택 위젯과 스마트 스택을 활용하면 공간을 최소화하면서 최대한 많은 정보를 담을 수 있습니다. 처음에는 날씨·캘린더·배터리 위젯부터 시작해 조금씩 자신만의 홈 화면을 꾸며보세요.
