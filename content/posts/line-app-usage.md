## 라인(LINE)이란?

**라인(LINE)**은 일본 기업이 개발한 메신저 앱으로, 전 세계 2억 명 이상이 사용하는 글로벌 소통 도구입니다. 한국에서는 카카오톡이 국민 메신저이지만, 일본·대만·태국 등 동남아시아 친구나 지인과 연락할 때, 또는 외국인 친구와 소통할 때 라인이 매우 유용합니다.

라인의 주요 특징:
- **무료 음성·영상 통화**: 인터넷만 있으면 국제 통화도 무료
- **스티커 메시지**: 다양하고 귀여운 스티커로 감정 표현
- **라인 오픈채팅**: 관심사 기반 커뮤니티 채팅방 참여
- **라인페이(LINE Pay)**: 간편결제 및 송금 기능
- **다양한 미니앱**: 쇼핑, 게임, 뉴스 등 다양한 서비스 연동

## 라인 앱 설치하기

### 스마트폰(안드로이드/아이폰) 설치

1. **구글 플레이스토어**(안드로이드) 또는 **앱스토어**(아이폰) 실행
2. 검색창에 **"LINE"** 또는 **"라인"** 입력
3. 개발사 "LINE Corporation" 앱 선택
4. **설치** 버튼 탭
5. 설치 완료 후 앱 실행

### PC(윈도우/맥) 설치

라인은 PC 버전도 제공합니다. 라인 공식 홈페이지에서 PC 버전을 다운로드할 수 있으며, 스마트폰과 동기화되어 PC에서도 동일한 채팅을 확인하고 주고받을 수 있습니다.

## 라인 회원가입 및 초기 설정

### 1단계: 전화번호로 가입

1. 라인 앱 실행 후 **로그인** 화면에서 **새 계정 만들기** 선택
2. 국가 선택 → **대한민국(+82)** 선택
3. 핸드폰 번호 입력 (앞에 0을 빼고 입력, 예: 01012345678 → 1012345678)
4. **SMS 인증번호** 받기 클릭
5. 문자로 받은 6자리 인증번호 입력
6. 이름 입력 (라인에서 표시될 이름)
7. 프로필 사진 설정 (나중에 설정 가능)
8. 비밀번호 설정 (대문자+소문자+숫자 포함 8자 이상 권장)

### 2단계: 친구 찾기 설정

가입 완료 후 연락처 동기화 여부를 묻는 화면이 나타납니다.

- **허용**: 스마트폰 연락처에 저장된 사람 중 라인 사용자를 자동으로 친구로 추가
- **나중에**: 수동으로 친구를 추가

개인정보 보호가 걱정된다면 **나중에**를 선택하고 필요한 사람만 개별적으로 추가하는 것을 권장합니다.

## 친구 추가하는 방법

### 방법 1: QR코드로 추가

가장 간편하고 많이 사용하는 방법입니다.

1. 라인 앱 하단의 **홈** 탭 선택
2. 오른쪽 상단의 **친구 추가(사람 모양+)** 아이콘 탭
3. **QR 코드** 선택
4. 상대방의 QR코드를 스캔하거나, 내 QR코드를 보여주기

### 방법 2: 라인 ID로 추가

1. 친구 추가 화면에서 **ID 검색** 선택
2. 상대방의 라인 ID 입력
3. **검색** 후 **추가** 버튼 탭

단, 상대방이 "ID로 친구 추가 허용" 설정이 켜져 있어야 검색됩니다.

### 방법 3: 전화번호로 추가

1. 친구 추가 화면에서 **전화번호** 선택
2. 상대방 전화번호 직접 입력

## 채팅 메시지 보내기

### 기본 문자 메시지

1. 라인 앱 하단 **채팅** 탭 선택
2. 오른쪽 상단 **+** 버튼 탭
3. 대화 상대 선택
4. 하단 입력창에 메시지 입력
5. 오른쪽 **전송** 버튼(화살표) 탭

### 스티커 전송

라인의 대표적인 기능 중 하나가 스티커입니다.

1. 채팅 입력창 왼쪽의 **☺** (이모티콘) 아이콘 탭
2. 스티커 탭에서 보내고 싶은 스티커 선택
3. 탭하면 바로 전송

**스티커 추가 다운로드:**
1. 스티커 창 오른쪽의 **+** 버튼 탭
2. 스티커 샵에서 원하는 스티커팩 선택
3. **무료** 스티커는 탭 후 **다운로드**로 바로 사용 가능
4. 유료 스티커는 라인 코인으로 구매

### 사진·동영상 전송

1. 채팅 입력창 왼쪽 **+** 버튼 탭
2. **앨범** 또는 **카메라** 선택
3. 전송할 사진/동영상 선택 후 전송

### 음성 메시지 전송

1. 채팅 입력창의 **마이크** 아이콘 길게 누르기
2. 말하는 동안 손가락 유지
3. 손가락을 떼면 자동 전송

## 무료 음성·영상 통화하기

라인의 가장 강력한 기능 중 하나가 무료 통화입니다. 와이파이나 데이터 연결만 있으면 국내외 어디서나 무료로 통화할 수 있습니다.

### 1:1 통화

1. 통화하고 싶은 친구의 채팅방 열기
2. 오른쪽 상단 **☎** (전화) 아이콘 탭
3. **음성 통화** 또는 **영상 통화** 선택
4. 상대방이 받으면 통화 시작

### 그룹 통화

1. 그룹 채팅방 열기
2. 오른쪽 상단 **☎** 아이콘 탭
3. **그룹 통화 시작**
4. 통화방에 참여를 원하는 멤버가 자유롭게 참여

## 그룹 채팅방 만들기

여러 명과 동시에 대화할 수 있는 그룹 채팅방을 만드는 방법입니다.

1. 채팅 탭에서 오른쪽 상단 **+** 버튼 탭
2. **그룹** 선택
3. 함께 채팅할 친구들 선택 (최대 500명)
4. 그룹 이름 입력
5. **만들기** 버튼 탭

그룹 채팅방에서는 공지사항 고정, 앨범 공유, 투표 기능 등 다양한 부가 기능을 활용할 수 있습니다.

## 라인 오픈채팅 활용하기

라인 오픈채팅은 친구가 아닌 사람들과도 관심사를 기반으로 대화할 수 있는 커뮤니티 기능입니다.

1. 홈 탭 상단 검색창에 관심 주제 검색 (예: "일본어 공부", "여행", "요리")
2. 원하는 오픈채팅방 선택
3. **참여하기** 버튼 탭
4. 닉네임 설정 (오픈채팅에서는 별도 닉네임 사용 가능)

## 라인 알림 설정하기

메시지 알림이 너무 많아 불편하다면 알림을 조절하세요.

### 특정 채팅방 알림 끄기

1. 알림을 끄고 싶은 채팅방 열기
2. 오른쪽 상단 메뉴(≡ 또는 ...) 탭
3. **알림 끄기** 선택
4. 1시간 / 8시간 / 24시간 / 완전히 끄기 중 선택

### 전체 알림 설정

1. 홈 탭 오른쪽 상단 **설정(⚙)** 아이콘 탭
2. **알림** 선택
3. 메시지 알림, 그룹 채팅 알림 등을 개별 설정

## 라인 프라이버시 보호 설정

### ID 검색 차단

낯선 사람에게 ID로 검색되는 것을 막고 싶다면:

1. **설정 → 개인 정보** 이동
2. **ID로 친구 추가 허용** 토글 끄기

### 메시지 수신 허용 범위

1. **설정 → 개인 정보 → 메시지 수신 허용** 이동
2. **친구만** 또는 **전체** 중 선택

## 자주 묻는 질문 (Q&A)

**Q. 라인 계정 비밀번호를 잊어버렸어요.**
A. 로그인 화면에서 **비밀번호를 잊으셨나요?** 를 탭하면 가입 시 사용한 이메일로 재설정 링크를 받을 수 있습니다. 이메일도 기억나지 않는다면 고객센터를 통해 계정 복구를 요청해야 합니다.

**Q. 라인 계정을 새 폰으로 이전하려면 어떻게 하나요?**
A. 먼저 기존 폰에서 **설정 → 계정 → 이메일 주소 등록**으로 이메일을 등록합니다. 새 폰에서 같은 전화번호와 이메일로 로그인하면 계정이 이전됩니다. 단, 채팅 내역은 기본적으로 이전되지 않으므로 중요한 내용은 미리 백업하거나 스크린샷을 찍어두세요.

**Q. 라인 메시지를 삭제하면 상대방 화면에서도 사라지나요?**
A. 자신의 화면에서만 삭제됩니다. 상대방 화면에서는 여전히 표시됩니다. 다만 **삭제가 아닌 "취소"** 기능을 사용하면 보낸 지 24시간 이내의 메시지는 양쪽에서 모두 삭제됩니다. 메시지를 길게 누른 후 **취소** 버튼을 사용하세요.

**Q. 라인 통화가 자꾸 끊겨요.**
A. 주로 인터넷 연결 상태 문제입니다. 와이파이 연결이 불안정하다면 모바일 데이터로 전환하거나, 라우터 가까이 이동해 보세요. 앱을 완전히 종료 후 재시작하거나, 앱 업데이트 여부도 확인해보세요.

**Q. 스팸 메시지가 너무 많이 와요. 차단하려면 어떻게 하나요?**
A. 해당 채팅방 또는 프로필에서 **차단** 기능을 사용하세요. 홈 탭 → 친구 목록에서 차단할 친구를 길게 누르거나, 채팅방 메뉴에서 **차단**을 선택할 수 있습니다. 차단된 상대는 메시지를 보낼 수 없고, 통화도 연결되지 않습니다.

**Q. 라인을 PC에서도 사용하고 싶어요.**
A. 라인 공식 홈페이지에서 PC 버전을 다운로드하거나, 크롬 확장 프로그램으로도 사용 가능합니다. 스마트폰에서 라인 앱 → 설정 → 계정 → PC/Mac에서 로그인에서 QR코드를 스캔해 연결하면 됩니다.

## 요약

라인(LINE)은 카카오톡과 더불어 유용하게 활용할 수 있는 메신저 앱입니다. 핵심 기능을 정리하면 다음과 같습니다.

- **설치 및 가입**: 전화번호로 간단히 가입 가능
- **친구 추가**: QR코드 스캔이 가장 편리하고 빠른 방법
- **무료 통화**: 와이파이로 국내외 음성·영상 통화 무료
- **스티커**: 다양한 무료·유료 스티커로 대화 풍성하게
- **오픈채팅**: 관심사 기반 커뮤니티 참여
- **프라이버시**: ID 검색 차단, 알림 설정으로 쾌적한 사용 가능

일본, 대만, 태국 등 아시아 친구들과 소통하거나, 해외에서 부담 없이 국제 통화를 해야 할 때 라인은 최고의 선택입니다. 설치부터 기본 기능까지 익혀두면 다양한 상황에서 유용하게 활용할 수 있습니다.
