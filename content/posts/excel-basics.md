## 엑셀 필수 단축키

마우스 없이 키보드만으로 엑셀을 빠르게 다루는 단축키 모음이에요.

## 기본 단축키

| 단축키 | 기능 |
|--------|------|
| Ctrl + C | 복사 |
| Ctrl + V | 붙여넣기 |
| Ctrl + Z | 실행 취소 |
| Ctrl + S | 저장 |
| Ctrl + F | 찾기 |
| Ctrl + H | 찾아 바꾸기 |

## 셀 이동 단축키

| 단축키 | 기능 |
|--------|------|
| Ctrl + Home | 첫 번째 셀(A1)로 이동 |
| Ctrl + End | 데이터 마지막 셀로 이동 |
| Ctrl + 방향키 | 데이터 끝으로 빠르게 이동 |
| Ctrl + Shift + End | 마지막 셀까지 선택 |

## 서식 단축키

| 단축키 | 기능 |
|--------|------|
| Ctrl + B | 굵게 |
| Ctrl + 1 | 셀 서식 창 열기 |
| Alt + H + H | 셀 배경색 변경 |
| Ctrl + Shift + % | 퍼센트 서식 적용 |

## 행/열 단축키

| 단축키 | 기능 |
|--------|------|
| Ctrl + Shift + + | 행/열 삽입 |
| Ctrl + - | 행/열 삭제 |
| Ctrl + Space | 열 전체 선택 |
| Shift + Space | 행 전체 선택 |

## 자주 쓰는 함수 단축키

- **Alt + =** : 자동 합계(SUM) 입력
- **Ctrl + Shift + Enter** : 배열 수식 입력
- **F2** : 셀 편집 모드로 전환
- **F4** : 절대참조($) 전환
