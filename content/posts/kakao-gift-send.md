## 카카오톡 선물하기란?

카카오톡 선물하기는 친구 주소를 몰라도, 직접 만나지 않아도 카카오톡으로 바로 선물을 보낼 수 있는 서비스예요. 스타벅스 커피, 케이크, 상품권, 화장품, 배달 쿠폰 등 수천 가지 상품이 있어요.

**선물하기의 장점**:
- 상대방 주소나 연락처를 몰라도 카카오톡 친구라면 누구에게나 선물 가능
- 받는 사람이 원하는 시간에 사용 가능한 모바일 쿠폰으로 전달
- 생일, 기념일, 감사의 마음을 즉시 전달 가능
- 다양한 가격대 (5천원 ~ 수십만원)

## 선물 보내는 방법

1. 카카오톡 하단 **더보기(···)** 탭 클릭
2. **선물하기** 클릭
3. 카테고리 또는 검색으로 원하는 상품 찾기
   - 예: '스타벅스', '배스킨라빈스', '상품권' 검색
4. 상품 선택 후 **선물하기** 클릭
5. 받을 친구 선택 (여러 명에게 동시 발송도 가능)
6. 축하 메시지 및 카드 선택 (생략 가능)
7. 결제 완료

## 결제 방법

- **카카오페이**: 등록된 카드나 카카오머니로 빠른 결제
- **신용/체크카드**: 카드 번호 직접 입력
- **카카오페이머니**: 미리 충전 후 사용
- **휴대폰 결제**: 월 통신비로 청구

**팁**: 카카오페이를 연결해두면 지문/얼굴 인식 한 번으로 바로 결제할 수 있어요.

## 선물 받는 방법

선물을 받으면 카카오톡 메시지로 알림이 와요.

1. 카카오톡 메시지에서 **선물 받기** 클릭
2. 내용 확인 후 **수락**
3. 모바일 쿠폰으로 저장됨
4. **더보기 → 선물하기 → 받은 선물함** 에서 확인 가능
5. 해당 매장에서 바코드 제시 후 사용

**참고**: 선물은 수락 기한이 있어요(보통 30일). 기한 내에 수락하지 않으면 선물이 반환돼요.

## 선물 유효기간 연장

받은 쿠폰의 사용 기한이 다가오면 연장할 수 있어요.

1. **더보기 → 선물하기 → 받은 선물함**
2. 만료 임박 쿠폰 선택
3. **유효기간 연장** 클릭
4. 연장 수수료 없이 1회 연장 가능 (보통 30~60일 연장)

**팁**: 유효기간이 짧은 쿠폰은 미리 연장해두는 게 좋아요. 만료 후에는 연장이 불가능해요.

## 받은 선물 환불

사용하지 않은 쿠폰은 환불 신청을 할 수 있어요.

1. **받은 선물함**에서 해당 선물 클릭
2. **환불 신청** 클릭
3. 카카오페이로 환불 처리 (1~3 영업일 소요)

**주의**: 일부 상품은 환불이 제한될 수 있어요. 환불 정책은 상품마다 다르니 상세 페이지에서 확인하세요.

## 선물하기 꿀팁

**단체 선물 기능 (선물 모으기)**
- 여러 명이 함께 돈을 모아 큰 선물을 보낼 수 있어요
- 상품 선택 → **선물 모으기** → 링크 공유 후 참여자 모집

**위시리스트 공유**
- 본인의 위시리스트를 만들어 친구에게 공유할 수 있어요
- 생일 등 선물을 받을 일이 있을 때 활용하면 유용해요

**B급 선물 / 할인 쿠폰 탭 활용**
- 선물하기 홈에서 **혜택/쿠폰** 탭을 확인하면 할인가에 구매할 수 있는 상품이 있어요

## 자주 묻는 질문

**Q. 카카오톡 친구가 아닌 사람에게도 선물할 수 있나요?**
전화번호를 입력해서 친구가 아닌 사람에게도 선물 발송이 가능해요.

**Q. 받은 선물을 다른 사람에게 다시 선물할 수 있나요?**
받은 선물을 다시 선물하는 기능은 지원하지 않아요. 직접 구매해야 해요.

**Q. 해외에서도 선물하기를 사용할 수 있나요?**
네. 카카오톡이 연결되는 해외에서도 선물하기를 이용할 수 있어요. 단, 국내 매장 쿠폰이라 사용은 귀국 후에 가능해요.
