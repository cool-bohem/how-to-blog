## 카카오톡 선물하기란?

카카오톡에서 바로 친구에게 커피, 케이크, 상품권 등을 선물할 수 있는 기능이에요. 주소 몰라도 돼요.

## 선물 보내는 방법

1. 카카오톡 하단 **더보기(···)** 탭
2. **선물하기** 클릭
3. 원하는 상품 검색 (예: 스타벅스 아메리카노)
4. 상품 선택 후 **선물하기** 클릭
5. 받을 친구 선택
6. 메시지 작성 후 결제

## 결제 방법

- 카카오페이
- 신용/체크카드
- 카카오페이 머니

## 선물 받는 방법

1. 카카오톡 메시지로 선물 알림 수신
2. **선물 받기** 클릭
3. 수락하면 모바일 쿠폰으로 저장됨
4. 해당 매장에서 바코드 제시 후 사용

## 선물 유효기간 연장

1. **더보기 → 선물하기 → 받은 선물함**
2. 만료 임박 쿠폰 선택
3. **유효기간 연장** 클릭 (1회 연장 가능, 무료)

## 받은 선물 환불

1. 받은 선물함에서 해당 선물 클릭
2. **환불 신청** 클릭
3. 카카오페이로 환불 처리 (1~3 영업일)
