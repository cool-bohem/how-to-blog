## 윈도우에서 프린터 설치가 필요한 경우

새 프린터를 구입했거나, PC를 새로 설정했거나, 윈도우를 재설치한 경우 프린터 드라이버를 새로 설치해야 합니다. 또한 회사나 학교에서 네트워크 공유 프린터를 사용해야 할 때도 설정이 필요합니다.

이 가이드에서는 윈도우 11 기준으로 USB 연결 프린터, Wi-Fi 무선 프린터, 그리고 네트워크 공유 프린터 연결 방법을 모두 설명합니다.

---

## 방법 1: USB 케이블로 프린터 연결하기

가장 간단한 방법으로, USB 케이블로 프린터와 PC를 직접 연결합니다.

### 1단계: 프린터 전원 켜기

프린터 전원 케이블을 콘센트에 연결하고 전원 버튼을 눌러 켭니다.

### 2단계: USB 케이블 연결

프린터와 함께 제공된 USB 케이블을 프린터와 PC의 USB 포트에 각각 연결합니다.

### 3단계: 자동 드라이버 설치 확인

윈도우 11은 프린터를 연결하면 자동으로 드라이버를 검색하고 설치하려고 시도합니다. 작업 표시줄 오른쪽 하단에 "드라이버 소프트웨어 설치 중" 또는 "장치를 사용할 준비가 되었습니다"라는 알림이 표시됩니다.

### 4단계: 자동 설치 확인

1. **시작 버튼** → **설정** (톱니바퀴 아이콘)을 클릭합니다.
2. **Bluetooth 및 디바이스** → **프린터 및 스캐너**를 클릭합니다.
3. 목록에 연결한 프린터가 표시되면 성공입니다.

### 5단계: 수동 드라이버 설치 (자동 설치가 안 될 경우)

자동으로 설치되지 않는 경우, 제조사 웹사이트에서 드라이버를 다운로드해야 합니다.

**주요 프린터 제조사 드라이버 다운로드 방법:**

- **삼성 프린터**: 삼성 공식 홈페이지 → 고객지원 → 소프트웨어/드라이버 검색
- **HP 프린터**: HP 공식 홈페이지 → 소프트웨어 및 드라이버 검색
- **캐논 프린터**: 캐논 공식 홈페이지 → 드라이버 다운로드 센터
- **엡슨 프린터**: 엡슨 공식 홈페이지 → 다운로드 → 드라이버 검색

제조사 홈페이지에서 본인 프린터 모델명을 검색하고, 운영체제를 "Windows 11"로 선택하여 드라이버를 다운로드합니다. 다운로드된 설치 파일을 실행하고 화면의 안내를 따라 설치를 완료합니다.

## 방법 2: Wi-Fi 무선으로 프린터 연결하기

요즘 출시되는 프린터 대부분은 Wi-Fi를 지원합니다. 케이블 없이 무선으로 연결하면 여러 기기에서 편리하게 사용할 수 있습니다.

### 1단계: 프린터를 Wi-Fi에 연결하기

프린터마다 무선 연결 방법이 다를 수 있으나, 일반적인 방법은 다음과 같습니다.

**프린터 조작판에서 Wi-Fi 설정:**
1. 프린터의 **설정** 또는 **메뉴** 버튼을 누릅니다.
2. **네트워크 설정** 또는 **무선 설정**을 선택합니다.
3. **무선 설정 마법사** 또는 **Wi-Fi 설정**을 선택합니다.
4. 연결할 Wi-Fi 네트워크 이름(SSID)을 선택합니다.
5. Wi-Fi 비밀번호를 입력합니다. (프린터에 따라 화면 키보드 또는 방향키로 입력)
6. 연결이 완료되면 프린터에 Wi-Fi 연결 확인 표시가 나타납니다.

**WPS 버튼으로 간편 연결:**
공유기에 WPS 버튼이 있다면 더 간편하게 연결할 수 있습니다.
1. 프린터에서 WPS 버튼을 누릅니다.
2. 2분 이내에 공유기의 WPS 버튼을 누릅니다.
3. 자동으로 연결이 완료됩니다.

### 2단계: PC에서 프린터 추가하기

1. **시작 버튼** → **설정** → **Bluetooth 및 디바이스** → **프린터 및 스캐너**를 클릭합니다.
2. **디바이스 추가** 버튼을 클릭합니다.
3. 같은 Wi-Fi 네트워크에 있는 프린터가 자동으로 검색됩니다.
4. 연결할 프린터 이름 옆의 **디바이스 추가** 버튼을 클릭합니다.
5. 드라이버가 자동으로 설치되고 프린터가 추가됩니다.

### 3단계: 프린터가 검색되지 않을 경우

1. 프린터와 PC가 동일한 Wi-Fi 네트워크에 연결되어 있는지 확인합니다.
2. **프린터가 목록에 없음**을 클릭합니다.
3. "Bluetooth, 무선 또는 네트워크 검색 가능한 프린터 추가"를 선택하고 다음을 클릭합니다.
4. 다시 검색을 기다립니다.

## 방법 3: 네트워크 공유 프린터에 연결하기

회사나 학교에서 다른 PC에 연결된 프린터를 공유해서 사용하는 경우입니다.

### 1단계: 프린터가 공유 설정되어 있는지 확인

프린터가 연결된 주 PC에서 공유 설정이 되어 있어야 합니다. 프린터가 연결된 PC에서:
1. **제어판** → **장치 및 프린터**를 열거나 **설정** → **프린터 및 스캐너**를 엽니다.
2. 공유할 프린터를 우클릭 → **프린터 속성** → **공유** 탭에서 **이 프린터 공유** 가 체크되어 있어야 합니다.

### 2단계: 공유 프린터에 연결

내 PC에서:
1. **시작 버튼** → **설정** → **Bluetooth 및 디바이스** → **프린터 및 스캐너**를 클릭합니다.
2. **디바이스 추가** → **프린터가 목록에 없음**을 클릭합니다.
3. **이름으로 공유 프린터 선택**을 선택합니다.
4. `\\컴퓨터이름\프린터공유이름` 형식으로 입력합니다.
   예: `\\OFFICE-PC\HP-LaserJet`
5. **다음** → **드라이버 설치** → **마침**을 클릭합니다.

## 테스트 페이지 출력하기

프린터 설치 후 제대로 작동하는지 확인합니다.

1. **설정** → **Bluetooth 및 디바이스** → **프린터 및 스캐너**를 클릭합니다.
2. 설치된 프린터를 클릭합니다.
3. **인쇄 테스트 페이지**를 클릭합니다.
4. 프린터에서 테스트 페이지가 출력되면 정상적으로 설치된 것입니다.

---

## 기본 프린터 설정하기

여러 프린터가 설치된 경우, 자주 사용하는 프린터를 기본 프린터로 설정합니다.

1. **설정** → **Bluetooth 및 디바이스** → **프린터 및 스캐너**를 클릭합니다.
2. 기본 프린터로 설정할 프린터를 클릭합니다.
3. **기본값으로 설정** 버튼을 클릭합니다.
4. 프린터 이름 아래에 "기본값" 표시가 나타나면 완료입니다.

---

## 인쇄 품질 및 용지 설정하기

프린터 설정을 최적화하여 더 좋은 결과물을 얻을 수 있습니다.

### 인쇄 기본 설정 변경

1. **설정** → **프린터 및 스캐너** → 해당 프린터 클릭합니다.
2. **인쇄 기본 설정**을 클릭합니다.
3. 여기서 다음 항목들을 설정할 수 있습니다:
   - **용지 크기**: A4, B5, Letter 등
   - **인쇄 품질**: 초안/일반/최고 품질 선택
   - **단면/양면 인쇄**: 종이 절약을 위해 양면 인쇄 권장
   - **컬러/흑백**: 잉크 절약을 위해 흑백 인쇄 설정 가능

## 자주 발생하는 문제와 해결 방법

### 프린터가 "오프라인" 상태로 표시될 때

1. 프린터 전원이 켜져 있는지 확인합니다.
2. USB 케이블 또는 Wi-Fi 연결 상태를 확인합니다.
3. **설정** → **프린터 및 스캐너** → 해당 프린터 → **인쇄 대기열 열기**를 클릭합니다.
4. 상단 메뉴 **프린터** → **오프라인으로 프린터 사용** 체크가 해제되어 있는지 확인합니다.
5. 체크되어 있다면 클릭하여 해제합니다.

### 인쇄 대기열에 작업이 걸려서 안 지워질 때

1. **프린터 서비스**를 재시작합니다.
2. **시작** 버튼을 우클릭 → **실행** → `services.msc`를 입력합니다.
3. **Print Spooler** 서비스를 찾아 우클릭 → **중지**를 클릭합니다.
4. 파일 탐색기를 열고 `C:\Windows\System32\spool\PRINTERS` 폴더에서 모든 파일을 삭제합니다. (폴더 자체는 삭제하지 마세요)
5. 다시 **Print Spooler** 서비스를 우클릭 → **시작**을 클릭합니다.

### 잉크가 충분한데 인쇄 품질이 나쁠 때

1. 프린터 드라이버를 열고 **유지보수** 또는 **서비스** 탭을 찾습니다.
2. **헤드 클리닝** 또는 **노즐 청소**를 실행합니다.
3. 청소 후 노즐 테스트 패턴을 인쇄하여 확인합니다.
4. 잉크가 굳어있거나 심하게 막힌 경우 2~3회 반복 청소가 필요할 수 있습니다.

## 자주 묻는 질문 (FAQ)

**Q. 드라이버 CD가 없는데 어떻게 설치하나요?**
제조사 공식 홈페이지에서 모델명으로 검색하면 최신 드라이버를 무료로 다운로드할 수 있습니다. CD 없이도 완벽하게 설치 가능합니다.

**Q. 스마트폰에서도 이 프린터로 출력할 수 있나요?**
대부분의 최신 프린터는 스마트폰 앱을 통한 무선 인쇄를 지원합니다. 삼성 프린터는 Samsung Mobile Print, HP는 HP Smart, 캐논은 Canon PRINT 앱을 사용하세요. Apple 기기라면 AirPrint를 지원하는 프린터에서 별도 앱 없이 인쇄 가능합니다.

**Q. 잉크가 없을 때 흑백으로만 인쇄하려면?**
인쇄 설정에서 "흑백" 또는 "그레이스케일"로 변경하면 컬러 잉크 없이도 인쇄할 수 있습니다. 단, 일부 프린터는 모든 카트리지가 있어야만 인쇄되는 경우가 있습니다.

**Q. 여러 대의 컴퓨터에서 하나의 프린터를 사용하려면?**
Wi-Fi 공유기에 연결된 무선 프린터를 사용하거나, 한 PC에 연결된 프린터를 네트워크 공유 설정하면 됩니다. 두 방법 모두 같은 Wi-Fi 네트워크에 있어야 합니다.

**Q. 프린터를 삭제하고 싶을 때는?**
설정 → 프린터 및 스캐너에서 해당 프린터를 클릭하고 "제거" 버튼을 클릭합니다. 더 이상 사용하지 않는 프린터는 삭제하여 목록을 깔끔하게 유지하세요.

## 마무리 정리

윈도우에서 프린터를 설치하는 방법은 크게 세 가지이며, 상황에 맞게 선택하시면 됩니다.

**핵심 요약:**
- **USB 연결**: 가장 안정적, 자동 드라이버 설치 시도 후 안 되면 수동 설치
- **Wi-Fi 무선 연결**: 편리하고 여러 기기에서 사용 가능, 같은 Wi-Fi 네트워크 필수
- **네트워크 공유**: 회사/가정에서 한 대를 여러 PC가 공유할 때 사용
- 설치 후 테스트 페이지로 정상 작동 확인
- 자주 쓰는 프린터는 기본 프린터로 설정하여 편리하게 사용

문제가 생겨도 당황하지 말고 이 가이드의 문제 해결 방법을 따라해보세요!
