## 네이버 비밀번호 찾기

네이버 비밀번호를 잊어버렸을 때 찾는 방법이에요.

## 방법 1. 휴대폰 번호로 찾기 (가장 빠름)

1. 네이버 로그인 화면에서 **아이디·비밀번호 찾기** 클릭
2. **비밀번호 찾기** 탭 선택
3. **아이디 입력** 후 **휴대폰으로 인증** 선택
4. 인증번호 받아서 입력
5. 새 비밀번호 설정

## 방법 2. 이메일로 찾기

1. **비밀번호 찾기** → **이메일로 인증** 선택
2. 가입 시 등록한 이메일로 인증 메일 발송
3. 메일 확인 후 링크 클릭 → 새 비밀번호 설정

## 방법 3. 아이디를 모를 때

1. **아이디 찾기** 탭 클릭
2. **이름 + 휴대폰 번호** 또는 **이름 + 이메일** 로 찾기
3. 아이디 확인 후 비밀번호 찾기 진행

## 방법 4. 위 방법이 모두 안 될 때

휴대폰 번호도 바뀌고 이메일도 모를 때는 **네이버 고객센터**를 통해 본인확인 후 복구 가능해요.

1. 네이버 고객센터 접속
2. **계정 → 아이디/비밀번호 찾기 실패** 선택
3. 본인 확인 서류 제출 후 처리 (1~3 영업일 소요)

## 비밀번호 분실 예방법

- **네이버 앱 → 설정 → 지문/얼굴 인식 로그인** 설정해두기
- 비밀번호 관리 앱(예: 아이클라우드 키체인, 삼성 패스) 활용
