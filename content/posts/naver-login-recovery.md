## 네이버 로그인이 안 될 때

네이버는 포털, 쇼핑, 블로그, 네이버페이 등 일상 생활 곳곳에서 쓰이는 계정이에요. 비밀번호를 잊어버리면 꽤 불편해지죠. 다행히 몇 가지 방법으로 빠르게 복구할 수 있어요.

**비밀번호를 잊어버린 경우**: 방법 1~3
**아이디까지 모르는 경우**: 아이디 찾기 먼저 → 이후 비밀번호 찾기

## 방법 1. 휴대폰 번호로 찾기 (가장 빠름)

가입 시 등록한 휴대폰 번호로 인증해서 새 비밀번호를 설정해요.

1. 네이버 로그인 화면에서 **아이디·비밀번호 찾기** 클릭
2. **비밀번호 찾기** 탭 선택
3. 아이디 입력 후 **휴대폰으로 인증** 선택
4. 휴대폰 번호 입력 후 인증번호 발송
5. 문자로 받은 인증번호 입력
6. 새 비밀번호 설정 완료

**주의**: 번호가 바뀌었다면 이 방법은 사용 불가예요. 방법 2로 시도해보세요.

## 방법 2. 이메일로 찾기

네이버 계정에 등록된 이메일로 인증 메일을 받아요.

1. **비밀번호 찾기** → **이메일로 인증** 선택
2. 가입 시 등록한 이메일 주소 입력
3. 해당 이메일 함에서 네이버 발송 메일 확인
4. 메일 내 링크 클릭 → 새 비밀번호 설정

**팁**: 이메일이 오지 않는다면 스팸함을 확인해보세요.

## 방법 3. 아이디를 모를 때 (아이디 찾기)

아이디까지 기억나지 않는다면 먼저 아이디를 찾아야 해요.

1. 로그인 화면 → **아이디·비밀번호 찾기** 클릭
2. **아이디 찾기** 탭 클릭
3. 선택:
   - **이름 + 휴대폰 번호** 로 찾기
   - **이름 + 이메일** 로 찾기
4. 인증 완료 후 아이디 확인
5. 확인된 아이디로 비밀번호 찾기 진행

## 방법 4. 위 방법이 모두 안 될 때 (고객센터)

휴대폰 번호도 바뀌고, 이메일도 기억나지 않는 경우에는 네이버 고객센터를 통한 본인 확인 복구를 이용할 수 있어요.

1. 네이버 고객센터 접속 (naver.com/help)
2. **계정 → 아이디/비밀번호 찾기 실패** 선택
3. 본인 확인 서류 제출 요청에 따라 진행
4. 처리까지 1~3 영업일 소요

**주의**: 본인 확인이 어렵거나 계정 정보와 불일치하면 복구가 거절될 수 있어요.

## 네이버 계정 보안 강화하기

비밀번호 분실을 예방하고 계정을 더 안전하게 보호하는 방법이에요.

**2단계 인증 설정**
1. 네이버 앱 → **설정 → 2단계 인증**
2. 로그인 시 휴대폰 인증을 추가로 요구
3. 비밀번호가 유출돼도 계정을 보호할 수 있어요

**지문/얼굴 인식 로그인 설정**
1. 네이버 앱 → **설정 → 생체 인증 로그인**
2. 지문 또는 얼굴 인식으로 간편 로그인
3. 비밀번호를 입력할 일이 줄어들어 분실 위험도 낮아져요

**비밀번호 관리 앱 활용**
- 아이클라우드 키체인(아이폰), 삼성패스(갤럭시), 구글 비밀번호 관리자 등을 활용하면 비밀번호를 안전하게 저장하고 자동 입력할 수 있어요

## 자주 묻는 질문

**Q. 네이버 아이디를 여러 개 만든 경우, 어떤 아이디를 쓰는지 모르겠어요.**
아이디 찾기를 통해 현재 휴대폰 번호 또는 이메일에 연결된 아이디를 찾을 수 있어요.

**Q. 비밀번호를 너무 자주 잊어버려요.**
비밀번호 관리 앱(Google 비밀번호 관리자, 네이버 앱 내 자동 저장)을 활용해보세요. 매번 입력 안 해도 자동으로 채워줘요.

**Q. 해킹을 당한 것 같아요. 어떻게 하나요?**
즉시 비밀번호를 변경하고, 네이버 보안 설정에서 **로그인 기록 확인** 후 모르는 기기에서 로그아웃하세요. 2단계 인증도 바로 설정하세요.
