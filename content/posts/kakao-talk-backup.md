## 카카오톡 백업이 필요한 이유

카카오톡을 오래 사용하면 중요한 대화, 사진, 파일들이 많이 쌓이게 돼요. 스마트폰을 바꾸거나 앱을 재설치하면 이 내용들이 모두 사라질 수 있어요.

**백업이 특히 중요한 상황**:
- 새 스마트폰으로 기기 변경 시
- 카카오톡 앱 재설치 또는 초기화 시
- 스마트폰 분실/파손으로 긴급 교체 시

백업은 5분이면 충분해요. 지금 바로 해두는 게 좋아요.

## Android 백업 방법

1. 카카오톡 **더보기(···) → 설정(톱니바퀴)**
2. **채팅** 메뉴 클릭
3. **대화 백업** 클릭
4. **지금 백업하기** 클릭
5. 카카오 계정 비밀번호 입력
6. 백업 완료까지 대기 (대화가 많으면 수 분 소요)

**백업 내용**: 1:1 채팅, 오픈채팅 제외한 그룹 채팅의 텍스트 메시지. 사진/동영상/파일은 별도 저장 필요.

**팁**: 정기적으로 백업하는 게 중요해요. **자동 백업** 설정을 켜두면 주기적으로 자동 백업돼요. 같은 메뉴에서 **자동 백업** 옵션을 확인하세요.

## iPhone 백업 방법

아이폰은 iCloud를 통해 백업해요.

1. 카카오톡 **더보기 → 설정 → 채팅 → 대화 백업**
2. **iCloud로 백업** 선택
3. Apple ID 로그인 상태 확인 후 백업 시작
4. 백업 완료까지 대기

**주의**: iCloud 여유 공간이 충분한지 확인하세요. iCloud 기본 무료 용량은 5GB예요. 대화가 많으면 부족할 수 있어요.

**팁**: 카카오 계정으로도 백업이 가능해요. **대화 백업 → 카카오 계정으로 백업** 을 선택하면 iCloud 대신 카카오 서버에 저장돼요.

## 새 폰에서 복구하는 방법

**Android에서 복구**
1. 새 갤럭시에 카카오톡 설치 후 **같은 카카오 계정**으로 로그인
2. 로그인 과정에서 **"이전 대화를 불러올까요?"** 팝업 확인
3. **불러오기** 클릭
4. 복구 완료까지 대기

**iPhone에서 복구**
1. 새 아이폰에 카카오톡 설치 후 같은 계정으로 로그인
2. **iCloud에서 불러오기** 선택
3. 복구 진행

**팁**: 기기 변경 전 마지막으로 한 번 더 백업해두는 것을 강력 추천해요. 가장 최신 대화까지 복구하려면 기기 교체 직전 백업이 필요해요.

## 개별 채팅방 내보내기

특정 채팅방 내용만 따로 파일로 저장하고 싶을 때 사용해요. 중요한 업무 대화나 추억이 담긴 채팅을 영구 보관할 수 있어요.

1. 해당 채팅방 열기
2. 오른쪽 상단 **메뉴(≡)** 클릭
3. **채팅방 설정** 클릭
4. **대화 내용 내보내기** 클릭
5. 저장 방식 선택:
   - **텍스트 파일(.txt)** 로 저장
   - 이메일, 카카오톡, 파일 앱 등으로 내보내기

**팁**: 텍스트 파일에는 이미지나 동영상은 포함되지 않아요. 중요한 파일은 따로 저장해두세요.

## 백업과 복구 시 주의사항

- **오픈채팅 대화**는 백업/복구 대상이 아니에요
- **첨부 파일(사진, 동영상, 파일)** 은 텍스트 메시지 백업에 포함되지 않아요. 중요한 파일은 따로 갤러리나 파일 앱에 저장해두세요
- 백업 후 새 기기에서 복구할 때 같은 카카오 계정을 사용해야 해요
- 복구된 대화는 복구 시점 이전의 메시지만 포함해요. 복구 후 새로 받은 메시지는 실시간으로 쌓여요

## 자주 묻는 질문

**Q. 카카오톡 백업은 어디에 저장되나요?**
안드로이드는 카카오 서버에, 아이폰은 iCloud 또는 카카오 서버에 저장돼요. 카카오 계정에 연결되어 있어요.

**Q. 백업 용량 제한이 있나요?**
카카오 서버 백업은 현재 용량 제한이 별도로 공지되지 않았어요. iCloud 백업은 iCloud 여유 공간에 따라 제한이 있어요.

**Q. 오래전 대화도 복구되나요?**
가장 최근 백업 시점까지의 대화가 복구돼요. 백업 전 삭제된 대화는 복구되지 않아요.
