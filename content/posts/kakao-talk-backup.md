## 카카오톡 백업이 필요한 이유

폰을 바꾸거나 앱을 재설치하면 채팅 내용이 사라질 수 있어요. 미리 백업해두면 언제든 복구할 수 있어요.

## Android 백업 방법

1. 카카오톡 **더보기(···) → 설정 → 채팅**
2. **대화 백업** 클릭
3. **지금 백업하기** 클릭
4. 카카오 계정 비밀번호 입력 후 백업 완료

## iPhone 백업 방법

1. 카카오톡 **더보기 → 설정 → 채팅 → 대화 백업**
2. **iCloud로 백업** 선택
3. 백업 완료까지 대기

## 새 폰에서 복구하는 방법

1. 새 폰에 카카오톡 설치 후 **같은 계정**으로 로그인
2. 로그인 시 **"이전 대화 불러오기"** 팝업 확인
3. **불러오기** 클릭 → 복구 완료

## 개별 채팅방 내보내기

특정 채팅방만 저장하고 싶을 때:

1. 채팅방 오른쪽 상단 **메뉴(≡)**
2. **채팅방 설정 → 대화 내용 내보내기**
3. 텍스트 파일로 저장됨
