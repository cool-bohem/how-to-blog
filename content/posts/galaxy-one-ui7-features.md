## One UI 7이란 무엇인가요?

삼성 갤럭시 스마트폰의 새로운 운영체제 인터페이스인 **One UI 7**은 기존 One UI 6에서 한 단계 진화한 버전입니다. 2026년 현재 갤럭시 S25 시리즈를 포함한 다양한 갤럭시 기기에 탑재되어 있으며, 더욱 직관적인 UI 디자인, 강화된 Galaxy AI 기능, 개선된 개인정보 보호 기능을 제공합니다.

이 가이드에서는 One UI 7의 핵심 기능을 처음 접하는 분들도 쉽게 이해하고 활용할 수 있도록 단계별로 설명합니다.

---

## One UI 7 주요 변경사항 한눈에 보기

One UI 7에서 가장 크게 달라진 부분은 다음과 같습니다.

- **새로운 잠금화면 디자인**: 시계, 날짜, 위젯 배치가 자유로워졌습니다.
- **Now Bar(나우 바)**: 화면 하단에 현재 진행 중인 활동(음악 재생, 타이머, 내비게이션 등)을 실시간으로 표시합니다.
- **Galaxy AI 기능 강화**: 통화 요약, 메모 정리, 이미지 편집 AI 기능이 대폭 향상되었습니다.
- **알림 패널 분리**: 빠른 설정과 알림이 완전히 분리되어 한 손 사용이 더욱 편리해졌습니다.
- **앱 서랍 기본 제공**: 설치된 앱을 별도의 서랍에서 관리할 수 있습니다.

## 새로운 홈화면 및 잠금화면 설정하기

### 1단계: 잠금화면 편집 시작하기

1. 잠금화면이 표시된 상태에서 화면을 **길게 누릅니다** (약 2초).
2. PIN 또는 패턴을 입력하라는 메시지가 뜨면 인증을 완료합니다.
3. 잠금화면 편집 모드로 진입됩니다. 상단에는 시계 스타일 선택 메뉴가 나타납니다.

### 2단계: 시계 스타일 변경하기

1. 잠금화면 편집 모드에서 시계 영역을 탭합니다.
2. 아날로그, 디지털, 월드 클락 등 다양한 시계 스타일 중 원하는 것을 선택합니다.
3. 시계 색상과 글꼴도 개인 취향에 맞게 변경할 수 있습니다.
4. 오른쪽 하단의 **완료** 버튼을 눌러 저장합니다.

### 3단계: 잠금화면 위젯 추가하기

1. 잠금화면 편집 모드에서 시계 아래 **위젯 추가** 영역을 탭합니다.
2. 날씨, 음악 플레이어, 달력, 피트니스 정보 등 원하는 위젯을 선택합니다.
3. 최대 2개의 위젯을 잠금화면에 추가할 수 있습니다.

### 4단계: 홈화면 그리드 변경하기

홈화면을 길게 눌러 편집 모드로 진입한 후, 하단의 **설정** 아이콘을 탭합니다. 여기서 앱 그리드 크기(4×6, 5×6 등)를 변경할 수 있습니다. 그리드가 작을수록 아이콘이 커져 사용이 편리하고, 그리드가 클수록 더 많은 앱을 한 화면에 배치할 수 있습니다.

## Now Bar(나우 바) 활용하기

Now Bar는 One UI 7의 가장 독창적인 기능 중 하나입니다. 화면 하단 중앙에 현재 진행 중인 작업을 표시하는 동적 인터페이스로, 아이폰의 Dynamic Island와 유사한 역할을 합니다.

### Now Bar가 표시하는 정보

- **음악/팟캐스트 재생 중**: 재생 중인 곡 제목과 재생/일시정지 버튼
- **타이머 실행 중**: 남은 시간 카운트다운
- **통화 중**: 현재 통화 상대방 이름과 통화 시간
- **내비게이션 중**: 다음 경로 안내 정보
- **녹음 중**: 음성 녹음 진행 상태

### Now Bar 설정 방법

1. **설정** 앱을 엽니다.
2. **디스플레이** → **Now Bar**를 선택합니다.
3. Now Bar를 표시할 앱을 개별적으로 켜거나 끌 수 있습니다.
4. Now Bar를 탭하면 해당 앱으로 바로 이동하고, 길게 누르면 빠른 컨트롤이 나타납니다.

## Galaxy AI 기능 완벽 활용하기

One UI 7에서 Galaxy AI 기능은 더욱 강력해졌습니다. 인터넷 연결 없이 기기 자체에서 처리되는 온디바이스 AI 기능도 크게 확대되었습니다.

### 통화 녹음 및 자동 요약

1. 전화 통화 중 화면 상단의 **녹음** 버튼을 탭합니다.
2. 통화 종료 후 자동으로 녹음 파일이 저장됩니다.
3. **녹음 앱**을 열어 해당 파일을 탭하면 AI가 자동으로 통화 내용을 텍스트로 변환하고 핵심 내용을 요약해줍니다.
4. 영어, 한국어, 일본어 등 다국어 통화도 자동 번역 요약이 지원됩니다.

### 메모 정리 기능 (노트 앱)

1. **Samsung Notes** 앱을 엽니다.
2. 메모를 작성한 후 오른쪽 하단의 **AI 아이콘**(별 모양)을 탭합니다.
3. **서식 지정**, **요약**, **철자 교정**, **번역** 중 원하는 기능을 선택합니다.
4. AI가 자동으로 메모를 정리하거나 요약해 제공합니다.

### 이미지 편집 AI (생성형 편집)

1. **갤러리** 앱에서 편집할 사진을 엽니다.
2. 하단의 **편집** 버튼 → **생성형 편집**을 탭합니다.
3. 손가락으로 지우고 싶은 부분을 선택하거나, 사진 속 사람을 이동시킬 영역을 표시합니다.
4. **생성** 버튼을 누르면 AI가 자연스럽게 배경을 채우거나 사람/사물의 위치를 조정해줍니다.

## 알림 패널 및 빠른 설정 사용하기

One UI 7에서는 알림 패널과 빠른 설정 패널이 완전히 분리되었습니다.

### 알림 패널 여는 방법

- 화면 **왼쪽 상단**에서 아래로 스와이프하면 **알림 목록**이 표시됩니다.
- 알림을 왼쪽으로 밀면 해당 앱의 알림을 모두 끄는 옵션이 나타납니다.
- 오른쪽으로 밀면 알림이 삭제됩니다.

### 빠른 설정 패널 여는 방법

- 화면 **오른쪽 상단**에서 아래로 스와이프하면 **Wi-Fi, 블루투스, 화면 밝기** 등 빠른 설정 버튼이 표시됩니다.
- 두 손가락으로 아래로 스와이프하면 전체 빠른 설정 화면으로 바로 이동합니다.

### 빠른 설정 버튼 순서 변경하기

1. 빠른 설정 패널을 연 상태에서 오른쪽 상단의 **편집(연필 아이콘)**을 탭합니다.
2. 자주 사용하는 버튼을 상단으로 드래그하여 이동합니다.
3. 사용하지 않는 버튼은 아래 영역으로 내려 숨길 수 있습니다.
4. **완료**를 눌러 저장합니다.

## 개인정보 보호 및 보안 설정

One UI 7은 개인정보 보호 기능도 크게 강화되었습니다.

### 앱 권한 관리

1. **설정** → **개인정보 보호** → **권한 관리자**로 이동합니다.
2. 카메라, 마이크, 위치 등 민감한 권한을 어떤 앱이 사용 중인지 확인할 수 있습니다.
3. 의심스러운 앱의 권한은 즉시 취소하는 것이 좋습니다.

### 개인정보 보호 대시보드 활용

1. **설정** → **개인정보 보호** → **개인정보 보호 대시보드**를 탭합니다.
2. 최근 24시간 동안 어떤 앱이 카메라, 마이크, 위치 정보에 접근했는지 타임라인으로 확인할 수 있습니다.
3. 불필요한 접근이 있었던 앱은 권한을 즉시 차단하세요.

### 최대 절전 모드 설정

배터리를 오래 유지하고 싶을 때 유용한 기능입니다.
1. **설정** → **배터리** → **절전 모드**로 이동합니다.
2. **최대 절전 모드**를 활성화하면 전화와 문자 등 기본 기능만 작동하여 배터리를 최대로 아낄 수 있습니다.

## 자주 묻는 질문 (Q&A)

**Q. One UI 7 업데이트는 어떻게 받나요?**
A. **설정** → **소프트웨어 업데이트** → **다운로드 및 설치**를 탭하면 됩니다. Wi-Fi 환경에서 진행하는 것을 권장하며, 배터리가 50% 이상인 상태에서 업데이트를 시작하세요.

**Q. One UI 7로 업데이트 후 기존 설정이 사라지나요?**
A. 기본적으로 앱과 데이터는 유지됩니다. 다만 홈화면 레이아웃이 일부 변경될 수 있으므로, 업데이트 전에 Smart Switch로 백업해두는 것을 권장합니다.

**Q. Now Bar가 표시되지 않아요.**
A. **설정** → **디스플레이** → **Now Bar**에서 기능이 켜져 있는지 확인하세요. 또한 해당 앱에 대한 Now Bar 권한도 개별적으로 켜져 있어야 합니다.

**Q. Galaxy AI 기능을 사용하려면 인터넷이 필요한가요?**
A. 일부 기능(통화 녹음 요약, 생성형 편집 등)은 인터넷 없이도 기기 자체적으로 동작합니다. 하지만 더 복잡한 AI 작업은 서버 연결이 필요할 수 있습니다.

**Q. One UI 7로 업그레이드하면 앱이 느려지나요?**
A. 최적화가 잘 되어 있어 갤럭시 S23 이상의 기기에서는 큰 성능 저하 없이 사용할 수 있습니다. 다만 구형 기기에서는 일부 AI 기능이 지원되지 않을 수 있습니다.

**Q. 잠금화면 위젯이 보이지 않아요.**
A. **설정** → **잠금화면 및 AOD** → **위젯**에서 위젯 표시 옵션이 켜져 있는지 확인하세요.
