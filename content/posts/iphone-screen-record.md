## 아이폰 화면 녹화란?

아이폰에는 화면에서 일어나는 모든 것을 영상으로 저장할 수 있는 화면 녹화 기능이 내장되어 있어요. 별도 앱 설치 없이 iOS 기본 기능으로 사용할 수 있어요.

**화면 녹화가 유용한 상황**:
- 앱 사용법을 누군가에게 설명할 때
- 게임 플레이를 기록하고 싶을 때
- 영상통화, 온라인 수업 등을 저장하고 싶을 때
- 버그나 오류 상황을 캡처해서 보여줄 때
- 인스타그램, 틱톡 등 직접 저장이 안 되는 콘텐츠를 남길 때

## 제어 센터에 화면 녹화 버튼 추가하기

처음 아이폰을 쓴다면 제어 센터에 화면 녹화 버튼이 없을 수 있어요. 먼저 추가해야 해요.

1. **설정 → 제어 센터** 이동
2. 아래쪽 **추가할 수 있는 컨트롤** 항목에서 **화면 녹화** 찾기
3. 왼쪽 **+** 버튼 클릭
4. 위쪽 **포함된 컨트롤** 목록에 추가됨

**팁**: 한 번만 설정하면 돼요. 이후엔 언제든 제어 센터에서 바로 녹화를 시작할 수 있어요.

## 화면 녹화 시작하기

1. 화면 오른쪽 위에서 **아래로 쓸어내리기** 로 제어 센터 열기
   (홈 버튼이 있는 구형 아이폰은 화면 아래에서 위로 쓸어올리기)
2. **화면 녹화 버튼(동그라미 안에 점)** 클릭
3. **3초 카운트다운** 후 녹화 시작
4. 상단 상태바가 **빨간색**으로 바뀌면 녹화 중이에요

**팁**: 3초 카운트다운 동안 제어 센터를 닫고 녹화할 화면으로 이동할 수 있어요.

## 화면 녹화 종료하기

**방법 1. 상태바 터치**
1. 상단 빨간색 상태바 클릭
2. **중단** 클릭

**방법 2. 제어 센터에서 종료**
1. 제어 센터 열기
2. 녹화 버튼(빨간색으로 변해 있음) 다시 클릭

녹화가 끝나면 **사진 앱**에 자동으로 저장돼요.

## 소리도 함께 녹화하려면 (마이크 소리 포함)

기본 설정에서는 화면 소리(앱 소리)는 녹음되지만, 마이크 소리(내 목소리)는 녹음되지 않아요. 설명 영상이나 통화 녹화 등을 할 때는 마이크를 켜야 해요.

1. 화면 녹화 버튼을 **꾹 누르기** (길게 누르기)
2. 마이크 아이콘이 표시됨
3. **마이크** 아이콘 클릭해서 **켜기** 상태로 변경 (빨간색으로 바뀜)
4. **녹화 시작** 클릭

**팁**: 마이크 설정은 다음 번 녹화 때도 유지되지 않아요. 매번 필요할 때만 켜주세요.

## 녹화된 영상 편집하기

사진 앱에서 바로 편집할 수 있어요.

1. **사진 앱** 에서 녹화된 영상 열기
2. **편집** 클릭
3. 앞뒤 불필요한 부분 잘라내기 (트림)
4. **완료** 클릭 후 저장

## 주의사항

- **알림 노출**: 녹화 중 카카오톡 등 알림이 뜨면 알림 내용도 영상에 담겨요. 방해금지 모드를 켜두는 게 좋아요
- **DRM 콘텐츠**: 넷플릭스, 왓챠 등 스트리밍 서비스는 화면 보호 기술로 인해 녹화해도 화면이 검게 나와요
- **저작권**: 화면 녹화한 영상을 SNS 등에 올릴 때는 저작권을 확인하세요
- **용량**: 고화질로 녹화하면 분당 수백 MB가 될 수 있어요. 저장 공간을 미리 확인하세요

## 자주 묻는 질문

**Q. 화면 녹화 중 전화가 오면 어떻게 되나요?**
전화를 받으면 녹화가 중단돼요. 중요한 녹화 중에는 방해금지 모드를 켜두는 게 좋아요.

**Q. 화면 녹화 영상을 유튜브나 카카오톡으로 보내려면?**
사진 앱에서 해당 영상을 선택 → 공유 버튼 → 원하는 앱 선택으로 보낼 수 있어요. 단, 영상 파일이 크면 압축되거나 전송이 실패할 수 있어요.

**Q. 녹화 중 화면 밝기를 바꿔도 되나요?**
네. 녹화 중에도 밝기 조절, 앱 전환 등 모든 조작이 가능해요.
