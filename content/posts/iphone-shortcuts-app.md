## 아이폰 단축어 앱이란?

아이폰의 **단축어(Shortcuts)** 앱은 여러 작업을 하나의 버튼으로 자동화할 수 있는 강력한 기능입니다. 예를 들어, 아침에 일어나면 자동으로 알람을 끄고, 날씨를 확인하고, 좋아하는 음악을 재생하는 과정을 한 번의 탭으로 처리할 수 있습니다.

단축어 앱은 iOS 12부터 기본 앱으로 포함되어 있으며, 2026년 현재 iOS 19에서는 더욱 강력하고 직관적인 기능을 제공합니다. 프로그래밍 지식이 전혀 없어도 누구나 쉽게 나만의 자동화 루틴을 만들 수 있습니다.

## 단축어 앱 기본 구조 이해하기

단축어 앱에는 크게 세 가지 탭이 있습니다.

- **단축어**: 내가 만들거나 저장한 단축어 목록
- **자동화**: 특정 조건(시간, 위치, 앱 실행 등)에 따라 자동으로 실행되는 작업
- **갤러리**: 애플이 미리 만들어둔 유용한 단축어 모음

## 나만의 단축어 만들기 (기초)

### 1단계: 단축어 앱 열기

홈 화면에서 **단축어** 앱을 찾아 탭합니다. 앱 아이콘은 알록달록한 정사각형 모양입니다. 만약 앱이 보이지 않는다면 앱 라이브러리에서 "단축어"를 검색하세요.

### 2단계: 새 단축어 만들기 시작

화면 오른쪽 상단의 **+** 버튼을 탭합니다. 새 단축어 편집 화면이 열립니다.

### 3단계: 작업 추가하기

**동작 추가** 버튼을 탭하면 수백 가지 동작 목록이 나타납니다. 검색창에 원하는 동작을 검색하거나 카테고리별로 찾을 수 있습니다. 예를 들어 "밝기"를 검색하면 화면 밝기를 조절하는 동작이 나타납니다.

### 4단계: 동작 순서 조정

여러 동작을 추가한 뒤에는 각 동작 왼쪽의 **≡** 아이콘을 길게 누른 채 위아래로 드래그해서 순서를 바꿀 수 있습니다.

### 5단계: 단축어 이름 설정 및 저장

화면 상단의 단축어 이름을 탭해서 알아보기 쉬운 이름을 입력합니다. 그다음 오른쪽 상단의 **완료** 버튼을 눌러 저장합니다.

## 유용한 단축어 예시 5가지

### 예시 1: 취침 모드 단축어

잠자리에 들기 전 한 번의 탭으로 여러 설정을 자동화할 수 있습니다.

1. 단축어 앱에서 + 탭
2. 다음 동작들을 순서대로 추가:
   - **밝기 설정** → 0%로 설정
   - **방해금지 모드 켜기**
   - **알람 설정** → 기상 시간 입력
   - **음악 일시정지**
3. 이름을 "취침 모드"로 저장
4. 홈 화면에 추가하면 바로 사용 가능

### 예시 2: 출근 준비 단축어

1. **날씨 가져오기** → 오늘의 날씨 정보 조회
2. **텍스트 말하기** → "오늘 날씨는 [날씨]입니다"
3. **음악 재생** → 아침 플레이리스트 시작
4. **밝기 설정** → 100%로 설정

이렇게 만들면 아침에 단축어를 실행할 때 오늘 날씨를 음성으로 알려주고 음악을 틀어줍니다.

### 예시 3: 와이파이 자동 연결 단축어

특정 장소(집, 회사)에 도착했을 때 자동으로 해당 와이파이에 연결하도록 설정할 수 있습니다. 이는 자동화 탭에서 **위치** 조건으로 설정합니다.

### 예시 4: 사진 크기 줄이기 단축어

공유 메뉴에서 바로 사진 용량을 줄일 수 있는 단축어입니다.

1. **사진 선택** 동작 추가
2. **이미지 크기 조정** → 원하는 크기(예: 1000px) 입력
3. **사진 앱에 저장** 동작 추가
4. 이름을 "사진 압축"으로 저장

이후 사진 앱에서 공유 버튼을 누르면 단축어 목록에 "사진 압축"이 나타납니다.

### 예시 5: 카카오택시 바로 부르기

자주 가는 목적지(예: 집)로 바로 카카오택시를 호출하는 단축어입니다.

1. **URL 열기** 동작 추가
2. URL에 카카오택시 딥링크 입력 (카카오T 앱의 딥링크 주소)
3. "집으로 택시" 이름으로 저장

## 자동화 설정 방법

자동화는 내가 직접 실행하지 않아도 특정 조건이 충족되면 자동으로 단축어가 실행되는 기능입니다.

### 자동화 만들기

1. 단축어 앱 하단의 **자동화** 탭 선택
2. 오른쪽 상단 **+** 버튼 탭
3. 조건 선택:
   - **시간대**: 특정 시간에 실행
   - **알람**: 알람이 울릴 때 실행
   - **앱**: 특정 앱을 열거나 닫을 때 실행
   - **위치**: 특정 장소에 도착하거나 떠날 때 실행
   - **충전**: 충전 시작/완료 시 실행
   - **배터리 잔량**: 배터리가 특정 % 이하일 때 실행

### 자동화 예시: 충전 완료 시 알림

1. 자동화 탭에서 + 탭
2. **배터리 잔량** 선택 → 100% 설정
3. **알림 보내기** 동작 추가 → "충전 완료! 충전기를 뽑아주세요" 입력
4. 저장

이제 아이폰 배터리가 100%가 되면 자동으로 알림이 울립니다.

## 갤러리에서 단축어 가져오기

처음부터 만들기 어렵다면 갤러리를 활용하세요.

1. 단축어 앱에서 **갤러리** 탭 선택
2. 카테고리를 둘러보거나 검색창에서 원하는 기능 검색
3. 마음에 드는 단축어 탭
4. **단축어 추가** 버튼 탭

갤러리에는 "GIF 만들기", "PDF 변환", "집으로 전화하기", "영수증 스캔" 등 수백 가지의 실용적인 단축어가 있습니다.

## 홈 화면에 단축어 추가하기

자주 사용하는 단축어는 홈 화면에 아이콘으로 추가할 수 있습니다.

1. 단축어 목록에서 원하는 단축어 길게 누르기
2. **세부사항** 선택 (또는 단축어 편집 후 ... 메뉴)
3. **홈 화면에 추가** 탭
4. 아이콘 이미지와 이름 커스터마이즈 가능
5. **추가** 버튼 탭

이렇게 하면 앱처럼 홈 화면에서 바로 단축어를 실행할 수 있습니다. 심지어 아이콘 이미지도 원하는 사진으로 바꿀 수 있어서 나만의 홈 화면을 꾸미는 데도 활용됩니다.

## 단축어 공유 및 가져오기

다른 사람이 만든 단축어를 인터넷에서 찾아 설치할 수 있습니다.

1. 단축어 공유 링크(일반적으로 icloud.com/shortcuts/... 형식)를 사파리로 열기
2. **단축어 가져오기** 버튼 탭
3. 단축어 내용 검토 후 **신뢰할 수 없는 단축어 추가** 허용

단, 출처를 알 수 없는 단축어는 보안 위험이 있을 수 있으니 신뢰할 수 있는 커뮤니티(예: Reddit의 r/shortcuts, 국내 애플 커뮤니티)에서 검증된 단축어만 설치하세요.

## 자주 묻는 질문 (Q&A)

**Q. 단축어가 실행은 되는데 원하는 대로 동작하지 않아요.**
A. 각 동작의 설정값을 다시 확인하세요. 예를 들어 '밝기 설정' 동작에서 퍼센트 값이 올바른지, '앱 열기' 동작에서 앱이 제대로 선택되어 있는지 점검하세요. 동작을 하나씩 테스트하면서 문제 되는 부분을 찾는 것이 좋습니다.

**Q. 자동화를 설정했는데 자동으로 실행되지 않아요.**
A. 자동화 설정 시 "실행 전 묻기" 옵션이 켜져 있으면 매번 확인을 요청합니다. 완전 자동 실행을 원한다면 자동화 편집 화면에서 이 옵션을 끄세요. 단, 일부 민감한 동작(메시지 보내기 등)은 항상 확인을 요구합니다.

**Q. 단축어 앱이 보이지 않아요.**
A. 설정 → 스크린 타임 → 콘텐츠 및 개인 정보 보호 제한에서 단축어 앱이 차단되어 있을 수 있습니다. 또는 앱 라이브러리(홈 화면을 왼쪽으로 쭉 밀면 나오는 화면)에서 "단축어"를 검색해보세요.

**Q. 단축어를 삭제하고 싶어요.**
A. 단축어 목록에서 삭제할 단축어를 길게 누른 후 **삭제** 를 선택하거나, 단축어를 왼쪽으로 스와이프하면 빨간 삭제 버튼이 나타납니다.

**Q. 시리(Siri)로 단축어를 실행할 수 있나요?**
A. 네, 가능합니다. 단축어 편집 화면에서 오른쪽 상단의 **...** 버튼을 탭하고 **시리에 추가**를 선택하면 음성 명령어를 녹음할 수 있습니다. 이후 "시리야, [녹음한 명령어]"라고 말하면 단축어가 실행됩니다.

## 요약

아이폰 단축어 앱은 처음에는 복잡해 보이지만, 기본 구조를 이해하면 누구나 쉽게 사용할 수 있습니다. 핵심 포인트를 정리하면:

- 단축어 앱의 **+** 버튼으로 새 단축어를 만들고 동작을 추가합니다.
- **갤러리**에서 미리 만들어진 단축어를 가져다 쓸 수도 있습니다.
- **자동화** 탭에서 시간, 위치, 충전 상태 등 조건에 따라 자동 실행을 설정합니다.
- 자주 쓰는 단축어는 **홈 화면에 추가**해 앱처럼 편하게 실행하세요.
- 다른 사람이 만든 단축어를 공유 링크로 가져올 수 있지만 신뢰할 수 있는 출처인지 반드시 확인하세요.

단축어 앱을 잘 활용하면 매일 반복하는 작업을 크게 줄여 스마트폰을 더 스마트하게 사용할 수 있습니다. 오늘 하나의 단축어부터 시작해 보세요!
