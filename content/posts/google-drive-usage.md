## 구글 드라이브란 무엇인가요?

**구글 드라이브(Google Drive)**는 구글이 제공하는 무료 클라우드 저장 서비스입니다. 구글 계정 하나만 있으면 무료로 **15GB**의 저장 공간을 사용할 수 있으며, 사진, 문서, 동영상 등 거의 모든 파일 형식을 저장하고 어디서든 접근할 수 있습니다.

2026년 현재 구글 드라이브는 PC 웹 브라우저, 스마트폰 앱, 태블릿 등 다양한 기기에서 사용할 수 있으며, 구글 문서, 스프레드시트, 프레젠테이션과 완벽히 연동되어 업무와 학습에 매우 유용합니다.

## 구글 드라이브 시작하기

### 1단계: 구글 계정 만들기 (이미 있다면 건너뛰세요)

1. 크롬 브라우저에서 `google.com`을 엽니다.
2. 오른쪽 상단의 **로그인** 버튼을 클릭합니다.
3. **계정 만들기**를 선택하고 이름, 이메일 주소, 비밀번호를 입력합니다.
4. 전화번호 인증을 완료하면 계정이 생성됩니다.

### 2단계: 구글 드라이브 접속하기

1. 크롬 브라우저에서 `drive.google.com`을 입력합니다.
2. 구글 계정으로 로그인되어 있다면 바로 드라이브 화면이 열립니다.
3. 로그인이 안 되어 있다면 이메일과 비밀번호를 입력해 로그인합니다.

처음 접속하면 **내 드라이브** 화면이 나타납니다. 왼쪽에는 폴더 트리, 오른쪽에는 저장된 파일 목록이 표시됩니다.

## 파일 업로드하기

### PC에서 파일 업로드하는 방법

**방법 1: 드래그 앤 드롭**
1. 업로드할 파일을 바탕화면이나 탐색기에서 찾습니다.
2. 파일을 마우스로 클릭한 채 구글 드라이브 화면으로 끌어다 놓습니다.
3. 화면에 파란색 테두리가 생기면 손을 놓으면 업로드가 시작됩니다.
4. 화면 오른쪽 하단에 업로드 진행 상황이 표시됩니다.

**방법 2: 업로드 버튼 사용**
1. 구글 드라이브 화면 왼쪽 상단의 **+ 새로 만들기** 버튼을 클릭합니다.
2. **파일 업로드**를 선택합니다.
3. 업로드할 파일을 선택하고 **열기**를 클릭합니다.

### 폴더 전체 업로드하기

1. **+ 새로 만들기** → **폴더 업로드**를 선택합니다.
2. 업로드할 폴더를 선택합니다.
3. 폴더 안의 모든 파일이 구글 드라이브에 업로드됩니다.

## 폴더 만들기 및 파일 정리하기

### 새 폴더 만들기

1. **+ 새로 만들기** → **폴더**를 클릭합니다.
2. 폴더 이름을 입력합니다. 예: "2026년 업무", "가족 사진", "학교 자료"
3. **만들기**를 클릭합니다.

### 파일을 폴더로 이동하기

1. 이동할 파일 위에서 **마우스 오른쪽 버튼**을 클릭합니다.
2. **이동**을 선택합니다.
3. 이동할 폴더를 클릭하고 **여기로 이동**을 클릭합니다.

또는 파일을 마우스로 드래그하여 원하는 폴더 위에 놓으면 됩니다.

### 별표(중요 표시) 기능 활용하기

자주 사용하는 파일이나 폴더에 별표를 달아두면 빠르게 찾을 수 있습니다.
1. 파일 위에서 마우스 오른쪽 클릭 → **중요 문서함에 추가**를 선택합니다.
2. 왼쪽 메뉴의 **중요 문서함**을 클릭하면 별표 표시한 파일만 모아볼 수 있습니다.

## 파일 공유하기

구글 드라이브의 가장 강력한 기능 중 하나가 바로 **파일 공유**입니다. 이메일로 파일을 주고받는 번거로움 없이, 링크 하나로 파일을 공유할 수 있습니다.

### 특정 사람에게 공유하기

1. 공유할 파일이나 폴더 위에서 마우스 오른쪽 버튼 → **공유**를 클릭합니다.
2. **공유** 창이 열리면 상대방의 이메일 주소를 입력합니다.
3. 권한을 설정합니다:
   - **뷰어**: 파일 보기만 가능
   - **댓글 작성자**: 보기 + 댓글 달기 가능
   - **편집자**: 파일 수정 가능
4. **보내기**를 클릭하면 상대방 이메일로 공유 알림이 발송됩니다.

### 링크로 공유하기 (카카오톡, 문자로 전달할 때)

1. 공유할 파일 위에서 마우스 오른쪽 버튼 → **링크 복사**를 클릭합니다.
2. 기본 설정은 **"링크가 있는 모든 사용자"로 변경**해야 링크를 받은 누구나 열 수 있습니다.
3. **링크 복사** 버튼을 클릭하면 클립보드에 링크가 복사됩니다.
4. 카카오톡이나 문자로 해당 링크를 붙여넣기하여 전송합니다.

> **주의**: "링크가 있는 모든 사용자"로 설정 시 링크를 아는 누구나 파일에 접근할 수 있습니다. 민감한 파일은 특정 이메일로만 공유하세요.

## 구글 문서, 스프레드시트 활용하기

구글 드라이브에서는 별도의 프로그램 설치 없이 브라우저에서 바로 문서를 작성하고 편집할 수 있습니다.

### 구글 문서(Google Docs) 만들기

1. **+ 새로 만들기** → **Google 문서**를 클릭합니다.
2. 새 탭에서 빈 문서가 열립니다. 마치 워드나 한글처럼 텍스트를 입력합니다.
3. **자동 저장** 기능이 기본으로 켜져 있어 별도로 저장 버튼을 누르지 않아도 됩니다.
4. 제목을 클릭하여 파일명을 변경하면 구글 드라이브에 그 이름으로 저장됩니다.

### 구글 스프레드시트(Google Sheets) 활용하기

1. **+ 새로 만들기** → **Google 스프레드시트**를 클릭합니다.
2. 엑셀과 거의 동일한 인터페이스로, SUM, AVERAGE 등 기본 함수를 모두 지원합니다.
3. 엑셀 파일(.xlsx)을 드라이브에 업로드한 후 더블클릭하면 구글 스프레드시트로 바로 열어 편집할 수 있습니다.

### 기존 Office 파일 편집하기

구글 드라이브에 업로드한 Word(.docx), Excel(.xlsx), PowerPoint(.pptx) 파일을 구글 앱으로 바로 열어 편집하고 다시 원본 형식으로 저장할 수 있습니다.

## 스마트폰 앱으로 구글 드라이브 사용하기

### 앱 설치 방법

- **안드로이드(갤럭시)**: 구글 플레이스토어에서 "구글 드라이브" 검색 후 설치 (대부분 기본 설치되어 있음)
- **아이폰**: 앱스토어에서 "Google Drive" 검색 후 설치

### 앱에서 파일 업로드하기

1. 드라이브 앱을 실행합니다.
2. 오른쪽 하단의 **+ 버튼**을 탭합니다.
3. **업로드**를 탭하고 갤러리나 파일 앱에서 업로드할 파일을 선택합니다.

### 오프라인에서 파일 사용하기

인터넷이 없는 상황에서도 파일을 열어볼 수 있도록 미리 오프라인 저장을 설정할 수 있습니다.
1. 오프라인으로 사용할 파일을 길게 누릅니다.
2. 점 세 개(⋮) 메뉴 → **오프라인으로 사용 가능하게 설정**을 탭합니다.
3. 이후 인터넷 없이도 해당 파일을 열고 편집할 수 있습니다.

### 카메라로 문서 스캔하기

1. 드라이브 앱에서 **+ 버튼** → **스캔**을 탭합니다.
2. 스마트폰 카메라로 문서, 영수증, 계약서 등을 촬영합니다.
3. 자동으로 PDF 파일로 변환되어 구글 드라이브에 저장됩니다.

## 저장 용량 관리하기

구글 계정 무료 용량은 15GB이며, Gmail, 구글 포토, 구글 드라이브가 이 공간을 공유합니다.

### 현재 저장 용량 확인하기

1. 구글 드라이브 왼쪽 하단의 **저장 용량** 섹션을 확인합니다.
2. 전체 사용량과 남은 용량이 표시됩니다.

### 용량 절약하는 방법

1. 크기가 큰 파일부터 삭제하기: **저장용량** 클릭 → 파일 크기 순으로 정렬하여 불필요한 파일 삭제
2. 휴지통 비우기: 삭제한 파일은 30일간 휴지통에 남아있으며 용량을 차지합니다. 왼쪽 메뉴의 **휴지통** → **휴지통 비우기**로 완전히 삭제
3. 구글 포토의 경우 고화질 저장 설정 시 구글이 압축하여 용량 절약 가능

### 추가 저장 공간 구매하기

무료 15GB로 부족하다면 구글 원(Google One)을 통해 추가 용량을 구매할 수 있습니다.
- 100GB: 월 2,900원
- 200GB: 월 3,900원
- 2TB: 월 11,900원 (2026년 기준 가격)

## 자주 묻는 질문 (Q&A)

**Q. 구글 드라이브에 올린 파일이 삭제되면 어떻게 되나요?**
A. 삭제한 파일은 **휴지통**에 30일간 보관됩니다. 30일 이내에 복원하려면 휴지통에서 해당 파일을 찾아 마우스 오른쪽 클릭 → **복원**을 선택하세요.

**Q. 가족이나 친구와 구글 드라이브를 같이 쓸 수 있나요?**
A. 네, **공유 드라이브** 또는 폴더 공유 기능을 사용하면 여러 명이 같은 폴더를 함께 사용할 수 있습니다. 단, 공유 드라이브는 구글 워크스페이스(유료) 기능입니다.

**Q. 인터넷이 없을 때는 구글 드라이브를 사용할 수 없나요?**
A. 미리 오프라인 사용 설정을 해둔 파일은 인터넷 없이도 열고 편집할 수 있습니다. 편집 내용은 인터넷 연결 후 자동으로 동기화됩니다.

**Q. 구글 드라이브에 올린 파일을 구글이 볼 수 있나요?**
A. 구글은 파일 내용을 직접 열람하지 않지만, 서비스 약관에 따라 일부 데이터를 처리할 수 있습니다. 민감한 개인정보나 기업 기밀 자료는 암호화 후 업로드하거나 별도 보안 솔루션을 사용하는 것을 권장합니다.

**Q. 드라이브에 업로드한 동영상을 그냥 재생할 수 있나요?**
A. 네, 구글 드라이브에서 동영상 파일을 클릭하면 별도 앱 없이 브라우저나 앱 내에서 바로 재생할 수 있습니다. MP4, MOV, AVI 등 주요 동영상 형식을 지원합니다.

**Q. 파일 이름으로 검색할 수 있나요?**
A. 네, 상단 검색창에 파일 이름, 파일 내용(문서 내 텍스트), 파일 유형 등을 입력하면 검색됩니다. 구글의 강력한 검색 기술이 적용되어 있어 파일 이름이 정확하지 않아도 찾을 수 있습니다.
