## 원드라이브란 무엇인가요?

원드라이브(OneDrive)는 마이크로소프트가 제공하는 클라우드 저장 서비스입니다. 쉽게 말해 인터넷상의 거대한 저장 공간으로, 사진·문서·동영상 등 어떤 파일이든 저장해 두고 PC, 스마트폰, 태블릿 등 다양한 기기에서 언제든지 꺼내 볼 수 있습니다.

윈도우 11(그리고 10)에는 원드라이브가 기본으로 내장되어 있어서 별도 설치 없이 바로 사용할 수 있습니다. 마이크로소프트 계정(Outlook, Hotmail, Live 계정)만 있으면 무료로 5GB 저장 공간이 제공됩니다. Microsoft 365 구독자라면 1TB(1,000GB)의 넉넉한 공간을 사용할 수 있습니다.

---

## 1단계: 원드라이브 로그인하기 (PC 기준)

**윈도우 11에서 원드라이브 시작하기**

1. 작업 표시줄 오른쪽 아래(시스템 트레이)에 있는 **구름 모양 아이콘**을 클릭합니다. 보이지 않으면 숨겨진 아이콘 표시(위쪽 화살표 버튼)를 클릭해 찾아보세요.
2. 원드라이브가 처음 실행되면 **로그인** 화면이 나타납니다.
3. 마이크로소프트 계정 이메일 주소를 입력하고 **로그인**을 클릭합니다.
4. 비밀번호를 입력합니다.
5. 원드라이브 폴더 위치를 확인하는 화면이 나옵니다. 기본값(C:\Users\사용자이름\OneDrive)을 그대로 사용하거나 원하는 위치로 변경할 수 있습니다. 특별한 이유가 없다면 기본값으로 진행하세요.
6. **다음**을 클릭하여 설정을 완료합니다.

**마이크로소프트 계정이 없다면?**

마이크로소프트 공식 홈페이지(account.microsoft.com)에서 무료로 계정을 만들 수 있습니다. 이미 Outlook.com 이메일이 있다면 그 계정을 그대로 사용하면 됩니다.

## 2단계: 원드라이브에 파일 저장하기

원드라이브에 파일을 저장하는 방법은 매우 간단합니다.

### 방법 1: 파일 탐색기를 통해 저장

1. **파일 탐색기**(폴더 아이콘)를 엽니다.
2. 왼쪽 패널에 **OneDrive**가 표시됩니다. 클릭하면 원드라이브 폴더가 열립니다.
3. 저장하고 싶은 파일을 이 폴더 안으로 **드래그 앤 드롭**하거나 복사(Ctrl+C) 후 붙여넣기(Ctrl+V)하면 됩니다.
4. 파일 아이콘 옆에 **초록색 체크 표시**가 나타나면 클라우드에 성공적으로 동기화된 것입니다. 파란색 화살표 아이콘이 표시되는 동안은 업로드 중입니다.

### 방법 2: 저장 시 원드라이브 선택

워드·엑셀·파워포인트 등 오피스 앱에서 파일을 저장할 때 **다른 이름으로 저장 → OneDrive** 경로를 선택하면 자동으로 클라우드에 저장됩니다.

### 방법 3: 웹 브라우저에서 업로드

1. 웹 브라우저에서 **onedrive.live.com**에 접속합니다.
2. 마이크로소프트 계정으로 로그인합니다.
3. 화면 상단의 **업로드** 버튼을 클릭합니다.
4. **파일** 또는 **폴더**를 선택하여 업로드합니다.

---

## 3단계: 파일 및 폴더 공유하기

원드라이브의 강력한 기능 중 하나는 파일을 링크 하나로 간편하게 공유할 수 있다는 점입니다.

1. 파일 탐색기에서 공유하고 싶은 파일 또는 폴더를 **오른쪽 클릭**합니다.
2. 메뉴에서 **OneDrive → 공유**를 선택합니다.
3. 공유 설정 창이 열립니다. 여기서 두 가지 방식을 선택할 수 있습니다:
   - **링크 복사**: 링크를 아는 모든 사람이 파일을 볼 수 있습니다.
   - **특정 사람에게 공유**: 이메일 주소를 입력하여 특정인에게만 공유합니다.
4. 공유 권한도 설정할 수 있습니다:
   - **보기 가능**: 파일 열람만 가능, 수정 불가
   - **편집 가능**: 파일을 열고 수정할 수 있음
5. 설정 후 **링크 복사** 버튼을 클릭하면 클립보드에 링크가 복사됩니다. 이 링크를 카카오톡, 이메일 등으로 전달하세요.

**주의:** 중요한 파일은 '링크를 아는 모든 사람'으로 설정하지 말고, 특정인에게만 공유하는 것이 안전합니다.

## 4단계: PC 폴더 자동 백업(동기화) 설정

원드라이브의 폴더 백업 기능을 사용하면 PC의 바탕화면, 문서, 사진 폴더가 자동으로 클라우드에 저장됩니다. PC가 고장 나도 파일을 잃어버리지 않는 가장 확실한 방법입니다.

1. 시스템 트레이의 **원드라이브 아이콘(구름)** 을 클릭합니다.
2. 오른쪽 위의 **도움말 및 설정(톱니바퀴 아이콘)** → **설정**을 클릭합니다.
3. 설정 창에서 **동기화 및 백업** 탭을 선택합니다.
4. **백업 관리** 버튼을 클릭합니다.
5. 백업할 폴더를 선택합니다. **바탕 화면**, **문서**, **사진** 모두 체크하는 것을 권장합니다.
6. **백업 시작**을 클릭합니다.

이제 선택한 폴더의 파일은 자동으로 원드라이브에 동기화됩니다. 새 파일을 저장하거나 수정할 때마다 실시간으로 클라우드에 업데이트됩니다.

---

## 5단계: 스마트폰에서 원드라이브 사용하기

PC뿐 아니라 스마트폰으로도 원드라이브를 사용할 수 있습니다.

### 앱 설치

- **안드로이드(갤럭시)**: 구글 플레이 스토어에서 'OneDrive' 검색 후 설치
- **아이폰**: 앱 스토어에서 'OneDrive' 검색 후 설치

### 스마트폰에서 사진 자동 백업 설정

1. OneDrive 앱을 실행합니다.
2. 마이크로소프트 계정으로 로그인합니다.
3. 앱 하단의 **나(Me)** → **카메라 업로드 설정**을 탭합니다.
4. **카메라 업로드** 토글을 켭니다.
5. Wi-Fi에서만 업로드할지, 모바일 데이터도 사용할지 선택합니다. 데이터 절약을 위해 **Wi-Fi에서만**을 권장합니다.

이제 스마트폰으로 사진을 찍으면 Wi-Fi에 연결될 때 자동으로 원드라이브에 백업됩니다.

---

## 6단계: 저장 공간 효율적으로 사용하기 – 주문형 파일

원드라이브의 **주문형 파일(Files On-Demand)** 기능을 사용하면 파일을 클라우드에만 저장하고 PC 저장 공간은 절약할 수 있습니다. 필요할 때만 파일을 다운로드해 열어 볼 수 있습니다.

1. 원드라이브 설정 → **동기화 및 백업** → **고급 설정**으로 이동합니다.
2. **주문형 파일** 항목이 켜져 있는지 확인합니다.
3. 파일 탐색기에서 특정 파일을 오른쪽 클릭하면 아래 옵션이 표시됩니다:
   - **이 장치에서 항상 유지**: 오프라인에서도 사용 가능하도록 PC에 저장
   - **공간 확보**: 클라우드에만 남기고 PC에서 삭제 (파일 자체는 삭제되지 않음)

## 자주 묻는 질문 (Q&A)

**Q. 원드라이브 5GB가 꽉 찼어요. 어떻게 하나요?**

A. 불필요한 파일을 삭제하거나, 유료 요금제로 업그레이드할 수 있습니다. Microsoft 365 Personal(월 약 9,900원)을 구독하면 1TB 저장 공간과 오피스 앱 사용권이 함께 제공됩니다. 또는 무료로 저장 공간을 추가하는 이벤트에 참여하거나 지인 초대 보너스를 활용해 보세요.

**Q. 인터넷이 없을 때도 파일을 볼 수 있나요?**

A. 네, '이 장치에서 항상 유지'로 설정된 파일은 오프라인에서도 열 수 있습니다. 반면 클라우드에만 있는 파일(구름 아이콘 표시)은 인터넷 연결이 필요합니다.

**Q. 실수로 파일을 삭제했어요. 복원할 수 있나요?**

A. 원드라이브는 삭제된 파일을 30일간 **휴지통**에 보관합니다. onedrive.live.com에 접속 후 왼쪽 메뉴의 **휴지통**에서 삭제된 파일을 찾아 복원할 수 있습니다.

**Q. 원드라이브와 구글 드라이브 중 어느 것이 더 좋나요?**

A. 두 서비스 모두 훌륭합니다. 윈도우 PC와 오피스(워드·엑셀)를 주로 사용한다면 원드라이브가 자연스럽게 통합됩니다. 구글 서비스(지메일, 구글 문서)를 주로 사용한다면 구글 드라이브가 편리합니다. 안드로이드 스마트폰 사용자라면 구글 드라이브, 아이폰 사용자라면 iCloud와 원드라이브를 함께 사용하는 경우가 많습니다.

**Q. 원드라이브 동기화를 잠시 멈출 수 있나요?**

A. 네, 시스템 트레이의 원드라이브 아이콘을 클릭 → **동기화 일시 중지**에서 2시간, 8시간, 24시간 중 선택할 수 있습니다. 중요한 작업 중 네트워크 부하를 줄이고 싶을 때 유용합니다.

## 마무리 – 원드라이브로 파일 걱정 없애기

원드라이브를 활용하면 "PC가 고장 났는데 중요한 파일이 다 날아갔어요"라는 최악의 상황을 예방할 수 있습니다. 처음에 자동 백업 설정만 해두면 별다른 조작 없이 파일이 자동으로 안전하게 보관됩니다.

오늘 배운 핵심 내용 정리:
- 윈도우 11에 원드라이브가 기본 내장되어 있으며, 마이크로소프트 계정으로 바로 시작할 수 있습니다.
- 파일 탐색기의 OneDrive 폴더에 파일을 넣으면 자동으로 클라우드에 동기화됩니다.
- 폴더 백업 설정으로 바탕화면, 문서, 사진을 자동 보호할 수 있습니다.
- 링크 공유 기능으로 대용량 파일도 간편하게 전달할 수 있습니다.
- 스마트폰 앱을 설치하면 사진을 자동으로 백업할 수 있습니다.

클라우드 저장소를 처음 사용하신다면 오늘부터 바로 원드라이브를 시작해 보세요!
