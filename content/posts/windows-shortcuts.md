## 단축키를 배워야 하는 이유

마우스로 메뉴를 하나씩 클릭하는 것보다 키보드 단축키를 활용하면 같은 작업을 훨씬 빠르게 처리할 수 있습니다. 처음에는 외우는 것이 번거롭게 느껴질 수 있지만, 자주 쓰는 단축키 10~20개만 익혀도 업무 속도가 눈에 띄게 빨라집니다.

윈도우 11에서는 새로운 단축키들이 추가되면서 멀티태스킹이 더욱 편리해졌습니다. 이 글에서는 기초 단축키부터 고급 기능까지 작업 유형별로 나누어 소개합니다.

## 창(윈도우) 관리 단축키

여러 프로그램을 동시에 사용할 때 창을 빠르게 정리하는 단축키입니다.

### 1단계: 기본 창 조작

| 단축키 | 기능 |
|---|---|
| Alt + F4 | 현재 창 닫기 |
| Alt + Tab | 열려 있는 창 전환 |
| Windows키 + D | 모든 창 최소화 (바탕화면 표시) |
| Windows키 + M | 모든 창 최소화 |
| Windows키 + Shift + M | 최소화한 창 모두 복원 |

**Alt + Tab 활용법:**
Alt를 누른 채로 Tab을 여러 번 누르면 열려 있는 프로그램 목록이 표시됩니다. 원하는 창에서 Alt를 놓으면 해당 창으로 전환됩니다.

### 2단계: 창 크기 조절

| 단축키 | 기능 |
|---|---|
| Windows키 + ← | 창을 화면 왼쪽 절반으로 배치 |
| Windows키 + → | 창을 화면 오른쪽 절반으로 배치 |
| Windows키 + ↑ | 창 최대화 |
| Windows키 + ↓ | 창 최소화 또는 이전 크기로 복원 |
| Windows키 + Shift + ↑ | 창을 세로로 최대화 |

**실전 활용법:** 엑셀과 웹 브라우저를 동시에 보고 싶다면,
1. 엑셀 창에서 **Windows키 + ←** 눌러 왼쪽에 배치
2. 브라우저 창에서 **Windows키 + →** 눌러 오른쪽에 배치
마우스 없이 화면을 분할할 수 있습니다.

## 검색 및 실행 단축키

### 빠른 실행

| 단축키 | 기능 |
|---|---|
| Windows키 | 시작 메뉴 열기 및 검색창 활성화 |
| Windows키 + R | 실행 창 열기 |
| Windows키 + S | 검색창 열기 |
| Ctrl + Shift + Esc | 작업 관리자 바로 열기 |

**Windows키 + R 활용법:**
실행 창에서 아래 명령어를 입력하면 빠르게 설정에 접근할 수 있습니다.

- `notepad` → 메모장 열기
- `calc` → 계산기 열기
- `mspaint` → 그림판 열기
- `control` → 제어판 열기
- `ms-settings:` → 설정 앱 열기
- `%temp%` → 임시 파일 폴더 열기 (PC 정리 시 유용)

|---|---|
| PrtSc | 전체 화면 캡처 후 클립보드에 복사 |
| Alt + PrtSc | 현재 활성 창만 캡처 후 클립보드에 복사 |

## 가상 데스크톱 단축키

윈도우 11의 가상 데스크톱 기능을 사용하면 작업별로 화면을 분리해서 관리할 수 있습니다.

| 단축키 | 기능 |
|---|---|
| Windows키 + Tab | 작업 보기 화면 열기 |
| Windows키 + Ctrl + D | 새 가상 데스크톱 만들기 |
| Windows키 + Ctrl + → | 다음 가상 데스크톱으로 이동 |
| Windows키 + Ctrl + ← | 이전 가상 데스크톱으로 이동 |
| Windows키 + Ctrl + F4 | 현재 가상 데스크톱 닫기 |

**실전 활용법:** 
- 데스크톱 1: 업무 관련 창 (엑셀, 이메일)
- 데스크톱 2: 웹 서핑
- 데스크톱 3: 개인 용도

이렇게 분리하면 집중력이 향상되고 화면이 깔끔해집니다.

## 자주 묻는 질문 (Q&A)

**Q. 단축키가 작동하지 않아요.**
A. 한/영 전환 상태를 확인해보세요. 일부 단축키는 한글 입력 상태에서는 작동하지 않을 수 있습니다. 또한 게임 중에는 일부 Windows 단축키가 비활성화될 수 있습니다.

**Q. Alt + F4를 눌렀는데 창이 닫히지 않아요.**
A. 저장되지 않은 내용이 있으면 저장 여부를 묻는 창이 먼저 뜹니다. 또는 해당 프로그램이 포커스(활성 상태)가 아닌 경우 작동하지 않을 수 있으니 먼저 해당 창을 클릭한 후 시도해보세요.

**Q. Windows키 단축키가 전혀 작동하지 않아요.**
A. 설정 → 접근성 → 키보드에서 "고정 키" 옵션이 켜져 있는지 확인하세요. 또는 일부 게임이나 소프트웨어가 Windows키를 차단하는 경우도 있습니다.

**Q. 단축키를 빠르게 외우는 좋은 방법이 있나요?**
A. 한 번에 모두 외우려 하지 말고, 하루에 2~3개씩 의식적으로 사용해보세요. 일주일만 지나도 자연스럽게 손에 익게 됩니다. 자주 쓰는 작업에 관련된 단축키부터 시작하는 것이 효율적입니다.

## 정리

윈도우 11의 단축키를 활용하면 반복적인 마우스 클릭을 줄이고 업무 효율을 크게 높일 수 있습니다. 오늘 소개한 단축키 중 **Ctrl+C/V/Z, Alt+Tab, Windows키+← →, Windows키+L, Windows키+Shift+S** 이 7가지만 먼저 익혀보세요. 이것만 해도 일상 PC 작업이 눈에 띄게 빨라질 것입니다. 나머지는 천천히 필요할 때마다 하나씩 추가하면 됩니다.
