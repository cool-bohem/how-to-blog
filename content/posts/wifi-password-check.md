## 와이파이 비밀번호 확인하기

연결은 되어 있는데 비밀번호가 기억 안 날 때 기기에서 직접 확인할 수 있어요.

## 아이폰에서 확인 (iOS 16 이상)

1. **설정 → 와이파이**
2. 현재 연결된 네트워크 옆 **정보(i) 아이콘** 클릭
3. **비밀번호** 항목을 꾹 눌러서 확인
4. Face ID 또는 Touch ID 인증 후 비밀번호 표시

## 갤럭시에서 확인

1. **설정 → 연결 → 와이파이**
2. 연결된 네트워크 클릭
3. **QR 코드** 버튼 클릭
4. QR 코드 아래 비밀번호가 텍스트로 표시돼요

## 윈도우에서 확인

1. 작업표시줄 **와이파이 아이콘** 우클릭
2. **네트워크 및 인터넷 설정 열기**
3. **고급 네트워크 설정 → 네트워크 어댑터 옵션**
4. 현재 연결된 와이파이 우클릭 → **상태 → 무선 속성**
5. **보안** 탭 → **문자 표시** 체크

## 공유기 관리자 페이지에서 확인

위 방법이 안 될 때는 공유기에서 직접 확인할 수 있어요.

1. 브라우저에서 **192.168.0.1** 또는 **192.168.1.1** 접속
2. 관리자 로그인 (기본값: admin/admin)
3. **무선 설정** 에서 비밀번호 확인
