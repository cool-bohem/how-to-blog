## 와이파이 비밀번호를 왜 확인해야 할까요?

처음 설치할 때 입력하고 나면 와이파이 비밀번호를 잊어버리는 경우가 많아요. 새로운 기기(새 스마트폰, 태블릿, 노트북)를 연결하거나, 손님에게 비밀번호를 알려줄 때 필요하죠.

연결된 기기에서 비밀번호를 찾는 방법과 공유기에서 직접 확인하는 방법까지 알려드릴게요.

## 아이폰에서 확인하는 방법 (iOS 16 이상)

iOS 16부터 아이폰에서 와이파이 비밀번호를 직접 확인할 수 있어요.

1. **설정 → 와이파이** 이동
2. 현재 연결된 네트워크 이름 옆 **정보(i) 아이콘** 클릭
3. **비밀번호** 항목까지 아래로 스크롤
4. 비밀번호 오른쪽 **점(●●●)** 을 터치
5. **Face ID 또는 Touch ID 인증** 후 비밀번호 표시

**팁**: iOS 15 이하라면 이 기능이 없어요. 이 경우 공유기 관리자 페이지에서 확인해야 해요.

**주변 기기와 비밀번호 공유하기**

아이폰끼리는 비밀번호 없이도 와이파이를 공유할 수 있어요.

1. 비밀번호를 공유할 상대방 아이폰이 가까이 있어야 해요
2. 상대방이 와이파이에 연결 시도하면 내 화면에 **"와이파이 비밀번호 공유"** 팝업 표시
3. **비밀번호 공유** 클릭 → 상대방 자동 연결

## 갤럭시에서 확인하는 방법

**방법 1. QR 코드로 확인**

갤럭시는 QR 코드 아래에 비밀번호를 텍스트로 보여줘요.

1. **설정 → 연결 → 와이파이**
2. 현재 연결된 네트워크 이름 클릭
3. **QR 코드** 버튼 클릭
4. QR 코드 하단에 **비밀번호가 텍스트로 표시됨**

**방법 2. 지인에게 QR코드로 공유**

1. 동일하게 QR 코드 화면 열기
2. 상대방 스마트폰의 카메라로 QR 코드 스캔
3. 스캔한 기기가 자동으로 해당 와이파이에 연결됨

**팁**: QR 코드 공유 방법을 쓰면 비밀번호를 직접 말하지 않아도 돼서 편리해요.

## 윈도우에서 확인하는 방법

**방법 1. 제어판 통해 확인**

1. 작업표시줄 오른쪽 아래 **와이파이 아이콘** 우클릭
2. **네트워크 및 인터넷 설정 열기** 클릭
3. **고급 네트워크 설정 → 네트워크 어댑터 옵션** 클릭
4. 현재 연결된 와이파이 아이콘 더블클릭
5. **무선 속성** 클릭
6. **보안** 탭 클릭
7. **문자 표시** 체크박스 클릭 → 비밀번호 표시됨

**방법 2. 명령 프롬프트 활용**

더 빠른 방법이에요.

1. **Win + R** → **cmd** 입력 후 Enter
2. 아래 명령어 입력:
   ```
   netsh wlan show profile name="와이파이이름" key=clear
   ```
   (와이파이이름 자리에 실제 네트워크 이름 입력)
3. 결과에서 **핵심 내용** 항목이 비밀번호예요

## 공유기 관리자 페이지에서 확인하기

위 방법이 모두 안 될 때나 정확한 비밀번호를 원할 때 공유기에서 직접 확인할 수 있어요.

1. 브라우저에서 **192.168.0.1** 또는 **192.168.1.1** 접속
   (공유기 제조사마다 다를 수 있어요. 공유기 뒷면 스티커 확인)
2. 관리자 로그인 (기본값: admin/admin 또는 admin/password)
3. **무선 설정** 또는 **Wi-Fi 설정** 메뉴
4. **비밀번호(PSK/보안키)** 확인 가능

## 자주 묻는 질문

**Q. 비밀번호를 확인하는 게 보안상 문제가 없나요?**
본인 기기에서 본인의 와이파이 비밀번호를 확인하는 것은 문제없어요. 단, 다른 사람의 와이파이 비밀번호를 무단으로 확인하는 것은 불법이에요.

**Q. 와이파이 비밀번호가 기억나지 않으면 공유기를 초기화해야 하나요?**
꼭 그럴 필요 없어요. 이미 연결된 기기에서 위 방법으로 확인하거나, 공유기 관리자 페이지에서 확인할 수 있어요.

**Q. 공유기 관리자 페이지 비밀번호도 잊어버렸어요.**
이 경우에는 공유기 뒷면의 리셋 버튼을 10초간 눌러 초기화한 후 재설정해야 해요. 초기화하면 기존에 연결된 모든 기기가 재연결해야 해요.
