## PDF 변환이 필요한 경우

- PDF → Word로 편집하고 싶을 때
- Word → PDF로 저장해서 제출할 때
- 여러 PDF를 하나로 합칠 때

## PDF → Word 변환 (무료)

**방법 1. iLovePDF 사이트 활용**
1. ilovepdf.com 접속
2. **PDF to Word** 선택
3. PDF 파일 업로드
4. **Word로 변환** 클릭 후 다운로드

**방법 2. Microsoft Word로 직접 열기**
1. Word 실행 → **파일 → 열기**
2. PDF 파일 선택
3. Word가 자동으로 변환해줌

## Word → PDF 변환

1. Word 문서 열기
2. **파일 → 다른 이름으로 저장**
3. 파일 형식에서 **PDF** 선택
4. 저장

## 여러 PDF 합치기

1. ilovepdf.com 접속
2. **Merge PDF** 선택
3. 합칠 PDF 파일들 업로드
4. 순서 조정 후 **병합** 클릭

## PDF 용량 줄이기

이메일 첨부 등으로 용량을 줄여야 할 때:

1. ilovepdf.com → **Compress PDF**
2. 파일 업로드 후 압축
3. 품질 선택 후 다운로드
