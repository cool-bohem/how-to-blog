## PDF 변환이 필요한 상황

직장이나 학교에서 PDF를 다루다 보면 다양한 변환이 필요한 상황이 생겨요.

- **PDF → Word**: 내용을 편집하거나 수정해야 할 때
- **Word → PDF**: 문서를 공식 제출용 파일로 만들 때
- **PDF 여러 개 → 하나의 PDF**: 여러 서류를 하나로 묶어서 제출할 때
- **PDF → JPG**: 특정 페이지를 이미지로 추출할 때
- **PDF 용량 줄이기**: 이메일 첨부 파일 크기 제한을 맞출 때

모두 무료 도구로 해결할 수 있어요.

## PDF → Word 변환

**방법 1. iLovePDF 사이트 활용 (가장 범용적)**

1. 브라우저에서 ilovepdf.com 접속
2. **PDF to Word** 선택
3. **PDF 파일 선택** 클릭하거나 파일을 끌어다 놓기
4. **Word로 변환** 클릭
5. 변환 완료 후 **Word 다운로드** 클릭

**팁**: 영어 PDF는 거의 완벽하게 변환돼요. 한글 PDF는 글꼴 등에 따라 일부 깨질 수 있어요.

**방법 2. Microsoft Word로 직접 열기**

Word 2013 이상 버전에서는 PDF를 Word로 직접 변환해서 열 수 있어요.

1. Word 실행 → **파일 → 열기**
2. PDF 파일 선택
3. "PDF를 편집 가능한 Word 문서로 변환합니다" 메시지 확인 후 **확인**
4. Word 형식으로 자동 변환되어 열림

## Word → PDF 변환

**방법 1. Word에서 직접 저장**

1. Word 문서 열기
2. **파일 → 다른 이름으로 저장**
3. 파일 형식 드롭다운에서 **PDF** 선택
4. 저장 위치 선택 후 저장

**방법 2. 인쇄 기능 활용 (프린터 없이)**

1. Word 문서에서 **Ctrl + P** (인쇄)
2. 프린터 선택에서 **Microsoft Print to PDF** 선택
3. **인쇄** 클릭 → PDF 파일로 저장됨

## 여러 PDF 합치기

1. ilovepdf.com 접속
2. **Merge PDF** 선택
3. 합칠 PDF 파일들을 모두 업로드
4. 드래그로 순서 조정 가능
5. **병합** 클릭 → 하나의 PDF 파일로 다운로드

**팁**: 제출 서류를 여러 장 묶을 때 매우 유용해요. 주민등록등본 + 재직증명서 + 통장 사본 등을 하나의 PDF로 만들어 제출할 수 있어요.

## PDF → 이미지(JPG) 변환

PDF 특정 페이지를 이미지로 저장하고 싶을 때:

1. ilovepdf.com 접속
2. **PDF to JPG** 선택
3. PDF 파일 업로드
4. 변환 방식 선택:
   - **페이지별 JPG 추출**
   - **모든 페이지를 ZIP으로**
5. **JPG로 변환** 클릭 후 다운로드

## PDF 용량 줄이기

이메일로 PDF를 보낼 때 용량이 너무 크다면:

1. ilovepdf.com → **Compress PDF** 선택
2. 파일 업로드
3. 압축 레벨 선택 (품질 ↔ 용량 조절)
4. **PDF 압축** 클릭 후 다운로드

**참고**: 일반적으로 품질 저하 없이 30~50% 용량 감소가 가능해요.

## 스마트폰에서 PDF 변환하기

PC 없이 스마트폰에서도 간단히 변환할 수 있어요.

**아이폰**:
- 파일 앱에서 PDF 선택 → 공유 → **마크업** 으로 편집 후 저장
- Microsoft Word 앱으로 PDF 열기 가능

**갤럭시**:
- 삼성 스마트폰의 **Galaxy PDF** 앱 활용
- 또는 모바일 브라우저에서 ilovepdf.com 접속 후 동일하게 사용 가능

## 자주 묻는 질문

**Q. PDF 변환 사이트에 개인정보가 담긴 파일을 올려도 안전한가요?**
ilovepdf 등 주요 사이트들은 변환 후 파일을 삭제한다고 명시하고 있어요. 하지만 민감한 개인정보 파일은 Microsoft Word나 Adobe Acrobat 같은 오프라인 프로그램을 사용하는 게 더 안전해요.

**Q. 무료 도구로 변환한 PDF는 품질이 낮나요?**
텍스트 기반 문서는 품질 차이가 거의 없어요. 이미지가 많거나 복잡한 레이아웃의 파일은 유료 도구(Adobe Acrobat)가 더 정확해요.

**Q. PDF에 비밀번호를 걸 수 있나요?**
네. ilovepdf에서 **Protect PDF** 기능으로 비밀번호를 설정할 수 있어요.
