## 한글(HWP)이란?

한글(HWP)은 한컴오피스에서 개발한 한국형 워드프로세서입니다. 공공기관, 학교, 기업에서 가장 널리 사용되는 문서 작성 프로그램으로, 정부 민원 서류나 공문서 대부분이 HWP 형식으로 제공됩니다. 마이크로소프트 워드(Word)와 비슷한 기능을 갖추고 있지만 한국어 처리에 특화되어 있으며, 국가 공인 전자문서 형식으로 사용되고 있습니다.

2026년 현재 한컴오피스 한글 2022 버전이 많이 사용되며, 구독형 한컴독스(HancomDocs)도 활용됩니다. 이 글에서는 취업 준비생, 직장인, 학생이 꼭 알아야 할 핵심 기능들을 알아보겠습니다.

## 한글 프로그램 기본 화면 구성

### 메뉴와 도구 모음 이해하기

한글을 처음 실행하면 다소 복잡해 보이지만 구조를 알면 쉽게 사용할 수 있습니다.

1. **제목 표시줄**: 화면 맨 위에 현재 작업 중인 파일 이름이 표시됩니다.
2. **메뉴 표시줄**: 파일, 편집, 보기, 입력, 서식, 쪽, 도구 등 주요 메뉴가 있습니다.
3. **도구 상자**: 자주 사용하는 기능 버튼들이 아이콘으로 배치되어 있습니다.
4. **편집 화면**: 실제로 문서를 작성하는 영역입니다.
5. **상태 표시줄**: 화면 아래쪽에 현재 커서 위치, 페이지 수, 글자 수 등이 표시됩니다.

> **팁:** 화면이 너무 복잡하게 느껴지면 **보기 → 작업 창** 메뉴에서 불필요한 패널을 숨겨 작업 공간을 넓힐 수 있습니다.

## 새 문서 만들기와 저장하기

### 새 문서 시작하기

1. 한글을 실행하면 자동으로 새 문서가 열립니다.
2. 이미 다른 문서를 작업 중일 때 새 문서를 만들려면 **Ctrl + N**을 누르거나, **파일 → 새 문서**를 클릭합니다.
3. 새 문서 대화상자가 나타나면 **빈 문서**를 선택하고 **만들기**를 클릭합니다.

### 문서 저장하기

1. 문서를 저장하려면 **Ctrl + S**를 누르거나 **파일 → 저장**을 클릭합니다.
2. 처음 저장할 때는 저장 위치와 파일 이름을 지정하는 대화상자가 나타납니다.
3. 저장 위치를 선택하고 파일 이름을 입력합니다.
4. **파일 형식**에서 **.hwp** (한글 파일) 또는 **.hwpx** (최신 형식)를 선택합니다.
5. 워드(Word)에서 열 수 있게 저장하려면 파일 형식을 **.docx**로 선택하세요.
6. **저장** 버튼을 클릭합니다.

> **중요:** 작업 중에는 **Ctrl + S**를 자주 눌러 저장하는 습관을 들이세요. 예기치 않은 프로그램 종료나 정전 시 데이터 손실을 예방할 수 있습니다.

## 글자 서식 설정하기

### 글꼴, 크기, 색상 변경하기

1. 서식을 변경할 텍스트를 **마우스로 드래그**하여 선택합니다.
2. 도구 모음에서 **글꼴 이름**을 눌러 원하는 글꼴을 선택합니다. (예: 맑은 고딕, 나눔고딕, 바탕체 등)
3. **글자 크기**를 입력하거나 크기 목록에서 선택합니다. 본문은 보통 10~12pt, 제목은 14~16pt를 사용합니다.
4. **굵게(Ctrl + B)**, **기울임꼴(Ctrl + I)**, **밑줄(Ctrl + U)** 버튼으로 강조 표시를 추가합니다.
5. 글자 색상을 바꾸려면 **A** 아이콘 옆의 화살표를 눌러 원하는 색을 선택합니다.

### 단락 정렬 설정하기

1. 정렬할 단락 안에 커서를 놓거나 여러 단락을 선택합니다.
2. 도구 모음의 정렬 버튼을 클릭합니다.
   - **왼쪽 정렬(Ctrl + L)**: 일반 본문에 주로 사용
   - **가운데 정렬(Ctrl + E)**: 제목이나 표 안의 내용에 사용
   - **오른쪽 정렬(Ctrl + R)**: 날짜나 서명란에 사용
   - **양쪽 정렬(Ctrl + J)**: 공문서나 보고서에 주로 사용

## 표 만들기

### 표 삽입하기

표는 데이터를 정리하거나 양식을 만들 때 필수적입니다.

1. 표를 삽입할 위치에 커서를 놓습니다.
2. **입력 → 표** 메뉴를 클릭하거나, 도구 모음의 **표 아이콘**을 클릭합니다.
3. 표 만들기 대화상자가 나타납니다.
4. **줄 수**(행 수)와 **칸 수**(열 수)를 입력합니다. 예를 들어 3행 4열의 표를 만들려면 줄 수 3, 칸 수 4를 입력합니다.
5. **만들기** 버튼을 클릭하면 표가 삽입됩니다.

### 표 편집하기

1. **셀 합치기**: 합칠 셀들을 드래그로 선택한 후, 마우스 오른쪽 버튼을 눌러 **셀 합치기**를 선택합니다.
2. **셀 나누기**: 나눌 셀을 선택하고 마우스 오른쪽 버튼 → **셀 나누기**를 선택한 후 행/열 수를 입력합니다.
3. **셀 크기 조절**: 행과 행 사이의 경계선이나 열과 열 사이의 경계선을 마우스로 드래그하면 크기를 조절할 수 있습니다.
4. **셀 배경색 설정**: 셀을 선택하고 마우스 오른쪽 버튼 → **셀 테두리/배경**을 클릭하면 배경색을 설정할 수 있습니다.

## 이미지 삽입하기

### 그림 파일 삽입하기

1. 그림을 삽입할 위치에 커서를 놓습니다.
2. **입력 → 그림** 메뉴를 클릭하거나, **Ctrl + N, I** 단축키를 사용합니다.
3. 그림 파일 선택 대화상자에서 삽입할 이미지 파일을 찾아 선택합니다.
4. **열기** 버튼을 클릭하면 그림이 삽입됩니다.
5. 삽입된 그림을 클릭하면 크기 조절 핸들이 나타납니다. 모서리의 핸들을 드래그하면 크기를 조절할 수 있습니다.

> **팁:** 그림을 선택하고 마우스 오른쪽 버튼 → **개체 속성**을 클릭하면 그림의 크기를 정확한 수치로 입력하거나, 글자와의 배치 방법을 설정할 수 있습니다.

## 쪽 설정과 인쇄하기

### 용지 설정하기

1. **쪽 → 쪽 설정** 메뉴를 클릭합니다.
2. 쪽 설정 대화상자에서 용지 크기를 설정합니다. 일반 문서는 **A4**, 이력서나 특수 양식은 별도 크기를 사용합니다.
3. 여백을 설정합니다. 일반적으로 위 30mm, 아래 30mm, 왼쪽 30mm, 오른쪽 30mm가 기본값이며, 필요에 따라 조정합니다.
4. 용지 방향을 **세로** 또는 **가로**로 선택합니다.
5. **확인**을 클릭합니다.

### 머리말과 꼬리말 설정하기

1. **쪽 → 머리말/꼬리말** 메뉴를 클릭합니다.
2. 머리말(페이지 위 고정 영역)이나 꼬리말(페이지 아래 고정 영역)을 선택합니다.
3. 원하는 내용(회사명, 문서 제목, 페이지 번호 등)을 입력합니다.
4. **닫기** 버튼을 클릭하면 설정이 완료됩니다.

### 문서 인쇄하기

1. **Ctrl + P**를 누르거나 **파일 → 인쇄**를 클릭합니다.
2. 인쇄 대화상자에서 프린터를 선택합니다.
3. 인쇄 범위(전체 쪽, 현재 쪽, 특정 쪽 등)를 설정합니다.
4. 인쇄 부수를 입력합니다.
5. **인쇄** 버튼을 클릭합니다.

> **팁:** 실제로 인쇄하기 전에 **미리 보기(Ctrl + F12)**로 인쇄 결과를 확인하면 용지와 잉크 낭비를 줄일 수 있습니다.

## 한글 필수 단축키 모음

자주 사용하는 단축키를 익혀두면 작업 속도가 크게 향상됩니다.

| 단축키 | 기능 |
|--------|------|
| Ctrl + S | 저장 |
| Ctrl + Z | 실행 취소 |
| Ctrl + Y | 다시 실행 |
| Ctrl + C | 복사 |
| Ctrl + V | 붙여넣기 |
| Ctrl + X | 잘라내기 |
| Ctrl + A | 전체 선택 |
| Ctrl + F | 찾기 |
| Ctrl + H | 찾아 바꾸기 |
| Ctrl + P | 인쇄 |
| F7 | 맞춤법 검사 |

## 자주 묻는 질문 (Q&A)

**Q. HWP 파일을 PDF로 변환하는 방법은?**
A. 파일 → PDF로 저장 메뉴를 이용하면 HWP 파일을 PDF로 저장할 수 있습니다. 또는 인쇄 대화상자에서 프린터를 'PDF'로 선택해도 PDF 파일을 만들 수 있습니다.

**Q. 한글 프로그램이 없을 때 HWP 파일을 열 수 있나요?**
A. 네, 한컴뷰어(무료)를 설치하면 HWP 파일을 볼 수 있습니다. 편집은 불가능하지만 내용 확인과 인쇄는 가능합니다. 또한 한컴독스(HancomDocs) 웹 서비스나 모바일 앱에서도 HWP 파일을 열 수 있습니다.

**Q. 워드(Word)에서 만든 DOCX 파일을 한글에서 열 수 있나요?**
A. 한글에서 파일 → 불러오기를 통해 DOCX 파일을 열 수 있습니다. 다만 일부 서식이나 기능이 변환 과정에서 달라질 수 있으므로, 열고 나서 서식을 확인하고 수정하는 것이 좋습니다.

**Q. 맞춤법 검사를 하고 싶은데 어떻게 하나요?**
A. 도구 → 맞춤법 검사 메뉴를 클릭하거나 F7 키를 누르면 맞춤법 검사가 실행됩니다. 틀린 단어가 발견되면 수정 제안과 함께 바꾸기 옵션이 나타납니다.

**Q. 자동 저장 기능은 어떻게 설정하나요?**
A. 도구 → 환경 설정 → 문서 탭에서 자동 저장 간격을 설정할 수 있습니다. 5~10분 간격으로 설정해두면 갑작스러운 오류 시 데이터 손실을 최소화할 수 있습니다.

## 정리

한글(HWP)은 대한민국에서 가장 많이 사용되는 문서 작성 프로그램으로, 기본적인 사용법만 익혀도 업무와 학업에 큰 도움이 됩니다. 새 문서 만들기, 글자 서식, 표 삽입, 이미지 삽입, 인쇄 설정 등 핵심 기능들을 차근차근 연습하면 누구나 쉽게 익숙해질 수 있습니다. 특히 자주 사용하는 단축키를 기억해두면 작업 속도를 크게 높일 수 있습니다. 처음에는 간단한 문서부터 시작해서 점차 복잡한 서식의 문서에 도전해보세요.
