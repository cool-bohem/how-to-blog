## 배달의민족이란?

배달의민족(배민)은 대한민국 최대 음식 배달 앱으로, 치킨, 피자, 중국음식, 분식, 한식 등 다양한 음식을 스마트폰으로 손쉽게 주문할 수 있는 서비스입니다. 2026년 현재 수천만 명이 사용하는 국민 배달 앱으로 자리잡았으며, 처음 접하는 분들도 몇 번만 해보면 금방 익숙해질 만큼 인터페이스가 직관적입니다.

이 가이드에서는 배달의민족 앱을 처음 사용하는 분들부터 더 똑똑하게 활용하고 싶은 분들을 위해 앱 설치부터 주문 완료까지 단계별로 설명해드리겠습니다.

## 배달의민족 앱 설치 및 회원가입

### 1단계: 앱 설치

1. 스마트폰의 **앱스토어(아이폰)** 또는 **플레이스토어(안드로이드)**를 엽니다.
2. 검색창에 **"배달의민족"** 또는 **"배민"**을 입력합니다.
3. 검색 결과에서 배달의민족 앱(파란색 배경에 흰색 글자 아이콘)을 찾아 **설치** 버튼을 누릅니다.
4. 설치가 완료되면 앱을 실행합니다.

### 2단계: 회원가입

1. 앱을 실행하면 로그인 화면이 나타납니다.
2. 처음 사용하는 경우 **"회원가입"** 버튼을 누릅니다.
3. **휴대폰 번호**를 입력하고 **인증번호 받기**를 누릅니다.
4. 문자로 받은 **6자리 인증번호**를 입력합니다.
5. **이름**, **이메일**, **비밀번호**를 입력하고 약관에 동의합니다.
6. **가입 완료** 버튼을 눌러 회원가입을 마칩니다.

**팁**: 카카오, 네이버, 애플 계정으로 소셜 로그인도 가능합니다. 소셜 로그인을 사용하면 별도의 비밀번호 없이 간편하게 로그인할 수 있어 추천합니다.

## 배달 주소 설정하기

음식을 배달받을 주소를 정확히 설정하는 것이 가장 중요합니다. 주소가 잘못되면 음식이 엉뚱한 곳으로 배달될 수 있으니 꼼꼼히 확인하세요.

1. 앱 하단 메뉴에서 **"홈"**을 누릅니다.
2. 화면 상단의 주소 표시 영역을 누릅니다.
3. **"새 주소 추가"**를 선택합니다.
4. **주소 검색창**에 배달받을 주소를 입력합니다. (예: 서울특별시 강남구 테헤란로 123)
5. 검색 결과에서 정확한 주소를 선택합니다.
6. 건물 내 위치를 더 정확히 알릴 경우 **상세주소**(동호수, 층수, 건물명 등)를 입력합니다.
7. 주소 별칭을 설정합니다. (예: "집", "회사", "학교")
8. **저장** 버튼을 누릅니다.

주소를 여러 개 저장해 두면 위치마다 빠르게 전환하며 주문할 수 있어 편리합니다.

## 음식 주문하기

### 1단계: 식당 찾기

1. 홈 화면에서 원하는 음식 카테고리를 선택합니다. (치킨, 피자, 중국음식, 한식 등)
2. 또는 상단 **검색창**에 원하는 음식이나 식당 이름을 직접 입력합니다.
3. 검색 결과 화면에서 각 식당의 정보를 비교합니다:
   - **별점**: 다른 이용자들의 평점
   - **리뷰 수**: 후기가 많을수록 신뢰도가 높습니다
   - **배달 예상 시간**: 배달까지 걸리는 예상 시간
   - **최소 주문 금액**: 주문 가능한 최소 금액
   - **배달비**: 배달 수수료 금액
4. 원하는 식당을 누릅니다.

### 2단계: 메뉴 선택 및 장바구니 담기

1. 식당 메뉴 목록에서 원하는 메뉴를 선택합니다. 카테고리별로 정리되어 있어 찾기 쉽습니다.
2. 메뉴를 누르면 상세 설명과 옵션이 나타납니다.
   - **맛 선택**: 순한맛, 보통맛, 매운맛 등
   - **사이드 추가**: 음료, 소스 추가 등
   - **필수 선택**: 빨간색으로 표시된 옵션은 반드시 선택해야 합니다
3. 수량을 조절(+ / - 버튼)한 뒤 **"장바구니에 담기"** 버튼을 누릅니다.
4. 추가 메뉴가 필요하면 계속 담습니다.
5. 모든 메뉴 선택이 끝나면 하단의 **"장바구니 보기"** 버튼을 누릅니다.

### 3단계: 장바구니 확인

1. 담은 메뉴와 수량을 다시 한번 확인합니다.
2. 최소 주문 금액을 충족했는지 확인합니다. 충족하지 못했다면 메뉴를 추가해야 합니다.
3. **사장님께 요청사항** 란에 특별한 요청 사항(예: "고추 빼주세요", "덜 맵게 해주세요")을 입력할 수 있습니다.
4. 이상이 없으면 **"주문하기"** 버튼을 누릅니다.

### 4단계: 결제하기

1. **할인 쿠폰**이 있다면 "쿠폰 적용" 부분을 눌러 선택합니다.
2. **배민 포인트**가 있다면 사용 여부를 선택합니다.
3. **결제 수단**을 선택합니다.
   - 카카오페이, 네이버페이, 토스페이, 신용/체크카드, 계좌이체, 현금(배달원에게) 등
   - 처음 주문 시 카드 정보를 등록해야 합니다.
4. 배달 주소와 총 금액을 최종 확인합니다.
5. **"결제하기"** 버튼을 눌러 주문을 완료합니다.

## 주문 현황 확인하기

주문을 완료한 후 앱에서 실시간으로 배달 상황을 확인할 수 있습니다.

1. 주문 완료 후 자동으로 **"주문 현황"** 화면으로 이동합니다.
2. 배달 상태는 다음 순서로 업데이트됩니다:
   - **주문 접수 대기** → **주문 접수 완료** → **조리 중** → **배달 중** → **배달 완료**
3. 일부 지역 및 식당에서는 **실시간 배달 위치 추적** 기능이 제공됩니다. 지도 위에서 라이더의 위치를 실시간으로 확인할 수 있습니다.
4. 배달 완료 후 앱 푸시 알림으로 알려줍니다.
5. 주문 내역은 앱 하단 **"주문"** 탭에서 언제든지 다시 확인할 수 있습니다.

## 쿠폰 및 할인 혜택 활용하기

배달의민족은 다양한 할인 혜택을 제공합니다. 적극 활용하면 배달비와 메뉴 가격 모두 절약할 수 있습니다.

1. **앱 홈 화면**의 이벤트 배너와 쿠폰 관련 알림을 자주 확인합니다.
2. **배민클럽** (유료 멤버십):
   - 월정액(약 3,990원~)을 내면 무제한 무료 배달, 할인 쿠폰, 배민스토어 할인 등 다양한 혜택을 받을 수 있습니다.
   - 배달 주문이 월 2~3회 이상이라면 멤버십이 더 경제적입니다.
3. **카드사 혜택**: 특정 카드로 결제 시 추가 할인이 적용됩니다. 결제 화면에서 각 카드의 혜택을 확인하세요.
4. **신규 가입 혜택**: 처음 가입하면 첫 주문 할인 쿠폰을 제공하는 경우가 많습니다.
5. **쿠폰 적용 방법**: 결제 화면 → "쿠폰 적용" 버튼 → 사용 가능한 쿠폰 목록에서 선택

## 리뷰 작성하기

리뷰는 식당 선정에 큰 도움이 되며, 작성하면 포인트도 쌓입니다.

1. 배달 완료 후 앱에서 **"리뷰 작성"** 알림이 옵니다. 또는 주문 내역에서 직접 작성할 수도 있습니다.
2. **별점**(1~5점)을 선택합니다.
3. 음식 맛, 양, 포장 상태, 배달 속도 등에 대한 솔직한 후기를 작성합니다.
4. **사진을 첨부**하면 다른 이용자들에게 더 도움이 됩니다.
5. **제출** 버튼을 누릅니다.

리뷰를 작성하면 **배민 포인트**를 받을 수 있어 다음 주문 시 할인에 활용할 수 있습니다. 또한 사장님이 직접 답글을 달아주는 경우도 있어 소통이 가능합니다.

## 자주 묻는 질문 (Q&A)

**Q. 주문 후 취소하려면 어떻게 하나요?**
A. 식당이 주문을 접수하기 전이라면 앱의 "주문 현황" 화면에서 취소 요청이 가능합니다. 조리가 시작된 후에는 식당에 직접 전화하거나 배민 고객센터(1600-0987)에 문의해야 합니다. 조리 시작 후 취소는 환불이 어려울 수 있습니다.

**Q. 배달이 너무 늦어요. 어떻게 하나요?**
A. 앱의 "주문 현황"에서 예상 도착 시간을 확인하세요. 예상 시간을 크게 초과했다면 앱 내 채팅으로 식당에 문의하거나, 배민 고객센터에 연락할 수 있습니다.

**Q. 음식이 잘못 오거나 빠진 메뉴가 있어요. 환불받을 수 있나요?**
A. 앱 주문 내역에서 "고객센터 문의" 또는 전화(1600-0987)로 문의하면 상황에 따라 환불이나 재주문 처리를 받을 수 있습니다. 증거 사진을 찍어두면 처리가 더 빠릅니다.

**Q. 포장 주문(테이크아웃)도 가능한가요?**
A. 네, 일부 식당에서는 포장 주문이 가능합니다. 식당 상세 페이지에서 "포장" 탭을 선택하면 됩니다. 포장 주문 시 배달비가 없어 더 저렴합니다.

**Q. 배달 주소를 잘못 입력했어요. 수정할 수 있나요?**
A. 주문 완료 전이라면 장바구니에서 주소를 변경할 수 있습니다. 결제 완료 후에는 앱 내 채팅이나 고객센터를 통해 빠르게 변경을 요청해야 합니다.

**Q. 배민 포인트는 어떻게 쌓이나요?**
A. 리뷰 작성, 이벤트 참여, 배민클럽 멤버십 혜택 등을 통해 쌓입니다. 결제 화면에서 적립된 포인트를 현금처럼 사용할 수 있습니다.

## 정리

배달의민족 앱은 설치부터 주문, 결제, 리뷰 작성까지 매우 간단하게 이용할 수 있습니다. 처음에는 낯설 수 있지만, 한 번 해보면 이후에는 자연스럽게 사용할 수 있습니다.

**핵심 요약**:
- **회원가입** → **주소 설정** → **식당·메뉴 선택** → **결제** → **배달 현황 확인** 순서로 진행
- 쿠폰, 포인트, 배민클럽 멤버십을 적극 활용하면 배달비와 메뉴 가격 모두 절약 가능
- 리뷰 작성으로 포인트를 쌓고 다음 주문에 활용
- 문제 발생 시 고객센터(1600-0987) 또는 앱 내 고객센터 이용

배달의민족으로 집에서도 맛있는 식사를 편리하게 즐겨보세요!
