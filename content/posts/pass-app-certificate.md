## PASS 앱이란?

PASS는 SKT, KT, LGU+ 세 개 통신사가 공동으로 만든 공식 본인인증 앱이에요. 공인인증서 없이도 통신사 회선을 이용해 다양한 인증과 서명을 처리할 수 있어요.

**PASS 앱으로 할 수 있는 것들**:
- 각종 사이트 본인인증 (은행, 공공기관, 쇼핑몰 등)
- 모바일 운전면허증 발급 및 사용
- 전자서명 (계약서, 의료, 금융 등)
- PASS 인증서 발급 (공동인증서 대체)
- 금융 인증서 관리

## 앱 설치 및 가입

1. 앱스토어(아이폰) 또는 플레이스토어(안드로이드)에서 **PASS** 검색 후 설치
   - 개발자: SKT/KT/LGU+ (통신사별 앱이 있어요)
2. 통신사 선택 (본인 가입 통신사)
3. 휴대폰 번호 입력 → 문자로 받은 인증번호 입력
4. 본인 확인 완료
5. **PIN 번호** 또는 **생체인증(지문/얼굴)** 설정

## 본인인증 사용 방법

각종 사이트에서 "PASS 인증" 또는 "휴대폰 인증"을 선택하면 돼요.

1. 인증이 필요한 사이트에서 **PASS 인증** 또는 **휴대폰 인증** 선택
2. 이름, 생년월일, 휴대폰 번호 입력
3. **PASS 앱으로 알림** 전송됨
4. 스마트폰에서 PASS 앱 알림 확인
5. 내용 확인 후 **수락** 클릭
6. 인증 완료

**팁**: PASS 앱을 열지 않아도 잠금화면에서 수락할 수 있어요.

## 모바일 운전면허증 등록하기

실물 운전면허증 없이도 편의점, 음식점 등에서 신분 확인을 할 수 있어요.

**등록 전 준비물**:
- 유효한 실물 운전면허증
- PASS 앱 설치 및 로그인 상태

**등록 방법**:
1. PASS 앱 하단 **신분증** 탭 클릭
2. **모바일 운전면허증 등록** 클릭
3. 실물 운전면허증 앞면 촬영
4. 경찰청 서버에서 면허 정보 확인 (수초 소요)
5. 얼굴 사진 촬영 (본인 확인용)
6. 등록 완료

**사용 방법**:
- PASS 앱 → **신분증 제시** 클릭
- QR코드 또는 바코드를 상대방에게 보여주기
- 편의점, 약국, 은행 창구 등에서 신분 확인 가능

**주의**: 일부 기관(정부 기관, 공항 등)은 실물 신분증만 받는 경우도 있어요. 중요한 자리에는 실물 신분증을 지참하세요.

## 전자서명 기능

금융 계약, 의료 동의서, 공공 서류 등에 사용할 수 있는 전자서명 기능이에요.

1. 은행이나 앱에서 전자서명 요청을 받으면 PASS 앱으로 알림 도착
2. PASS 앱 열기 → 서명 내용 확인
3. **서명** 버튼 클릭 → 생체인증 또는 PIN 입력
4. 서명 완료

## PASS 인증서 발급

공동인증서(구 공인인증서) 대신 사용할 수 있는 PASS 전용 인증서예요.

1. PASS 앱 → **인증서** 탭
2. **PASS 인증서 발급** 클릭
3. 본인인증 완료 후 발급
4. 은행, 정부 기관 등에서 공동인증서 대신 선택해서 사용 가능

**장점**: USB나 파일 없이 스마트폰 안에 저장되며, 갱신이 자동이에요.

## 자주 묻는 질문

**Q. PASS 앱은 무료인가요?**
네. 앱 설치 및 기본 본인인증 기능은 무료예요. 일부 고급 기능(전자계약 등)은 서비스 제공 기관에서 별도로 비용을 낼 수 있어요.

**Q. 모바일 운전면허증은 모든 곳에서 통용되나요?**
편의점, 약국, 주점 등 대부분의 나이 확인 장소에서 사용 가능해요. 단, 공항 탑승수속, 일부 관공서 등 실물 신분증만 받는 곳도 있어요.

**Q. 폰을 바꿨을 때 PASS 앱을 다시 설정해야 하나요?**
네. 새 기기에 앱을 다시 설치하고 인증을 다시 진행해야 해요. 모바일 운전면허증도 새 기기에서 다시 등록해야 해요.
