## PASS 앱이란?

통신 3사(SKT, KT, LGU+)가 만든 본인인증 앱이에요. 공인인증서 없이도 각종 인증이 가능해요.

## 앱 설치 및 가입

1. 앱스토어/플레이스토어에서 **PASS** 검색 후 설치
2. 통신사 선택 후 휴대폰 번호 인증
3. PIN 번호 또는 생체인증 설정

## 본인인증 방법

각종 사이트에서 본인인증 시:

1. **PASS 인증** 선택
2. 휴대폰 번호 입력
3. PASS 앱 알림 확인 후 **수락**
4. 인증 완료

## 모바일 운전면허증 등록

1. PASS 앱 하단 **신분증** 탭
2. **모바일 운전면허증 등록** 클릭
3. 실물 운전면허증 촬영
4. 경찰청 서버 인증 후 등록 완료

편의점, 음식점 등에서 실물 신분증 대신 사용 가능해요.

## 전자서명 기능

1. **전자서명 요청** 알림 수신
2. 내용 확인 후 **서명**
3. 금융, 의료, 공공기관 등에서 활용 가능

## PASS 인증서 발급

1. PASS 앱 **인증서** 탭
2. **PASS 인증서 발급** 클릭
3. 본인인증 완료 후 발급
4. 공동인증서 대신 사용 가능
