## 스마트폰 저장공간 확보하기

저장공간이 부족하면 사진도 안 찍히고 앱도 안 깔려요. 아래 순서대로 정리해보세요.

## 방법 1. 캐시 데이터 삭제

1. **설정 → 배터리 및 디바이스 케어 → 저장공간**
2. **캐시 데이터 정리** 클릭
3. 수백 MB ~ 수 GB까지 확보 가능해요

## 방법 2. 사용하지 않는 앱 삭제

1. **설정 → 앱**
2. 오른쪽 상단 **정렬** → **저장공간 순** 으로 정렬
3. 큰 용량 앱부터 확인 후 불필요한 앱 삭제

## 방법 3. 사진/영상 구글 포토로 이전

1. **구글 포토** 앱 설치
2. **백업 켜기** 설정
3. 백업 완료 후 **저장 공간 확보** 기능으로 기기 내 원본 삭제
4. 사진은 구글에 저장되어 언제든 볼 수 있어요 (무료 15GB)

## 방법 4. 카카오톡 캐시 삭제

카카오톡은 오래 쓸수록 캐시가 많이 쌓여요.

1. 카카오톡 **설정 → 개인/보안 → 저장공간 관리**
2. **캐시 데이터 삭제** 클릭

## 방법 5. 다운로드 폴더 정리

1. **내 파일 앱 → 다운로드** 폴더 열기
2. 오래된 파일, 중복 파일 선택 후 삭제
3. 특히 영상 파일이 용량을 많이 차지해요

## 방법 6. SD카드 활용 (지원 기기)

일부 갤럭시 모델은 SD카드 슬롯이 있어요.

1. microSD 카드 구매 후 삽입
2. **설정 → 저장공간 → SD카드로 이동** 으로 사진/파일 이전
