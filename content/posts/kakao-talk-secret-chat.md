## 카카오톡 비밀채팅이란?

카카오톡 비밀채팅은 대화 내용을 암호화하고 화면 캡처를 차단하며 일정 시간이 지나면 메시지를 자동으로 삭제하는 보안 강화 채팅 기능입니다. 중요한 개인정보나 민감한 내용을 주고받을 때 일반 채팅보다 훨씬 안전하게 대화할 수 있습니다.

2026년 현재 카카오톡 비밀채팅은 **1:1 채팅**에서만 사용할 수 있으며, 양쪽 모두 최신 버전의 카카오톡을 사용해야 정상적으로 작동합니다.

---

## 비밀채팅 시작 전 준비사항

비밀채팅을 시작하기 전에 다음 사항을 확인하세요.

- 카카오톡 앱이 최신 버전인지 확인 (앱스토어 또는 플레이스토어에서 업데이트)
- 상대방도 최신 버전의 카카오톡을 사용 중인지 확인
- 스마트폰 운영체제가 iOS 14 이상 또는 Android 8 이상인지 확인

## 카카오톡 비밀채팅 시작하는 방법

### 1단계: 카카오톡 앱 실행 후 친구 찾기

카카오톡 앱을 실행합니다. 하단 메뉴에서 **채팅** 탭을 누릅니다. 비밀채팅을 하고 싶은 친구와의 기존 채팅방이 있다면 그 채팅방으로 이동하고, 처음 시작하는 경우라면 **친구** 탭에서 해당 친구를 찾아 채팅방을 엽니다.

### 2단계: 채팅방 상단 메뉴 열기

채팅방에 들어간 뒤 화면 오른쪽 상단의 **햄버거 메뉴(≡)** 또는 **점 세 개(···)** 아이콘을 누릅니다. 기종과 앱 버전에 따라 아이콘 모양이 다를 수 있습니다. 아이콘을 누르면 채팅방 설정 화면이 열립니다.

### 3단계: 비밀채팅 전환 선택

설정 화면 하단 또는 메뉴 목록에서 **비밀채팅으로 전환** 또는 **비밀채팅 시작** 항목을 찾아 누릅니다. 항목이 보이지 않는다면 앱을 최신 버전으로 업데이트한 뒤 다시 시도해 보세요.

### 4단계: 비밀채팅 안내 확인 및 동의

비밀채팅의 주요 기능과 제한 사항을 설명하는 안내 화면이 표시됩니다. 내용을 읽고 **시작하기** 또는 **확인** 버튼을 누릅니다. 이 시점에서 상대방에게도 비밀채팅 초대 알림이 전송됩니다.

### 5단계: 상대방의 수락 대기

상대방이 비밀채팅 초대를 수락해야 채팅이 시작됩니다. 상대방의 화면에 **비밀채팅 초대** 알림이 표시되며, 상대방이 **수락**을 누르면 비밀채팅 방이 개설됩니다. 상대방이 수락하기 전까지 메시지를 보낼 수 없습니다.

## 비밀채팅 주요 기능 설정하기

### 메시지 자동 삭제 타이머 설정

비밀채팅의 가장 강력한 기능 중 하나는 메시지 자동 삭제입니다.

1. 비밀채팅방 하단의 **+ (더하기)** 아이콘을 누릅니다.
2. **타이머** 또는 **메시지 삭제 시간** 항목을 선택합니다.
3. 삭제 시간을 설정합니다. 선택 가능한 시간은 다음과 같습니다:
   - 6시간
   - 12시간
   - 24시간
   - 3일
   - 7일
   - 삭제 안 함
4. 원하는 시간을 선택하면 이후 주고받는 메시지는 설정한 시간이 지나면 양쪽 모두에서 자동으로 사라집니다.

**주의:** 타이머를 설정하면 상대방에게도 이를 알리는 시스템 메시지가 표시됩니다.

### 화면 캡처 차단 확인

비밀채팅방에서는 기본적으로 화면 캡처가 차단됩니다. 상대방이 캡처를 시도하면 검은 화면만 저장되거나 캡처 자체가 차단됩니다. Android 기기에서는 캡처 시도 시 알림이 표시될 수 있습니다.

단, iOS(아이폰) 기기에서는 운영체제 특성상 캡처 차단이 완전하지 않을 수 있으므로 이 점에 유의하세요.

### 비밀채팅방 잠금 설정

비밀채팅방 자체에 별도의 잠금을 설정할 수도 있습니다.

1. 채팅방 목록에서 비밀채팅방을 **길게 누릅니다**.
2. 팝업 메뉴에서 **채팅방 잠금** 또는 **잠금** 항목을 선택합니다.
3. 비밀번호 또는 지문/Face ID를 설정합니다.
4. 설정 완료 후 해당 채팅방은 잠금 해제를 해야만 내용을 볼 수 있습니다.

## 비밀채팅 사용 시 주의사항

### 전송 가능한 콘텐츠 제한

비밀채팅에서는 일반 채팅과 달리 일부 콘텐츠 전송이 제한될 수 있습니다.

- **가능:** 텍스트, 이모티콘, 사진, 동영상, 파일
- **제한 또는 불가:** 카카오페이 송금, 선물하기, 일부 카카오 서비스 연동 기능

비밀채팅방에서 송금이나 쇼핑 기능이 필요하다면 일반 채팅방을 별도로 사용해야 합니다.

### 백업 및 기록 남기지 않기

비밀채팅 내용은 카카오 서버에 저장되지 않으며 기기에만 임시로 보관됩니다. 따라서:

- 카카오톡 대화 백업을 해도 비밀채팅 내용은 백업되지 않습니다.
- 앱을 삭제하거나 폰을 초기화하면 내용이 영구적으로 사라집니다.
- 상대방이 채팅방을 나가면 채팅 기록이 소멸됩니다.

### 비밀채팅 종료 방법

비밀채팅방을 나가는 방법은 일반 채팅방과 동일합니다. 채팅방 설정에서 **채팅방 나가기**를 선택하면 됩니다. 방을 나가면 채팅 기록이 모두 삭제되며 복구가 불가능합니다.

## 자주 묻는 질문 (Q&A)

**Q. 비밀채팅방인데 상대방이 캡처를 했다는 알림이 왔어요. 어떻게 된 건가요?**

A. Android 기기에서는 상대방이 캡처를 시도하면 알림이 전송됩니다. 하지만 iOS 기기는 운영체제 정책상 알림이 전송되지 않을 수 있습니다. 또한 외부 카메라로 화면을 촬영하는 방법은 기술적으로 차단할 수 없으므로 민감한 정보는 가능하면 텍스트로 공유하지 않는 것이 좋습니다.

**Q. 비밀채팅으로 전환 메뉴가 보이지 않아요.**

A. 다음을 확인해 보세요. 첫째, 카카오톡 앱을 최신 버전으로 업데이트합니다. 둘째, 상대방이 카카오톡 미니 버전이나 구 버전을 사용하는 경우 비밀채팅이 지원되지 않을 수 있습니다. 셋째, 해외 카카오톡 계정과는 비밀채팅이 제한될 수 있습니다.

**Q. 비밀채팅 중에 상대방이 카카오톡을 탈퇴했어요. 내 기록은 어떻게 되나요?**

A. 상대방이 카카오톡을 탈퇴하면 비밀채팅방은 자동으로 종료되고 채팅 기록은 내 기기에서도 삭제됩니다. 중요한 내용이 있다면 미리 다른 방법으로 저장해 두시기 바랍니다.

**Q. 타이머를 설정했는데 중간에 해제할 수 있나요?**

A. 네, 가능합니다. 타이머 설정 메뉴로 다시 들어가서 **삭제 안 함**을 선택하면 타이머가 해제됩니다. 단, 이미 타이머가 시작된 메시지는 설정된 시간에 따라 삭제됩니다.

**Q. 비밀채팅방을 실수로 나갔어요. 다시 들어갈 수 있나요?**

A. 비밀채팅방은 한번 나가면 재입장이 불가능합니다. 다시 대화하려면 상대방에게 새로운 비밀채팅을 다시 시작해 달라고 요청해야 합니다.

## 마무리 정리

카카오톡 비밀채팅은 개인정보 보호가 필요한 대화에 매우 유용한 기능입니다. 2026년 현재 기준으로 주요 기능을 다시 정리하면 다음과 같습니다.

1. 비밀채팅은 1:1 채팅에서만 가능하며 양쪽 모두 수락이 필요합니다.
2. 메시지 자동 삭제 타이머를 설정하면 일정 시간 후 메시지가 양쪽에서 사라집니다.
3. 화면 캡처는 Android에서 효과적으로 차단되지만 iOS에서는 제한이 있습니다.
4. 채팅 내용은 서버에 저장되지 않아 백업이 되지 않습니다.
5. 채팅방을 나가면 모든 기록이 삭제되며 복구할 수 없습니다.

민감한 정보를 주고받을 때는 비밀채팅을 적극적으로 활용해 개인정보를 보호하시기 바랍니다.
