## 다크모드란?

화면 배경을 어둡게 바꿔주는 기능이에요. 눈의 피로를 줄이고 AMOLED 화면에서는 배터리도 절약돼요.

## 다크모드 켜는 방법

**방법 1. 빠른 설정 패널**
1. 화면 위에서 아래로 두 번 쓸어내리기
2. **다크모드** 아이콘 클릭

**방법 2. 설정 메뉴**
1. **설정 → 디스플레이**
2. **다크모드** 토글 켜기

## 자동으로 켜고 끄기 (일몰/일출 기준)

1. **설정 → 디스플레이 → 다크모드 설정**
2. **예약 설정** 켜기
3. **일몰부터 일출까지** 선택
4. 해가 지면 자동으로 다크모드, 해가 뜨면 자동으로 해제

## 앱별로 다크모드 적용

카카오톡, 유튜브 등 일부 앱은 자체 다크모드 설정이 있어요.

- **카카오톡**: 설정 → 화면 → 테마 → 다크
- **유튜브**: 계정 → 설정 → 일반 → 다크 테마

## 배터리 절약 효과

갤럭시 AMOLED 화면은 검은색 픽셀이 꺼지기 때문에 다크모드 사용 시 **배터리를 최대 30%** 까지 절약할 수 있어요.
