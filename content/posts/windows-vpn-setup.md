## VPN이란 무엇인가요?

**VPN(Virtual Private Network, 가상 사설망)**은 인터넷 연결을 암호화하여 외부에서 내 인터넷 사용 내용을 볼 수 없게 보호하는 기술입니다. 쉽게 말해 내가 인터넷을 쓸 때 나의 실제 IP 주소 대신 VPN 서버의 주소가 노출되어 개인정보를 보호할 수 있습니다.

VPN이 필요한 대표적인 상황은 다음과 같습니다.

- 카페, 도서관 등 공용 와이파이 사용 시 해킹 방지
- 재택근무 시 회사 내부 네트워크 접속
- 해외에서 국내 서비스 이용 또는 해외 콘텐츠 접근

이 글에서는 윈도우 11에서 VPN을 설정하고 연결하는 방법을 단계별로 설명합니다.

---

## 윈도우 11 내장 VPN 설정 방법

윈도우 11에는 별도 앱 없이 VPN을 설정할 수 있는 기능이 내장되어 있습니다. 회사나 학교에서 VPN 서버 주소와 계정 정보를 제공한 경우 아래 방법으로 바로 연결할 수 있습니다.

### 1단계: VPN 설정 화면 열기

1. 화면 왼쪽 하단의 **시작 버튼(Windows 로고)**을 클릭합니다.
2. **설정(톱니바퀴 아이콘)**을 클릭합니다. 또는 키보드 단축키 **Win + I**를 누릅니다.
3. 설정 창이 열리면 왼쪽 메뉴에서 **네트워크 및 인터넷**을 클릭합니다.
4. 오른쪽 목록에서 **VPN**을 클릭합니다.

### 2단계: VPN 연결 추가하기

1. VPN 화면에서 **VPN 연결 추가** 버튼(+ 아이콘)을 클릭합니다.
2. 다음 항목을 입력합니다.
   - **VPN 공급자**: Windows(기본 제공)를 선택합니다.
   - **연결 이름**: 나중에 구분하기 쉬운 이름을 입력합니다. 예: "회사 VPN", "학교 VPN"
   - **서버 이름 또는 주소**: VPN 서버 주소를 입력합니다. (예: vpn.company.com 또는 IP 주소)
   - **VPN 종류**: 서버에 맞는 프로토콜을 선택합니다. 모르면 **자동**으로 두세요.
   - **로그인 정보 형식**: 일반적으로 **사용자 이름 및 암호** 선택
   - **사용자 이름**, **암호**: 회사 또는 VPN 서비스에서 제공한 계정 정보 입력
3. 모두 입력 후 **저장** 버튼을 클릭합니다.

### 3단계: VPN 연결하기

1. VPN 목록에 방금 만든 연결이 표시됩니다.
2. 연결 이름 아래의 **연결** 버튼을 클릭합니다.
3. 잠시 후 **연결됨** 상태로 바뀌면 VPN이 성공적으로 연결된 것입니다.
4. 작업 표시줄 오른쪽 하단의 네트워크 아이콘에 자물쇠 표시가 나타납니다.

### 4단계: VPN 연결 해제하기

1. **설정 → 네트워크 및 인터넷 → VPN**으로 이동합니다.
2. 연결된 VPN 항목 아래 **연결 끊기** 버튼을 클릭합니다.
3. 또는 작업 표시줄의 네트워크 아이콘을 클릭하고 VPN 항목에서 연결을 끊을 수 있습니다.

## 빠른 설정에서 VPN 켜고 끄기

매번 설정 앱을 열지 않아도 빠르게 VPN을 전환할 수 있습니다.

1. 작업 표시줄 오른쪽의 **시스템 트레이(시계 옆 아이콘 묶음)**를 클릭합니다.
2. **빠른 설정** 패널이 열리면 **VPN** 타일을 클릭합니다.
3. VPN 목록이 표시되면 연결할 VPN을 선택합니다.
4. VPN 타일이 보이지 않으면 패널 오른쪽 하단의 **편집(연필 아이콘)**을 클릭해 VPN 타일을 추가하세요.

## 무료 VPN 앱 활용하기

직접 VPN 서버를 갖고 있지 않은 일반 사용자라면 VPN 전용 앱을 설치하는 방법이 더 편리합니다.

### 대표적인 무료 VPN 서비스

| 서비스명 | 특징 |
|---|---|
| Proton VPN | 무료 플랜 제공, 스위스 기반 강력한 보안 |
| Windscribe | 월 10GB 무료 데이터 |
| TunnelBear | 월 500MB 무료, 직관적인 UI |

### 앱 설치 및 사용 방법 (Proton VPN 기준)

1. 공식 홈페이지에서 Windows용 앱을 다운로드합니다.
2. 설치 파일을 실행하고 안내에 따라 설치를 완료합니다.
3. 앱을 실행하고 계정을 만들거나 로그인합니다.
4. 화면에서 연결할 서버 위치를 선택합니다. (무료 플랜은 일부 국가만 선택 가능)
5. **연결** 버튼을 클릭하면 VPN이 활성화됩니다.
6. 상단에 **보호됨(Protected)** 메시지가 뜨면 연결 성공입니다.

## VPN 프로토콜 종류와 선택 방법

VPN 설정 시 **VPN 종류(프로토콜)**를 선택해야 하는 경우가 있습니다. 각 프로토콜의 특징을 간단히 알아두면 도움이 됩니다.

| 프로토콜 | 특징 | 권장 상황 |
|---|---|---|
| IKEv2/IPsec | 빠르고 안정적, 모바일에 강함 | 일반 사용 |
| L2TP/IPsec | 호환성 높음, 속도 보통 | 구형 서버 |
| PPTP | 가장 빠름, 보안 취약 | 보안 불필요한 경우 |
| OpenVPN | 강력한 보안, 설정 복잡 | 높은 보안 필요 시 |
| WireGuard | 최신 프로토콜, 빠르고 안전 | 최신 서버 |

모르는 경우 **자동** 또는 **IKEv2/IPsec**를 선택하면 대부분의 환경에서 잘 작동합니다.

## 재택근무 시 회사 VPN 연결 팁

### 회사 VPN 정보 확인하기

회사 VPN을 처음 설정할 때는 IT 담당자 또는 관리자에게 다음 정보를 확인하세요.

- VPN 서버 주소 (예: vpn.yourcompany.com)
- VPN 프로토콜 종류 (IKEv2, L2TP, OpenVPN 등)
- 로그인 계정(사용자 이름, 초기 비밀번호)
- 회사 전용 앱 설치 여부 (Cisco AnyConnect, GlobalProtect 등)

### 자동 연결 설정하기

1. VPN 설정 화면에서 등록된 VPN 항목의 오른쪽 **V 버튼**을 클릭하고 **고급 옵션**을 선택합니다.
2. **셀룰러 데이터를 사용하여 로밍할 때 VPN에 연결** 또는 **신뢰할 수 없는 네트워크에서 자동 연결** 옵션을 활성화할 수 있습니다.
3. 이렇게 하면 카페 등 외부 네트워크에 연결될 때 자동으로 VPN이 켜집니다.

## 자주 묻는 질문(Q&A)

**Q. VPN을 사용하면 인터넷이 느려지나요?**
A. VPN을 사용하면 암호화 과정과 서버를 경유하는 특성상 속도가 다소 느려질 수 있습니다. 서버 위치가 가까울수록, 프로토콜이 최신일수록(WireGuard 등) 속도 저하가 적습니다. 유료 VPN이 일반적으로 무료보다 빠릅니다.

**Q. VPN 연결 중 오류가 발생해요.**
A. 다음을 순서대로 확인해 보세요. ① 서버 주소와 계정 정보가 정확한지 확인 ② 인터넷 연결이 정상인지 확인 ③ 윈도우 방화벽이 VPN을 차단하고 있지 않은지 확인 (**설정 → Windows 보안 → 방화벽 및 네트워크 보호**에서 VPN 앱 허용) ④ VPN 프로토콜을 다른 것으로 변경해 재시도

**Q. VPN 연결이 자꾸 끊겨요.**
A. 인터넷 연결이 불안정하거나 서버 문제일 수 있습니다. VPN 앱의 '킬 스위치(Kill Switch)' 기능을 활성화하면 VPN이 끊겼을 때 인터넷 연결 자체를 차단해 데이터 유출을 방지합니다.

**Q. 무료 VPN은 안전한가요?**
A. 무료 VPN은 편리하지만 일부 서비스는 사용자 데이터를 수집하거나 판매하는 경우가 있습니다. 신뢰할 수 있는 서비스(Proton VPN 등 오픈소스 또는 감사 완료 서비스)를 선택하고, 광고 기반의 출처 불명 VPN은 피하는 것이 좋습니다.

**Q. 회사 VPN에 연결하면 회사에서 내 인터넷 사용 내역을 볼 수 있나요?**
A. 네, 회사 VPN을 통해 접속할 경우 회사의 네트워크 관리자가 트래픽을 모니터링할 수 있습니다. 개인 인터넷 사용은 VPN 연결을 끊고 이용하는 것을 권장합니다.

## 정리

윈도우 11에서 VPN 설정하는 방법을 정리합니다.

- **회사·학교 VPN**: 설정 → 네트워크 및 인터넷 → VPN에서 서버 주소·계정 입력 후 연결
- **일반 사용자**: Proton VPN 등 전용 앱 설치 후 클릭 한 번으로 간편 연결
- **빠른 전환**: 작업 표시줄 빠른 설정 패널에서 VPN 타일 추가

카페나 공공 와이파이를 자주 사용한다면 VPN은 선택이 아닌 필수입니다. 오늘 소개한 방법으로 안전한 인터넷 환경을 만들어 보세요.
