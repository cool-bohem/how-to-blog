## 블루스크린(BSOD)이란?

블루스크린(Blue Screen of Death, BSOD)은 윈도우에서 **심각한 시스템 오류**가 발생했을 때 컴퓨터가 더 이상 정상 작동할 수 없다고 판단해 강제로 멈추고 파란 화면을 표시하는 것을 말합니다.

화면에는 슬픔 이모티콘(:()과 함께 오류 코드(예: `MEMORY_MANAGEMENT`, `IRQL_NOT_LESS_OR_EQUAL` 등)와 **중지 코드(Stop Code)**가 표시됩니다. 이 코드를 통해 원인을 파악할 수 있습니다.

블루스크린은 단순히 귀찮은 현상이 아니라 하드웨어나 드라이버에 심각한 문제가 생겼다는 신호이므로 반드시 원인을 찾아 해결해야 합니다.

---

## 블루스크린의 주요 원인

| 원인 | 설명 |
|------|------|
| 드라이버 문제 | 그래픽카드, 사운드카드 등 드라이버가 오래되었거나 손상됨 |
| RAM 불량 | 메모리 불량 또는 접촉 불량 |
| 저장장치 문제 | HDD/SSD의 손상된 섹터나 파일 시스템 오류 |
| 과열 | CPU·GPU 온도가 지나치게 높아짐 |
| 윈도우 시스템 파일 손상 | 바이러스 감염, 강제 종료 등으로 시스템 파일이 손상됨 |
| 소프트웨어 충돌 | 최근 설치한 프로그램이나 업데이트와 충돌 |

## 블루스크린 발생 직후 해야 할 일

### 1단계: 오류 코드 메모하기

블루스크린 화면에서 반드시 다음 정보를 확인하고 메모하세요.

1. **중지 코드(Stop code)**: 예) `SYSTEM_THREAD_EXCEPTION_NOT_HANDLED`
2. **실패한 파일 이름**: 중지 코드 아래에 표시되기도 합니다. 예) `nvlddmkm.sys`
3. QR 코드를 스마트폰으로 스캔하면 마이크로소프트 공식 도움말 페이지로 연결됩니다.

이 코드들은 원인 파악에 핵심적인 단서가 됩니다.

### 2단계: 재시작 후 정상 부팅 확인

1. 블루스크린이 지속되면 컴퓨터가 자동으로 재시작됩니다.
2. 재시작 후 정상적으로 윈도우가 부팅되면 일단 로그인합니다.
3. 재시작 자체가 안 된다면 전원 버튼을 5초 이상 눌러 강제 종료 후 다시 켜세요.

## 원인별 해결 방법

### 방법 1: 최근 드라이버 업데이트 롤백

블루스크린이 특정 드라이버 업데이트 후 발생했다면 이전 버전으로 되돌립니다.

1. **윈도우 키 + X** → **"장치 관리자"**를 클릭합니다.
2. 문제가 있는 장치 항목(주로 디스플레이 어댑터, 사운드 등)을 확장합니다.
3. 해당 드라이버를 **마우스 오른쪽 버튼**으로 클릭 → **"속성"**을 선택합니다.
4. **"드라이버"** 탭 → **"드라이버 롤백"** 버튼을 클릭합니다.
5. 이유를 선택하고 **"예"**를 누르면 이전 버전으로 복원됩니다.
6. 컴퓨터를 재시작합니다.

> **팁:** 그래픽 드라이버(`nvlddmkm.sys` = 엔비디아, `amdkmdag.sys` = AMD) 문제로 블루스크린이 자주 발생합니다. 파일명이 표시된다면 해당 제조사 공식 홈페이지에서 안정화 버전 드라이버를 직접 다운로드해 설치하세요.

### 방법 2: 윈도우 업데이트 확인

윈도우 업데이트로 블루스크린 원인인 버그가 수정되었을 수 있습니다.

1. **설정(윈도우 키 + I)** → **"Windows 업데이트"**를 클릭합니다.
2. **"업데이트 확인"** 버튼을 누릅니다.
3. 대기 중인 업데이트가 있으면 모두 설치합니다.
4. 설치 후 재시작합니다.

반대로 특정 업데이트 후에 블루스크린이 발생했다면:

1. **설정 → Windows 업데이트 → "업데이트 기록"**을 클릭합니다.
2. **"업데이트 제거"**를 선택합니다.
3. 문제가 된 업데이트를 선택 후 **"제거"**합니다.

### 방법 3: 시스템 파일 검사기(SFC) 실행

손상된 윈도우 시스템 파일을 자동으로 복구합니다.

1. **윈도우 키**를 누르고 **"cmd"** 또는 **"명령 프롬프트"**를 검색합니다.
2. 검색 결과에서 **"관리자 권한으로 실행"**을 클릭합니다.
3. 아래 명령어를 입력하고 **Enter**를 누릅니다:
   ```
   sfc /scannow
   ```
4. 검사가 완료될 때까지 기다립니다. (5~15분 소요)
5. "보호된 시스템 파일 중 일부가 손상된 것을 발견했으며 복구했습니다"라는 메시지가 나오면 성공입니다.
6. 컴퓨터를 재시작합니다.

SFC 이후에도 문제가 지속되면 DISM 명령어도 실행합니다:
```
DISM /Online /Cleanup-Image /RestoreHealth
```

### 방법 4: 디스크 검사 실행

저장장치(HDD/SSD) 오류를 검사하고 복구합니다.

1. **윈도우 키 + E**로 파일 탐색기를 엽니다.
2. 왼쪽에서 **"내 PC"**를 클릭합니다.
3. C: 드라이브를 **마우스 오른쪽 클릭 → "속성"**을 선택합니다.
4. **"도구"** 탭 → **"오류 검사"** 항목의 **"검사"** 버튼을 누릅니다.
5. **"드라이브 검사"**를 클릭합니다.
6. 검사 중 오류가 발견되면 자동 복구 여부를 묻는 메시지가 나타납니다. **"복구"**를 선택합니다.

### 방법 5: RAM 메모리 검사

메모리 불량이 블루스크린의 흔한 원인입니다.

1. **윈도우 키**를 누르고 **"Windows 메모리 진단"**을 검색합니다.
2. 앱을 실행 후 **"지금 다시 시작하여 문제 확인(권장)"**을 클릭합니다.
3. 컴퓨터가 재시작되며 자동으로 메모리 검사가 진행됩니다. (수 분 소요)
4. 검사 완료 후 윈도우에 로그인하면 결과가 알림으로 표시됩니다.
5. 오류가 발견되면 메모리 모듈 교체를 고려해야 합니다.

### 방법 6: 안전 모드에서 문제 프로그램 제거

블루스크린으로 일반 부팅이 안 될 때는 안전 모드를 이용합니다.

**안전 모드 진입 방법:**
1. 컴퓨터 전원을 켜는 즉시 **F8 키**를 반복해서 누릅니다.
   (또는 윈도우가 3번 부팅 실패하면 자동으로 복구 모드로 진입)
2. "고급 부팅 옵션" 또는 "복구 환경"이 표시되면 **"안전 모드"**를 선택합니다.
3. 안전 모드에서 로그인 후 **제어판 → 프로그램 제거**에서 최근 설치한 프로그램을 삭제합니다.

## 블루스크린 오류 코드 별 빠른 해결법

| 오류 코드 | 주요 원인 | 해결 방법 |
|-----------|----------|---------|
| MEMORY_MANAGEMENT | RAM 불량/오류 | 메모리 진단 실행, RAM 교체 |
| IRQL_NOT_LESS_OR_EQUAL | 드라이버 충돌 | 드라이버 업데이트/롤백 |
| SYSTEM_THREAD_EXCEPTION_NOT_HANDLED | 드라이버 파일 손상 | 해당 드라이버 재설치 |
| CRITICAL_PROCESS_DIED | 핵심 프로세스 손상 | SFC /scannow 실행 |
| PAGE_FAULT_IN_NONPAGED_AREA | RAM 또는 드라이버 문제 | 메모리 검사 + 드라이버 점검 |
| NTFS_FILE_SYSTEM | 하드 드라이브 오류 | 디스크 검사(chkdsk) 실행 |

## 블루스크린 재발 방지 방법

1. **드라이버를 정기적으로 업데이트하세요.** 특히 그래픽카드 드라이버.
2. **윈도우 업데이트를 미루지 마세요.** 보안 패치와 버그 수정이 포함됩니다.
3. **컴퓨터 내부 청소를 주기적으로 하세요.** 먼지로 인한 과열이 블루스크린을 유발합니다.
4. **RAM 및 저장장치 건강 상태를 주기적으로 확인하세요.** 무료 도구(CrystalDiskInfo 등)를 활용할 수 있습니다.
5. **신뢰할 수 없는 프로그램 설치를 피하세요.** 불법 소프트웨어나 출처 불명 드라이버는 위험합니다.
6. **중요 파일은 정기적으로 백업하세요.** 블루스크린이 심각해지면 데이터 손실로 이어질 수 있습니다.

## 자주 묻는 질문 (Q&A)

**Q. 블루스크린이 하루에도 여러 번 발생해요. 어떻게 해야 하나요?**
A. 해결 방법을 순서대로 모두 시도해보세요. 그래도 지속된다면 윈도우를 초기화(설정 → 시스템 → 복구 → PC 초기화)하거나 전문 수리점에 문의하는 것을 권장합니다.

**Q. 블루스크린 후 파일이 사라졌어요.**
A. 블루스크린 자체로 파일이 삭제되지는 않지만, 갑작스러운 종료로 저장 중이던 파일이 손상될 수 있습니다. 파일 탐색기에서 **임시 파일 폴더**(C:\Users\사용자명\AppData\Local\Temp)나 해당 앱의 자동 복구 기능을 확인해보세요.

**Q. 블루스크린 없이 컴퓨터가 갑자기 꺼져요. 이것도 같은 문제인가요?**
A. 블루스크린 없는 갑작스러운 종료는 주로 **과열** 또는 **전원 공급 장치(PSU) 문제**일 가능성이 높습니다. CPU/GPU 온도를 모니터링하고, 내부 청소를 먼저 해보세요.

**Q. 블루스크린이 발생했을 때 데이터는 안전한가요?**
A. 저장된 파일은 대부분 안전합니다. 다만 블루스크린 직전에 저장하지 않은 작업 내용은 잃을 수 있습니다. 중요한 문서는 자동 저장 기능을 활성화해두세요.

**Q. 새 컴퓨터인데 블루스크린이 발생해요. 불량인가요?**
A. 새 제품도 블루스크린이 발생할 수 있습니다. 드라이버 업데이트와 윈도우 업데이트를 모두 적용해보세요. 여러 번 반복된다면 제조사 A/S를 요청하는 것이 좋습니다.

## 마무리 정리

블루스크린은 무섭게 느껴지지만 대부분 드라이버 업데이트나 시스템 파일 복구로 해결할 수 있습니다.

**해결 순서 요약:**
1. 오류 코드 메모 → 재시작
2. 드라이버 롤백 또는 업데이트
3. 윈도우 업데이트 확인
4. `sfc /scannow` 명령으로 시스템 파일 복구
5. 디스크 검사 실행
6. RAM 메모리 진단
7. 안전 모드에서 의심 프로그램 제거

위 방법을 순서대로 시도하면 대부분의 블루스크린을 직접 해결할 수 있습니다. 해결이 되지 않는다면 윈도우 초기화 또는 전문 수리점 방문을 고려하세요.
