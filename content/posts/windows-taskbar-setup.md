## 윈도우 11 작업 표시줄이란?

작업 표시줄(Taskbar)은 윈도우 화면 하단에 위치한 가로 막대로, 시작 메뉴, 검색창, 실행 중인 앱 목록, 시스템 트레이(시계, 볼륨, 네트워크 등)가 표시되는 영역입니다.

윈도우 10과 달리 윈도우 11은 기본적으로 작업 표시줄 아이콘이 **가운데 정렬**되어 있으며, 일부 기능이 변경되었습니다. 이 글에서는 윈도우 11 작업 표시줄을 자신의 작업 스타일에 맞게 커스터마이징하는 방법을 단계별로 알려드립니다.

---

## 작업 표시줄 설정 열기

모든 작업 표시줄 설정은 하나의 메뉴에서 관리할 수 있습니다.

### 설정 화면 진입 방법

1. 작업 표시줄의 **빈 공간**에서 **마우스 오른쪽 버튼**을 클릭합니다.
2. 나타나는 메뉴에서 **작업 표시줄 설정**을 클릭합니다.
3. 설정 앱의 **개인 설정 → 작업 표시줄** 화면이 열립니다.

또는 **설정(윈도우 키 + I)** → **개인 설정** → **작업 표시줄**으로 직접 이동할 수도 있습니다.

## 작업 표시줄 아이콘 정렬 방향 변경 (가운데 → 왼쪽)

윈도우 11의 기본 설정은 아이콘이 가운데 정렬되어 있어 윈도우 10에 익숙한 사용자에게 어색하게 느껴질 수 있습니다. 왼쪽 정렬로 변경하는 방법을 알아봅니다.

### 정렬 방향 변경 방법

1. **작업 표시줄 설정** 화면을 엽니다(위의 방법 참조).
2. 아래로 스크롤하여 **작업 표시줄 동작**을 클릭하여 펼칩니다.
3. **작업 표시줄 맞춤**의 드롭다운 메뉴를 클릭합니다.
4. **왼쪽**을 선택합니다.
5. 즉시 작업 표시줄 아이콘이 왼쪽으로 이동합니다.

다시 가운데로 돌아오고 싶다면 같은 방법으로 **가운데**를 선택하면 됩니다.

## 작업 표시줄에 표시되는 항목 설정

작업 표시줄에 표시할 항목을 선택적으로 켜거나 끌 수 있습니다.

### 작업 표시줄 항목 설정 방법

1. **작업 표시줄 설정** 화면에서 **작업 표시줄 항목** 섹션을 확인합니다.
2. 다음 항목의 토글을 켜거나 끌 수 있습니다:
   - **검색**: 검색창 또는 검색 아이콘 표시 여부
   - **작업 보기**: 가상 데스크톱 전환 버튼
   - **위젯**: 날씨, 뉴스 등 위젯 버튼
   - **채팅(Microsoft Teams)**: Teams 빠른 실행 버튼
3. 불필요한 항목은 토글을 끄면 해당 아이콘이 작업 표시줄에서 사라집니다.

### 검색창 표시 방식 변경

1. **작업 표시줄 항목**에서 **검색** 항목을 클릭합니다.
2. 다음 중 원하는 표시 방식을 선택합니다:
   - **숨기기**: 검색창 완전히 숨김
   - **검색 아이콘만**: 돋보기 아이콘만 표시(공간 절약)
   - **검색 아이콘 및 레이블**: 아이콘과 "검색" 텍스트 표시
   - **검색 상자**: 전체 검색 입력창 표시(기본값)

## 시스템 트레이 아이콘 관리

작업 표시줄 오른쪽의 시스템 트레이에는 볼륨, 네트워크, 배터리 등 시스템 아이콘과 백그라운드 앱 아이콘들이 모여 있습니다.

### 시스템 트레이 아이콘 설정

1. **작업 표시줄 설정** 화면에서 아래로 스크롤합니다.
2. **다른 시스템 트레이 아이콘** 섹션을 찾아 클릭하여 펼칩니다.
3. 백그라운드에서 실행 중인 각 앱의 아이콘 표시 여부를 **토글**로 설정합니다.
4. 항상 보이게 하고 싶은 앱은 **켬**, 숨기고 싶은 앱은 **끔**으로 설정합니다.
5. 숨겨진 아이콘은 시스템 트레이의 **^ 화살표**를 클릭하면 확인할 수 있습니다.

### 고정 아이콘(볼륨, 네트워크 등) 표시 설정

1. **작업 표시줄 설정** → **작업 표시줄 모서리 아이콘**을 확인합니다.
2. 다음 아이콘의 표시 여부를 설정할 수 있습니다:
   - **펜 메뉴**: 태블릿/터치 기기에서 펜 관련 메뉴
   - **화상 키보드**: 터치 키보드 버튼
   - **가상 터치패드**: 터치패드 기능

## 작업 표시줄에 앱 고정하기

자주 사용하는 앱을 작업 표시줄에 고정해 두면 빠르게 실행할 수 있습니다.

### 앱 고정 방법

**방법 1: 실행 중인 앱 고정**
1. 작업 표시줄에 이미 실행 중인 앱 아이콘을 **마우스 오른쪽 버튼**으로 클릭합니다.
2. **작업 표시줄에 고정**을 클릭합니다.

**방법 2: 시작 메뉴에서 고정**
1. **시작 메뉴**를 열고 고정하고 싶은 앱을 찾습니다.
2. 앱 아이콘을 **마우스 오른쪽 버튼**으로 클릭합니다.
3. **자세히** → **작업 표시줄에 고정**을 클릭합니다.

**방법 3: 파일 탐색기에서 고정**
1. 파일 탐색기에서 고정하고 싶은 앱의 실행 파일(.exe)을 찾습니다.
2. 파일을 **마우스 오른쪽 버튼**으로 클릭합니다.
3. **작업 표시줄에 고정**을 클릭합니다.

### 고정 앱 순서 변경

작업 표시줄에 고정된 앱의 순서는 마우스로 **드래그하여** 원하는 위치로 이동할 수 있습니다.

### 고정 해제 방법

1. 작업 표시줄의 고정된 앱 아이콘을 **마우스 오른쪽 버튼**으로 클릭합니다.
2. **작업 표시줄에서 제거**를 클릭합니다.

## 작업 표시줄 자동 숨기기 설정

작업 화면을 최대한 넓게 사용하고 싶다면 작업 표시줄을 자동으로 숨기도록 설정할 수 있습니다.

### 자동 숨기기 설정 방법

1. **작업 표시줄 설정** 화면을 엽니다.
2. **작업 표시줄 동작** 섹션을 클릭하여 펼칩니다.
3. **데스크톱 모드에서 작업 표시줄 자동 숨기기** 체크박스를 체크합니다.
4. 작업 표시줄이 사라지며, 마우스를 화면 하단으로 이동하면 다시 나타납니다.

자동 숨기기를 해제하려면 같은 체크박스의 체크를 해제하면 됩니다.

## 여러 모니터에서 작업 표시줄 설정

모니터를 2개 이상 사용하는 경우, 각 모니터에 작업 표시줄을 표시하는 방법을 설정할 수 있습니다.

### 다중 모니터 작업 표시줄 설정

1. **작업 표시줄 설정** → **작업 표시줄 동작**을 펼칩니다.
2. **모든 디스플레이에 작업 표시줄 표시** 체크박스를 체크합니다.
3. 추가 옵션이 나타납니다:
   - **작업 표시줄 앱 표시 위치**: 모든 작업 표시줄 / 기본 작업 표시줄 및 창이 열린 작업 표시줄 / 창이 열린 작업 표시줄
4. 원하는 옵션을 선택합니다.

## 자주 묻는 질문(Q&A)

**Q. 윈도우 11에서 작업 표시줄을 위쪽이나 옆으로 이동할 수 있나요?**
A. 아쉽게도 윈도우 11은 기본적으로 작업 표시줄 위치 변경(상단, 좌측, 우측)을 지원하지 않습니다. 위쪽으로 이동하려면 레지스트리를 수정하거나 'StartAllBack' 같은 서드파티 프로그램을 사용해야 합니다.

**Q. 작업 표시줄 시계 옆에 날짜가 표시되지 않아요.**
A. 작업 표시줄이 너무 작거나 해상도 설정에 따라 날짜가 숨겨질 수 있습니다. 설정 → 개인 설정 → 작업 표시줄 → 작업 표시줄 동작에서 특별한 설정이 없다면, 날짜/시간 아이콘을 클릭하면 날짜가 포함된 팝업 캘린더가 표시됩니다.

**Q. 작업 표시줄에서 날씨 위젯을 없애고 싶어요.**
A. 작업 표시줄 설정 → 작업 표시줄 항목 → 위젯의 토글을 끄면 날씨 위젯 버튼이 사라집니다.

**Q. 작업 표시줄이 갑자기 보이지 않아요.**
A. 자동 숨기기가 활성화되어 있을 수 있습니다. 마우스를 화면 맨 아래쪽으로 이동해 보세요. 그래도 안 보인다면 Ctrl+Alt+Delete → 작업 관리자 → 프로세스에서 'Windows 탐색기'를 찾아 오른쪽 클릭 → 다시 시작을 선택해 보세요.

**Q. 작업 표시줄 아이콘 크기를 작게 만들 수 있나요?**
A. 윈도우 11에서는 기본 설정으로는 아이콘 크기 변경이 제한적입니다. 전체 화면 배율을 낮추려면 설정 → 시스템 → 디스플레이 → 배율에서 조정할 수 있습니다(예: 125% → 100%). 단, 이렇게 하면 화면 전체 요소가 작아집니다.

## 작업 표시줄 활용 꿀팁

- **점프 목록 활용**: 작업 표시줄의 앱 아이콘을 마우스 오른쪽 버튼으로 클릭하면 최근 파일 목록과 자주 사용하는 기능의 '점프 목록'이 나타납니다.
- **빠른 설정**: 작업 표시줄 오른쪽 하단의 **Wi-Fi/볼륨/배터리** 아이콘 영역을 클릭하면 빠른 설정 패널이 열려 Wi-Fi, 블루투스, 비행기 모드 등을 빠르게 전환할 수 있습니다.
- **알림 센터**: 시스템 트레이의 **날짜/시간** 옆 **알림 아이콘**을 클릭하면 앱 알림과 캘린더를 한 번에 확인할 수 있습니다.
- **창 미리보기**: 작업 표시줄의 앱 아이콘에 마우스를 올리면 해당 앱의 열린 창 미리보기가 나타납니다. 닫고 싶은 창의 미리보기에서 X 버튼을 클릭하면 바로 닫을 수 있습니다.
