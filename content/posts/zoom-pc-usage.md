## 줌(Zoom)이란?

줌(Zoom)은 전 세계에서 가장 널리 사용되는 화상회의 프로그램 중 하나입니다. 코로나19 이후 재택근무와 비대면 교육이 일상화되면서 국내에서도 직장인, 학생, 강사, 모임 운영자 등 다양한 분야에서 폭넓게 활용되고 있습니다. 줌은 무료로도 기본적인 화상회의 기능을 충분히 사용할 수 있어 진입 장벽이 낮습니다.

이 글에서는 윈도우 PC에서 줌을 설치하고, 다른 사람이 개설한 회의에 참여하는 방법, 직접 회의를 개설하는 방법, 화면 공유·채팅·가상 배경 등 자주 사용하는 기능들을 처음 사용자 기준으로 상세하게 안내해 드리겠습니다.

## 줌 PC 설치하기

### 1단계: 줌 공식 사이트 접속

인터넷 브라우저(크롬, 마이크로소프트 엣지 등)를 열고 주소창에 **zoom.us**를 입력하여 줌 공식 홈페이지에 접속합니다. 반드시 공식 사이트에서 다운로드하여야 안전합니다. 검색 엔진에서 "줌 다운로드"로 검색하면 사설 사이트로 연결될 수 있으니 주의하세요.

### 2단계: 다운로드 페이지 이동

홈페이지 상단 또는 하단 메뉴에서 **"다운로드"(Download)** 항목을 클릭합니다. 다운로드 센터 페이지가 열리면 상단에 있는 **"Zoom Desktop Client"** 항목에서 **"다운로드"** 버튼을 클릭합니다.

### 3단계: 설치 파일 실행

다운로드된 설치 파일(**ZoomInstaller.exe**)을 더블클릭하여 실행합니다. 사용자 계정 컨트롤(UAC) 확인 창이 나타나면 **"예"**를 클릭합니다. 잠시 기다리면 줌이 자동으로 설치됩니다. 별도의 설치 과정 입력 없이 자동으로 진행되며, 완료까지 1~2분 정도 소요됩니다.

### 4단계: 설치 완료 및 바탕화면 아이콘 확인

설치가 완료되면 줌 앱이 자동으로 실행됩니다. 바탕화면에 줌 아이콘(파란색 카메라 모양)이 생성됩니다. 다음부터는 이 아이콘을 더블클릭하여 실행하면 됩니다.

## 줌 계정 만들기

줌은 계정 없이도 초대받은 회의에 참여할 수 있지만, 직접 회의를 개설하거나 회의를 예약하려면 계정이 필요합니다. 무료로 계정을 만들 수 있습니다.

### 5단계: 회원가입하기

줌 실행 화면에서 **"무료로 가입"** 버튼을 클릭합니다. 가입 방법을 선택합니다:

- **이메일로 가입**: 이메일 주소 입력 → 인증 메일 수신 → 링크 클릭 → 이름·비밀번호 설정 완료
- **구글 계정으로 가입**: 구글 계정 선택 후 바로 가입 완료
- **SSO(기업/학교 계정)**: 소속 기관 도메인을 입력하여 기관 계정으로 로그인

이메일로 가입하는 경우, 입력한 이메일 주소로 인증 메일이 발송됩니다. 이메일 받은 편지함을 확인하여 **"활성화"** 또는 **"이메일 확인"** 링크를 클릭하면 가입이 완료됩니다.

### 6단계: 줌 앱에서 로그인

줌 앱 시작 화면에서 **"로그인"**을 클릭합니다. 이메일 주소와 비밀번호를 입력하거나, **"Google로 로그인"** 버튼을 클릭하여 소셜 로그인합니다.

로그인 상태를 유지하려면 "로그인 유지" 체크박스를 선택해 두면, 다음번에 앱을 열 때 자동으로 로그인됩니다.

## 화상회의에 참여하기

줌에서 가장 기본적인 기능은 초대받은 회의에 참여하는 것입니다. 참여 방법은 두 가지입니다.

### 7단계: 회의 ID로 직접 참여

회의 주최자로부터 **회의 ID**(9~11자리 숫자)와 **비밀번호**(있는 경우)를 전달받습니다.

1. 줌 앱 메인 화면에서 **"참가"** 버튼을 클릭합니다
2. 회의 ID를 입력하고 본인 이름을 확인합니다
3. **"참가"**를 클릭합니다
4. 비밀번호 입력 창이 나타나면 입력합니다

### 8단계: 초대 링크로 참여

카카오톡, 이메일 등으로 받은 줌 초대 링크(https://zoom.us/j/... 형식)를 클릭합니다. 브라우저에서 팝업이 나타나면 **"Zoom Meetings 열기"** 또는 **"허용"**을 클릭합니다. 줌 앱이 자동으로 열리면서 회의에 참여됩니다.

### 9단계: 오디오 및 비디오 설정

회의 참여 직전 또는 대기실에서 오디오와 비디오를 설정합니다.

- **비디오 미리보기**: 본인 얼굴이 제대로 나오는지 확인합니다. 카메라를 켜고 싶지 않다면 "비디오 없이 참가"를 선택하세요.
- **오디오 연결**: 반드시 **"컴퓨터 오디오로 참여"**를 클릭해야 마이크와 스피커가 연결됩니다. 이를 클릭하지 않으면 소리가 들리지 않을 수 있습니다.

## 화상회의 중 주요 기능 사용법

회의에 참여하면 화면 하단에 도구 모음이 표시됩니다. 마우스를 화면 하단으로 이동하면 도구 모음이 나타납니다.

### 마이크 조절 (음소거/해제)

화면 왼쪽 하단의 **마이크 아이콘**을 클릭하여 마이크를 켜거나 끌 수 있습니다.
- 마이크 아이콘에 **빨간 사선**이 표시되면 현재 음소거 상태입니다
- 단축키: **Alt + A** (음소거 토글)
- 잠깐 말할 때는 **스페이스바를 누르고 있는 동안만** 일시적으로 음소거가 해제됩니다

마이크 옆의 **화살표(∧)**를 클릭하면 마이크·스피커 기기를 변경하거나 오디오 테스트를 할 수 있습니다.

### 비디오 조절 (카메라 켜기/끄기)

**카메라 아이콘**을 클릭하여 카메라를 켜거나 끌 수 있습니다.
- 단축키: **Alt + V**
- 카메라를 끄면 프로필 사진 또는 이름이 표시됩니다

### 참가자 목록 확인

도구 모음에서 **"참가자"** 버튼을 클릭하면 현재 회의에 참여 중인 모든 사람의 목록이 오른쪽 패널에 표시됩니다. 각 참가자의 마이크·카메라 상태도 아이콘으로 확인할 수 있습니다.

### 채팅 기능

**"채팅"** 버튼을 클릭하면 오른쪽에 채팅창이 열립니다. 텍스트 메시지를 입력하여 **전체 참가자** 또는 **특정 참가자**에게만 메시지를 보낼 수 있습니다. 파일(문서, 이미지 등)도 채팅을 통해 공유할 수 있습니다.

채팅창 상단의 받는 사람 옵션에서 "모두에게" 또는 특정 이름을 선택하여 귓속말처럼 1:1 메시지를 보낼 수 있습니다.

### 손들기 및 반응 표현

**"반응"** 버튼에서 박수, 좋아요, 놀람 등의 이모지 반응을 표시할 수 있습니다. **"손들기"** 기능을 사용하면 주최자에게 발언을 요청할 때 유용합니다. 손들기를 취소하려면 같은 버튼을 다시 클릭합니다.

## 화면 공유하기

화면 공유는 줌의 핵심 기능으로, 프레젠테이션, 문서, 영상 등을 다른 참가자들에게 보여줄 때 사용합니다.

### 10단계: 화면 공유 시작

하단 도구 모음에서 **"화면 공유"** 버튼을 클릭합니다. 공유할 내용을 선택하는 창이 열립니다:

- **화면 전체**: PC에 열려 있는 모든 화면을 공유합니다. 다른 창의 내용도 모두 보이므로 개인정보나 민감한 내용이 없는지 주의하세요.
- **특정 창**: 현재 열려 있는 특정 프로그램 창만 선택하여 공유합니다. 파워포인트나 특정 문서만 보여주고 싶을 때 유용합니다.
- **화이트보드**: 빈 화이트보드를 열어 그림을 그리거나 판서할 수 있습니다. 강의나 설명할 때 편리합니다.

원하는 항목을 선택하고 **"공유"** 버튼을 클릭합니다. 공유가 시작되면 화면 테두리에 초록색 테두리가 표시됩니다.

### 11단계: 화면 공유 중 제어

화면 공유 중에는 화면 상단(또는 이동 가능한 툴바)에 작은 도구 모음이 나타납니다:
- **공유 중지**: 빨간색 "공유 중지" 버튼으로 공유를 종료합니다
- **일시 중지**: 현재 공유 화면을 정지 상태로 유지합니다
- **새 공유**: 다른 창이나 화면으로 공유 대상을 변경합니다
- **주석 달기**: 공유 화면 위에 펜, 형광펜, 화살표 등으로 표시할 수 있습니다
- **원격 제어**: 주최자가 허용한 경우, 다른 참가자가 내 화면을 직접 제어할 수 있습니다

## 직접 회의 개설하기 (주최자)

### 12단계: 즉석 회의 시작

줌 메인 화면에서 **"새 회의"** 버튼을 클릭합니다. 카메라 켜기/끄기를 선택하면 즉시 회의가 시작됩니다. 혼자만 있는 회의가 열리며, 이후 참가자를 초대합니다.

### 13단계: 참가자 초대

회의 중 하단 도구 모음에서 **"참가자"** → **"초대"**를 클릭합니다. 또는 회의 정보에서 회의 링크와 회의 ID를 복사하여 카카오톡, 이메일 등으로 참가자에게 전달합니다.

상단 메뉴바에서 **"참가자 관리"**를 이용하면 특정 참가자의 마이크를 음소거하거나, 대기실에서 참가자를 수락하는 등의 관리 기능을 사용할 수 있습니다.

### 14단계: 회의 예약하기

나중에 진행될 회의를 미리 예약할 수 있습니다. 줌 메인 화면에서 **"예약"** 버튼을 클릭합니다.

설정 항목:
- **회의 이름**: 회의 제목 입력 (예: "팀 주간 미팅", "영어 스터디")
- **날짜 및 시간**: 회의 시작 날짜와 시간 설정
- **기간**: 예상 회의 시간 설정
- **비밀번호**: 보안을 위한 비밀번호 설정 (권장)
- **대기실 활성화**: 참가자가 회의에 입장하기 전 대기실에서 기다리도록 설정

예약이 완료되면 회의 링크가 생성됩니다. 참가자들에게 이 링크를 미리 전달해 두세요.

## 가상 배경 설정하기

집이나 사무실 배경을 숨기고 싶을 때 가상 배경을 사용하면 편리합니다.

1. 비디오 아이콘 옆의 **화살표(∧)**를 클릭합니다
2. **"가상 배경 선택"**을 클릭합니다
3. 줌에서 기본 제공하는 배경 이미지 중 원하는 것을 선택하거나, **'+'** 버튼으로 직접 사진을 업로드하여 배경으로 사용합니다
4. **"영상 필터"**를 이용하면 화면에 다양한 필터 효과를 적용할 수 있습니다

그린스크린(초록색 배경 천)이 없어도 기본적으로 가상 배경을 사용할 수 있지만, 그린스크린을 사용하면 훨씬 자연스럽게 배경이 분리됩니다.

## 핵심 설정 최적화

줌을 처음 설치한 후 설정(톱니바퀴 아이콘)에서 몇 가지 옵션을 조정해 두면 더욱 편리하게 사용할 수 있습니다.

### 오디오 설정

설정 → **"오디오"** 탭에서:
- **"회의 참여 시 컴퓨터 오디오로 자동 연결"** 체크: 매번 오디오 연결 버튼을 클릭하지 않아도 자동으로 마이크·스피커가 연결됩니다
- **"회의 참여 시 내 마이크 음소거"** 체크: 대규모 회의 참여 시 자동으로 음소거 상태로 입장합니다
- 마이크 및 스피커 볼륨을 테스트하여 적절한 수준으로 조절합니다

### 비디오 설정

설정 → **"비디오"** 탭에서:
- **"HD 비디오 사용"** 체크: 더 높은 화질로 영상을 전송합니다 (인터넷 속도가 충분한 경우)
- **"내 비디오 미러링"** 체크: 내 화면에서 좌우가 반전되어 거울을 보는 것처럼 자연스럽게 보입니다 (상대방에게는 정상으로 보임)
- **"내 모습 수정"** 또는 **"스튜디오 효과"**: 얼굴 보정 효과를 적용할 수 있습니다

### 일반 설정

설정 → **"일반"** 탭에서:
- **"Windows 시작 시 Zoom 실행"** 체크를 해제하면 PC 시작 시 자동 실행을 막을 수 있습니다

## 자주 묻는 질문(Q&A)

**Q. 줌 무료 계정의 제한사항은 무엇인가요?**
A. 무료 계정은 3명 이상이 참여하는 그룹 회의 시 40분 제한이 있습니다. 1:1 회의는 시간 제한이 없습니다. 40분 후 회의가 자동 종료되면 같은 링크를 사용해 다시 시작하면 되며, 이렇게 재연결하는 방식으로 계속 이용할 수 있습니다. 유료 계획(Pro 이상)을 사용하면 시간 제한이 없어집니다.

**Q. 상대방 목소리가 안 들립니다. 어떻게 해야 하나요?**
A. 먼저 하단 마이크 아이콘 옆 화살표(∧)를 클릭하여 올바른 스피커 기기가 선택되어 있는지 확인하세요. PC 볼륨이 음소거 상태인지, 헤드셋이 제대로 연결되어 있는지도 확인하세요. 그래도 안 들린다면 "스피커 & 마이크 테스트"를 실행하여 문제를 파악할 수 있습니다.

**Q. 화면이 자꾸 끊기고 버퍼링이 발생합니다.**
A. 인터넷 연결이 불안정할 때 주로 발생합니다. 가능하다면 유선 랜선으로 연결하세요. 비디오를 끄면 데이터 사용량이 크게 줄어 연결이 안정될 수 있습니다. 줌 설정 → 비디오 → "비디오 해상도 제한"을 낮추거나, 다른 프로그램의 인터넷 사용량을 줄이는 것도 방법입니다.

**Q. 회의 내용을 녹화할 수 있나요?**
A. 네, 무료 계정도 PC에 직접 저장하는 **로컬 녹화**가 가능합니다. 회의 중 하단 도구 모음에서 **"녹화"**를 클릭하면 됩니다. 녹화 파일은 회의 종료 후 지정한 폴더에 MP4 형식으로 저장됩니다. 클라우드 녹화(줌 서버에 저장)는 유료 계정에서만 지원됩니다.

**Q. 줌을 설치하지 않고 브라우저로 참여할 수 있나요?**
A. 네 가능합니다. 회의 링크를 클릭하면 나타나는 페이지에서 **"브라우저에서 시작"**을 클릭하면 앱 설치 없이 크롬 등 브라우저에서 참여할 수 있습니다. 다만 화면 공유, 가상 배경 등 일부 기능이 제한될 수 있습니다.

**Q. 회의 중에 참가자가 입장할 때 알림음이 거슬립니다.**
A. 설정 → "오디오" → "참가자 입장/퇴장 시 소리 재생" 옵션을 끄면 됩니다. 또는 회의 중 "참가자" 탭 하단의 옵션에서 조절할 수도 있습니다.

## 마무리 정리

줌은 처음에는 낯설게 느껴질 수 있지만, 기본 기능만 익혀두면 누구나 편리하게 화상 회의에 참여하고 개설할 수 있습니다. 오늘 알려드린 내용을 정리하면 다음과 같습니다:

- 줌 공식 사이트(zoom.us)에서 설치 파일 다운로드 후 자동 설치
- 계정 없이도 회의 ID나 초대 링크로 참여 가능
- 하단 도구 모음으로 마이크, 카메라, 채팅, 반응 등 쉽게 제어
- 화면 공유로 프레젠테이션이나 특정 창을 참가자와 공유
- 가상 배경으로 주변 환경을 숨기고 더 깔끔한 모습으로 참여 가능
- 무료 계정은 3인 이상 그룹 회의 40분 제한, 재연결로 계속 이용 가능
- 회의 녹화는 무료 계정에서도 PC 로컬 저장으로 가능

줌을 잘 활용하여 재택근무, 온라인 수업, 비대면 모임을 더욱 효율적으로 진행하세요!
